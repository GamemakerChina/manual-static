virtual_key_hide

一旦你为你的设备创建了虚拟键，你可能需要调试和测试它是否正确放置并正常工作，为此你可以调用函数virtual_key_show()。但是，您可能需要打开和关闭此视图，因此您还可以使用此函数隐藏虚拟键，使用要隐藏的虚拟键的索引(以前使用virtual_key_add()创建和存储)。一旦您对事物感到满意，建议您创建并在其区域中放置自己的图形。

 

语法

virtual_key_hide(index);

参数 类型 描述
index Virtual Key ID 要隐藏的虚拟键的索引。

 

返回:

N/A

 

例子:

if global.Debug
{
    virtual_key_show(global.Left);
}
else
{
    virtual_key_hide(global.Left);
}

上面的代码检查全局变量“ global.debug ”，如果测试结果为true，则在屏幕上绘制变量“ global.left ”中索引的虚拟键，如果测试结果为false，则隐藏该键。
