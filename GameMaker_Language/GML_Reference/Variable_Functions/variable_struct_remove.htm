struct_remove

此函数从结构中删除变量。您提供要从中删除变量的结构ID和要删除的变量的字符串名称。

 

语法:

struct_remove(struct, name);

参数 类型 描述
struct Struct 从中删除变量的结构引用
name String 要删除的变量的名称 (作为字符串)

 

返回:

N/A

 

例子:

if (struct_exists(mystruct, "shields"))
{
    struct_remove(mystruct, "shields");
}

上面的代码将检查给定的变量是否存在于给定的结构中，如果它存在，那么它将被删除。
