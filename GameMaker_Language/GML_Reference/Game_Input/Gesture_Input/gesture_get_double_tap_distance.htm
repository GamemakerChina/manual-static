gesture_get_double_tap_distance

此函数用于获取单击后必须再次触摸/单击屏幕的距离，以触发双击手势。距离以英寸为单位，默认值为0.1。

 

语法:

gesture_get_double_tap_distance();

 

返回:

Real (inches)

 

例子:

if (gesture_get_double_tap_distance() != 0.1)
{
    gesture_double_tap_distance(0.1);
}

上面的代码检查手势的双击距离是否设置为0.1英寸，如果不是，则将其设置为该值。
