layer_sprite_get_angle

此函数可用于获取精灵元素的当前角度。给定精灵元素 ID(使用layer_sprite_create()或使用函数layer_sprite_get_id()创建精灵元素时获得的 ID)，函数将返回角度的实际值。此值将介于 0 和 360 之间，其中 0 代表右侧、90 向上、180 左和 270 向下。

 

语法:

layer_sprite_get_angle(sprite_element_id);

参数类型 描述
sprite_element_idSprite Element ID 获取信息的精灵元素的唯一ID值

 

返回:

Real

 

例子:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_angle(spr_id) != 0
{
    layer_sprite_yscale(spr_id, 0);
}

上面的代码将获取名为"sprite_sky"的层的层ID，然后使用该代码来获取该层上的精灵元素的ID。然后，此 ID 用于检查角度，如果它不是 0，则设置为 0。
