instance_id_get

使用此函数，您可以从当前活动实例列表中获取任何实例的唯一ID值。您在实例列表中给出索引以从中获取ID，函数将返回用于存储在变量中的值。

 

语法:

instance_id_get(index);

参数 类型 描述
index Real 实例列表中从0到实例计数-1的索引。

 

返回:

Object Instance

 

例子:

for (var i = 0; i < instance_count; ++i;)
{
    var temp_id = instance_id_get(i);
    with (temp_id)
    {
        speed += 0.1;
    }
}

上面的代码将遍历房间内的所有实例，并将它们的速度增加0.1。
