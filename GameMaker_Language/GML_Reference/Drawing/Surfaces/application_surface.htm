application_surface

此全局内置变量可用于访问应用程序表面，用于任何表面函数。此表面是永久可用的，并且是GameMaker绘制游戏的地方。

默认情况下，常规绘制事件(绘制开始、绘制和绘制结束)将绘制到此表面。其他绘制事件(如绘制前/绘制后和绘制Gui)不会在应用程序表面上绘制。有关这些事件如何工作的详细信息，请参阅绘制事件。

 

语法:

application_surface;

 

返回:

Surface ID

 

例子:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

上述代码将重新调整应用程序表面的大小，使其与ui层的比例为1:1。
