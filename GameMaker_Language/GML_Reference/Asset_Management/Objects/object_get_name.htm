object_get_name

此函数将以指定对象的字符串的形式返回名称。此名称是在主GameMaker窗口的"资源浏览器"中为对象指定的名称。请注意，这只是字符串并且不能用于直接引用该对象，因为您需要对象索引。 但是，您可以使用此字符串获取对象索引，使用返回的字符串和函数asset_get_index()。

 

语法:

object_get_name(obj);

参数类型 描述
objObject Asset 要检查的对象的索引。

 

返回:

String

 

例子:

str = object_get_name(object_index);

上面的代码将获取运行该代码的实例的对象索引的名称，并将返回值存储在变量" str "中。
