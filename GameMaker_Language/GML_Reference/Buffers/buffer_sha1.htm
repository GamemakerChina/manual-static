buffer_sha1

在密码学中，Sha-1是由美国国家安全局设计的加密哈希函数，并在几个广泛使用的应用程序和协议中使用，如流行的Git，用于检查文件更改。此函数将从缓冲区获取输入数据，并以ASCII格式返回160位消息摘要。通过这种方式，您可以生成一个安全密钥，该密钥可以存储并用于检查完整性。发送到外部服务器(或从外部服务器接收)的信息(例如)。使用此函数将其应用于缓冲区时，必须指定要使用的缓冲区的缓冲区ID，然后指定从何处开始的偏移值(以字节为单位)，然后指定要散列的区域的大小(同样以字节为单位)。

注意SHA-1 并不完全安全并且可能被破解。请参阅 此页面 了解更多信息。

 

语法:

buffer_sha1(buffer, offset, size);

参数 类型 描述
buffer Buffer ID 要使用的缓冲区的索引。
offset Real 数据偏移值。
size Real 缓冲区的大小。

 

返回:

String

 

例子:

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

上述代码将为存储在由变量"buff"索引的缓冲区中的完整数据创建一个sha1哈希，并将返回的哈希存储在变量"check_string"中。
