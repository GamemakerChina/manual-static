Android 首选项

Android 首选项分为下面列出的各个部分：

在首选项顶部，您可以设置 JDK 内存堆的最大大小(默认为 2 GB)。接下来，您必须设置各种不同的路径，以便GameMaker知道在哪里查找构建最终可执行程序包和测试游戏所需的各种工具。分为 Android SDK、Android NDK 和 Java JDK 三个部分。这些路径应该已经为您自动填充，但如果您有任何问题，您可以单击右侧的按钮打开文件资源管理器，并手动选择每个元素的位置。您可以在此处查看每个 SDK 元素需要的版本的详细信息。应在应用路径时对其进行验证，一条消息将显示"Found"或告诉您任何错误(在这种情况下，您应修改所有信息以确保其正确无误)。

下面有一个打包选项部分，其中只包含一个复选框：

  • 始终完全安装 APK：标记此选项将使 GameMaker 在安装游戏的新版本之前，从您的设备中删除所有以前运行游戏的程序，以及测试设备中的所有关联文件 (例如，*.ini 文件)。

最后，我们来到了密钥库。此文件将用于对您的所有 Android 应用程序进行"签名"，因此请正确填写所有详细信息，并记住，将来创建和更新您的所有 Android 应用程序时将需要此文件，这意味着在设置它之后，建议您对所使用的设置截屏并备份最终文件。如果丢失此文件，您将无法更新已上载到商店的任何现有游戏。

要创建新的密钥库文件，需要以下详细信息：

  • 文件名称：这是密钥库文件的名称。

注意：这不是您的姓名或公司名称，而是要生成的文件的名称！如果您已经拥有以前项目中的密钥库文件，则可以单击此处的文件浏览器按钮，并将 GameMaker 定向到先前创建的密钥库文件(或单击导入按钮)。在这种情况下，您仍然必须填写所需的其余信息(与您第一次生成文件时的信息完全相同)，但您必须不能单击生成密钥库按钮！这将生成一个新的密钥库文件，覆盖以前的文件。

  • 常用名称：这通常是您的名字。
  • 密码：密钥库文件的安全密码，长度必须至少为六个字符。
  • 别名：这是另一个可用于密钥库文件的名称，应该与上面输入的名称不同。
  • 别名密码：密钥库文件的别名安全密码，必须至少六个字符。
    注：如果您的 Android Studio 版本是 4.2 或更高版本，则上面的密码和别名密码字段必须使用相同的密码。使用不同的密码和别名密码会导致构建失败。
  • 组织单位：您所在的公司内部的部门。
  • 组织：公司名称。
  • 位置：您所在的城镇或城市的名称。
  • 国家/地区：您所在国家/地区的标准双字母代码。

填写此信息并满意后，按下标记为生成密钥库的按钮，以让 GameMaker 创建所需的密钥库文件。这可能需要一段时间，但一旦完成，通常不需要再次更改。请注意，您可以单击导入按钮来导入以前创建的密钥库文件(请参见上面的文件名部分)，您还可以通过单击生成密钥哈希按钮来检索密钥库 SHA-1 哈希。按此将生成两个直接链接到您的密钥库文件的特殊安全哈希密钥，某些网站需要这些密钥才能正常运行(例如，Facebook 或 Google Play 要求 SHA-1 哈希)。如果需要，只需按下按钮，然后复制并粘贴生成的哈希键。
