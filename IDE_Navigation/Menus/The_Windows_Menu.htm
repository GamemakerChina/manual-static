窗口菜单

此菜单用于打开常规停靠的 IDE 窗口。由于可以重新排列停靠窗口，并且可以将其内容作为独立窗口删除，甚至完全关闭，因此，如果您发现需要使用停靠窗口执行任何操作，可以使用此窗口重新打开停靠窗口。从此处可以打开的窗口包括"  资源浏览器  "窗口和所有不同的 " 输出  "窗口 (输出、搜索、源代码管理、断点和编译 / 编译错误)。请注意，如果您 启用了 Feather，则会显示 Feather 消息而不是编译错误。

接下来，您可以打开检查器窗口。此窗口用于检查当前选定IDE元素的属性，然后可以根据需要编辑这些属性。

在检查器选项之后，您可以看到最近使用的窗口列表，还有打开环境变量窗口的选项，您可以在其中查看首选项中使用的各种环境变量所指向的不同路径。这些变量无法编辑(既不能编辑其名称也不能编辑其路径/设置)，但您可以选择并复制/粘贴其中任何一个以在其他地方使用。

Finally you have an option to open the Project Tool Output window. This windows shows the output of the Project Tool.
