file_bin_size

这个函数将返回已打开的文件的大小 (以字节为单位)。文件 ID 是函数 file_bin_open() 返回的值。

注意 : 当目标模块是 HTML5 时，这些函数 不能工作。

 

语法:

file_bin_size(binfile);

参数类型 描述
binfileBinary File ID 要获取其大小的文件的 ID。

 

返回:

Real

 

例子:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_close(file);

这将从游戏的本地目录中打开一个文件，并将其索引分配给变量“ file”。然后它会得到该文件的大小并再次关闭它。
