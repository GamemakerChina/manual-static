layer_sequence_is_paused

使用此函数，您可以提供由layer_sequence_create()或层元素函数之一返回的序列元素ID，它将检查当前是否暂停序列，如果序列暂停，则返回true;如果未暂停，则返回false。

 

语法:

layer_sequence_is_paused(sequence_element_id)

参数类型 描述
sequence_element_idSequence Element ID 序列元素到目标的唯一ID值

 

返回:

Boolean

 

例子:

if (layer_sequence_is_paused(title_sequence) != 0)
{
    layer_sequence_play(title_sequence);
}

上述代码检查存储在变量"title_sequence"中的序列元素ID是否暂停，如果是，则开始播放。
