ds_grid_get

此函数可用于从给定DS网格内的任何单元格获取值。如果将无效的网格坐标传递给函数，则返回的值将为未定义的，并且输出窗口中将显示错误。

 

语法:

ds_grid_get(index, x, y);

参数类型 描述
indexDS Grid 网格的索引。
xReal 要查找其值的单元格的x位置。
yReal 要查找其值的单元格的y位置。

 

返回:

Any

 

例子:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

上述代码从变量"grid"中索引的DS网格中选择随机单元格，并将其值存储在变量"val"中。
