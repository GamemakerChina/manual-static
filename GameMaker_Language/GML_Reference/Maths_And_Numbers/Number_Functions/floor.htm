floor

返回 n 的底数，即 n 向下舍入为整数。 这类似于 round() 函数，但它只会向下舍入，无论十进制值是多少，因此 floor(5.99999) 将返回 5， floor(5.2)、floor(5.6457) 等...

 

语法:

floor(n);

参数类型 描述
nReal 向下舍入的数。

 

返回:

Real

 

例子:

val = floor( 3.9 );

这将把 val 设置为3。
