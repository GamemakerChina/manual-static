GX.games 首选项

GX.games 首选项有以下选项：

  • Emscripten SDK 位置 ：这是为 GX.games 制作的 YYC 版本所必需的 Emscripten SDK 的位置。有关设置 SDK 的信息，请阅读 设置 GX.games。
  • 默认 Web 服务器端口：这允许您更改 GX.games Web 服务器的默认端口。默认值为 51264。
  • 默认 Web 服务器地址：您可以在此处更改 GX.games Web 服务器的默认 Web 服务器地址。 默认值为 127.0.0.1。
  • 默认 Web 服务器允许的 IP：您可以在此处添加 Web 服务器的任何允许 IP，这些 IP 将被设置为通过目标管理器添加的所有设备的默认 IP。
  • 默认打包类型：这是默认创建的打包类型： 游戏 或 动态壁纸。默认的显示对话框在每次构建之前显示一个对话框，让您在类型之间进行选择。
