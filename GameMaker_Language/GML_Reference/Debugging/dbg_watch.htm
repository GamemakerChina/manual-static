dbg_watch

此函数为当前调试部分中的变量创建监视。

这只是用来监视变量的值，不能用来改变它。查看调试视图函数参考以了解允许修改变量的控件。

每个值都将转换为字符串并显示。

注意 此调试控制跨越两列。

 

语法:

dbg_watch(ref[, label]);

参数 类型 描述
ref Reference 对使用ref_create创建的变量的引用
label String 可选 要显示的标签

 

返回:

N/A

 

例子:

Create Event

counter = 0;

dbg_watch(ref_create(self, "counter"), "Counter");

Step Event

counter += 1;

上面的代码首先在Create事件中初始化变量counter。然后使用dbg_watch为该变量添加一个监视。

在步事件中，计数器递增，监视将显示此更改。
