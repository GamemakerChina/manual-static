audio_emitter_get_gain

此函数返回为给定音频发射器设置的当前增益(音量)，通常在0和1之间，其中0为静音，1为全音量。请注意，在某些平台上，增益可以大于1，但1被视为"全音量"，任何更大的值都可能引入音频削波。

 

语法:

audio_emitter_get_gain(emitter);

参数 类型 描述
emitter Audio Emitter ID 要使用的发射器的索引。

 

返回:

Real

 

例子:

if audio_emitter_get_gain(emitter_player) < 1
{
    audio_emitter_gain(emitter_player, 1);
}

上述代码检查给定发射器的当前增益，如果它小于1，则设置为1。
