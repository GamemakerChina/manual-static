sprite_get_speed_type

此函数可用于检索精灵编辑器中针对精灵资源定义的精灵动画类型。返回的值将是以下常量之一:

Sprite Speed常量
常量 描述
spritespeed_framespersecond The sprite was defined with animation in frames per second.
spritespeed_framespergameframe The sprite was defined with in animation in frames per game frame.

以下两个示例说明了在计算精灵的动画速度时可以产生的差异:

  • 如果您有一个精灵，每 秒绘制 1 帧 并将图像速度设置为 0.5，它将以每秒 0.5 帧的速度绘制。如果你的游戏帧速率是每秒 60 帧，那么精灵将为每 120 个游戏帧绘制 1 帧。
  • 如果您有一个精灵，每个游戏帧绘制 1 帧，并将图像速度设置为 0.5，它将每个游戏帧绘制 0.5 帧。如果你的游戏帧速率是每秒 60 帧，那么精灵将每 60 个游戏帧绘制 30 帧。

您可以使用函数sprite_get_speed()找出用于精灵动画的速度值，并且可以使用函数sprite_set_speed()设置动画速度和类型。

 

语法:

sprite_get_speed_type(index)

参数 类型 描述
index Sprite Asset 的精灵的索引，以获得的动画类型

 

返回:

Sprite Speed常量 (see above)

 

例子:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

上面的代码获取了精灵速度和精灵动画类型，并将它们存储在实例变量中，以备将来使用。
