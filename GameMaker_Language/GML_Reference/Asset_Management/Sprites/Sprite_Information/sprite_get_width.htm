sprite_get_width

使用此函数，您可以找到基本资源精灵的宽度，没有变换，以像素为单位。

 

语法:

sprite_get_width(index);

参数类型 描述
indexSprite Asset 精灵的索引，以找到的宽度。

 

返回

Real

 

例子:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

上面的代码检查精灵的宽度，因为它在当前实例中，如果它与原始基础精灵之间存在差异，它将重置x轴比例。
