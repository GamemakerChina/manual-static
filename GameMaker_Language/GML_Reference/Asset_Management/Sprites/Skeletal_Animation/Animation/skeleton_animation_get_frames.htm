skeleton_animation_get_frames

此函数可用于检索任何给定骨骼动画的帧数。您提供骨骼动画名称(作为字符串，在用于制作动画的程序中定义，或使用函数skeleton_animation_get()返回)，该函数将以整数值的形式返回它具有的帧。如果指定的动画不存在，函数将返回0。

 

语法:

skeleton_animation_get_frames(anim_name);

参数 类型 描述
anim_name String 要获取其帧的动画名称。

 

返回:

Real

 

例子:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

上面的代码将获取动画中的帧数，然后将精灵设置为最后一帧并停止动画。
