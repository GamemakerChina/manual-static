video_open

此函数使用指定的路径打开视频文件。路径可以指向项目的"包含的文件"中的文件、用户设备上本地存在的文件或从Internet加载文件的URL(可能不适用于所有平台)。

由于一次只能加载一个视频，因此此函数会将指定的视频加载到"唯一可用的"视频插槽中，然后通过其他视频函数操作该插槽。

存在与视频播放相关联的异步回调。有关详细信息，请参阅异步回调。

 

语法:

video_open(path);

参数 类型 描述
path String 视频文件的路径

 

返回:

N/A

 

例子:

video_open("splash.mp4");

上面的代码从游戏的"包含的文件"加载splash.mp4。
