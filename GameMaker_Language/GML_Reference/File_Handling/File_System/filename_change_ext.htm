filename_change_ext

此函数返回扩展名 (包括点) 更改为新扩展名的指示文件名。通过使用空字符串作为新的扩展，您可以一起删除扩展部分。

 

语法:

filename_change_ext(fname, newext);

参数类型 描述
fnameString 要使用的文件。
newextString 要使用的新扩展。

 

返回:

String

 

例子:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

上面的代码获取删除了扩展部分的文件名 (作为字符串)。
