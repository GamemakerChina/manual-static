layer_background_get_vtiled

此函数可用于查找背景元素是否垂直图块。给定背景元素 ID(当使用layer_background_create()创建背景元素时或使用函数layer_background_get_id()时获得该 ID)，如果元素当前图块，则函数将返回true，否则返回false。

 

语法:

layer_background_get_vtiled(background_element_id);

参数类型 描述
background_element_idBackground Element ID 获取信息的背景元素的唯一ID值

 

返回:

Boolean

 

例子:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (!layer_background_get_vtiled(back_id))
{
    layer_background_vtiled(back_id, true);
}

上面的代码将获取名为"Background_sky"的层的层ID，然后使用该层来获取该层上的背景元素的ID。然后使用此 ID 来检查元素是否横穿垂直轴，如果未图块，则将其设置为图块。
