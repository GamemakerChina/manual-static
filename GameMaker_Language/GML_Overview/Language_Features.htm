语言特点

GameMaker 语言(GML) 具有许多函数，这些函数是所有使用该语言编写的程序的基本结构的一部分。所有这些都将用于你多次编写的任何代码中，因此在开始编程项目之前，你应该知道它们是什么以及它们是如何工作的。

  • repeat 重复
  • while 循环
  • do / until 循环
  • for 循环
  • switch
  • break 结束
  • continue 跳过
  • exit 退出
  • with
  • return 返回
  • begin / end
  • try / catch / finally 异常捕捉
  • throw 抛出
  • new 创建
  • delete 删除
