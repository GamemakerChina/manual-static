part_emitter_stream

此函数允许您将发射器设置为流式处理特定类型的粒子，并且通常使用创建事件，因为它是一个一次性代码，用于设置发射器以生成游戏中每个步骤后指定的粒子数。粒子按照part_emitter_region()函数设置的分布、形状和位置发射。您可以将正在流动的粒子数设置为零，然后再次调用此函数以"关闭"粒子流，并且该函数还将接受该数量的负数，在这种情况下，发射器将基于随机机会生成粒子。例如，如果将粒子数设定为-5，则每个步骤都有1:5的几率生成粒子。如果需要粒子一次出现而不是每个步骤，则应使用函数part_emitter_burst()。

 

语法:

part_emitter_stream(ps, ind, parttype, number);

参数 类型 描述
ps Particle System ID 发射器所在的粒子系统。
ind Particle Emitter ID 要从中进行流式发射器的索引。
parttype Particle Type ID 要创建的粒子的索引(类型)。
number Real 每个步骤要创建的粒子数。

 

返回:

N/A

 

例子:

part_emitter_stream(global. Sname, p_emit1, p1, 1);

上述代码将在游戏的每个步骤中流1粒子，直到发射器被破坏或流被设置为0。
