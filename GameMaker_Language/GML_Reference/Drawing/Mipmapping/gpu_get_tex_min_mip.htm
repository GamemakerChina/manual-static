gpu_get_tex_min_mip

使用此函数，您可以获得当前使用的最低mipmap级别，其中值0表示最高分辨率，1表示使用第一个mipmap，2表示第二个mipmap，等等...

 

语法:

gpu_get_tex_min_mip();

 

返回:

Real (default: 0)

 

例子:

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

上述代码将检查当前最小mipmap级别，如果不是0，则设置为0。
