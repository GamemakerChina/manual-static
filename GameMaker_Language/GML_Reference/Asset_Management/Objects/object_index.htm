object_index

此只读变量返回从中创建实例的对象的索引。不是与对象名称相同，它是一个字符串，可以使用object_get_name()找到，因为此函数返回索引号，它是GameMaker在创建时分配给每个对象的唯一值。

 

语法:

object_index;

 

返回:

Object Asset

 

例子:

obj_name = object_get_name(object_index);

上面的代码将使用 object_index 来查找创建当前实例的对象的名称。
