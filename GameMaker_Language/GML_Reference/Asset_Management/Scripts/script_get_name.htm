script_get_name

此函数将以指定脚本的字符串的形式返回名称。此名称是在主GameMaker窗口的"资源浏览器"中为脚本指定的名称。有关脚本的更多信息，请参阅脚本编辑器。

 

语法:

script_get_name(scr);

参数 类型 描述
scr Script Asset 要获取其名称的脚本的索引。

 

返回:

String

 

例子:

scr_name = script_get_name(Help_File);

上述示例代码将在变量"scr_name"中存储所指示脚本索引的名称。
