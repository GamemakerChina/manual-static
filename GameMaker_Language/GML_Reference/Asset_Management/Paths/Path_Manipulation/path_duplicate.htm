path_duplicate

此函数获取路径并将其复制到新路径中。在进程中创建新路径，并返回其索引，以便在使用此新路径的所有进一步调用中使用。

 

语法:

path_duplicate(index);

参数类型描述
indexPath Asset要复制的现有路径的索引。

 

返回:

Path Asset

 

例子:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

上面的代码从四个路径资源中选择一个并复制它，将新路径的索引存储在变量" myPath "中。
