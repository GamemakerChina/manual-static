layer_background_sprite

使用此函数可以设置背景元素的精灵索引。提供背景元素 ID(当使用layer_background_create()创建背景元素时或使用函数layer_background_get_id()时获得该 ID)，然后给出要使用的 sprite 索引。背景元素图像将替换为新精灵。如果您给出的值为-1，则元素将没有指定sprite(但仍然存在，以后可以再次分配sprite)。

This function also has the alias layer_background_change().

 

语法:

layer_background_sprite(background_element_id, sprite_index)

参数 类型 描述
background_element_id Background Element ID 要更改的背景元素的唯一ID值
sprite_index Sprite Asset 要用于背景元素的精灵的精灵索引

 

返回:

N/A

 

例子:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
    layer_background_sprite(back_id, spr_Clouds);
}

上面的代码将获取名为"Background_sky"的层的层ID，然后使用该层来获取该层上的背景元素的ID。然后，此 ID 用于检查分配给元素的精灵，如果尚未设置，则将其设置为 sprite" spr_Clouds"。
