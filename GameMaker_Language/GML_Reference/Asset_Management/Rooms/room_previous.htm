room_previous

使用此函数，可以在房间输入函数之前检索房间的索引。例如，可以使用变量room获取当前房间的索引，然后使用此函数查找它前面的房间，如房间管理器中所列。 如果输入前没有空间，则返回-1。请注意，该函数不会识别或考虑使用room_add()或room_duplicate()动态添加的房间。

 

语法:

room_previous(numb);

参数类型描述
numbReal要从中获取前一个的房间的索引。

 

返回:

Room Asset

 

例子:

if (room_previous(room) != -1)
{
    room_goto_previous();
}

上面的代码将检查前一个房间是否存在，如果存在，它将转到该房间。
