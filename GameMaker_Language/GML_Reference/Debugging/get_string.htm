get_string

这将创建一个显示标准消息的弹出窗口，其中有一个标记为" OK "的按钮，提示用户输入字符串。如果没有输入任何内容，该函数将返回输入字符串或默认值。

注意此函数仅在Windows目标上用于调试，但在所有其他目标上已弃用。

 

语法:

get_string(str, def);

参数 类型 描述
str String 要在弹出消息中显示的字符串。
def String 文本框中的默认字符串。

 

返回:

String

 

例子:

global.test_name = get_string("Test highscore name:", "Anonymous");

上面的代码将提示用户给出一个名称，然后将其存储在全局变量 "test_name" 中。如果没有输入任何内容，并且用户只按 "OK"，则将返回默认值 "Anonymous"。
