application_get_position

当您在目标平台的"游戏选项"中选中"保持纵横比"时，GameMaker将自动设置应用程序表面的绘制位置，以便它正确居中显示并缩放到给定的显示器。但是，如果您正在操作此表面，并希望自己绘制它，则此函数可以为您提供一种简单的方法，在显示器或窗口中查找正在绘制表面的确切位置，以便您可以您可以自己在那里绘制，也可以将ui图像或后期绘制图像与之对齐。

该函数将返回带有四个键的数组，其中键0和1是表面左上角的x和y位置，键2和3是表面右下角的x和y。表面，全部与显示器或窗口的大小相关。

 

语法:

application_get_position();

 

返回:

Array

 

例子:

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

上述代码将获取应用程序表面的位置以及相对于显示窗口的绝对宽度和高度，并将它们存储在四个变量中以供将来使用。
