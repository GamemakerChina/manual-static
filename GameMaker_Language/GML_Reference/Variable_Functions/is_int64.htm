is_int64

此函数返回给定变量是否为64位整数。在某些情况下，您想检查并查看变量在GameMaker中包含的数据类型，这就是您使用此函数的时间。根据值是否为int64，它返回true或false。

 

语法:

is_int64(n);

参数 类型 描述
n Any 要检查的参数。

 

返回:

Boolean

 

示例:

if !is_int64(val)
{
    show_debug_message("Not a 64bit integer!");
}

上面的代码检查变量 “val” 以查看它是否包含int64，如果不包含int64，则在调试控制台中显示消息。
