vertex_format_delete

每当您使用任何创建的顶点格式完毕后，都必须调用此函数。您提供格式ID值(由函数vector_format_end()返回), 此函数将释放与其关联的内存。请注意，如果您在调用此函数后再次尝试使用此格式，您将收到错误。

 

语法:

vertex_format_delete(formatID);

参数 类型 描述
formatID Vertex Format ID 要删除的顶点格式

 

返回:

N/A

 

例子:

vertex_format_delete(myFormat);

上述代码将从内存中删除在变量"myFormat"中创建的顶点格式。
