direction

GameMaker中的所有实例都具有某些"内置"属性，您可以使用这些属性并设置这些属性来管理它们的外观和行为。方向是这些属性之一，可用于设置实例具有0以外的速度时的实例。请注意，GameMaker中的方向通常计算为 0° 为右侧、90° 为向上、180° 为左侧和 270° 为向下，重力和重力方向变量在游戏中使用时可以修改方向值。

 

语法:

direction;

 

返回:

Real (single precision floating point value)

 

例子:

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上面的代码将根据按下的键(向左或向右)更改实例的移动方向。
