vertex_float3

此函数将向顶点数据添加三个浮点值。要使用vertex_format_add_custom()函数接受此操作，必须正确设置顶点的格式。

 

语法:

vertex_float3(buffer, float, float, float);

参数 类型 描述
buffer Vertex Buffer ID 写入信息的缓冲区。
float Real 第一个输入值。
float Real 第二个输入值。
float Real 第三个输入值。

 

返回:

N/A

 

例子:

vertex_float3(buff, 0.05, 0.01, room_width / x);

上述代码将向正在定义的顶点数据添加三个浮点值。
