gesture_pinch_angle_towards

此函数用于设置在可能开始捏手势之前，触摸必须向另一触摸移动的角度。角度以度为单位测量，默认值为45°。

 

语法:

gesture_pinch_angle_towards();

 

返回:

Real (inches)

 

例子:

if gesture_get_pinch_angle_towards() != 45
{
    gesture_pinch_angle_towards(45);
}

上面的代码检查是否将手势的收缩角度设置为45°，如果不是，则将其设置为该值。
