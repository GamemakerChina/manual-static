physics_set_density

当您使用 physicals_fixture_bind 将夹具绑定到实例时，这会返回绑定夹具的 "id"。您可以随时使用此函数使用此 ID 设置绑定夹具 (而不是 " 基础 " 夹具) 的密度值。

注意 要使物理引擎在所有情况下都使用新值，您需要使用 phy_active 停用并重新激活物理实例。请参阅 更新现有联系人 。

 

语法:

physics_set_density(fixture, density)

参数 类型 描述
fixture Physics Fixture ID 绑定装置的 ID
density Real 要应用的新密度值

 

返回:

N/A

 

例子:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

上面的代码获取实例的绑定物理属性的当前密度值，然后将其设置为不同的值。
