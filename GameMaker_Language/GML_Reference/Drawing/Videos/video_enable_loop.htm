video_enable_loop

此函数启用或禁用当前加载的视频的循环。将enable参数设置为true以启用循环，将false设置为禁用循环。

默认情况下，循环被禁用。

 

语法:

video_enable_loop(enable);

参数 类型 描述
enable Boolean 是启用(true)还是禁用(false)循环

 

返回:

N/A

 

例子:

video_open("Loading.mp4");
video_enable_loop(true);

上述代码加载视频并在其上启用循环。
