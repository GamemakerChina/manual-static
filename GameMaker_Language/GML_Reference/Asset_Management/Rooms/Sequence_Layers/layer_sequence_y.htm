layer_sequence_y

使用此函数，可以设置给定序列元素沿房间的 Y(垂直)轴的位置。提供由layer_sequence_create()或层元素函数之一返回的序列元素 ID 以及要设置的 Y 位置，序列将移动到新位置。

 

语法:

layer_sequence_y(sequence_element_id, pos_y)

参数类型 描述
sequence_element_idSequence Element ID 序列元素到目标的唯一ID值
pos_yReal 将序列元素移动到的Y位置

 

返回:

N/A

 

例子:

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

上面的代码检查变量"my_seq"中引用的sequence元素是否存在，如果存在，它将x/ y位置设置为运行代码的实例的x/ y位置。
