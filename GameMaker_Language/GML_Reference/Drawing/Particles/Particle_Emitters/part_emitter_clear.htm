part_emitter_clear

使用此函数，可以将指定粒子系统中的给定发射器清除回其默认状态。此操作还将停止当时从发射器流出的任何粒子，如果希望再次使用发射器，则需要使用part_emitter_region()函数设置区域位置和粒子类型。

 

语法:

part_emitter_clear(ps, ind);

参数 类型 描述
ps Particle System Instance 发射器所在的粒子系统。
ind Particle Emitter ID 要清除的发射器的步进。

 

返回:

N/A

 

例子:

part_emitter_clear(Sname, p_emit1 );

上述代码将清除变量"p_emit 1"中的粒子发射器索引。
