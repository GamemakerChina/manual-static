audio_sound_get_audio_group

此函数用于返回给定声音资源或声音实例所属的音频组的 ID。

 

语法:

audio_sound_get_audio_group(sound_index);

参数 类型 描述
sound_index Sound Asset or Sound Instance ID 音频资源和实例的索引

 

返回:

Audio Group ID

 

例子:

var _group = audio_sound_get_audio_group(snd_animal);

上面的代码获取分配给声音的音频组的 ID snd_animal 并将其存储在临时变量 _group 中。
