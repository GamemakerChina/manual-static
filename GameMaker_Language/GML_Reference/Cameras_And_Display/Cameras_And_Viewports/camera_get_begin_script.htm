camera_get_begin_script

此函数可用于检索分配为给定相机的开始脚本的脚本函数的 ID。如果未分配脚本函数，则该函数将返回 -1。

 

语法:

camera_get_begin_script(camera_id)

参数类型描述
camera_idCamera ID创建相机时返回的唯一相机 ID 值。

 

返回:

Script Function or -1 if no function is assigned

 

例子:

var scr = camera_get_begin_script(camera_view[0]);
if (scr != cutScene)
{
    camera_set_update_script(camera_view[0], cutScene);
}

上述代码检查分配给 viewport[0] 的相机的脚本函数作为开始脚本，如果不是"cutScene"，则设置为该函数。
