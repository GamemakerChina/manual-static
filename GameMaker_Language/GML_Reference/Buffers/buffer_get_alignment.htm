buffer_get_alignment

使用此函数，您可以获得给定缓冲区 ID 的 a字节对齐。

 

语法:

buffer_get_alignment(buffer);

参数类型 描述
bufferBuffer ID 要检查的缓冲区索引。

 

返回:

Real

 

例子:

alignment = buffer_get_alignment(buff);

上面的代码将从变量"buff"中索引的值获取缓冲区的对齐，并将其存储在变量中。
