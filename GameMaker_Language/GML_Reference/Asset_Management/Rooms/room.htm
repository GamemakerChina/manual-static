room

此变量保存游戏正在运行的当前房间的房间索引。这不是只读变量，但更改此变量不会更改当前房间的索引值，而是更改房间以匹配已设置该变量的索引。执行此操作时应小心，因为如果将变量更改为无效的索引，游戏将抛出错误并关闭。一般来说，使用room_goto()更改房间是更好的做法。

注意：房间ID不基于其在资源浏览器或房间管理器中的顺序，因此应避免直接提供数值。相反，对于要引用的资源(只是其名称)，请使用房间常量或通过函数检索它。

语法:

room;

 

返回:

Room Asset

 

例子:

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

上面的代码将检查当前房间是否是第一层，并在这种情况下播放该层的音乐。
