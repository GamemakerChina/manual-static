GameMaker更新日志

最近的 GameMaker 版本的所有更改都列在本网站上。它分别列出了每月版、LTS 版和测试版的发行说明和下载。

更深入的更改可在 GitHub 上找到。链接页面列出了 GameMaker 的所有主要版本。

上面的链接包含对 GameMaker 2023.8 及更高版本的更改。旧版本的发行说明和下载可在此处获得：IDE/Runtime

访问 GitHub

链接的 GitHub 页面有两类版本：Open (包含正在进行的版本和未来的版本)和 Closed (已向公众发布)。

点击 "[N]Closed" 查看已发布的版本：

单击某个版本可查看其中的所有问题，这些问题可以是 Open (表示该版本正在处理中) 或 Closed (表示已完成)。

如果一个版本处于测试阶段，则关闭的问题可能已经进入最新的测试版本，或者将在下一个测试更新中进行。

GitHub 问题列表总是最新的，所以在报告之前检查已知问题是一个很好的地方。
