array_filter_ext

此函数类似于 array_filter，但它不返回新数组，而是修改作为参数传递的原始数组。

您提供了一个对数组中的所有元素运行的判断方法。它应根据条件返回 true 或 false。

判断函数返回 true 的第一个元素写入 offset 给出的索引处，其他元素写入后续索引 (按照 length 参数的符号给出的方向)。

此函数返回新的有效元素数，从给定的偏移量位置开始，并沿长度参数设置的方向。对于此函数，它是判断返回 true 并被写回数组的元素的数量。

判断函数判断函数

此函数接受在给定数组上操作的判断方法。向判断函数传递以下参数：

语法:

function(element, index);

参数 类型 描述
element Any 当前数组元素的值
index Real 当前数组索引

判断函数应返回布尔值，这会影响原始函数修改或读取数组的方式。

请参阅有关判断方法的信息和示例。

重要 此函数对输入数组进行突变，即直接更改。偏移量和长度参数指定的范围内的现有数组元素可能会被覆盖。出于性能原因，数组未调整大小，而是返回新的有效元素数。

语法:

array_filter_ext(array, function, [offset], [length]);

参数 类型 描述
array Array 要使用的数组
function Function The Predicate Method to run on each element
offset Real 可选 数组中的偏移量或起始索引。设置负值将从数组的末尾开始计数。然后，起始索引将是 array_length(array) + offset。请参阅：偏移量和长度
length Real 可选 要遍历的元素数。负值将向后遍历数组(即以索引的降序，例如 2、1、0 而不是 2、3、4)。请参阅：偏移量和长度

返回:

Real (the number of valid elements in the array)

 

例子:

var _is_even = function(_element, _index) {
    return (_element mod 2) == 0;
}

var _values = [1, 2, 3, 4, 5, 6, 7, 8, 9];
var _valid_elements = array_filter_ext(_values, _is_even, -2, -infinity);

上面的代码首先定义了一个函数 _is_even，该函数接受 element 和 index 参数，并在元素为偶数时返回 true。

然后使用从 1 到 9 的数字对数组 values 进行初始化。

最后，使用判断函数在此数组上调用 array_filter_ext。它的起始索引为 -2( 倒数第二个索引)，长度为 -infinity。这将从 8 开始检查，并向后返回到数组的开头。

该函数从中找到 4 个偶数 (8，6，4，2)，因此它返回 4 作为新的有效元素数。

由于此函数会改变原始数组，因此 _values 变为 [1，2，3，4，2，4，6，8，9]，将偶数从它开始搜索的索引写入数组。
