array_create

使用此函数，您可以创建给定大小的数组。

您告诉函数要创建的数组的长度，它将返回数组的 “句柄”，然后您可以将其分配给变量。

除非您指定 (可选) 初始化值，否则以这种方式创建的数组将使每个条目初始化为 0。如果您提供用于初始化数组的额外值，那么新数组中的所有索引都将设置为该值，而不是 0，但请注意，在这种情况下，该函数将具有更大的性能开销。

注意 要使用函数初始化数组的每个条目，可以使用 array_create_ext。

语法:

array_create(size, [value]);

参数 类型 描述
size Real 要创建的数组的大小。
value Any 可选用于初始化所有数组索引的值。

 

返回:

Array

 

例子:

instance_array = array_create(100, noone);

上面的代码将创建一个新的100条目数组，每个条目都设置为关键字 noone，然后将其分配给变量 “instance_array”。
