timeline_speed

通常，在每个步骤中，时间线中的位置增加1，但是您可以通过将此变量设置为不同的值来更改此量。您可以使用实数 (例如0.5或2.4)，如果该值大于1，则可以在同一时间步长内发生多个时刻 (它们都将按照时间线定义的相同顺序执行，因此不会跳过任何操作)。

 

语法:

timeline_speed;

 

返回:

Real (single precision floating point value)

 

例子:

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

上面的代码将检查并查看实例是否正在运行其关联的时间线，如果没有，它将以一半的速度启动时间线。
