The Track Struct

track 结构体可用于顶级 资源轨道 或子轨道 参数轨道 ，并且 track 的行为由两件事定义：其 名称 和 类型 。轨道 结构 是使用函数 sequence_track_new() 创建的，并且可以使用 序列实例结构 中的 activeTracks 属性或 序列对象结构 中的 tracks 属性从序列资产中检索。

TRACK结构中可用的属性有：

序列轨道结构
变量 类型 描述
type 序列轨道类型常数 其中包含序列轨迹类型常数，它描述了此轨迹的类型。这可以是下文"类型"部分中给出的任一常数。
name String 创建"顶级"资源轨迹时，此处指定的名称可以是识别该轨迹所需的任何字符串。但是，对于参数轨迹，您需要指定特定的字符串来告诉GameMaker您正在创建哪种参数轨迹。下面的"名称"部分对此进行了详细说明。
tracks Array of 序列轨道结构s 此属性允许访问此曲目的子轨迹列表。获得此属性时，将返回序列跟踪结构的数组，设置此属性时，应指定序列跟踪结构的数组。
visible Boolean 这指示此轨迹是否可见(值为true)(值为false)。 您可以获取或设置此值，如果轨迹不可见，则也不会绘制其子轨迹。
keyframes Array of Sequence Keyframe Structs 此属性允许访问轨迹的关键帧结构列表。获取此属性时，返回关键帧结构数组，设置此属性时应指定关键帧结构数组。

Name

name属性可以是以下任何字符串：

  • "Position"-当设置在类型为SeqTrackType_Real的轨道上时，这表示该轨道的内容应用作位置数据。
  • "Scale"-当设置在类型为SeqTrackType_Real的轨道上时，这表示该轨道的内容应用作缩放数据。
  • "rotation"-当设置在类型为SeqTrackType_Real的轨道上时，这表示该轨道的内容应用作旋转数据。
  • "blend_multiply" 或 "image_blend" - 当在 seqtracktype_colour 类型的轨道上设置时，这表示该轨道的内容应用作颜色乘法数据。此类轨道的关键帧数据应该是由四个值组成的数组，格式为 [A, R, G, B]。请注意，每个分量的值表示为 0 到 1 之间，其中 0 对应于十六进制值 #00，1 对应于十六进制值 #FF(0 - 255，如 序列编辑器 中图像混合轨迹的颜色选择器中所示) )。
  • "image_speed"在类型为seqtracktype_real的磁道上设置时，表示该磁道的内容应用作图像速度值。
  • "image_index"在类型为seqtracktype_real的磁道上设置时，表示此磁道的内容应用作图像索引值。
  • "image_angle" - 当在 seqtracktype_real 类型的轨道上设置时，这表示该轨道的内容应用作图像角度值。

文本轨道参数名称 (仅在文本轨道下使用)

  • "frameSize" - 当在 seqtracktype_real 轨道上设置时，这表示该轨道的内容应用作文本的帧大小。该轨道的每个关键帧应有两个通道：0 和 1，其中 0 存储帧的 X 尺寸，1 存储帧的 Y 尺寸。
  • "characterSpacing" - 也与 seqtracktype_real 轨道一起使用，用作文本的字符间距。该轨道的每个关键帧都应有一个包含间距值的通道。
  • "lineSpacing" - 也与 seqtracktype_real 轨道一起使用，用作文本的行间距。该轨道的每个关键帧都应有一个包含间距值的通道。
  • "paragraphSpacing" - 也与 seqtracktype_real 轨道一起使用，用作文本的段落间距。该轨道的每个关键帧都应有一个包含间距值的通道。

音频效果轨道参数名称

  • "audioEffect_bus"- 您可以通过该轨道访问和修改为此音轨效果创建的 AudioBus 结构 的属性。它不包括总线的音频效果，因为每个效果都位于单独的参数轨道上。
  • "audioEffect_bitcrusher"- 用于 失真 (AudioEffectType.Bitcrusher) 效果的轨道。
  • "audioEffect_compressor"- 压缩器 (AudioEffectType.Compressor) 效果的轨道。
  • "audioEffect_delay"- 用于 延迟 (AudioEffectType.Delay) 效果的轨道。
  • "audioEffect_gain"- 用于 增益 (AudioEffectType.Gain) 效果的轨道。
  • "audioEffect_hishelf"-HiShelf(AudioEffectType.HiShelf) 效果的轨道。
  • "audioEffect_hpf2"-HPF(AudioEffectType.HPF2) 效果的轨道。
  • "audioEffect_loshelf"-LoShelf(AudioEffectType.LoShelf) 效果的轨道。
  • "audioEffect_lpf2"-LPF(AudioEffectType.LPF2) 效果的轨道。
  • "audioEffect_peakeq"-PeakEQ(AudioEffectType.PeakEQ) 效果的轨道。
  • "audioEffect_reverb1"- 混响 (AudioEffectType.Reverb1) 效果的轨道。
  • "audioEffect_tremolo"- 用于 Tremolo(AudioEffectType.Tremolo) 效果的轨道。

序列轨迹插补常数
常量 描述 值
