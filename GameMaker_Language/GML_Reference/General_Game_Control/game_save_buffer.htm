game_save_buffer

这是game_save()函数的变体，因此请首先阅读其页面，因为它包含与其使用以及此函数相关的重要信息。

使用此函数，您可以将游戏的当前状态保存到以前创建的"增长"缓冲区(请参见缓冲区)，然后可以使用game_load_buffer()函数随时重新加载该缓冲区。此函数设计为一次在单个房间中使用(即，不建议在一个房间中执行缓冲区保存，然后尝试从另一个房间加载缓冲区)，理想情况下应仅用于检查点或级别重新启动。

注意：此函数非常有限，专为初学者设计，可让检查点系统快速启动并运行，但更高级的用户可能更喜欢使用文件函数对自己的系统进行编码，因为游戏不会保存任何动态资源，如数据结构、表面、添加的精灵等。

 

语法:

game_save_buffer(buffer);

参数类型 描述
bufferBuffer ID 要保存到的缓冲区ID。

 

返回:

N/A

 

例子:

if keyboard_check_pressed(ord("S"))
{
    global.Checkpoint = true;
    game_save_buffer(save_buff);
}

上述代码将全局变量设置为true，然后在按键"S"时将当前游戏状态保存到变量"save_buffs"中索引的缓冲区。
