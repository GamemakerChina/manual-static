string_replace_all

您可以使用此函数解析字符串以查找特定部分，然后可以用您在它们出现的所有位置指定的新字符串替换该字符串。

 

语法:

string_replace_all(str, substr, newstr);

参数 类型 描述
str String 要复制的字符串。
substr String 在所有情况下要替换的字符串中的子字符串。
newstr String 替换先前子字符串的新子字符串。

 

返回:

String

 

例子:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

这将把str2设置为str1，但将其“ X ”替换为“ l ”，从而导致str2为“ Hello World ”。
