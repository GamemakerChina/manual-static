tilemap_set_width

This function can be used to resize a tile map element.

You give the tile map element ID (which you get when you create a tile map element using layer_tilemap_create() or when you use the function layer_tilemap_get_id()), and the new width of the tile map in tile cells.

 

语法:

tilemap_set_width(tilemap_element_id, width);

参数类型描述
tilemap_element_idTile Map Element ID图块映射元素的唯一ID值，用于设置
widthReal宽度值(以"单元格"表示)

 

返回:

N/A

 

例子:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_width(map_id) != room_width div 16)
{
    tilemap_set_width(map_id, room_width div 16);
}

上面的代码检查特定图块贴图的宽度，如果其大小不正确，则设置该宽度。
