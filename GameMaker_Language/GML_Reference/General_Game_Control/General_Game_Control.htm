高分榜

为了帮助您创建和维护游戏的高分，GameMaker创建了一个包含10个位置的全局高分数组，您可以访问、添加和更改该数组，以创建您自己的自定义高分列表，而不会有太多麻烦。此函数是游戏的本地函数，适用于所有平台，因此您可以使用这些函数以及文件处理函数轻松存储和维护基本的高分表。

存在以下处理内部高分列表的函数。

  • highscore_add
  • highscore_name
  • highscore_value
  • highscore_clear

GameMaker 还包含一个非常基本的功能，用于将高分作为列表绘制在屏幕上。这比其他任何事情都更适合调试，因为制作自己的高分表很容易，并且可以让您更好地控制它的显示方式：

  • draw_highscore
