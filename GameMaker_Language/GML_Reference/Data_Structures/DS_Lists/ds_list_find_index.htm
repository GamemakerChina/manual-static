ds_list_find_index

使用此函数，您可以检查给定列表中的某个值，并将返回该值在列表中的位置。注意，如果列表中有多个具有相同值的条目，则可以返回其中任何一个的位置，并且如果该值不存在，则将返回-1。

 

语法:

ds_list_find_index(id, val);

参数类型描述
idDS List要使用的列表的ID。
valAny要查找的值。

 

返回:

Real

 

例子:

pos = ds_list_find_index(list, "Player1");

上述代码检查变量 "list" 中索引的列表中的值 "Player1"，并将返回的位置存储在变量 "pos" 中。
