string_concat

此函数连接(联接在一起)传递给它的所有参数的字符串表示形式，并将结果作为新字符串返回。

不是字符串的参数将具有隐式运行的 string() 函数。有关如何转换这些数据类型的信息，请参阅 从非字符串类型转换  。

如果要连接包含某些字符的字符串，请使用 string_join()。

 

语法:

string_concat(value1 [, value2, ... max_val]);

参数 类型 描述
value1 Any 要连接的第一个值
[, value2, ... max_val] Any 可选 要连接的其他值

 

返回:

String

 

例子:

result = string_concat("W", "o", "r", "d", "s");

上面的代码对一系列字母调用 string_concat 组成一个单词。新字符串存储在变量 result 中。
