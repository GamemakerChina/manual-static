gesture_double_tap_time

此函数用于设置两次触摸/单击之间触发双击手势事件所需的时间。时间以秒为单位测量，默认值为0.16。

 

语法:

gesture_double_tap_time(time);

参数 类型 描述
time Real 为双击手势事件检测设置的时间 (以秒为单位)。

 

返回:

N/A

 

例子:

if gesture_get_double_tap_time() != 0.16
{
    gesture_get_double_tap_time(0.16);
}

上述代码检查手势的双击时间是否设置为0.16秒，如果不是，则将其设置为该值。
