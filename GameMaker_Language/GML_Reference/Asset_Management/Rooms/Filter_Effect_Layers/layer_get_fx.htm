layer_get_fx

此函数用于检索图层的 FX 结构。可以指定要目标的层的 ID 或名称，函数将返回包含其应用效果信息的结构。此结构与您从fx_create()中获得的结构类似，函数fx_get_parameter/s和fx_set_parameter/s可用于读取和修改其参数。

如果指定层没有应用滤镜/效果，则函数将返回-1。

 

语法:

layer_get_fx(layer_name_or_id);

参数 类型 描述
layer_name_or_id String or Layer 要读取的层的名称或ID

 

返回:

FX Struct (or -1 if not found)

 

例子:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(layer_fx, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

上面的代码运行for循环遍历房间中的所有层，并检查每个层的FX结构。如果层具有FX结构，则调用fx_get_name()来检查该过滤器/效果的名称; 如果它等于"_filter_tintfilter"，意思是它是"着色"滤镜，则它将其色调颜色更改为蓝色。
