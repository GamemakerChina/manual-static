gpu_set_texfilter

该函数可用于设置游戏画面上所有图像的纹理过滤 (线性插值)。如果启用此选项 (true)，则绘制时将平滑所有纹理 (包括精灵，因为它们也被视为纹理)，这意味着当缩放或移动时，如果没有 1:1 的像素比率，则会在各个像素之间出现 " 污迹"，这可能会使图像显示模糊，具体取决于所使用的艺术风格。如果禁用此选项 (false)，则缩放或移动时将基于最接近的像素绘制图像，这可能会导致"阻塞"图像。默认值为 false，也可以在单个目标平台的 全局游戏选项 中更改此值。

注意：此设置将使用函数 gpu_set_texfilter_ext() 覆盖为着色器设置的任何纹理载物台插值。

注意：在 HTML5 目标上，此函数只能在启用 WebGL 的情况下使用。

 

语法:

gpu_set_texfilter(enable);

参数类型 描述
enableBoolean 启用或禁用纹理过滤(true/false)

 

返回:

N/A

 

例子:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

以上代码检查纹理过滤是打开还是关闭，并相应地将其切换。
