path_get_point_speed

通过此函数，您可以获得以百分比表示的点的速度 (如 路径编辑器 中定义的或使用 path_add_point() 动态添加路径点时的速度)。因此，如果您在路径编辑器中将路径点设置为 50，则该函数在使用时将返回 50。

 

语法:

path_get_point_speed(index, n);

参数 类型 描述
index Path Asset 要检查的路径的索引。
n Real 要检查的点编号。

 

返回:

Real

 

例子:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

上述代码检查由"pth_Patrol"索引的路径的路径点"0"，以查看速度是否不等于100。如果不是，则设置该点的速度为100。
