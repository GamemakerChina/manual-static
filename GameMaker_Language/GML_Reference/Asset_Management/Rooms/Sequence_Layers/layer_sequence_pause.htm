layer_sequence_pause

使用此函数，可以暂停播放给定序列。layer_sequence_create()或层元素函数之一返回的序列元素Id将暂停该序列，直到您使用函数layer_sequence_play()再次开始播放。

 

语法:

layer_sequence_pause(sequence_element_id)

参数 类型 描述
sequence_element_id Sequence Element ID 序列元素到目标的唯一ID值

 

返回:

N/A

 

例子:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if (global.Pause)
{
    layer_sequence_pause(_seq);
}

上面的代码创建一个新的序列并将其ID存储在局部变量中。然后，它检查游戏是否暂停，如果是，则暂停播放序列。
