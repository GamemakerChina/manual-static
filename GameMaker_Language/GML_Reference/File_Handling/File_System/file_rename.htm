file_rename

此函数将使用指定的名称重命名指定的文件。 如果文件已成功重命名，该函数将返回 true，或者在任何其他情况下返回 false。

警告！ 由于 GameMaker 被沙盒化，此函数可能无法按您预期的那样工作！有关详细信息，请参阅 文件系统 部分。

 

语法:

file_rename(oldname, newname);

参数类型 描述
oldnameString 要更改的文件的名称。
newnameString 文件的新名称。

 

返回:

Boolean

 

例子:

if file_exists("level1.txt")
{
    file_rename("level1.txt", "level.txt");
}

这将检查文件，如果存在，则重命名。
