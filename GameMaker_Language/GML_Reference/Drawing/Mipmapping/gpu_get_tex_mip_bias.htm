gpu_get_tex_mip_bias

使用此函数可以检索mipmap偏移值，其中0值等于无偏移，1等于第一个mipmap，2等于第二个mipmap等等...这样可以控制mip贴图的交换速率，并且通常会使渲染纹理越模糊，值越高，查看的"距离"越大。请注意，这也可以返回负值，在这种情况下，渲染纹理在较大距离内越锐化，值越低。

 

语法:

gpu_get_tex_mip_bias();

 

返回:

Real (default: 0)

 

例子:

if (gpu_get_tex_mip_bias() != 0)
{
    gpu_set_tex_mip_bias(0);
}

上述代码将检查当前mipmap偏移，如果不为0，则设置为0。
