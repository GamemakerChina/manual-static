physics_remove_fixture

此函数从一个或多个实例中删除 (或 " 解除绑定 ") 固定装置。它需要绑定固定装置的唯一 "id"( 由函数 physicals_fixture_bind 返回)，并且它将删除实例的所有当前定义的物理属性，允许您重新定义新的固定装置并绑定该固定装置通过这种方式，您可以更改实例的物理属性，而无需销毁并重新创建它。

 

语法:

physics_remove_fixture(id, fixture)

参数 类型 描述
id Object Instance 要从中删除装置的实例的 ID
fixture Physics Fixture ID 要从实例中删除的装置的 ID

 

返回:

N/A

 

例子:

physics_remove_fixture(id, my_fix);

上面的代码将从实例中删除存储在变量 "my_fix" 中的装置 "id" 。
