tileset_get_name

此函数将返回指定图块集资源的名称作为字符串。此名称是在 GameMaker 主窗口的资源浏览器中为图块集指定的名称。

注意 这只是一个字符串，不能用于直接引用图块集-为此，您将需要图块集索引。但是，您可以使用此字符串来获取tile set index，方法是使用返回的字符串以及函数asset_get_index()。

 

语法:

tileset_get_name(index);

参数 类型 描述
index Tile Set Asset 要获取其名称的图块集的索引。

 

返回

String

 

示例:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

上面的代码将获取给定层的图块集索引的名称，并将返回字符串存储在变量tileset_name中。
