room_goto_next

使用此函数，您可以在游戏编译时进入房间管理器中列出的下一个游戏。如果此房间不存在，将抛出错误并迫使游戏关闭。请注意，在调用函数的事件结束之前，房间不会改变，因此，如果在相同事件中，调用后的任何代码仍将运行。

注意调用此函数后，您将无法在同一事件中创建新实例。

 

语法:

room_goto_next();

 

返回:

N/A

 

例子:

if room_exists(room_next(room))
{
    room_goto_next();
}

上面的代码将检查当前房间之后是否有另一个房间，如果有，它将转到该房间。
