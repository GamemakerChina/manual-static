program_directory

这将返回存储游戏可执行文件的目录。 然而，这可能并不总是有用，特别是当某些设备从 *.zip 文件运行 exe 时，因此无论游戏实际从何处运行，这都会返回相同的结果。

警告！由于 GameMaker 被沙盒化，此函数可能无法按您预期的那样工作！ 有关详细信息，请参阅文件系统部分。

注意：如果您禁用沙盒，那么您仍然无法写入此目录以防止删除所需游戏文件时可能出现的任何问题。

 

语法:

program_directory

 

返回:

String

 

例子:

dir = program_directory;

这会将可执行文件的存储目录存储在一个变量中。
