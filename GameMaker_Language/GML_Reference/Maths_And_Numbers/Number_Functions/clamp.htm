clamp

通过这个函数，您可以将输入值保持在指定的范围内。

 

语法:

clamp(val, min, max)

参数类型 描述
valReal 要夹紧的值。
minReal 夹紧之间的最小值。
maxReal 夹紧之间的最大值。

 

返回:

Real

 

例子:

speed = clamp(speed, 1, 10);

上面的代码将箝制速度，使它永远不会低于1或超过10。
