os_lock_orientation

使用此函数，您可以将设备"锁定"到当前方向，直到您"释放"它以允许该目标平台再次启用所有游戏选项方向。请注意，您可能希望在锁定之前确认方向是否如所需。

 

语法:

os_lock_orientation(flag)

参数 类型 描述
flag Boolean 设置为true或false以启用或禁用方向锁定。

 

返回:

N/A

 

例子:

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

上面的代码检查操作系统类型，如果它是Android或iOS则方向锁定被标记为true。
