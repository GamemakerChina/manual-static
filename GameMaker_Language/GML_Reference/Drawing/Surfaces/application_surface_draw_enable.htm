application_surface_draw_enable

可以使用此函数启用或禁用应用程序表面的自动绘制。默认情况下，此选项处于启用状态，但在许多情况下，您希望在绘制表面的时间和方式(例如，使用着色器时)占据主导地位，在这种情况下您可以使用此函数将其设置为false，以便可以在需要的时间和方式自己绘制它。请务必注意，当您关闭自动绘制并自己绘制应用程序表面时，可能会看到某些问题精灵的阿尔法分量和表面本身。这是因为GameMaker将在启用自动绘制时绘制不带Alpha混合的应用程序表面。如果关闭自动绘制，则需要使用以下代码(例如)自行处理此问题：

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

 

语法:

application_surface_draw_enable(flag);

参数类型 描述
flagBoolean 该标志将为 true (启用，默认值) 或 false (禁用)。

 

返回:

N/A

 

例子:

application_surface_draw_enable(false);

上述代码将关闭应用程序表面的自动绘制。
