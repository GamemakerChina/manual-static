chr

此函数返回一个字符串，其中包含与用于显示的输入Unicode代码相关的字符。此字符取决于当前图形字体字符集代码页，如果未设置字体，则将使用默认核心代码页。

 

语法:

chr(val);

参数 类型 描述
val Real 要从中获取字符串的Unicode代码值

 

返回:

String

 

例子:

mystring = chr(53) + chr(48);

这会将mystring设置为“50”(字符串，而不是整数)chr(53)为“5”，chr(48)为“0”。
