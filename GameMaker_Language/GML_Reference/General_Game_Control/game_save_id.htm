game_save_id

此 只读 变量指向在目标设备上为您的游戏保留的本地存储区域。请参阅 保存区域位置 。

一般来说，您不需要使用它来写入或读取保存区域中的文件，因为 GameMaker 在读取 / 写入文件时会自动解析保存区域的路径。例如，这个：

buffer_save(buf, game_save_id + "my_file.dat");

最好这样做：

buffer_save(buf, "my_file.dat");

该目录对于其他应用程序可能是可见的，也可能是不可见的，具体取决于平台。

注意 在 HTML5 目标上，它将返回一个空字符串。

阅读： 文件系统

 

语法:

game_save_id

 

返回:

String

 

例子:

save_dir = game_save_id;

这将存储用于将文件保存到变量以供将来使用的目录。
