math_get_epsilon

此函数将返回当前的精度值。有关精度的更多信息，请参见函数 math_set_epsilon()。

 

语法:

math_get_epsilon();

 

返回:

Real

 

例子:

var e = math_get_epsilon();
if e != 0.000001
{
    math_set_epsilon(0.000001);
}

这将检索当前的精度值并将其存储在局部变量中，然后检查该变量并在需要时设置新的精度。
