GX.games 游戏设置

GX.games 游戏选项允许您修改游戏和 mod 的属性。提供以下部分：

常规

此部分显示在为GX.games创建可执行文件时选择"上传为游戏"时使用的选项。

此窗口显示您的 GX.games 登录状态。您可以单击 刷新图标以使用您的 Opera 帐户登录。

然后，您可以选择以下选项：

  • 选择游戏：从此列表中，您可以选择要更新的现有游戏，或者选择上传新游戏。
  • 游戏名称：这是你的游戏在上传到 GX.games 上时游戏的名称（标题）。

    重要 为了成功上传到 GX.games，请务必只在游戏名称中使用允许的字符(默认情况下，该字符设置为 GameMaker 项目的名称，因此请确保选择有效的项目名称，或者稍后手动更改游戏名称)。

    如果您上传的游戏标题中包含禁止使用的字符，则会出现以下错误：Incorrect Data. Check the output

    允许的字符包括：
    • a-z、A-Z（支持方言字母，例如å）
    • 0-9
    • 空白区域
    • : - ! ? . , ' ; ( )
  • 版本：这是上次上传的游戏版本。您只能在游戏上传到 GX.games 之前更改此字段。
  • 下一版本：这是用于下一次 GX.games 上传的版本号。您只能在游戏上传到 GX.games 后才能更改此设置。您不需要为每次上传手动修改此内容，因为 GameMaker 将自动递增版本号的最后一位。
  • 在 GX.games 上注册：单击此按钮可在 GX.games 上注册一个新的游戏条目。仅当“选择游戏”设置为“新游戏”时，此选项才有效。
  • 选择组 : 在上传新游戏之前，您可以选择游戏所属的工作室。
  • 在 Opera 上编辑游戏 ：这将打开 GX.create 并将您带到游戏的 " 游戏详细信息 " 页面。只有在您登录并上传游戏时才可以单击此页面。
  • 内部共享 URL：这将打开您的游戏的私人版本，仅当它已在 GX.NET 上启用时。
  • 公共共享 URL：这将打开您的游戏的公开版本，仅当它已在 GX. NET 上启用时。

    这两个 URL 都可以与其他人共享，以允许他们玩您的游戏。

图形

这里您可以更改下列与游戏显示方式相关的细节:

  • 显示光标 : 控制游戏开始时鼠标光标是否可见。对于不需要鼠标输入或使用自定义鼠标精灵的游戏非常有用。
  • 在像素之间插入颜色 : 打开线性插值 ，基本上是“平滑”像素。对于清晰的像素图形，它应该关闭，但如果你有很好的Alpha混合和平滑的边缘图形，它最好开启。这是默认开启的。
  • 缩放比例：您的游戏可以配置为自动缩放绘制画布以在浏览器中保持纵横比，或者您可以选择使其运行满比例。满比例选项将不会在浏览器中完全显示游戏，而是拉伸绘制的内容以适应画布大小，如游戏第一个房间所定义的画布。默认情况下，此选项设置为保持纵横比。

最后，可以选择设置纹理页面的大小。默认（最兼容）大小为2048 x 2048，但您可以在256 x 256和8192 x 8192之间的任意位置进行选择。还有一个标记为预览的按钮，它将生成此平台的纹理页面，然后打开一个窗口，以便您可以看到它们的外观。如果希望了解纹理页面的结构以及防止纹理页面大于或小于所需的页面，这将非常有用。有关纹理页面的详细信息，请参阅此处。

Mod

此部分显示在为GX. games创建可执行文件时选择"上传为动态壁纸模块"时使用的选项。

此窗口还显示您的GX.games登录状态。您可以单击刷新图标以使用您的Opera帐户登录。

然后，您可以选择以下选项：

  • 选择模组：从此列表中，您可以选择要更新的现有模组，或选择上传新模组。
  • 模组名称：这是您在GX.games上的模组名称(标题)。

    重要提示为了成功上传到GX. games，请务必在游戏名称中仅使用允许的字符(默认情况下，设置为您的GameMaker项目名称，因此请确保选择有效的项目名称或之后手动更改游戏名称)。

    如果您上传的游戏标题包含禁用字符，您将收到此错误：不正确的数据。检查输出

    允许的字符是：
    • a-z，A-Z
    • 0-9
    • 空格
    • ：-！？.，"&()
  • 版本：这是上次上传的mod版本。您只能在mod上传到GX. games之前更改此字段。
  • 下一个版本：这是您下次上传GX.games时使用的版本号。您只能在您的mod上传到GX. games后更改此版本号。您不需要在每次上传时手动修改此版本号，因为GameMaker会自动为您增加版本号的最后一位。
  • 在GX.games上注册：单击此按钮可在GX. games上注册新条目。仅当"Select Mod"设置为"New Mod"时才有效。
  • 选择组：在上传新模组之前，您可以选择模组所属的工作室。
  • 在Opera上编辑模组：打开GX.create并进入模组详情页面。只有在登录并上传模组后才能点击。
  • 内部共享URL：这将打开您的mod的私人版本，仅当它已在GX. NET上启用时。
  • 公共共享URL：这将打开您的mod的公共版本，只有当它已在GX. NET上启用时。

    这两个URL都可以与其他人共享，以允许他们获得您的mod。
