File Directories

在处理文件时，有时您需要引用存储文件的目录。GameMaker 具有一些与您可能需要的不同可能目录相关的特殊函数和变量。

注意 您无法在 HTML5 上创建或销毁目录。

功能参考

基本

  • directory_exists
  • directory_create
  • directory_destroy

变量引用

路径

  • temp_directory
  • working_directory
  • program_directory
  • cache_directory

其他

  • game_save_id ( 来自 通用游戏控制 )
