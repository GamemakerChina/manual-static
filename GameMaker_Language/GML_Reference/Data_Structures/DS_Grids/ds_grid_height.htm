ds_grid_height

此函数将返回给定网格的高度。此值是网格沿y轴的单元格数，始终为整数，如下图所示：

 

语法:

ds_grid_height(index);

参数类型 描述
indexDS Grid ID 此网格索引用于查找网格的高度。

 

返回:

Real

 

例子:

for (var i = 0; i < ds_grid_width(grid); ++i)
{
    for (var j = 0; j < ds_grid_height(grid); ++j)
    {
        if (ds_grid_get(grid, i, j) == 1)
        {
            instance_create_Layer(i * 32, j * 32, "Walls", obj_Wall);
        }
    }
}

上述代码将循环通过变量"网格"中索引的DS网格，如果在任何特定单元格中找到的值等于1，则将在房间内的适当位置创建"obj_Wall"实例。
