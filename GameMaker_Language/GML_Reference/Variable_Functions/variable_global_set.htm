variable_global_set

使用此函数，您可以设置给定全局变量的值。您提供全局变量的名称以将的值设置为字符串(请参见下面的示例代码)，然后提供要设置的值 (可以是任何有效的数据类型)。如果全局变量在游戏中不存在，它将被创建，然后分配值。

 

语法:

variable_global_set(name, val);

参数类型描述
nameString要设置的全局变量的名称 (作为字符串)
valAny将全局变量设置为

 

返回:

N/A

 

例子:

if (!variable_global_exists("enemy_num"))
{
    variable_global_set("enemy_num", 0);
}

上面的代码将检查是否存在全局变量，如果不存在，则创建并设置为0。
