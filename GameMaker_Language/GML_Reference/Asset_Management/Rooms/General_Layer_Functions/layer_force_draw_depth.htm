layer_force_draw_depth

This function forces all layers to be drawn at the specified z depth.

This does not change the order the layers are rendered in and they'll still be drawn in depth order, it simply changes what z value is used. In general you do not need to worry about this, but if you have layers that have a depth outside of the legal range (-16000 to 16000) then they won't be rendered, so you can force the Z depth to a reasonable value - 0 for example - and they will all be rendered fine. Note that this is generally only for use with legacy projects from previous version of GameMaker where you could have draw depths higher or lower than the permitted layer range.

 

语法:

layer_force_draw_depth(force, depth)

参数 类型 描述
force Boolean 是启用(设置为true)还是禁用(设置为false)Z深度强制
depth Real 新Z深度

 

返回:

N/A

 

例子:

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

上面的代码检查层Z深度是否被强制，如果不是，它将Z深度设置为0并启用它。
