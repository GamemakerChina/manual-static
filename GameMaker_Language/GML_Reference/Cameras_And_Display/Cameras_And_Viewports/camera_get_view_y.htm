camera_get_view_y

此函数可用于检索给定相机的视图的 y 位置。请注意，此函数仅对于使用 camera_create_view() 创建的相机或添加到房间编辑器中的相机有效。

 

语法:

camera_get_view_y(camera_id)

参数类型 描述
camera_idCamera ID 创建相机时返回的唯一相机 ID 值。

 

返回:

Real

 

例子:

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

以上代码检索分配给 viewport[0] 的相机的位置，然后相对于该位置绘制文本。
