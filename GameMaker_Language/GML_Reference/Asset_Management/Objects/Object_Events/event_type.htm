event_type

此只读变量返回当前正在执行的事件类型，可以是以下常量之一：

  • ev_create
  • ev_destroy
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_mouse
  • ev_collision
  • ev_other
  • ev_gesture
  • ev_draw


有关与不同事件相关的常量的完整列表，请参阅event_perform，如果需要事件编号(即子事件)，则应检查event_number。

 

语法:

event_type;

 

返回:

Event Type Constant

 

例子:

show_debug_message("Current Event = " + string(event_type));

上面的代码将显示当前正在执行的事件类型的调试消息。
