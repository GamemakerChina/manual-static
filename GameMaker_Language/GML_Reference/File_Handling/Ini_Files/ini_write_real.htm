ini_write_real

您可以使用此函数将值 (数字) 写入 ini 数据文件。 Ini 文件被分成 部分 ，然后每个部分随后被分成 键 - 值 对。 所以一个典型的 ini 文件看起来像这样：

 

语法:

ini_write_real(section, key, value);

参数类型 描述
sectionString 要写入的 .ini 节。
keyString 要写入的 .ini 的相关部分中的键。
valueReal 写入相关目的地的真正值。

 

返回:

N/A

 

例子:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

这将设置 score 为 1000，然后打开“savedata.ini”，并将该值写入“save1”>“score”。然后，它将把 score2 设置为其中“save1”>“score”下的实数值 1000，并再次关闭 .ini。如果它没有找到这个值，score2 将最终设置为 0。
