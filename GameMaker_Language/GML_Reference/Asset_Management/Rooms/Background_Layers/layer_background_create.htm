layer_background_create

使用此函数，可以将精灵资源分配给要在项目中用作背景的层。提供层 ID(使用layer_create()创建层时获得的 ID)或层名称(作为字符串，这将产生性能影响)和 sprite 索引(如"资源浏览器"中显示的 sprite 的名称)，并将其添加到层中。函数返回元素的唯一ID值，然后，该ID值可用于背景的其他层函数中。

 

语法:

layer_background_create(layer_id, sprite)

参数类型描述
layer_idString or Layer目标层的唯一ID值(或作为字符串的层名)
spriteSprite Asset拟使用的精灵指数

 

返回:

Background Element ID

 

例子:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

上面的代码创建一个新层，然后添加一个新的背景元素，将一个精灵设置为所使用的背景图像。
