room_width

此变量以像素为单位保存当前房间的宽度。您可以随时更改此变量以更改房间的宽度。

 

语法:

room_width;

 

返回:

Real

 

例子:

if (bbox_right > room_width)
{
    x += room_width - bbox_right;
}

上面的代码检查当前实例的精灵边界框是否大于房间的宽度，如果是，则将实例向上移动。
