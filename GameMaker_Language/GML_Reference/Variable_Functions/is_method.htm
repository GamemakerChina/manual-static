is_method

此函数可用于检查并查看变量是否是方法变量 (它将返回true) 或不返回 (在这种情况下，它将返回false)。

注意 要检查值是否可调用(即引用方法或函数)，请参见 is_callable。

语法:

is_method(n);

参数 类型 描述
n Any 要检查的变量。

 

返回:

Boolean

 

例子:

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

上面的代码检查一个变量以查看它是否为方法，并且如果函数返回true, 则将调试消息输出到控制台。
