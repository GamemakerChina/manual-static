weak_ref_create

使用此函数，您可以创建一个指向结构的弱引用，然后可以用来检查该结构是否仍在 "活着"(引用)在游戏中。您提供对要跟踪的结构的引用，该函数将返回另一个结构，该结构是对该结构的弱引用。

使用说明

  • 您可以使用函数 instanceof() 来检查引用是 "强" 还是 "弱"，因为强引用将返回 "struct" 或创建该结构的构造函数的名称，如果是弱引用，则返回 "weakref"
  • 弱引用结构将有一个变量 "ref"，可以访问该变量来获得对相关结构的强引用，除非它已被垃圾回收，在这种情况下，它将返回 undefined。

 

语法:

weak_ref_create(struct_to_track);

参数 类型 描述
struct_to_track Struct or Method 要为其创建弱引用的结构或方法

 

返回:

Struct Weak Reference

 

例子:

inventory_ref = weak_ref_create(inventory);

上面的代码创建了对结构的弱引用，并将其存储在实例变量中供以后使用。
