layer_sprite_get_xscale

此函数可用于获得精灵元素沿 x 轴的当前比例乘数。给定精灵元素 ID(使用layer_sprite_create()或使用函数layer_sprite_get_id()创建精灵元素时得到的 ID)，该函数将返回 x 比例的真实值，其中 1 不是缩放。

 

语法:

layer_sprite_get_xscale(sprite_element_id);

参数类型 描述
sprite_element_idSprite Element ID 获取信息的精灵元素的唯一ID值

 

返回:

Real

 

例子:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_xscale(spr_id) != 1)
{
    layer_sprite_xscale(spr_id, 1);
}

上面的代码将获取名为"sprite_sky"的层的层ID，然后使用该代码来获取该层上的精灵元素的ID。然后，此 ID 用于检查 xscale 值，如果它不是 1，则设置为 1。
