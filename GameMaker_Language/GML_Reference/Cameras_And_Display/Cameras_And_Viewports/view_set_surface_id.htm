view_set_surface_id

使用此变量，可以将要绘制的视口内容设置为表面。使用表面时，通常需要将屏幕的整个可见区域捕获到表面，因此您可以使用此函数将其指定给视口。这意味着，所选视口中显示的所有内容现在都将绘制到指定的表面，并且该视口的内容将不再显示，这意味着您需要：

  • 启用新视图并仅在该视图中绘制表面(使用 view_current 检查正在绘制的视图)
  • 在实例的"绘制GUI"或"后期绘制"事件中绘制表面，因为这些事件与视图无关。

使用此函数时，可以给出视口索引(从0到7)和表面索引 (application_surface 或函数 surface_create() 返回的唯一索引值)，或者，如果之前已指定表面并且要删除它，则给出 -1 的值。有关将视口设置为表面的详细示例，请参见变量 view_surface_id。

重要 在为具有指定表面的视图绘制表面或纹理时必须小心，因为如果您尝试在同一视图内绘制视图的指定表面(或其纹理)，则会遇到一个错误，因为您正在尝试将纹理绘制到自身上。

 

语法:

view_set_surface_id(view_port, surf)

参数类型描述
view_portReal到目标的视口 (0 - 7)
surfSurface要应用于视图的表面

 

返回:

Real

 

例子:

if (view_get_surface_id(0) == -1)
{
    view_set_surface_id(0, global.vSurf);
}

以上代码将检查是否已将表面指定给 viewport[0] ，以及是否尚未指定表面。
