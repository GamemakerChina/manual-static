window_set_caption

使用此函数，您可以更改或设置当前所在房间的窗口标题。当游戏未处于全屏模式时，此标题将出现在窗口顶部的游戏图标旁边。

 

语法:

window_set_caption(caption);

参数类型描述
captionString新的标题。

 

返回:

N/A

 

例子:

if (window_get_caption() != "")
{
    window_set_caption("");
}

上述代码将检查窗口标题以查看其是否包含文本，以及是否包含文本，并将其设置为空字符串以便不显示标题。
