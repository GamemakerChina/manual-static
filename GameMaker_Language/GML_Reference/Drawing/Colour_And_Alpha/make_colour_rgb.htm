make_colour_rgb

GameMaker 提供此函数(以及其他函数)以允许用户创建自己的颜色。此特定函数包含三个组成部分，即您希望制作的颜色的红色、绿色和蓝色组件。这些值介于 0 和 255 之间，因此您可以使用此值生成 16,777,216(256 * 256 * 256) 种颜色！下面您可以看到这些组件分开时的外观图像：

左边的图像是函数各个组件的细分，右边的图像说明了更改这些组件如何影响端点颜色。

 

语法:

make_colour_rgb(red, green, blue);

参数类型 描述
redReal 颜色的红色分量
greenReal 颜色的绿色成分
blueReal 颜色的蓝色分量

 

返回:

Colour

 

例子:

col = make_colour_rgb(100, 145, 255);

上述代码使用该函数创建颜色并将其值存储在变量"col"中以供以后使用。
