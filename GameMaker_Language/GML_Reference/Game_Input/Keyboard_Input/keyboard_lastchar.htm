keyboard_lastchar

该变量存储最后按下的键的字符串。该变量 不是 只读的，您可以更改它，例如，如果您已经处理过它，则将其设置为 ""( 空字符串)。

注意 在 Xbox GDK 上，此变量将不包含任何输入。请参阅 这篇帮助台文章 了解更多信息。

 

语法:

keyboard_lastchar

 

返回:

String

 

示例:

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

上面的代码检查 keyboard_lastkey 变量是否大于 -1，如果是，则将最后一个键作为字符串添加到变量 str。
