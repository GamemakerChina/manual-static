GML 代码首选项

GML 代码首选项用于定义编码的某些语言功能将如何在 GameMaker 的 IDE 中显示。GML 代码的选项如下：

  • 在状态栏中显示参数类型：此选项与您何时使用自己的自定义脚本并已使用 JSDoc 表示法来标识不同的参数等有关……。(有关详细信息，请参阅此处)。选中此选项并在脚本中指定参数类型后，它们将显示在编辑器底部的状态栏中，并包含在{}中。默认情况下，此选项处于禁用状态。
  • 在状态栏中显示参数描述：如果选中此选项，则使用 JSDoc 表示法为您的自定义脚本提供的任何描述都将显示在编辑器窗口底部的状态栏中。默认情况下，此选项处于禁用状态。
  • 显示系统函数的美式拼写：默认情况下，GameMaker 对函数使用英式英语拼写，例如，draw_text_color()，但这可能不是您所习惯的，因此您可以使用此选项切换到美式拼写，使函数变为 draw_text_color()。默认情况下，此选项处于禁用状态。
  • 要显示的最大语法错误数：该值设置一次显示的最大语法错误数。如果您发现 IDE 滞后，则可以尝试减小此值。默认情况下，该值设置为 50。
  • 启用变量引用警告：启用时，这将标记变量的任何潜在问题，例如已在某个地方定义了变量，但实际上并未在代码中使用。禁用时，将不再显示这些警告。默认情况下，这是启用的。
  • 启用语句折叠为区域：启用后，代码中的方括号可以像区域一样折叠。这在默认情况下是启用的。
