camera_get_view_angle

此函数可用于检索给定摄像机的角度。返回值为0到360之间。注意，本函数只对使用 camera_create_view() 创建的相机或在房间编辑器中添加的相机有效。

 

语法:

camera_get_view_angle(camera_id)

参数类型描述
camera_idCamera ID创建相机时返回的唯一相机 ID 值。

 

返回:

Real

 

例子:

var ang = camera_get_view_angle(view_camera[0]);
if (ang != 0)
{
    camera_set_view_angle(view_camera[0], 0);
}

以上代码检索分配给 viewport[0] 的相机角度，然后检查该角度是否与给定值匹配。如果不是，则将视图的相机设置为该角度。
