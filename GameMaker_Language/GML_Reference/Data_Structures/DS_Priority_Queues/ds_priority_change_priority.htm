ds_priority_change_priority

此函数将采用给定的值，并在引用的优先级队列中更改其优先级。给定的值应该已存在于优先级队列中。

 

语法:

ds_priority_change_priority(id, val, priority);

参数类型描述
idDS Priority要更改的优先级队列的ID。
valAny要更改其优先级的值。
priorityReal值的新优先级。

 

返回:

N/A

 

例子:

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

上述代码检查全局变量，如果该变量低于某个值，则会更改保存在优先级队列中的脚本函数索引的优先级。
