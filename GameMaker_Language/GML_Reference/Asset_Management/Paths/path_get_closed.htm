path_get_closed

此函数可用于返回路径是否标记为关闭(true)或打开(false)，即路径是否循环或是否具有明确的开始和结束。

 

语法:

path_get_closed(index);

参数 类型 描述
index Path Asset 要检查的路径的索引。

 

返回:

Boolean

 

例子:

state = path_get_closed(pth_Patrol);

这将根据在"pth_Patrol"中索引的路径的关闭状态，将"state"设置为true或false。
