surface_free

在处理表面时，应始终在完成表面的使用后使用此函数。表面占用内存空间，因此需要移除，通常在房间的尽头，但可以随时移除，具体取决于您对表面的使用情况。如果不这样做，可能会导致内存泄漏，从而最终降低速度并使游戏崩溃。

注意：使用表面时，由于表面存储在纹理内存中，因此表面可能随时停止存在。在直接引用表面之前，应该始终使用surface_exists()检查表面是否存在。

 

语法:

surface_free(surface_id);

参数类型 描述
surface_idSurface ID 要释放的表面的ID。

 

返回:

N/A

 

例子:

if keyboard_check_pressed(vk_escape)
{
    surface_free(surf);
    room_goto(rm_Menu);
}

上述代码检查是否按键，如果检测到按键，它将释放为可变"浏览"中的索引表面保留的内存，然后更换空间。
