camera_get_proj_mat

此函数可用于检索投影矩阵。该函数返回矩阵数组，该矩阵数组随后可用于其他矩阵函数或设置另一个相机的投影矩阵(使用camera_set_proj_mat())。

 

语法:

camera_get_proj_mat(camera_id)

参数类型描述
camera_idCamera ID创建相机时返回的唯一相机 ID 值。

 

返回:

Matrix Array

 

例子:

mat = camera_get_proj_mat(camera_view[0]);

以上代码获取分配给 viewport[0] 的相机的投影矩阵。
