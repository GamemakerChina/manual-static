ds_map_size

使用此函数，您可以找到(先前创建的)DS映射包含的密钥/值对数。

 

语法:

ds_map_size(id);

参数类型 描述
idDS Map 要检查的数据结构的ID。

 

返回:

Real

 

例子:

if (ds_map_size(inventory) > 49)
{
    full = true;
}

上述代码将检查DS映射的大小(即：密钥/值对的数量)，如果大于49，则将变量"full"设置为true。
