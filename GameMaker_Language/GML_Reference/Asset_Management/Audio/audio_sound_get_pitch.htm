audio_sound_get_pitch

此函数可用于获取给定声音的音调。声音可以是使用audio_play_sound()或audio_play_sound_at()函数时存储在变量中的正在播放的单个声音的索引引用的声音，也可以是来自资源浏览器的实际声音资源。

 

语法:

audio_sound_get_pitch(index);

参数 类型 描述
index Sound Asset or Sound Instance ID or Audio Queue ID 声音的索引得到的音调。

 

返回:

Real

 

例子:

if audio_sound_get_pitch(snd_Explode) != 1
{
    audio_sound_pitch(snd_Explode, 1);
}

如果音频的音调不等于1，则上述代码将从索引为"snd_Explode"的声音改变所播放的音频的音调。
