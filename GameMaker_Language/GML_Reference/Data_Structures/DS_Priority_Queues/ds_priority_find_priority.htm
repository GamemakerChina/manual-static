ds_priority_find_priority

使用此函数可以检索任何给定值的优先级。如果优先级队列中不存在该值，则将返回未定义的。

 

语法:

ds_priority_find_priority(id, val);

参数 类型 描述
id DS Priority ID 要使用的优先级队列的ID。
val Any 要查找其优先级的值。

 

返回:

Real or undefined

 

 

 

例子:

p = ds_priority_find_priority(ai_priority, "intelligence");

上述代码将在实例变量"p"中存储给定值的返回优先级。
