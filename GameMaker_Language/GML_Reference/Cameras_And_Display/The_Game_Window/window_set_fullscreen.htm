window_set_fullscreen

使用此函数，您可以将游戏窗口设置为全屏 (true) 或非全屏 (false)。请注意，对于 macOS 目标，您必须取消选中"全屏启动"选项，并在游戏选项中选中"允许玩家调整游戏窗口大小"选项，否则此函数将无效。另请注意，此函数在 HTML5 上不起作用，除非它作为"可单击"回调添加(有关详细信息，请参阅此处了解更多)。

 

语法:

window_set_fullscreen(full);

参数类型描述
fullBoolean是否将屏幕设置为全屏(true)或不全屏(false)。

 

返回:

N/A

 

例子:

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

上面的代码检查鼠标按键是否被按下，如果还没有被按下，则将窗口设置为全屏，如果已经被按下，则将其设置为窗口。
