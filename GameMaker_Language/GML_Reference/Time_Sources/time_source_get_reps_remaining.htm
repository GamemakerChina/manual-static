time_source_get_reps_remaining

此函数返回给定时间源在完全停止之前所保留的重复次数。

如果给定的时间源无限期运行，或者它是内置时间源，则函数将返回 undefined。

 

语法:

time_source_get_reps_remaining(id);

参数类型描述
idTime Source要获取时间源的信息

 

返回:

Real
