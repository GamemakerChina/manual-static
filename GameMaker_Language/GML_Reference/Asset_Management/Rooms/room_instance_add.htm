room_instance_add

使用该函数，您可以将实例添加到当前房间以外的任何房间以及该房间中的任何位置。函数返回实例的唯一id，然后可用于进一步函数调用以设置属性等... 但只在游戏进入指定房间时。 如果要在当前房间中创建实例，应使用函数instance_create_layer()。 请注意，在资源浏览器中创建的房间资源上调用此函数将永久地将实例添加到房间，甚至调用game_restart()也不会将房间返回到其原始状态(只有结束游戏并再次打开它，房间将再次以原始状态开始)。

 

语法:

room_instance_add(index, x, y, obj);

参数类型 描述
indexRoom Asset 要向其中添加对象实例的房间的索引。
xReal 新实例的X位置。
yReal 新实例的Y位置。
objObject Asset 要添加其实例的对象。

 

返回:

Object Instance

 

例子:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

上面的代码将给游戏添加一个新房间，然后复制被索引为"rm_Base"的房间的内容。然后，它将在这个新房间的位置100，100添加对象"obj_player"的实例。
