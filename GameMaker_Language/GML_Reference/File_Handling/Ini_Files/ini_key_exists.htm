ini_key_exists

此函数检查当前打开的 ini 中是否存在键，如果存在，则返回 true ，否则返回 false 。这不是防止错误的必要检查，因为当一个键不存在时，从一个不存在的键读取只会返回一个默认值。但是，查看 ini 文件是否保存了特定的数据和其他一些内容会很有用。

 

语法:

ini_key_exists(section, key);

参数类型描述
sectionString打开的 .ini 文件中用于检入密钥的部分。
keyString要检查的键。

 

返回:

Boolean

 

例子:

ini_open("savedata.ini");
if (!ini_key_exists("save1", "name"))
{
    global.name = "Player1";
}
ini_close();

如果不存在诸如“name”之类的键，这将把变量 global.name 设置为“Player1”。
