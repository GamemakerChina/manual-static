tag_get_assets

This function retrieves the names of all assets that have been assigned the given tag or tags.

You supply either a single tag string or an array, where each item in the array is a tag string. The function will return an array where each entry is the name (as a string) of the asset with the given tag. If you need the unique index for the asset, then you can use the function asset_get_index() along with the returned name. If there are no assets with the given tag(s), or if there is an error (eg: the given tags do not exist), then an empty array will be returned.

 

语法:

tag_get_assets(tags);

参数 类型 描述
tags String or Array of Strings 单个资源标签字符串或具有各种资源标签的数组。

 

返回:

Array

 

例子:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

上面的代码创建一个列表，然后检索带有标签 “background” 的所有资源的名称。它循环遍历返回的名称数组，检查是否有任何名称是精灵资源，如果它们是资源的唯一索引值，则将其添加到列表中以备将来使用。
