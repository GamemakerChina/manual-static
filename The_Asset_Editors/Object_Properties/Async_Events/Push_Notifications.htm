推送通知

推送通知事件由设备操作系统上来自 推送通知 的 回调 触发，无论是来自本地源还是远程源。它生成此事件独占的 DS Map，并存储在特殊变量 async_load 中。

此事件仅由扩展程序 (例如 Firebase Cloud Message 扩展程序) 使用，因此请参阅扩展程序的留档以获取有关使用此事件的说明。
