parameter_count

命令行参数是您可以添加到 exe 中以更改程序运行方式的额外命令。您可以使用此函数找到当前游戏的参数数量，其中第一个参数的索引为 1，最后一个参数的索引为函数返回的索引 (0 值很特殊，因为它是游戏可执行文件的文件名，包括路径)。需要注意的是，该函数适用于 HTML5 平台，检索 URL 参数。

 

语法:

parameter_count();

 

返回:

Real

 

例子:

p_num = parameter_count();

上述代码会将游戏中使用的命令行参数的数量存储在变量 p_num 中。
