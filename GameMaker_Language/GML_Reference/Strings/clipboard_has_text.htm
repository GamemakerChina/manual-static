clipboard_has_text

如果剪贴板包含文本，此函数将返回 true，如果不包含文本，则返回 false。

注意 此函数仅在Windows，Android，MacOS，iOS，HTML5和Opera GX目标上有效。在HTML5上，在Firefox不使用扩展的情况下 不支持剪贴板，并且在Internet Explorer上完全不支持剪贴板。

 

语法:

clipboard_has_text();

 

返回:

Boolean

 

例子:

if (clipboard_has_text())
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

上面的代码检查剪贴板中的文本，如果其中包含任何文本，则将其作为字符串读取到变量 “str” 中。最后，通过将剪贴板设置为空字符串来清除剪贴板。
