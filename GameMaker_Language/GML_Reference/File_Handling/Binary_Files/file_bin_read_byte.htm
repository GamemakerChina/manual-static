file_bin_read_byte

此函数将从具有给定文件 ID 的文件中的当前位置返回一个字节的数据。 您提供函数 file_bin_open() 返回的文件 ID 值。

注意 : 当目标模块是 HTML5 时，这些函数 不能工作。

 

语法:

file_bin_read_byte(binfile);

参数类型描述
binfileBinary File ID要从中获取数据的文件的 ID。

 

返回:

Real

 

例子:

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

这将从与游戏相同的目录中打开一个文件，并从中获取一个字节的数据，然后再次关闭该文件。
