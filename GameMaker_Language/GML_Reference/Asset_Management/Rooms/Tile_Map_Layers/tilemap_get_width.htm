tilemap_get_width

使用此函数可以检索图块映射元素的宽度(以单元格为单位)。可以指定图块贴图元素 ID(当使用layer_tilemap_create()创建图块贴图元素时或当您使用函数layer_tilemap_get_id())，函数将返回宽度(单元格宽度的数量)。

 

语法:

tilemap_get_width(tilemap_element_id);

参数类型 描述
tilemap_element_idTile Map Element ID 图块映射元素的唯一ID值，以获取

 

返回:

Real

 

例子:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

上面的代码使用检索到的图块图ID来获取图块图的宽度和高度，然后使用这些值来设置实例变量。
