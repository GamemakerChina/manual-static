window_center

当目标模块为 Windows、Ubuntu(Linux) 或 MacOS 时，您可以使用此函数将游戏窗口居中显示，如果目标模块为 HTML5，则可以将其居中显示在浏览器中。此函数不影响任何其他设备。

注意 如果您在当前步骤中调整了游戏窗口的大小(例如，通过从全屏切换到窗口或使用 window_set_size() )，并希望使新窗口居中，则至少应在一步之后调用此函数(例如，使用 alarm)，否则它将无法正常工作。

 

语法:

window_center();

 

返回:

N/A

 

例子:

if !window_get_fullscreen()
{
    window_center();
}

如果不是全屏模式，上述代码将使游戏窗口在显示屏内居中。
