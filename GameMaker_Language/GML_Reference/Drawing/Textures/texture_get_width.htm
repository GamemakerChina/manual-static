texture_get_width

返回给定ID的纹理宽度，该值始终在0-1范围内。然后，可以在将纹理映射到模型或基本体时使用此选项。

 

语法:

texture_get_width(tex);

参数 类型 描述
tex Texture 要使用的纹理页面资源指针

 

返回:

Real

 

例子:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

以上代码将获取从精灵资源获取的纹理宽度。
