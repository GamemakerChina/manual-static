gesture_get_drag_time

此函数用于获取触摸或单击触发拖动开始手势事件所需的时间。这个时间也会影响Tap事件是如何触发的，因为在这个时间结束之前，触摸/点击和释放将被视为一个Tap。时间以秒为单位测量，默认值为0.16。

 

语法:

gesture_get_drag_time();

 

返回:

Real (time in seconds)

 

例子:

if (gesture_get_drag_time() != 0.16)
{
    gesture_drag_time(0.16);
}

上述代码检查手势的拖动时间是否设置为0.16秒，如果不是，则将其设置为该值。
