scheduler_resolution_get

此函数用于以毫秒为单位检索Windows线程调度程序的解析度。如果调度程序的解析度设置为默认值(由Windows设置)，则该函数将返回-1。

有关线程调度程序的信息，请参见scheduler_resolution_set().

注意：此函数仅适用于Windows。

 

语法:

scheduler_resolution_get();

 

返回:

Real (or -1 for default)

 

例子:

scheduler_resolution = scheduler_resolution_get();

此示例检索Windows线程调度程序的解析度，并将其存储在"scheduler_resolution"变量中。
