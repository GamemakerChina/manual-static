display_mouse_get_x

此函数将返回屏幕中的鼠标 X 位置。应注意，此函数仅在 Windows 和 macOS 目标平台上使用时才能正常工作。

它也可以用于 HTML5，但只会向画布本身的 0,0 返回值 relative，并且在鼠标位于画布外部时不会返回任何值。

对于其他设备，它将返回 0，您应改用 device_mouse_raw_x() 和 device_mouse_raw_y() 函数。

 

语法:

display_mouse_get_x();

 

返回:

Real

 

例子:

my_x = display_mouse_get_x();

这会将 my_x 设置为鼠标相对于显示器的 X 坐标。
