filename_drive

此函数返回文件名的驱动器信息。

 

语法:

filename_drive(fname);

参数类型 描述
fnameString 要使用的文件。

 

返回:

String

 

例子:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

上面的代码获取指定文件的驱动器信息(作为字符串)。
