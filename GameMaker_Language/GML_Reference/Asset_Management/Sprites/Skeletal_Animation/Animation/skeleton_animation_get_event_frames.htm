skeleton_animation_get_event_frames

此函数可用于检索给定动画中给定事件的所有帧。您可以提供骨骼动画名称(作为字符串，在用于制作动画的程序中定义，或使用函数skeleton_animation_get()返回)和事件名称。该函数返回一个数组，其中包含该事件的帧。

如果指定的事件名称不存在，则函数将返回一个以-1作为其第一个(也是唯一一个)元素的数组。

 

语法:

skeleton_animation_get_event_frames(anim_name, event_name);

参数类型描述
anim_nameString要获取其帧的动画名称。
event_nameString要为其获取帧的事件名称。

 

返回:

Array of Reals

 

例子:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

上面的代码获取"Footstep"事件的帧。如果它返回一个包含有效帧的数组，它将遍历该数组，并检查当前帧是否等于数组中的任何帧。在这种情况下，它会播放足迹声音效果并中断(停止)循环。
