gpu_get_alphatestref

您可以使用此函数查找 Alpha 测试引用的当前值 (默认值为 0，但可以使用 gpu_set_alphatestref() 将此值设置为非此值)。

 

语法:

gpu_get_alphatestref();

 

返回:

Real (from 0 - 255)

 

例子:

if (gpu_get_alphatestenable())
{
    if (gpu_get_alphatestref() < 254)
    {
        gpu_set_alphatestref(254);
    }
}

上述代码检查Alpha测试是否启用，如果启用，则检查当前Alpha测试参考值，如果小于254，则设置该参考值。
