ini_section_delete

使用这个函数，您可以删除一个 ini 文件的整个部分，这也将删除与之关联的所有键-值对。

 

语法:

ini_section_delete(section);

参数类型 描述
sectionString 要删除的部分。

 

返回:

N/A

 

例子:

ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
ini_section_delete("save1");
ini_close();

这个例子将打开 "savedata.ini" 并将值写入 "save1">"Score"。然后它将删除 "save1" 节并关闭 .ini 文件。
