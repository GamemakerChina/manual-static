room_first

此只读变量返回游戏中第一个房间的索引(这由房间在房间管理器中的显示顺序定义，而不是按创建房间的顺序定义)。

 

语法:

room_first;

 

返回:

Room Asset

 

例子:

if lives < 1
{
    room_goto(room_first);
}

上面的代码将检查" Lives "变量，如果它小于1，则转到游戏中的第一个房间。
