skeleton_animation_set_frame

此函数将指定给给定轨迹的动画设置为您提供的帧。帧索引应该是一个介于0和动画帧数之间的实数值(可以使用函数skeleton_animation_get_frames)。如果提供的值超出此范围，则将循环动画以弥补差异，例如，如果动画有5帧，而您将帧设置为8，则实际显示的帧将为3。

 

语法:

skeleton_animation_set_frame(track, index);

参数 类型 描述
track Real 要设置帧的动画轨迹。
index Real 动画帧的新索引。

 

 

返回:

N/A

 

例子:

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

当全局"暂停"变量为true(实质上是暂停动画)时，上面的代码将把轨道0中的动画的帧索引设置为负图像速度。
