layer_fx_is_enabled

此函数返回是否启用图层的过滤器/效果。

可以指定要目标的层的 ID 或名称，函数将返回一个布尔值，指示层的 FX 是已启用(true)还是禁用(false)。

 

语法:

layer_fx_is_enabled(layer_name_or_id);

参数 类型 描述
layer_name_or_id String or Layer 要检查的层的名称或ID

 

返回:

Boolean

 

例子:

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

上面的代码通过将实例移动到其原始位置并填充其HP来"重新生成"实例。然后检查是否启用了去饱和FX层，如果启用，则禁用它。
