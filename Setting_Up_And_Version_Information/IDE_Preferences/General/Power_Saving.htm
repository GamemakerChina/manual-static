节能首选项

此处的节能首选项将影响将使用多少 CPU 功率和 GPU 功率来生成 GameMaker IDE (本质上，这将降低程序运行的基本帧速率，从而降低功耗)。

这里的选项包括：

  • 交流电源帧速率 (%)：连接到电源输入时的基本 IDE 帧速率修改器。默认值为 100%。
  • 电池电源帧速率 (%)：使用电池为机器供电时的基本 IDE 帧速率修改器。默认为 35%。
  • 后台帧速率 (%)：当 GameMaker 未对焦时 (与电源无关)，基本 IDE 帧速率修改器。默认为 15%。
