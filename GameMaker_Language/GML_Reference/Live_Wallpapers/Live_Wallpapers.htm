动态壁纸

此页面列出了与动态壁纸功能一起使用的功能。

有关如何设置的信息，请参阅 GX动态壁纸上的帮助台文章。

函数

动态壁纸具有以下功能：

  • wallpaper_set_config
  • wallpaper_set_subscriptions

另请参阅： 壁纸事件

限制

通过商店安装后，动态壁纸将受到以下限制：

  • 鼠标输入 ：默认禁用，通过 wallpaper_set_subscriptions 启用
  • 文件访问 ：适用标准 文件沙箱 ; 禁用沙盒 不会执行任何操作
  • 网络 ：完全禁用 (包括 TCP、UDP、HTTP 等)
  • 麦克风 ：已禁用
  • 扩展 ：禁用对 扩展 中包含的 DLL 函数的调用 (包括 external_call、external_define 等)
