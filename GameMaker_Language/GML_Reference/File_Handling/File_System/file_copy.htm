file_copy

此函数将复制指定的文件，并为其指定您选择的新名称。

如果您在 Android 上从 内含文件 区域 (将在您的 APK 内) 复制文件，它将首先复制到内存中，
因此请注意，从 APK 复制非常大的文件可能会导致设备运行内存不足。

在 HTML5 目标上，我们使用 HTML5 Local Storage API 进行客户端保存，
并且这些函数不支持二进制文件操作，因此假设您正在复制文本文件。

如果 newname 文件已经存在，则函数将失败。

警告 由于 GameMaker 处于沙盒状态，这可能无法按您的预期工作！请参阅有关 文件系统 的部分了解更多信息。

 

语法:

file_copy(fname, newname);

参数 类型 描述
fname String 要复制的文件的名称。
newname String 新复制文件的名称。

 

返回:

N/A

 

例子:

if (file_exists("level1.txt"))
{
    file_copy("level1.txt", "level2.txt");
}

这将检查一个文件，如果它存在，它将被复制。
