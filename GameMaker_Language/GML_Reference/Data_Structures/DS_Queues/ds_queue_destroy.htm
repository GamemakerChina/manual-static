ds_queue_destroy

此函数将从内存中删除给定的队列数据结构，释放它正在使用的资源并删除它包含的所有值。当您使用完 DS 队列后，应始终使用此函数，以防止内存泄漏导致游戏速度减慢和崩溃。

注意 调用此函数后，您应始终将保存数据结构引用的变量设置为 -1，因为该引用将不再有效。

 

语法:

ds_queue_destroy(id);

参数类型描述
idDS Queue要删除的数据结构的ID。

 

返回:

N/A

 

例子:

if (lives == 0)
{
    ds_queue_destroy(AI_queue);
    AI_queue = -1;
    room_goto(rm_Menu);
}

上述代码将检查内置全局变量 lives 的值，如果为 0，则会销毁变量 AI_queue 中索引的 DS 队列，然后更改房间。
