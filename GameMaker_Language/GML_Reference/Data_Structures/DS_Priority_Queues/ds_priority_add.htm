ds_priority_add

使用此函数，您可以向优先级队列添加值(实数或字符串)，同时为其分配优先级值。

 

语法:

ds_priority_add(id, val, priority);

参数 类型 描述
id DS Priority ID 要添加到的优先级队列的ID。
val Any 要添加到优先级队列的值。
priority Real 要添加的值的优先级。

 

返回:

N/A

 

例子:

ds_priority_add(ai_priority, AI_Search, 5);

上述代码将脚本函数添加到变量"ai_priority"中索引的优先级队列，并为其分配优先级5。
