skeleton_attachment_destroy

此函数用于销毁使用 skeleton_attachment_create/skeleton_attachment_create_colour 或 skeleton_attachment_replace/skeleton_attachment_replace_colour 函数之一创建的自定义附件.

销毁附件后，该名称将再次可用，您可以使用该名称并通过 skeleton_attachment_create 创建新附件.

语法:

skeleton_attachment_destroy(name);

参数 类型 描述
name String 要销毁的附件的名称

 

返回:

N/A

 

例子:

skeleton_attachment_destroy("leg_left");

上面的代码销毁了名为 "leg_left" 的现有自定义附件。
