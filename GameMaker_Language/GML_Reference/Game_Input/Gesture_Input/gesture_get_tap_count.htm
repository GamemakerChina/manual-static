gesture_get_tap_count

该函数用于检查点击计数是否已启用或禁用。如果启用，该函数将返回true，否则返回false，您可以使用函数启用或禁用点击计数gesture_tap_count()。启用后，每个tap事件将具有一个附加的 “tapcount” DS map条目，该条目将注册tap的数量，并且启用后，这意味着所有tap事件将被触发，即: 两次tap将同时触发单次tap事件和双击事件，单击事件抽头计数为1，双击事件抽头计数为2。在为双重抽头检测设置的时间过去之后，抽头计数值将重置为0 (请参阅函数gesture_double_tap_time())。如果禁用了轻击计数，则直到双击时间过去且未检测到第二次轻击之前，才会注册初始轻击。请注意，默认情况下，这是启用的。

 

语法:

gesture_get_tap_count();

 

返回:

Boolean

 

例子:

if (!gesture_get_tap_count())
{
    getsure_tap_count(true);
}

上面的代码检查是否启用了点击计数，如果没有启用，则将其打开。
