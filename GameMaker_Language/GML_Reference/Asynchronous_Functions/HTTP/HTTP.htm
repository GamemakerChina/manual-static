HTTP

本部分列出了 GameMaker 中可用的所有不同异步 HTTP 函数。 这些函数将在所有具有它的实例中生成一个异步 HTTP 事件：

  • http_request
  • http_get
  • http_get_file
  • http_post_string

跨域问题

请注意，由于 跨域安全 问题，上述 http_* 函数可能无法按预期运行。这意味着向您的服务器发出的请求或跨域加载资源的尝试将被阻止，并且可能会返回空白结果或 404 错误。

对于需要确定或设置跨域类型的镜像加载或 HTTP 请求，存在以下函数：

  • http_get_request_crossorigin
  • http_set_request_crossorigin

在大多数情况下，应该不需要使用这些函数，但如果游戏存储在安全的服务器上-在该服务器上，某些资源可能需要进行基本身份验证才能访问，并在加载时生成安全错误-可能需要将图像请求的跨域类型设置为 "use-credentials"( 与默认的 "anonymous" 设置相反)。
