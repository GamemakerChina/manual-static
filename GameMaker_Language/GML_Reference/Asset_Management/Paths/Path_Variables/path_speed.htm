path_speed

使用函数path_start()，可以使用此函数获取或设置路径的速度。 您可以使用负值来表示实例应反向的路径。

 

语法:

path_speed;

 

返回:

Real

 

例子:

path_speed = -1 + random(2);

上面的代码将实例设置为以-1和1之间的随机速度行进路径。
