常规首选项

常规首选项 用于定义 GameMaker IDE 的常规行为。最常见的是在主页上列出的，而许多其他的则被分成多个子类别，本页下面将对此进行进一步解释。

  • IDE 字体：此选项允许您更改在整个 IDE 中使用的字体。默认为 Open Sans。
  • IDE 语言：此选项允许您从提供的选项列表中选择 IDE 的语言。默认为 English。
  • GameMaker 的最大帧速率编辑器：此选项允许您设置 GameMakerIDE 更新时的最大帧速率，允许您设置帧速率以匹配显示器的更新速度。默认值为 60。
  • IDE 外观：您可以在此处从 IDE 可以使用的不同外观中进行选择。默认情况下，GameMaker 有两种皮肤，一种浅色和一种深色，初始设置使用深色皮肤。
  • 默认构建目标：这是用于项目的默认构建目标。默认为 GX.games。
  • 默认音频设备：您可以在此处选择供 GameMaker IDE 使用的默认音频设备。您可以通过单击刷新按钮来刷新列表。
  • 自动响应退出确认：此选项允许您设置在退出 GameMaker 时是否显示确认消息。默认情况下，该选项被设置为显示消息，但也可以设置为跳过消息，就像您单击了"确定"一样。
  • 自动响应对系统字体更改的对话框：此选项允许您设置在添加、删除或编辑系统字体时是否显示确认消息。默认情况下，该选项被设置为显示消息，但也可以设置为跳过消息，就像您单击了"确定"一样。
  • 启用通知：选中此选项将使 IDE 能够显示有关新运行时的通知，等等...。默认情况下，该选项处于启用状态。
  • 通知超时(毫秒)：设置显示通知的时间长度(以毫秒为单位)。默认为 6000。
  • 退出时注销：如果选中此选项，则在关闭 GameMaker 时，IDE 将自动注销用户会话。这主要是针对教育用户，因为-选中-意味着每当程序关闭时，用户将被注销，从而释放会话席位供其他用户使用。默认情况下，此选项处于禁用状态。
  • 允许通过键盘快捷键关闭空选项卡：如果启用此功能，则可以通过其键盘快捷键 ( / + W) 关闭空工作区选项卡。禁用时，不能关闭空选项卡。
  • 在将文件拖入 IDE 时打开新资源以进行编辑：某些资源(如精灵或声音)可以拖到 GameMaker IDE 中的主工作区，并将被检测并作为新资源添加到资源浏览器中。默认情况下，发生这种情况时，也会打开相应的资源编辑器进行编辑，但取消选中此选项将阻止打开编辑器，并要求您手动打开它们。
  • 使用美国日期格式：此设置更改用于在 IDE 中显示日期的格式。启用时，日期显示为例如。"Thursday, April 27, 2023"，禁用时，日期显示为例如 "27 April 2023"。默认情况下，此首选项处于未选中状态。
  • 创建时打开新资源以进行编辑：如果选中此选项，则每次在资源浏览器中创建新资源时，都会自动打开相应的资源编辑器。默认情况下，该选项处于选中状态，取消选中意味着您必须手动打开所有新创建的资源。
  • 自动重新加载更改的文件：如果选中此选项，IDE 将自动扫描和检查文件，以查看其中是否有任何文件在 GameMaker 之外进行了更改，如果有，则重新加载。
  • 项目文件不区分大小写模式(可能会影响性能)：旨在避免非 Windows 设备 (MacOS 和 Linux) 上的文件名区分大小写问题。在将项目从 Windows 传输到非 Windows 系统、使用启用了 忽略大小写 的 Git (您应该先尝试禁用它) 和 / 或仅重命名文件以更改大小写时，可能会发生此类问题。这在默认情况下是禁用的，因为这是一种性能开销，在上述情况之外不应该要求这样做。
  • 禁用IDE过渡动画：GameMaker IDE 有许多动画过渡，例如打开窗口等。然而，并不是每个人都喜欢它们，如果在较老的或性能不足的计算机上禁用它们，您可能会得到性能提升。您可以在此处禁用它们，默认情况下它们处于打开状态。
  • 启用 DPI 覆盖：此选项允许您使用您自己的设置覆盖默认 DPI 设置。您必须首先启用该选项，然后设置原始 DPI 覆盖值 (缺省值为 96) 或设置本机 DPI 的 百分比 ( 缺省值为 100%)，一旦更改，系统将提示您重新启动 IDE，这可能需要您保存在当前打开的项目上所做的任何工作。请注意，DPI 的最小百分比为 50%，最大百分比为 500%，如果您将鼠标悬停在信息图标上，您可以看到正在使用的显示器的建议最大设置：

    请注意，如果您超出显示器的建议 DPI 设置，则会显示允许您重置 DPI 的通知。
