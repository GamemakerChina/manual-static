network_resolve

此函数将返回给定URL的IP地址。

 

语法:

network_resolve(url);

参数 类型 描述
url String 获取(字符串)的IP的网址。

 

返回:

String

 

例子:

game_ip = network_resolve("www.macsweeneygames.com");

上述代码将返回给定URL的IP地址并将其存储在变量"game_ip"中。
