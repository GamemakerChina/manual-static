DS 优先队列

DS优先队列是一种类似于DS队列的特殊数据结构，只有在这里，值队列才按照已分配值的优先级(始终为实数)排序。此结构对于构建可以权衡每个条目的重要性的排行榜或信息列表非常有用。

注意：与所有动态资源一样，数据结构占用内存，因此当不再需要防止内存泄漏时，应始终销毁，这将降低速度并最终导致游戏崩溃。

注意：如果需要检查是否存在数据结构，可以使用ds_exists()函数。

存在以下处理Ds优先级队列的函数：

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
