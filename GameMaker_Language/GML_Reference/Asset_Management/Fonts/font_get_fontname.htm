font_get_fontname

使用该函数，可以获取给定字体资源的实际系统名称。此函数返回字符串而不是索引，返回的名称将取决于所使用的字体。例如，您可能在资源浏览器中有一个名为"fnt_Main"的字体资源，字体本身可能是Windows系统字体Arial。 在这种情况下，函数将返回" Arial"，因为这是字体的系统名称。如果需要资源浏览器中显示的名称，请使用font_get_name()。

上述行为仅适用于通过IDE添加的字体资源。当在运行时加载的字体上使用时(使用font_add())，该函数将返回该字体文件的完整路径而不是其字体名称。

 

语法:

font_get_fontname(ind);

参数类型描述
indFont Asset要检查的字体的索引。

 

返回:

String

 

例子:

fnt_Name = font_get_fontname(font0);

上述代码将获取字体资源的系统名称，并将其存储为变量"fnt_Name"中的字符串。
