part_emitter_stream

此函数将发射器设置为传输特定类型的粒子。

注意如果您需要粒子同时出现而不是每一步出现，则应使用函数part_emitter_burst。

用法注释

  • 此函数通常用于Create事件，因为它是一次性代码，用于设置发射器以生成此后游戏中每一步指定的粒子数。
  • 粒子按照函数part_emitter_region设置的分布、形状和位置发射。
  • 您可以将粒子流的数量设置为零，并再次调用此函数以"关闭"粒子流，并且该函数还将接受数量的负数，在这种情况下，发射器将基于随机机会产生粒子。例如，如果将粒子数设置为-5，则每一步生成粒子的概率为1：5。
  • 如果使用part_emitter_relative启用了发射器的相对模式，则number参数不是指创建的粒子的实际数量，而是指示发射器区域的覆盖百分比。

 

语法:

part_emitter_stream(ps, ind, parttype, number);

参数 类型 描述
ps Particle System Instance 发射器所在的粒子系统。
ind Particle Emitter ID 要从中进行流式发射器的索引。
parttype Particle Type ID 要创建的粒子的索引(类型)。
number Real 每个步骤要创建的粒子的数量，或密度(即，粒子的密度)。发射器区域的覆盖百分比)，并启用相对模式(参见part_emitter_relative)

 

返回:

N/A

 

例子:

part_emitter_stream(global.Sname, p_emit1, p1, 1);

上述代码将在游戏的每个步骤中流1粒子，直到发射器被破坏或流被设置为0。
