date_date_string

使用这个函数，您可以创建一个包含给定日期的字符串，格式为日/月/年。

 

语法:

date_date_string(date);

参数类型 描述
dateDatetime 要使用的日期。

 

返回:

String

 

例子:

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

这将设置"str"以保存系统显示的当前日期和时间的格式化字符串。
