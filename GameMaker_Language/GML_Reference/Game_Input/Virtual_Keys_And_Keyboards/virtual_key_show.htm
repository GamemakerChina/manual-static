virtual_key_show

为设备创建虚拟按键后，可能需要进行调试并测试其位置是否正确并且工作正常。为此，您可以调用此函数，使用要查看的虚拟按键的索引(以前使用virtual_key_add()创建和存储的虚拟密钥)，该索引将绘制一个彩色矩形来表示屏幕上的按键。一旦您对某些事情感到满意，建议您在其区域中创建并放置自己的图形。

注意正在绘制的矩形将受到相应绘制函数设置的混合、颜色和alpha选项的影响。

 

语法

virtual_key_show(index);

参数 类型 描述
index Virtual Key ID 要显示的虚拟键的索引。

 

返回:

N/A

 

例子:

if (global.Debug)
{
    virtual_key_show(global.Left);
}
else
{
    virtual_key_hide(global.Left);
}

上面的代码检查全局变量“ global.debug ”，如果测试结果为true，则在屏幕上绘制变量“ global.left ”中索引的虚拟键，如果测试结果为false，则隐藏该键。
