string_height

该函数将返回输入字符串的高度 (以像素为单位)，同时考虑到行分隔和文本可能具有的任何换行符。根据最高的字母计算文本元素之间的距离非常方便组成字符串，因为它将使用 draw_text() 使用当前定义的字体绘制。

 

语法:

string_height(string);

参数 类型 描述
string String 要测量的高度的字符串。

 

返回:

Real

 

例子:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

上面的代码将获取给定字符串的高度，然后绘制两行文本，使用返回的字符串高度作为分隔符。
