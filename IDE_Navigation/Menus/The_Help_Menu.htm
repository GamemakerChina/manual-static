帮助菜单

此菜单与使用 GameMaker 时获取帮助有关。它包含以下菜单项：

  • 打开手册 - 打开帮助手册。请注意，单击菜单栏中的帮助按钮也会打开文档，按下也会打开文档。
  • 知识库 - 这将把您带到 GameMaker 知识库，在那里您可以找到有关平台设置、已知问题、使用 GameMaker 以及其他许多内容的文章。
  • 报告错误 - 如果您在使用 GameMaker 时发现任何错误或有任何问题，您可以在此处报告它们。
  • 在资源管理器中打开日志 - 这将打开您系统上的目录，您可以在其中找到 ui.log 文件，该文件通常用于提交与 IDE 相关的错误报告。
  • GameMaker.io - 打开 GameMaker 网站 。
  • GameMaker 社区 - 这将在您的浏览器中打开 GameMaker 社区论坛。请注意，这是一个社区运作的论坛，所以如果您有错误报告或需要技术帮助，您应该首先转到官方支持站点(见上文)。
  • 发行说明 - 这将在您的浏览器中打开适用于 GameMaker 的发行说明。如果您需要回滚更新，您可以通过单击此页面上的版本号链接获取以前版本的安装程序。
  • 当前运行时发行说明 - 由于运行时可以独立于IDE进行更新，因此它有自己的一套运行时发行说明。选择此选项将在您的浏览器中打开它们，但请注意，最新的运行时可能不是您正在使用的运行时。要了解您使用的运行时，或有关如何添加、删除或激活运行时的信息，请参阅运行时首选项部分。
  • 所需SDK - 由于 GameMaker 的跨平台特性，需要许多第三方 SDK。单击此链接将转到一篇支持文章，其中详细介绍了您在每个平台上需要什么。
  • 在资源管理器中打开项目 - 这将在操作系统文件资源管理器中打开当前加载项目的保存位置。
  • 刷新系统字体 - 这将刷新 GameMaker 能够看到的系统字体，并有助于检测您在打开 IDE 时安装的新字体。
  • 许可证 - 这将打开一个窗口，其中包含 GameMaker 中使用的所有第三方库的许可证。
  • 关于 - 这将打开"关于 GameMaker"窗口，您可以在其中获取产品的版本信息(报告任何错误或问题所必需的)并查看制作者名单。
