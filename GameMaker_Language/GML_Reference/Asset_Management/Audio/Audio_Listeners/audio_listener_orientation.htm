audio_listener_orientation

使用此函数，您可以更改收听者在 3D 音频空间中的方向。查看方向和向上方向基于从给定相对 x、y 和 z 位置解析的向量，默认为( 0， 0， 1)表示查看方向，(0， 1， 0)表示向上方向，如下图所示：

注意如果您有多个监听器，应使用函数audio_listener_set_orientation()。

使用此函数更改收听者方向将改变游戏室周围的音频发射器产生的声音被游戏玩家感知的方式。在下面的示例中，当收听者位于默认位置时，发射器产生的声音似乎来自收听者的右下方，但在收听者的新位置和方向下，它们现在将被感知为来自右上方。

语法:

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

参数 类型 描述
lookat_x Real X 环视向量 (默认为 0)。
lookat_y Real Y 环视向量 (默认为 0)。
lookat_z Real Z 环视向量 (默认为 1)。
up_x Real X 向上向量 (默认为 0)。
up_y Real Y 向上向量 (默认为 1)。
up_z Real Z 向上向量 (默认为 0)。

 

返回:

N/A

 

例子:

dx = dcos(direction);
dy = - dsin(direction);
dz = - dsin(zdirection);
audio_listener_position(x, y, z);
audio_listener_orientation(dx, dy, dz, 0, 0, 1);

上面的代码将3D音频监听器位置设置为实例的x和y变量以及额外的z变量。然后，它将音频收听者的方向设置为从实例的方向计算的三个变量和一个额外的zdirection变量。
