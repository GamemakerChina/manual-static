clickable_delete

这个函数必须用于从游戏窗口中删除之前用clickable_add()创建的可点击图标。

 

语法:

clickable_delete(index);

参数类型描述
indexClickable ID要删除的可单击图标的索引。

 

返回:

N/A

 

例子:

if (clickable_exists(global.Help_Icon))
{
    clickable_delete(global.Help_Icon);
}

以上代码从游戏窗口中删除了全局变量“Help_Icon”中的可点击图标(如果它存在的话)。
