texture_get_height

此函数返回具有给定 ID 的纹理的高度，该高度始终是 0 - 1 范围内的值。然后可以在将纹理映射到模型或图元时使用该高度。

 

语法:

texture_get_height(tex);

参数类型描述
texTexture要使用的纹理页面资源指针

 

返回:

Real

 

例子:

tex_h = texture_get_height(surface_get_texture(global.Surf));

以上代码将获取从先前创建的表面获取的纹理高度。
