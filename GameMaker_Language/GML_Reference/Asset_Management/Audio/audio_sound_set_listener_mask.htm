audio_sound_set_listener_mask

此函数可用于设置声音的位掩码，以便仅从指定的收听者播放。可以使用 audio_get_listener_info() 创建位掩码，然后使用位或("|") 与需要播放声音的收听者创建自定义掩码，然后在播放声音后将此自定义掩码应用于声音。此掩码将覆盖默认掩码或使用函数 audio_set_listener_mask() 设置的掩码。

此函数只能在声音设置为播放后使用，并且需要播放声音的函数返回的声音 ID。

 

语法:

audio_sound_set_listener_mask(soundID, mask);

参数 类型 描述
soundID Sound Instance ID 声音的唯一ID，用于设置的掩码
mask Real 为声音设置的位掩码数据

 

返回:

N/A

 

例子:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

上面的代码播放声音，然后设置收听者使用存储在全局变量中的掩码数据来播放。
