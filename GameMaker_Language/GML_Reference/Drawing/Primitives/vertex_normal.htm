vertex_normal

此函数将普通数据添加到当前为自定义基元定义的顶点。

您提供用于写入数据的顶点缓冲区以及法线的 x、y 和 z 分量。

 

语法:

vertex_normal(buffer, nx, ny, nz);

参数 类型 描述
buffer Vertex Buffer 要写入信息的顶点缓冲区。
nx Real 法线的 x 分量。
ny Real 法线的 y 分量。
nz Real 法线的 z 分量。

 

返回:

N/A

 

例子:

vertex_normal(buff, 0, 1, 1);

上述代码将设置所定义顶点的表面法线。
