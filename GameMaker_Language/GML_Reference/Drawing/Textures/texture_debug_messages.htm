texture_debug_messages

此函数可用于启用或禁用纹理调试消息。启用时(设置为true)，有关纹理页面使用的其他信息将发送到输出窗口。设置为false以再次禁用此输出。

 

语法:

texture_debug_messages(enable);

参数类型 描述
enableBoolean 启用或禁用纹理调试消息

 

返回:

N/A

 

例子:

if global.debug
{
    texture_debug_messages(true);
}

以上代码将检查全局变量的值，如果设置为true，则启用纹理调试消息。
