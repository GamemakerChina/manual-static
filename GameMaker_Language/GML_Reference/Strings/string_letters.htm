string_letters

使用此函数，可以删除所有未分类为字母的字符。这便于防止人们在文本条目中输入不需要的字符(如输入“#”以强制输入新行)。

注意 此函数仅检测并返回从 A 到 Z (大写或小写)的 26 个字母的英文字母。

 

语法:

string_letters(string);

参数 类型 描述
string String 要转换为字母的字符串。

 

返回:

String

 

例子:

username = string_letters(username);

上面的代码将 "username" 变量设置为仅保存原始字符串的 字母 字符 (原始字符串的 ID 为 "macsw33ny"，返回的字符串将为 "macswny")。
