gc_enable

使用此函数，您可以启用或禁用垃圾回收器。使用true作为参数调用函数将启用它，使用false将禁用它(不推荐)。默认情况下，它处于启用状态。

 

语法:

gc_enable(enable);

参数类型 描述
enableBoolean 启用(true)或禁用(false)垃圾回收器。

 

返回:

N/A

 

例子:

if (global.debug == true)
{
    gc_enable(false);
}

如果给定的全局变量为true，上述代码将禁用垃圾收集。
