md5_file

在密码学中，MD5( 消息摘要算法 5) 是一种广泛使用的加密哈希函数，其哈希值为 128 位，已被广泛应用于各种安全应用程序中。它还通常用于检查文件和字符串的完整性。该函数将接受给定的文件并为该文件生成唯一的 MD5，然后将其存储起来供以后使用。

注意: MD5 不是完全安全的，可以被破坏。参见本页获取更多信息。

 

语法:

md5_file(filename)

参数类型 描述
filenameString 要为其生成 MD5 哈希的文件。

 

返回:

String

 

例子:

hash = md5_file(working_directory + "game_data.ini")

上面的代码将为指定的文件生成一个 MD5 散列，并将返回的值存储在变量“ hash”中。
