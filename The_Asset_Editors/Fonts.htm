字体编辑器

When you want to draw text in your game this text is drawn in a standard Arial 12-point font, but to make more interesting or unique looking texts you will probably want to use different fonts. So, to use different fonts that you have on your computer you must create a font asset in GameMaker. This is done by right clicking  on the Asset Browser and selecting "Create" -> "Font". For each font asset you create you specify a particular type of font from your computer which can then be used in your game using the action or code to set a font for drawing.

Fonts in GameMaker can make use of SDF rendering so text stays sharp as it's scaled up. Enable it under Rendering Options (listed below), or run font_enable_sdf on the font at runtime.

Once SDF is enabled, you can apply various effects to your font at runtime.

范围控制范围控制

当创建一个字体资源时，你很少会想要包括字体的每一个字形。这就是为什么GameMaker默认只使用所选字体中从32到127的字符，因为这给了我们数字和 小写和大写字母，以及某些标点符号。然而，你可以改变使用的字符范围，以帮助优化你的游戏，例如：如果你只需要某个特定字体的数字，那么只选择数字。要做到这一点 要做到这一点，你应该首先点击删除按钮来清除当前的范围（你可能需要通过点击预览窗口中的 来选择这个范围），然后点击添加按钮来添加一个新的范围。这将打开以下链式窗口：

这个窗口有一些按钮可以帮助你为你的字体建立一个标准范围。

   - 普通范围从32到127

   - ASCII范围从0到255

   - 数字范围仅包含10位数字

   - 字母范围仅包含所有大小写字母

其他范围可以通过在字符范围输入框中输入你想设置的范围的第一个和最后一个字符索引来使用（如果一个字符不在范围内，则用空格代替）。除了这些按钮和输入框外，你还有两个选项。 你还有两个特别有用的选项：

  • 来自代码：如果你点击这个按钮，然后点击 OK，GameMaker 会自动解析你的游戏代码中的字符串，然后创建不同的字符范围来覆盖你游戏中的所有文本。请注意，它寻找的是所有的字符串（在 \"\" 或 \u0027\u0027 内），因此也可能在字符范围内包括文件名。
  • 来自文件：这与上面解释的 "来自代码" 选项的作用类似。如果你点击它，你将被要求提供一个文件，然后一旦完成，你应该点击确定。然后，GameMaker将解析该文件并为文本创建字符范围并为文件中的文本创建字符范围。

注意，你可以通过直接在预览窗口中添加或删除文本字形来手动编辑文本范围。

纹理控制纹理控制

如上所述（在详情一节），默认情况下，所有的字体字形将被渲染 到一个纹理页中，作为纹理组的一部分，将其转换为图形资源，当游戏被编译时，字体将被添加到一个临时纹理中，以创建一个包含所有所需字体字形的单一图像，然后这将被添加到主纹理组中进行导出。这里的按钮是用来调整字体纹理的生成方式或重新生成一个新的字体纹理。

点击选项按钮将打开高级字体纹理选项：

这些选项有时可以从根本上改变字体在游戏中的显示方式，而且并非所有字体都需要相同的设置。以下选项旨在为你的字体提供最大程度的定制服务。

  • 指示：字体提示（也被称为指示）是指使用数学指令来调整轮廓字体的显示，使其与光栅化的网格相一致。在低屏幕分辨率下，提示对于产生清晰可读的文本至关重要，在这里你可以选择不同的提示算法来获得最佳效果。默认是 "正常"。
  • 字形选项。这些选项可用于通过启用/禁用不同的操作来改变字形的加载方式。对于常见的计算机操作系统，字体的显示算法需要一些字体的信息来优化屏幕显示效果，英文称为Hinting。这些选项是：
    1. Disable Hinting: 这将禁用字体的本机Hinting和自动Hinting。
    2. Disable Auto-Hinter。这将禁用自动Hinting，而使用字体的本地Hinting（如果它有的话）。请注意，这将覆盖 "首选自动Hinting "选项。
    3. Prefer Auto-Hinter。这将告诉GameMaker更倾向于使用自动Hinting，而不是字体可能有的任何本地Hinting。请注意，这个选项会被 "Disable Auto-hinter "和 "Disable Hinting "所覆盖。
    4. Warping。启用该选项将对字形进行轻微扭曲，以改善网格对齐。请注意，这需要自动Hinting和 "正常 "提示模式。
    5. No Scale。当该选项被选中时，它将保持字形的字体单位，而忽略要求的尺寸。这意味着提示功能被禁用。
    上述所有选项的默认状态都是禁用。
  • 解释器：在这里你可以选择GameMaker应该使用的TrueType版本来进行字体渲染。默认是TrueType v40。
  • 凑整：该选项用于设置从定点小数转换为整数时的舍入方法。默认为 "floor"。
  • 应用字距调整：该选项控制是否使用字体文件中的字符间距数据在游戏中进行渲染。如果源字体的字符间距数据不正确，并导致字体无法正确显示，则应该禁用该功能。

更新按钮可以用来重新生成字体纹理。如果你想更新你对现有字体所做的任何改动，或者你已经创建了一个新的字体，并且你想看看它在预览窗口中的样子，那么就需要这样做。注意，你也可以从资源浏览器中的字体（或字体）的右键 菜单中重新生成字体。

需要注意的是，当你第一次创建一个新的字体时，如果你不在编辑器中重新生成纹理，那么在你第一次运行或测试你的游戏时，该纹理将被创建，但这是唯一一次自动生成。这意味着，一旦一个字体有了生成的纹理，对字体资源所做的任何改变都需要重新生成字体，以便在游戏中看到这些改变，否则，"旧的 "字体纹理将仍然被使用。

GameMaker将把生成的字体纹理作为PNG文件与项目的其他文件一起渲染，如果你加载项目时没有找到基本字体，那么游戏仍然会使用存储的纹理正确显示文本。然而，一个缺失的字体将意味着你无法再改变它的任何属性，除了它的名字、字体家族和纹理组（纹理组编辑器）之外。

 

预览模式预览模式

在这一部分，你可以选择让预览窗口显示所选字体的一些样本文本，或为该字体生成的渲染纹理图像。

 

预览预览

预览窗口是你可以看到字体的预览，可以是文本样本，也可以是纹理。当预览模式被设置为示例，你可以直接在预览窗口中输入任何你想测试的文本，看看它看起来如何，同时选择游戏预览将显示为该字体生成的渲染纹理图像。如果没有图像，那么你应该点击重新生成按钮来生成纹理。

请注意，纹理图像可以与样本不同，因为纹理将 不被重新生成，除非明确告知。这意味着在你最初生成一个字体纹理后，对字体资源所做的改变将不会被带到游戏中，除非纹理被重新生成。
