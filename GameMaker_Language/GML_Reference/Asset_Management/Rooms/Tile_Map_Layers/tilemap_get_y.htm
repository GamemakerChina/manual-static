tilemap_get_y

使用此函数，您可以检索图块地图元素的 y 位置 (在房间内)。

您提供图块地图元素 ID(在使用 layer_tilemap_create() 创建图块地图元素或使用函数 layer_tilemap_get_id() 时获得)，然后该函数将返回 y 轴位置。

 

语法:

tilemap_get_y(tilemap_element_id)

参数类型描述
tilemap_element_idTile Map Element ID图块图元素的唯一ID值，用于获取

 

返回:

Real

 

例子:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_x(map_id);
var _y = tilemap_get_y(map_id);
tilemap_x(map_id, _x + 10); tilemap_y(map_id, _y + 10);

上面的代码使用检索到的图块图ID来获取图块图的图块x和y位置，然后使用这些值来移动它。
