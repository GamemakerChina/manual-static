string_ord_at

您可以使用此函数在字符串内的特定位置返回特定字符代码，第一个字符的索引从1开始。如果找不到字符或字符串短于给index的值，则返回-1。

 

语法:

string_ord_at(str, index);

参数类型描述
strString要检查的字符串。
indexReal从中获取字符代码的位置。

 

返回:

Real

 

例子:

str = "Hello World";
char_code = string_ord_at(str, 7);

这将获得字符串 “str” 中第七个字符 (其中 “H” 计为第一个) 的字符代码，并将其存储在变量 “char_code” 中。
