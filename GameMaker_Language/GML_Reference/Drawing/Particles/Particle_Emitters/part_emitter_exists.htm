part_emitter_exists

使用此函数，可以检查给定系统中是否存在索引的给定粒子发射器。请注意，如果要检查的变量是未初始化的变量(否则粒子发射器会将其索引指定给该变量)，则会引发错误。

 

语法:

part_emitter_exists(ps, ind);

参数 类型 描述
ps Particle System Instance 用于检查发射器的粒子系统。
ind Particle Emitter ID 要搜索的发射器的索引。

 

返回:

Boolean

 

例子:

if (part_emitter_exists(sname, p_emit1))
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

上面的代码将检查变量"permit"中索引的发射器，如果它存在，它将从中爆发30个粒子。
