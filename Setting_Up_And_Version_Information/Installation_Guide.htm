安装指南

GameMaker 专为在 macOS 和 Windows 台式计算机上使用而设计，其最低规格在 订阅比较页面 中提到。

您可以从您的账户页面下载GameMaker网站的GameMaker安装程序并进行安装。安装后，首次运行GameMaker时，系统可能会提示您进行更新，然后才能使用该产品。您将看到运行时模块对话框，该对话框允许您选择要安装的运行时模块。

安装任何更新后，您需要提供您的 YoYo 游戏账户的许可详细信息 (请参阅 许可信息 页面) 才能使用该产品，这意味着您第一次使用 GameMaker 时必须连接到互联网 。初次登录后，您应该能够脱机使用该产品，但它会定期要求您连接以验证登录账户。

值得注意的是，根据您拥有的许可证类型，您可能需要安装其他组件或执行其他步骤，才能为特定平台设置和使用 GameMaker。有关每个平台需要什么的完整信息，请参阅 YoYo 游戏帮助台上的这篇文章：GameMaker Required SDK。

稳定版和测试版

您从 GameMaker 网页下载的版本应该是当前的稳定版 GameMaker，但也可能有另一个 "Beta" 版本，用于在新功能进入最终稳定版本之前对其进行展示或测试。完整版本详细信息如下：

  • Stable: 此版本是您将用于一般开发的版本，建议与任何商业产品一起使用。它可能缺少Beta版本中的某些功能，但这是因为它们还没有经过完全测试或需要额外的工作。
  • Beta: GameMaker的Beta版本是为那些希望在添加到稳定版本之前尝试新功能的人准备的，或者是为那些需要对某个问题进行修复的人准备的。不建议你在最终的游戏制作中使用Beta版本，除非它包含了一个重要的补丁，可以修复阻碍你项目的bug。你可以从你的YoYo账户“下载”页面获得GameMaker的Beta版本，它可以与稳定版本一起安装和运行。请注意，如果你通过Steam运行GameMaker，那么你将无法使用Steam本身访问该产品的Beta版本，尽管你仍然可以下载和使用桌面版本。

运行时模块对话框

GameMaker 中的每个运行时都由一组模块组成。由于您可能不需要在系统上安装所有这些应用程序，因此您可以选择要下载和安装的应用程序，例如，在磁盘空间有限或互联网连接速度较慢的计算机上工作。

注意 从 2022.11 开始，第一次安装任何版本的 GameMaker 时，此对话框将始终显示给您。默认设置是记住您的选择，并且以后不会再次显示该对话框，但您可以将运行时源首选项中的记住上次选择的模块选项设置为"显示消息"，以便每次再次显示该对话框。
