device_get_tilt_z

此函数根据设备的 “倾斜” 角度返回-1到1之间的值。倾斜度与返回的值之间的实际相关性取决于它使用的设备和OS，但通常值为1或-1与/- 90 ° 相同。下图显示了每个可用函数与设备的关系:

语法:

device_get_tilt_z()

 

返回:

Real

 

例子:

y += sign(device_get_tilt_z());

上面的代码使用倾斜值沿y轴移动玩家。
