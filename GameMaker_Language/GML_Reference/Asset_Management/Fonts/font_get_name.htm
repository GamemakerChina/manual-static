font_get_name

此函数可用于获取在将字体作为资源添加到GameMakerAsset Browser时为字体指定的名称(作为字符串)。请注意，这是唯一的字符串，并且不能用于直接引用字体-为此，您需要字体索引。但是，您可以使用此字符串来获取字体索引(使用返回的字符串和函数asset_get_index()。

 

语法:

font_get_name(ind);

参数类型 描述
indFont Asset 要检查的字体的索引。

 

返回:

String

 

例子:

fnt_Name = font_get_name(font0);

上面的代码将获取字体资源在"资源浏览器"中显示的名称，并将其作为字符串存储在变量" fnt_名称"中。
