code_is_compiled

如果平台在虚拟机外部编译，例如YYC和JS平台，此函数将返回true。

 

语法:

code_is_compiled();

 

返回:

Boolean

 

示例:

if (code_is_compiled())
{
    show_debug_message("Compiler okay!");
}

上面的代码检查游戏是否是使用YoYo编译器编译的，如果是，则显示调试消息。
