network_destroy

This function removes a network socket connection from your game.

NOTE This function will not work when used in a project running on the HTML5 target.

 

语法:

network_destroy(socket);

参数 类型 描述
socket Network Socket ID 要删除的套接字的ID。

 

返回:

N/A

 

例子:

if !os_is_network_connected()
{
    network_destroy(sock);
}

以上代码将检查是否存在数据连接，如果未找到任何连接，则销毁ID为"sock"的套接字。
