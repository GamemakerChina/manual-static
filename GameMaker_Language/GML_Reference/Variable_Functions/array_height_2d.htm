array_height_2d 已删除

警告！ 此函数已弃用(替换为 array_length())，因为数组不再局限于 1 或 2 维，因此提供此函数仅用于支持传统项目。所有新项目都应使用当前创建和访问多维数组的方式(有关更多信息，请参见此处 )。

使用此函数，您可以获得二维数组第一维的高度(条目数)。您提供要检查的数组，该函数的输出会告诉您它包含多少初始条目。你可以使用函数 array_length_2d 获取数组第二维的条目数。

 

语法:

array_height_2d(array_index);

参数 类型 描述
array_index Array 要检查的数组的索引。

 

返回:

Real

 

例子:

for (var i = 0; i < array_height_2d(a); ++i;)
{
    for (var j = 0; j < array_length_2d(a, i); ++j;)
    {
        a[i, j] = -1;
    }
}

上面的代码将循环通过一个2D数组，并将每个条目设置为-1。
