layer_background_visible

使用此函数可以切换背景元素可见性。提供背景元素 ID(当使用layer_background_create()创建背景元素时或使用函数layer_background_get_id()时获得)，然后将可见参数设置为true或false。 当设置为true时，将显示元素，当设置为false时，将不显示该元素。请注意，这取决于层可见性，即使背景元素被标记为可见，如果其上的层被标记为不可见，也不会绘制背景元素。

 

语法:

layer_background_visible(background_element_id, visible)

参数类型描述
background_element_idBackground Element ID要更改的背景元素的唯一ID值
visibleBoolean可见性切换，可以是true或false

 

返回:

N/A

 

例子:

if (mouse_check_button_pressed(mb_left))
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_visible(back))
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

上述代码检查是否按下鼠标按钮，如果检测到鼠标按钮按下，它将切换分配给运行代码的实例所在的层的背景元素的背景可见性。
