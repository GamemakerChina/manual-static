ds_map_write

此函数会将指定映射的DS映射数据转换为字符串格式，然后可以将其写入*.ini或a*.txt文件以便于存储。然后，可以使用ds_map_read()将此字符串读回新的ds_map。

注意 返回的字符串不是人类可读的字符串，而是数据结构内容的转储。

 

语法:

ds_map_write(id);

参数类型描述
idDS Map要使用的数据结构的ID

 

返回:

String

 

例子:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

上述代码将打开一个可以写入的ini文件。然后使用ds_map_write()生成存储在临时变量"t_string"中的字符串。最后，它会在关闭ini文件之前将该字符串写入。
