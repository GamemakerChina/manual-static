layer_get_id_at_depth

可以使用此函数获得指定特定深度的所有层的 ID。指定要检查的深度，函数将返回具有一个或多个条目的数组，这取决于在给定深度处是否存在任何层。如果在给定深度处没有层，则数组将在[ 0]位置有一个条目，值为-1，但如果深度处有层，则在数组中为每个找到的层创建一个条目，条目值将是层的唯一ID值。

 

语法:

layer_get_id_at_depth(depth)

参数 类型 描述
depth Real 要从中检查和检索层ID的深度。

 

返回:

Array

 

例子:

var a = layer_get_id_at_depth(0);
if a[0] != -1
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

上面的代码检索关于深度为0的层的数据。进行检查以查看该深度是否存在任何层，如果存在，则解析返回的数组，并销毁所找到的每个层。
