video_is_looping

此函数返回当前加载的视频是否设置为loop(true)或not(false)。您可以通过调用video_enable_loop()将视频设置为循环。

 

语法:

video_is_looping();

 

返回:

Boolean

 

例子:

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

上述代码通过将自定义实例变量设置为false来检查当前加载的视频是否正在循环，并在这种情况下禁用玩家的控件。
