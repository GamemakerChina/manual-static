fx_get_parameter

此函数用于从 FX Struct 中检索参数值。指定要读取的 FX 结构(从fx_create()或layer_get_fx()返回)和参数名称为字符串，函数返回其当前值。这可能是实值，也可以是数组(如果参数是向量)(它是一组值，例如存储 RGBA 颜色的vec4)。

在本页查找滤镜/效果的参数名称：外汇类型 参数。

 

语法:

fx_get_parameter(filter_or_effect, parameter_name);

参数 类型 描述
filter_or_effect FX Struct FX 结构用于从中获取参数
parameter_name String 获取值的参数名称(作为字符串)

 

返回:

Real or Array

 

例子:

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

上面的代码从假定对其应用了"Color Tint"过滤器的层中检索FX结构，并获取其"g_TintCol"参数的值，该参数返回一个数组，该数组包含4个其色调颜色(红色、绿色、蓝色和alpha)值。然后，它将该值打印到输出日志中。
