ds_queue_clear

使用此函数，您可以清除给定队列数据结构中的所有数据。这不会不破坏数据结构(对于应使用ds_queue_destroy()的数据结构)，它只会擦除其中的所有数据并返回空队列。

 

语法:

ds_queue_clear(id);

参数 类型 描述
id DS Queue 要清除的数据结构的ID。

 

返回:

N/A

 

例子:

if (count = 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = game_get_speed(gamespeed_fps);
    ai_count = 0;
}

上述代码检查变量是否达到特定值，以及是否清除变量"command_queue"中的DS队列索引，设置报警，并将变量重置为0。
