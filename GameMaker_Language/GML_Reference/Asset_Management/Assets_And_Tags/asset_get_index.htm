asset_get_index

您可以使用此函数从其名称中获取游戏资源的唯一标识索引。如果找不到资源，函数将返回值-1，否则将返回被检查资源的唯一索引id。然后，可以像其他索引值一样在其他函数中使用此id，例如sprite_index或path_index。请注意，尽管此函数可用于从字符串中引用资源 (请参见下面的示例)，但在使用资源之前，应始终确保该资源存在，否则可能会出现使游戏崩溃的错误。

 

语法:

asset_get_index(name);

参数类型 描述
nameString 获取 (一个字符串) 的索引的游戏资源的名称。

 

返回:

Asset (any asset type)

 

例子:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

上面的代码将从一个字符串中获取一个对象索引，如果该索引存在，则在游戏中创建该对象的实例。
