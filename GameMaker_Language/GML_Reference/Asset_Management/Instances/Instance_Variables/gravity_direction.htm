gravity_direction

gravity_direction是所有实例具有的"内置"属性之一，当实例重力大于 0 时，可用于设置移动方向。请注意，GameMaker中的方向通常计算为 0° 为右、90° 为上、180° 为左、270° 为向下。

 

语法:

gravity_direction;

 

返回:

Real (single precision floating point value)

 

例子:

if (place_meeting(x, y, obj_switch))
{
    gravity_direction += 180;
}

如果玩家对象遇到"obj_switch"实例，上述代码将改变重力方向。
