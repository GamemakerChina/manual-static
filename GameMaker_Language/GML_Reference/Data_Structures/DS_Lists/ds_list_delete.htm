ds_list_delete

使用此函数，您可以删除存储在列表中特定位置的值。

 

语法:

ds_list_delete(id, pos);

参数类型 描述
idDS List 要更改的列表的ID。
posReal 列表中删除值的位置。

 

返回:

N/A

 

例子:

if (ds_list_size(sc_list) > 10)
{
    while (ds_list_size(sc_list) > 10)
    {
        ds_list_delete(sc_list, 0);
    }
}

上面的代码检查DS列表的大小，如果大于10，它将在列表中循环移除顶部值(位置0)，直到列表只有10个条目。
