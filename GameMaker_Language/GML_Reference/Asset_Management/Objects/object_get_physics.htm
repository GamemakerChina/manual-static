object_get_physics

此函数将告诉您正在检查的对象是否已标记为"物理已启用"，在这种情况下它将返回true，否则将返回false。

 

语法:

object_get_physics(obj);

参数类型 描述
objObject Asset 要检查的对象的索引。

 

返回:

Boolean

 

例子:

if (object_get_physics(object_index))
{
    phy_active = true;
}

上面的代码将检查运行它的实例，以查看从中创建它的对象是否启用了物理，如果启用了，则激活实例的物理模拟。
