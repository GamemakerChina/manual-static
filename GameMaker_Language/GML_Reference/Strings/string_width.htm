string_width

该函数将返回输入字符串的宽度 (以像素为单位)，同时考虑到文本可能具有的任何换行符。根据组成字符串的字母的总宽度来计算文本元素之间的距离非常方便因为它将使用 draw_text() 使用当前定义的字体绘制。

 

语法:

string_width(string);

参数 类型 描述
string String 要测量的宽度的字符串。

 

返回:

Real

 

例子:

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

上面的代码将获取给定字符串的宽度，然后绘制两行文本，使用返回的字符串宽度作为分隔符。
