part_system_colour

此函数更改渲染给定粒子系统的混合色和 Alpha。这与实例中的 image_blend 和 image_alpha 类似。

 

语法:

part_system_color(ind, color, alpha);

参数 类型 描述
ind Particle System Instance 要更改的粒子系统的索引。
color Colour 粒子系统的新颜色。
alpha Real 粒子系统的新alpha。

 

返回:

N/A

 

例子:

part_system_colour(pt_sys, c_red, 0.5);

这会将粒子系统的颜色更改为红色，α值为 0.5。
