array_length_1d 已删除

警告！ 此函数已弃用(替换为 array_length())，因为数组不再局限于 1 或 2 维，因此提供此函数仅用于支持传统项目。所有新项目都应使用当前创建和访问多维数组的方式(有关更多信息，请参见此处 )。

使用此函数，您可以获得一维数组的长度(条目数)。对于二维数组，你应该使用 array_height_2d 和 array_length_2d 函数。

重要！: 如果数组有超过 32,000 个条目，则此函数将返回错误值，不应使用。

 

语法:

array_length_1d(array_index);

参数 类型 描述
array_index Array 要检查的数组的索引。

 

返回:

Real

 

例子:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
{
    a[i] = -1;
}

上面的代码将遍历一个数组并将每个条目设置为 -1。
