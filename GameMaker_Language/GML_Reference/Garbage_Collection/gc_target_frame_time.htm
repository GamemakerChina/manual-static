gc_target_frame_time

使用此函数，您可以向垃圾收集器指示它应该花费不超过运行每个帧指定的时间。该函数采用以微秒为单位指定的时间值(其中1,000,000微秒等于1秒)，默认目标帧时间为100微秒。

请注意，这只是一个目标值，因为某些操作仍需要在一帧内完全运行，有时可能需要比目标时间更长。另请注意，i增加目标时间将使垃圾回收器对内存使用量的快速变化做出更快的响应，并将导致更快地释放内存，尽管实际上在大多数情况下不太可能需要这样做。另请注意，将目标帧时间设置为 0 将而不是会导致收集器完全停止——它每帧所做的工作量仍然很少。要完全禁用收集器，请使用gc_enable()函数。

 

语法:

gc_target_frame_time(time);

参数类型 描述
timeReal 垃圾收集器每帧工作的目标时间(以微秒为单位)

 

返回:

N/A

 

例子:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

上面的代码检查垃圾收集器的当前帧时间目标，如果不是 50 微秒，则将其设置为该值。
