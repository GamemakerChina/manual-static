string_digits

您可以使用此函数解析给定的字符串并从中获取任何数字。例如，假设您有此文本-“我已经81岁了”。有了这个函数，你会得到一个返回字符串 “81”。

 

语法:

string_digits(string);

参数 类型 描述
string String 从中获取数字的字符串。

 

返回:

String

 

例子:

var t_str = string_digits(input_str);
age = real(t_str);

上面的代码将采用输入字符串，将其除数字以外的所有字符剥离，然后将变量 age 设置为保存返回字符串的实数值 (因此，如果输入字符串为-例如-“I am 18”，则函数将返回 “18”)。
