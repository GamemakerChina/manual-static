audio_get_type

将音频添加到GameMaker时，它可以流式传输或在内存中。如果您需要知道给定的声音索引是否用于流式音频，您可以使用此函数，将为流返回1，为内存中的声音返回0，如果有任何错误或索引未指向有效的声音资源，则返回-1。

 

语法:

audio_get_type(index);

参数类型描述
indexSound Asset要检查的声音的索引。

 

返回:

Real (1: streamed, 0: in memory, -1: any error)

 

例子:

type = audio_get_type(snd_Music_1);

上面的代码检查变量"snd_Music_1"中索引的音频类型，并将返回值存储在变量"type"中。
