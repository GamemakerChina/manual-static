exit

exit 语句具有以下语法：

exit;

exit 只是结束当前脚本函数、方法或事件的执行。请注意，根据范围，此处的使用略有不同：

  • 如果您在脚本函数或方法中使用 exit，它只会退出该函数并返回调用该函数的代码。
  • 如果您在对象事件的代码块中使用 exit，它将退出整个事件，即使在调用 exit 后有多个单独的代码块也是如此。
  • 如果您在父事件中使用 exit 并在子对象中使用 event_inherited() 调用该事件，则只有父事件会退出，而子事件（调用 event_inherited()） 将继续。

在事件中使用时，exit 通常用于避免实例在满足（或不满足）特定条件时运行任何进一步的代码。下面的代码概述了如何使用它的示例，在本例中是在碰撞事件中，尽管它可以在任何事件中使用。

if (!visible)
{
    exit;
}

other.hp -= attack;
other.coins -= 4;
coins += 4;

上面的代码检查当前实例是否不可见，在这种情况下它退出代码块，否则它继续运行其余代码。

注意它不会结束游戏的执行。为此，您需要使用函数game_end()。
