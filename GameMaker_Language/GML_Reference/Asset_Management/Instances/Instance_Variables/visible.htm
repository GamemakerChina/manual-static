visible

通过将此变量设置为 true( 可见) 或 false( 不可见)，可以将实例标记为 visible 或不标记。这是通过告诉 GameMaker 跳过此实例的绘制事件来实现的，因此在使用此方法时必须小心，因为这意味着当实例不可见时，将不会运行放置在绘制事件中的代码。另请注意，如果将实例分配到的 层 标记为不可见，则将此变量设置为 true 将不起作用，直到层本身也标记为可见。

 

语法:

visible;

 

返回:

Boolean

 

例子:

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

上面的代码将检查碰撞事件中"other"实例的可见标志，如果它被设置为true，则将该实例移回其先前的位置。
