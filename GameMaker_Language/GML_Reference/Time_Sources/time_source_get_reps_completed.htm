time_source_get_reps_completed

此函数返回给定时间源自上次重置以来已完成的重复次数。这将是它已过期并执行其回调方法的次数。

 

语法:

time_source_get_reps_completed(id);

参数 类型 描述
id Time Source 要获取时间源的信息

 

返回:

Real
