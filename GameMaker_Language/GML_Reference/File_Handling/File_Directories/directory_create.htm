directory_create

此函数在保存区域中创建一个具有给定名称的目录。

注意 此功能不适用于 HTML5，因为您无法在浏览器本地存储中创建或删除目录。

警告 由于 GameMaker 处于沙盒状态，这可能无法按您的预期工作！请参阅有关 文件系统 的部分了解更多信息。

 

语法:

directory_create(dname)

参数类型描述
dnameString要创建的目录的名称。

 

返回:

N/A

 

例子:

if (!directory_exists("Games"))
{
    directory_create("Games");
}

这将检查指定的目录是否存在于本地数据文件夹中，如果不存在，它将为您创建该目录。
