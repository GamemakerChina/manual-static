string_char_at

您可以使用此函数在字符串内的特定位置返回特定字符，第一个字符的索引从1开始。如果找不到任何字符或字符串短于给定的索引值，则返回一个空字符串 ""，但是如果给定的索引等于或小于0，则返回该字符串的第一个字符。

 

语法:

string_char_at(str, index);

参数 类型 描述
str String 要检查的字符串。
index Real 从中获取角色的位置。

 

返回:

String

 

例子:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

这将 str2 设置为字符串 str1 中的第七个字符 (“H” 计数为第1个)，在本例中为 “W”。
