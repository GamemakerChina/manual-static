filename_path

使用此函数返回指定文件路径的路径部分，包括最后一个反斜杠。

 

语法:

filename_path(fname);

参数类型 描述
fnameString 要使用的文件名和路径。

 

返回:

String

 

例子:

path = filename_path(working_directory + "Test.ini");

以上代码获取指定文件的路径(作为字符串)。
