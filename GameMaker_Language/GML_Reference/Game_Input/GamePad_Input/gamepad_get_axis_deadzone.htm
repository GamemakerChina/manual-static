gamepad_get_axis_deadzone

此函数可用于获取操纵杆轴的 “死区” 值。您指定要获取的设备插槽，该函数将返回一个介于 0 到 1 之间的值，其中值反映了阈值，在该阈值下，操纵杆轴被认为处于 0。

 

语法:

gamepad_get_axis_deadzone(device);

参数类型描述
deviceReal要检查哪个游戏手柄设备 “插槽”。

 

返回:

Real

 

例子:

if (gamepad_get_axis_deadzone(global.PadId) != 0.5)
{
    gamepad_set_axis_deadzone(global.PadId, 0.5);
}

上面的代码检查连接到存储在全局变量中的设备 “插槽” 的游戏手柄的模拟轴阈值，如果它不是默认值 0.5，则将其设置为该值。
