fx_get_single_layer

此函数返回提供的过滤器/效果是否使用单层模式(true)(false)。 有关单层模式的信息，请参见fx_set_single_layer()。

指定 FX 结构(如从fx_create()或layer_get_fx()返回)，并且该函数返回布尔，该布尔{}指示 FX 是否启用单层模式。

 

语法:

fx_get_single_layer(filter_or_effect);

参数类型 描述
filter_or_effectFX Struct 要读取的FX结构

 

返回:

Boolean

 

例子:

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

上述代码从名为"Distort"的层获取FX结构，并检查其上是否启用了单层模式。如果不是，它将启用该FX的单层模式。
