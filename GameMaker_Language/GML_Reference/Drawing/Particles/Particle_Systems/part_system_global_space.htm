part_system_global_space

该函数为给定的粒子系统启用全局空间粒子。

禁用全局空间粒子时（默认设置），粒子的位置相对于粒子系统的位置。 移动这样的粒子系统时，其粒子将随之移动。

启用全局空间允许您移动粒子系统，而不移动其中的粒子。这可以用于例如要创建粒子轨迹，执行以下操作：

禁用 启用

在上面的例子中，单个粒子在粒子系统中的相同位置创建，粒子系统本身在每帧移动到船的位置。

第一个视频显示禁用了全局空间，因此所有粒子都粘在其系统上，但在第二个视频中，粒子停留在创建它们的房间位置。

启用后，可以分别使用 part_system_position 和 part_system_angle 移动和旋转 粒子系统 ，而无需移动粒子系统中的单个粒子。

注意如果在内部已有粒子的粒子系统上启用全局空间，您可能会看到这些现有粒子从相对于粒子系统切换到相对于房间原点时跳到屏幕上的新位置。

 

语法:

part_system_global_space(ind, enable);

参数 类型 描述
ind Particle System Instance 粒子系统的指数
enable Boolean 是否启用或禁用全局空间粒子

 

返回:

N/A

 

示例：跟随实例的粒子系统

创建事件

ps = part_system_create();
part_system_global_space(ps, true);
pe = part_emitter_create(ps);

pt = part_type_create();
part_type_shape(pt, pt_shape_flare);
part_type_direction(pt, 0, 360, 0, 0.3);
part_type_speed(pt, 0.1, 0.2, 0, 0.01);
part_type_scale(pt, 0.3, 0.3);

part_emitter_stream(ps, pe, pt, 2);

步骤事件

part_system_position(ps, x, y);
part_system_angle(ps, direction);

清理事件

part_emitter_destroy(pe);
part_system_destroy(ps);
part_type_destroy(pt);

上面的代码显示了如何将粒子系统添加到实例中，

在“创建”事件中，将设置一个基本粒子系统，并向其中添加一个发射器。使用 part_system_global_space，它被配置为在全局空间中拥有其粒子。 还添加了一个基本粒子类型，该粒子类型配置为具有光斑形状（pt_shape_flare）、可以是 0 到 360 之间的任意随机值的起始方向、每步 0.1 到 0.2 像素之间的任意起始速度以及 0.3 的比例因子。

在Step事件中，粒子系统的位置和角度分别设置为实例的（x，y）位置及其方向。

最后，在清理事件中，粒子发射器、系统和类型将被销毁，以防止内存泄漏。
