dbg_text

此函数在当前调试部分中创建只读文本标签。您可以使用它来解释周围的控件。

对包含字符串的变量的 String 和 Reference 都可以传递给函数。文本可以是多行的。

注意有关允许修改文本的控件，请参见dbg_text_input。

注意此控件占用一列，可以使用dbg_same_line与另一个单列控件显示在同一行上。

 

语法:

dbg_text(ref_or_string);

参数 类型 描述
ref_or_string Reference or String 一个字符串或一个可以转换为字符串的变量的引用，由ref_create返回

 

返回:

N/A

 

例子:

Create Event

text = "text";
ref_to_text = ref_create(self, "text");
dbg_text(ref_to_text);
dbg_text(text);
dbg_text("More text");

上面的代码将一些文本赋给 实例变量 text。然后使用 ref_create 创建对该变量的引用，并将其存储在 ref_to_text 中。接下来，它将三个文本控件添加到将在新节 "Default" 下创建的新调试视图 "Default" 中。对 dbg_text 的三次调用以三种不同的方式添加文本条目：第一个提供对 text 变量的引用，第二个传递变量 text 本身 (它为变量当前具有的 值 赋值)，最后一个直接传递字符串。
