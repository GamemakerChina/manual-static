layer_sprite_get_yscale

该函数可用于获得精灵元素的沿y轴的当前比例乘数。给定精灵元素 ID(当使用layer_sprite_create()创建精灵元素时或使用函数layer_sprite_get_id()时获得)，该函数将返回 y 比例的真实值，其中 1 表示无缩放。

 

语法:

layer_sprite_get_yscale(sprite_element_id);

参数类型描述
sprite_element_idSprite Element ID获取信息的精灵元素的唯一ID值

 

返回:

Real

 

例子:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_yscale(spr_id) != 1)
{
    layer_sprite_yscale(spr_id, 1);
}

上面的代码将获取名为"sprite_sky"的层的层ID，然后使用该代码来获取该层上的精灵元素的ID。然后，此 ID 用于检查 yscale 值，如果它不是 1，则设置为 1。
