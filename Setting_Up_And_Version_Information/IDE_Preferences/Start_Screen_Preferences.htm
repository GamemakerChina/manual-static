开始屏幕首选项

开始屏幕首选项允许您更改起始页的行为方式。主要选项包括：

  • 显示 Tour：这允许您选择在启动 IDE 时是否显示介绍性教程。当您第一次跳过 Tour 时，此选项将自动禁用。
  • 显示安装向导：这允许您选择在启动 IDE 时是否显示安装向导。安装向导基本上会引导您完成创建新项目所需的步骤。
  • 最近项目列表样式：这允许您在起始页上将最近项目列表的显示方式在"列表"和"网格"之间更改。
