keyboard_lastkey

这个变量是指 keyboard_key 在上一帧中的值，返回该键的键码 (返回所有标准键码常量)。

该变量 不是 只读的，您可以更改它，例如，如果您已经处理过它，则将其设置为 vk_nokey。

 

语法:

keyboard_lastkey

 

返回:

Virtual Key Constant (vk_*)

 

示例:

if (keyboard_lastkey != vk_nokey)
{
    str += keyboard_lastchar;
    keyboard_lastkey = vk_nokey;
}

上面的代码检查 keyboard_lastkey 变量是否不等于 vk_nokey，如果不等于，则将最后一个键作为字符串添加到变量 str 中，然后重置 keyboard_lastkey 变量以接受进一步的输入。
