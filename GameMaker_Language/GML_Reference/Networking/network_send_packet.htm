network_send_packet

通过此函数，您可以通过网络发送数据"数据包"。该函数使用套接字ID进行连接，然后必须提供包含要发送的数据的缓冲区ID(有关缓冲区的详细信息，请参阅参考-缓冲区)，最后是数据包的大小(以字节为单位)。使用此函数发送的数据包将被格式化，以便接收数据的GameMaker游戏可以正确"拆分"数据包，并且此函数将返回发送的数据的字节数，如果发送失败，则返回小于0的数字。值得注意的是，该函数返回的正在发送的数据的最终大小还将包括GameMaker头信息，即额外的12个字节。

 

语法:

network_send_packet(socket, bufferid, size);

参数类型 描述
socketNetwork Socket ID 要使用的套接字的ID。
bufferidBuffer ID 从中获取数据的缓冲区的ID。
sizeReal 数据的大小(以字节表示)。

 

返回:

Real

 

例子:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

上述信息将先前保存的缓冲区数据加载到内存中，并返回要存储在变量"buffe"中的缓冲区ID。然后，使用由变量"sock"标识的套接字，将该完整缓冲区作为数据包通过网络发送。
