layer_get_vspeed

您可以使用此函数来检索当前范围房间内图层的垂直速度(每游戏帧的像素数)。提供层 ID(使用layer_create()创建层时获得的 ID)或层名称(作为字符串，这将对性能产生影响)，并且函数返回垂直速度的实数，其中正值向下，负值向上。默认值为0(除非在房间编辑器中设置)。

 

语法:

layer_get_vspeed(layer_id)

参数类型描述
layer_idString or Layer要从中获取垂直速度的层的唯一ID值

 

返回:

N/A

 

例子:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

上面的代码检查给定层的水平和垂直速度，如果它们没有都设置为0，则将它们设置为0。
