layer_sequence_get_headdir

使用此函数，您可以提供layer_sequence_create或层元素函数之一返回的序列元素 IDas，它将返回当前播放头方向，这是下列常量之一。

序列方向常量
常量 描述 值
seqdir_right The sequence will play frames in an incremental order from left to right 1
seqdir_left The sequence will play frames in a decremental order from right to left -1

 

语法:

layer_sequence_get_headdir(sequence_element_id)

参数 类型 描述
sequence_element_id Sequence Element ID 序列元素到目标的唯一ID值

 

返回:

Constant

 

例子:

if layer_sequence_get_headdir(title_sequence) != seqdir_left
{
    layer_sequence_headdir(title_sequence, seqdir_left);
}

上述代码检查存储在变量 title_sequence 中的序列元素 ID 的当前播放头方向，如果未设置为 seqdir_left，则设置为该值。
