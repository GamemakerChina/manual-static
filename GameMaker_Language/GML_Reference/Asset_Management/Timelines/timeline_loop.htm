timeline_loop

此变量将返回时间线是否循环(true)或不循环(false)。您可以更改此变量以打开或关闭循环，它以负时间线速度工作(如果时间线位置低于0，它将在最后定义的时刻重新开始)。

 

语法:

timeline_loop;

 

返回:

Boolean

 

例子:

 

if !timeline_loop
{
    timeline_loop = true;
}

上面的代码检查当前分配给实例的时间线是否会循环，如果不循环，则将其设置为true.
