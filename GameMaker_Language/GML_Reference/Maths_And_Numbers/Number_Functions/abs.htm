abs

这个函数返回输入参数的绝对值，所以如果它是正值，那么它将保持不变，但是如果它是负值，它将乘以 -1，使其为正值。

 

语法:

abs(val)

 

参数类型 描述
valReal 要转为绝对值的数。

 

返回:

Real

 

例子:

x += abs (x - mouse_x)

这将添加一个等于实例当前 x 位置和鼠标 x 位置的绝对值的量。
