timeline_running

此变量保留分配的时间线的当前状态，如果正在运行，则返回true，如果不运行，则返回false。您也可以将此变量设置为true或false，以随时启动和停止时间线。应该注意的是，停止的时间线是不重置的，因此在以后的时间再次启动它将从停止的确切时刻开始。

 

语法:

timeline_running;

 

返回:

Boolean

 

例子:

if (!timeline_running)
{
    timeline_position = 0;
    timeline_running = true;
}

上面的代码将检查实例是否正在运行时间线，如果不是，则将分配的时间线重置为在第一时刻启动，然后启动它。
