ds_grid_multiply

此函数将给定网格单元格的值乘以指定的数量。

注意 此函数仅适用于实数，不适用于字符串。

 

语法:

ds_grid_multiply(index, x, y, val);

参数类型描述
indexDS Grid网格的索引。
xReal网格中单元格的x位置。
yReal单元格在网格中的y位置。
valReal要与单元格相乘的值。

 

返回:

N/A

 

例子:

ds_grid_multiply(mygrid, 5, 5, 2)

上述代码将把存储在给定DS网格单元中的值乘以2。
