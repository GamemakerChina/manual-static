game_end

此函数结束游戏并触发 游戏结束事件 。

这不会立即发生，而是在当前步骤结束时发生，因此调用此函数后同一步骤中的任何代码仍将运行。

使用说明

  • 在 Android 设备上，调用 game_end 会将应用推入后台，但 不会 关闭应用。这必须由用户完成。
  • 在iOS上它将什么都不做，默默地失败。
  • 在所有控制台上，它可能会使游戏崩溃，或者最好是静默失败，而且可能还会导致提交失败，因此不能使用此函数。
  • 在HTML5上，它将结束游戏，但让用户盯着网页上的空白画布，因此不建议在该目标平台上使用它。
  • 在Windows、Linux和macOS上，该函数只需结束游戏并关闭游戏窗口(游戏结束事件也将被触发)。

    •  

 

语法:

game_end([return_code]);

参数 类型 描述
return_code Real 可选 应用程序的返回代码，默认为 0。

 

返回:

N/A

 

例子:

if keyboard_check_pressed(vk_escape) game_end();

如果玩家按下"逃逸"键，这将结束游戏。
