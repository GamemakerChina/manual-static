最近使用的窗口

可以从 Windows 菜单中打开 " 最近使用的窗口 " 列表，该列表将显示一个简短列表 (默认情况下，它最多有 10 个项目)。开始时，此列表将为空，但当您在 IDE 中打开不同的窗口时，(例如精灵、脚本或对象，等等)。列表将被填充。如果您需要返回到这些先前打开的窗口中的任何一个，那么您只需在列表中的项目上单击鼠标左键 ，工作区将平移到窗口。如果您在项目上单击鼠标右键 ，则将打开以下菜单： 在这里您可以选择删除项目 (或项目，因为你可以使用 + 或 / + 来选择多个) 或者你可以把它们移到列表的顶部。人民币菜单也有一个额外的选项，允许你完全清除列表。这是值得的请注意，"最近使用的窗口 " 列表可以停靠在 IDE 的两侧，您还可以调整某些 " 首选项 " 以更改其行为方式。
