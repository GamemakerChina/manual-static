path_start

路径由一系列链接在一起的定义点创建，然后用于规划实例的移动。可以在路径编辑器中使用代码、或创建它们，并将它们分配给一个实例以在游戏中使用。然后，您可以使用此函数告诉您的实例要遵循的路径、遵循路径的速度(以每步像素为单位)、到达路径末端时的行为以及是遵循绝对还是相对路径位置。这最后一部分意味着，它要么完全按照您在路径编辑器中设计并放置的路径开始并跟随该路径(绝对)，要么从创建实例的位置开始并跟随该路径(相对)。

正如您在上图中所看到的，两个实例遵循以path_start()开头的相同路径，但是其中一个实例遵循绝对路径，而另一个则遵循相对路径。需要注意的是，当RELATIVE设置为FALSE(ABSOLUTE)时，实例将"跳转"到路径的起点，无论它位于房间中的哪个位置。

本页给出了用于定义实例到达路径末尾时的行为的常量。

 

语法:

path_start(path, speed, endaction, absolute);

参数 类型 描述
path Path Asset 要启动的路径索引。
speed Real 以每步像素为单位跟踪路径的速度，负数表示后退。
endaction Path End Action Constant 到达路径终点时要执行的操作(请参阅上面列出的常量)。
absolute Boolean 调用实例应遵循编辑器中的绝对路径(true)还是其当前位置的相对路径(false)。

 

返回:

N/A

 

例子:

path_start(path, 4, path_action_reverse, 0);

这将使调用实例以每步4个像素的速度跟随变量" path "中索引的路径。它会沿着相对于自己位置的路径前进，当它到达终点时，它会改变方向，回到它来的地方。
