ds_grid_create

此函数创建指定单元格宽度和高度的新 DS 网格数据结构。该函数返回一个 ID，必须在处理此 DS 网格的所有其他函数中使用该 ID。

重要信息 创建数据结构时，用于标识该数据结构的索引值是从 0 开始的整数值。这意味着不同类型的数据结构可以具有 相同 的索引值，因此如果有疑问，您应该在访问它们之前使用 ds_exists 函数。另请注意，索引是重复使用的，因此被破坏的数据结构索引值可能会被随后新创建的索引值使用。

 

语法:

ds_grid_create(w, h);

参数 类型 描述
w Real 要创建的网格的宽度。
h Real 要创建的轴网的高度。

 

返回:

DS Grid

 

例子:

mygrid = ds_grid_create(10, 10);

这将创建一个高10个单元格和宽10个单元格的栅格。
