文件系统

在 GameMaker 中存在一些函数，允许你以不同的方式操作所有类型的文件:

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

 

以下函数可用于更改文件名:

注意 这些函数对实际的文件不起作用!它们只处理与不同路径元素相关的字符串，应该与其他文件函数一起使用。

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

 

最后，还有一些用于获取文件名的特殊函数，可用于使用文件类型过滤器加载和保存文件。
这些函数会打开一个资源管理器窗口，从而导致游戏失去焦点。

这意味着游戏所依赖的任何输入都可能不会被正确注册，因此我们建议你
使用 键释放(key up) 事件而不是按住或按下事件 。(如果在键盘事件中使用这些函数)

注意 这些函数仅适用于 Windows 和 macOS。

警告 由于 GameMaker 处于沙盒状态，这些函数可能无法按您的预期运行！请参阅有关 文件系统 的部分了解更多信息。

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
