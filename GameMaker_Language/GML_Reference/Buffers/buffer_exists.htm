buffer_exists

此函数可用于检查变量，以查看其是否保存有效的缓冲区ID值。如果这样做，函数将返回true，否则将返回false。

 

语法:

buffer_exists(buffer)

参数类型 描述
bufferBuffer ID 要检查的缓冲区的索引。

 

返回:

Boolean

 

例子:

if buffer_exists(buff)
{
    buffer_delete(buff);
}

上面的代码检查变量 "buff" 是否包含缓冲区ID，如果包含，则删除缓冲区。
