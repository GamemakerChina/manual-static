设置 Git 插件

在使用 Git 插件之前，我们需要设置要使用的 Git 可执行文件，方法是进入首选项的插件-源代码管理 (Git)部分：

The Git executable path should be auto-filled if it's found by GameMaker, however if the field is empty, you can click on the  button and manually select the git.exe file included in cmd folder of your Git installation, which is usually inside C:\Program Files\Git\.
