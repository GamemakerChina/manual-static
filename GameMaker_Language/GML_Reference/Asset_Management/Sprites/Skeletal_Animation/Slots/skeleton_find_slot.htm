skeleton_find_slot

使用此函数，您可以在与当前实例关联的Spine Sprite中找到位于指定房间空间位置的插槽。您创建一个DS list并提供其ID以及要检查的X/Y位置，该列表将使用精灵的每个可用附件槽的名称字符串填充(包括任何附件修改)。注意，列表总是从最上面的槽开始按降序排序。

 

语法:

skeleton_find_slot(x, y, list);

参数 类型 描述
x Real 要检查的房间中的X位置。
y Real 要检查的房间中的Y位置。
list DS List ID 要用DS map填充的DS list的ID.

 

返回:

N/A

 

例子:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

上面的代码创建了一个DS list，然后用鼠标在房间中的位置找到的插槽名称填充它。
