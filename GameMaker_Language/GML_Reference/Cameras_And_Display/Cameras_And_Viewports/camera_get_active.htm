camera_get_active

此函数可用于检索当前活动相机的唯一相机 ID 值。

 

语法:

camera_get_active()

 

返回:

Camera ID

 

例子:

var active = camera_get_active();
if active != view_camera[0]
{
    view_camera[0] = active;
}

以上代码获取活动相机的相机ID，并设置 port[0] 使用的视图的相机。
