directory_exists

如果指定的目录存在，此函数将返回 true; 如果不存在，则返回 false。指定的名称必须包含完整路径，而不是相对路径，并且默认情况下，您无法从游戏包外部访问任何目录，因为所有游戏都已沙盒化 (请参阅有关 文件系统 的部分，了解更多信息)。

 

语法:

directory_exists(dname)

参数类型描述
dnameString要查找的目录的名称。

 

返回:

Boolean

 

例子:

if (directory_exists(working_directory + "Saves/"))
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

这将检查指定的目录是否存在，如果存在，则返回找到的第一个“只读”doc 文件。
