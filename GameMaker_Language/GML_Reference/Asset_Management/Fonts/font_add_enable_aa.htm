font_add_enable_aa

该函数可用于对使用font_add()添加的字体启用或禁用抗锯齿 (AA)。这个函数需要在添加任何字体之前调用并将其设置为true会启用AA，将其设置为false会禁用它。默认情况下，已启用AA。请注意，这对在调用函数之前已经添加的字体没有影响，并且该函数只需要在添加字体时调用一次，而不是在使用该字体的每个绘制/步骤。

 

语法:

font_add_enable_aa(enable);

参数类型 描述
enableBoolean 对于已添加的字体，是启用 (true) 还是禁用 (false) AA。

 

返回:

N/A

 

例子:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

上面的代码检查添加字体的抗锯齿状态，如果未启用，则启用它。
