path_mirror

此函数获取所有路径点并沿垂直轴镜像它们。该函数更改了实际路径资源，因此将永久影响该路径从函数开始被游戏中所有实例使用的方式直到游戏结束。如果这不是您需要的，那么您应该先使用像path_duplicate()这样的函数来创建路径的副本，然后在重复资源(当不再需要时，别忘了在资源上调用path_delete())。

语法:

path_mirror(index);

参数类型 描述
indexPath Asset 要镜像的路径的索引。

 

返回:

N/A

 

例子:

path_mirror(mypath);

这将镜像在变量"mypath"中沿垂直轴索引的路径。
