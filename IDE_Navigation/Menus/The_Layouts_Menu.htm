布局菜单

此菜单与保存和加载不同的 IDE 布局相关。您可以随心所欲地设置 IDE，其中某些元素 (如资源浏览器或输出窗口) 的排列方式与默认设置不同。这允许您设置 GameMaker 以适合您的工作流程，并且使用此菜单中的选项，您可以将当前项目布局保存为 *.xml 文件，然后可以将该文件加载到任何项目中，以将布局设置为相同的。此菜单中的选项如下：

  • 加载布局 - 这将打开一个文件资源管理器，您可以在其中选择要加载的先前保存的布局 *.xml 文件。
  • 保存布局 - 这将打开一个文件资源管理器，您可以在其中命名当前的 IDE 布局并将其保存为 *.xml 文件。
  • 重置布局 - 这会将当前的 IDE 布局重置为默认设置。
  • 显示所有窗口 - 此选项设置工作区缩放，以便显示所有窗口。该选项仅在工作区中可用。
  • 关闭所有窗口 - 这将关闭工作区中所有打开的窗口。该选项仅在工作区中可用。
