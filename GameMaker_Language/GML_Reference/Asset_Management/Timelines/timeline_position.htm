timeline_position

该变量保持时间线当前处于的当前位置 (时刻)。您可以更改此值以跳过时间线的部分，或者重复部分或从头开始重新开始时间线。

 

语法:

timeline_position;

 

返回:

Real (single precision floating point value)

 

例子:

if (!timeline_running)
{
    timeline_position = 0;
    timeline_running = true;
}

上面的代码将检查实例是否正在运行时间线，如果不是，则将分配的时间线重置为在第一时刻启动，然后启动它。
