show_question

此函数创建一个带有"是"和"否"两个按钮的弹出消息框。根据用户按下两个按钮中的哪一个，它返回true或false。

注意此函数仅在Windows目标上用于调试，但在所有其他目标上已弃用。

 

语法:

show_question(str);

参数 类型 描述
str String 要在弹出问题中显示的字符串。

 

返回:

Boolean

 

例子:

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

上面的代码将检查分数，如果分数超过500，它将询问用户是否希望继续，如果点击"是"按钮，它将转到另一个房间，但如果选择"否"按钮，它将结束游戏。
