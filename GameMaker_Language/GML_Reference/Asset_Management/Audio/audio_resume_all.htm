audio_resume_all

使用此函数，您可以恢复以前暂停的所有声音。

 

语法:

audio_resume_all();

 

返回:

N/A

 

例子:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

上述代码检查是否按下键盘键"P"，如果检测到一个，则将全局变量"暂停"设置为真或假，然后暂停所有声音或重新启动所有先前暂停的声音。
