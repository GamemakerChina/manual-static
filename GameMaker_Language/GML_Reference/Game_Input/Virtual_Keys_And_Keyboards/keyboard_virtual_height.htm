keyboard_virtual_height

此函数将根据显示的大小返回虚拟键盘的当前高度(以像素为单位)。如果键盘不可见，则返回0。

 

语法

keyboard_virtual_height();

 

返回:

Real

 

例子:

if (keyboard_virtual_status() == true)
{
    key_h = keyboard_virtual_height();
}

上面的代码将检查操作系统虚拟键盘的状态，如果可见，则将一个变量设置为键盘的高度。
