buffer_poke

使用buffer_write()函数，您可以在当前 "seek" 位置向给定缓冲区写入数据，每条数据都会按正在写入或读取的字节向前移动。但是，您可能需要在不希望更改当前搜索位置的情况下更改给定的数据片段，而这正是您使用此函数的时候。您只需为函数提供一个缓冲区索引，然后在该缓冲区内从缓冲区起点的偏移位置(以字节为单位)，以及要写入的数据类型和值即可。

 

语法:

buffer_poke(buffer, offset, type, value);

参数 类型 描述
buffer Buffer ID 要使用的缓冲区的索引。
offset Real 缓冲区内的偏移位置 (以字节为单位)，用于将给定数据写入其中。
type 缓冲区数据类型常量 要写入缓冲区的数据类型 (请参见常量列表此处)。
value Any 根据指定的类型添加到缓冲区的数据。

 

返回:

N/A

 

例子:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

上述代码会将用于图像混合的颜色的蓝色分量值添加到变量 "buff" 中索引的缓冲区中，在缓冲区中的第三个位置并作为无符号的8bit值。
