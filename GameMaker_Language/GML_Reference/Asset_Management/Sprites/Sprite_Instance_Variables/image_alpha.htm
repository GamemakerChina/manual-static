image_alpha

该变量用于获取或设置精灵的 alpha 值。 Alpha 始终计算为 0 到 1 之间的值，其中 0 表示完全透明，1 表示完全不透明。请注意，要使该变量的更改可见，实例应具有无绘制事件(因此GameMaker将默认绘制精灵)或使用扩展绘图函数之一进行绘制，例如draw_self或draw_sprite_ext。

 

语法:

image_alpha

 

返回:

Real

 

例子:

image_alpha = clamp(image_alpha - 0.01, 0, 1);

上面的代码将缓慢地减小图像_Alpha，直到它达到0。
