video_close

此函数可关闭当前加载的视频文件。请确保仅在video_open()调用之后调用此函数，否则不会执行任何操作。

 

语法:

video_close();

 

返回:

N/A

 

例子:

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

按下Escape(退出)键时，上述代码将关闭视频。
