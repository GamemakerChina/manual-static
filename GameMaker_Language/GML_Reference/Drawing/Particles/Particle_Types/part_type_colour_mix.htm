part_type_colour_mix

使用此函数，可以将给定的粒子类型设置为两种颜色的随机混合。

 

语法:

part_type_colour_mix(ind, colour1, colour2);

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
colour1 Colour 混合将采用的第一种颜色。
colour2 Colour 混合将使用的第二种颜色。

 

返回:

N/A

 

例子:

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

上述代码将全局变量"P_Damage"中索引的粒子类型的每个发射粒子的颜色设置为红色和黄色之间的随机混合。
