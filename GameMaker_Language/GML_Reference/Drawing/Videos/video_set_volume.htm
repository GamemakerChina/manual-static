video_set_volume

此函数可更改当前加载的视频的音频音量。它采用介于0和1之间的值，其中0是静默的，1是最大卷。

请确保仅在video_open()调用之后但在video_close()调用之前调用此函数，否则它将不执行任何操作(因为不会加载视频)。

在某些平台上，此函数只能在视频完全加载后才能工作，因此请使用异步回调查看视频何时准备就绪，然后更改音量。

 

语法:

video_set_volume(value);

参数 类型 描述
value Real 卷值介于0和1之间

 

返回:

N/A

 

例子:

my_video = video_open("splash.mp4");
video_set_volume(0.5);

上面的代码从游戏的"包含的文件"中加载splash.mp4，并将其卷更改为0.5(最大卷的50%)。
