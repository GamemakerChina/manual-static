buffer_base64_decode

使用此函数，您可以将base64编码的字符串 (使用buffer_base64_encode()函数创建) 解码到缓冲区中。此函数将创建缓冲区 (作为1字节对齐的 "增长" 缓冲区)，并返回缓冲区的唯一索引，该索引应在所有其他函数调用中使用。

注意: 重要的是，当您不再需要它们以防止内存泄漏时，从内存中删除任何动态创建的资源，因此，当您完成创建的缓冲区时，您应该使用buffer_delete(){}再次释放它。

 

语法:

buffer_base64_decode(string);

参数类型 描述
stringString 要解码的base64编码字符串

 

返回:

Buffer ID

 

例子:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

上面的代码将打开一个ini文件，然后从中读取一个字符串到解码函数中。该函数将返回一个缓冲区索引，该索引存储在变量 "buff" 中，其中包含先前编码和保存的数据。然后关闭ini文件。
