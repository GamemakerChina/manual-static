audio_is_paused

此函数将检查给定声音以查看当前是否暂停。声音可以是声音的单个实例(当使用audio_play_sound()或audio_play_sound_at()函数时，播放的单个声音的索引可以存储在变量中)或声音资源中，在这种情况下将检查给定声音的所有实例，如果其中任何实例暂停，函数将返回true，否则将返回false。

 

语法:

audio_is_paused(index);

参数 类型 描述
index Sound Asset or Sound Instance ID 检查声音的索引。

 

返回:

Boolean

 

例子:

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

上述代码检查变量"snd_Waterfall"中索引的声音当前是否暂停，如果它返回true，则将恢复声音的播放。
