part_emitter_enable

此功能启用或禁用给定的粒子发射器。

用法注释

  • 由禁用的发射器创建的粒子将变为不可见且不会更新。
  • 与发射器延迟(part_emitter_delay)或间隔(part_emitter_interval)相关的定时器暂停。
  • 禁用的发射器在流模式下不会创建新粒子。 在调用part_emitter_burst后，它也不会创建任何粒子。
  • 函数part_particles_burst不会在粒子系统资源中创建粒子或任何禁用的发射器。

 

语法:

part_emitter_enable(ps, ind, enable);

参数 类型 描述
ps Particle System Instance 发射器所在的粒子系统的索引
ind Particle Emitter ID 要更改的发射器的索引
enable Boolean 是否启用发射器

 

返回:

N/A

 

例子:

 

Create Event

ps = part_system_create();

pe1 = part_emitter_create(ps);
part_emitter_region(ps, pe1, 100, 200, 100, 200, ps_shape_rectangle, ps_distr_gaussian);
part_emitter_enable(ps, pe1, false);
pe2 = part_emitter_create(ps);
part_emitter_region(ps, pe2, 200, 300, 100, 200, ps_shape_rectangle, ps_distr_gaussian);

pt = part_type_create();
part_type_speed(pt, 2, 2, 0, 0);
part_type_direction(pt, 90, 90, 0, .2);

part_emitter_stream(ps, pe1, pt, 2);
part_emitter_stream(ps, pe2, pt, 2);

Cleanup Event

part_emitter_destroy(ps, pe1);
part_emitter_destroy(ps, pe2);
part_system_destroy(ps);
part_type_destroy(pt);

上面的代码在 Create 事件中创建了一个粒子系统 ps，并向其中添加了两个发射器： pe1 和 pe2 发射器各自被配置为使用高斯分布（即，高斯分布）在 100x100 像素的矩形区域中发射粒子。更多的颗粒在中心）。 使用 part_emitter_enable 将发射器 pe1 设置为禁用。 然后创建粒子类型 pt，并配置其速度和方向，以便此类型的所有粒子将以每步 2 像素的恒定速度向上移动。

然后使用 part_emitter_stream“打开”两个发射器：它们中的每一个被配置为每步产生两个粒子。 由于 pe1 设置为禁用，因此它不会发射任何粒子。 pe2 被启用并正常发射粒子。

最后，由于它们是动态资源，粒子发射器、系统和类型都将在清理事件中销毁。
