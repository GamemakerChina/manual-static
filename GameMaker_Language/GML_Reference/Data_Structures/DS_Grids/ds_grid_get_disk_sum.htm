ds_grid_get_disk_sum

此函数可用于将在网格的已定义磁盘区域内找到的所有单元格的值相加，如下图所示：

 

语法:

ds_grid_get_disk_sum(index, xm, ym, r);

参数类型描述
indexDS Grid网格的索引。
xmReal磁盘在网格上的x位置。
ymReal磁盘在网格上的y位置。
rReal网格上磁盘的半径。

 

返回:

Real or String

 

例子:

val = ds_grid_get_disk_sum(grid, 5, 5, 2)

上述代码将变量"val"设置为包含在变量"grid"中索引的DS网格的给定磁盘中的所有值的总和。
