image_yscale

此值设置应用于已指定给当前实例的精灵的垂直缩放(沿Y轴)。比例为1表示不缩放(1：1)，较小的值将缩小(例如，0.5将缩小为精灵高度的一半)，较大的值将放大，负值将镜像精灵并缩放它，除非所使用的值正好为-1(在这种情况下，精灵仅沿Y轴镜像而不缩放)。

 

语法:

image_yscale;

 

返回:

Real (single precision floating point value)

 

例子:

if (image_xscale < 5)
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

上面的代码缩放精灵，然后一旦它被缩放到其原始大小的5倍，就会创建另一个对象的新实例，并销毁该实例。
