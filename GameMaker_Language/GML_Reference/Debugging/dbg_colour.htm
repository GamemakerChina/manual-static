dbg_colour

此函数在当前调试部分中创建颜色控制。

该控件接受Colour变量的值，并显示R、G和B分量：

你可以通过双击值并输入新值来更改R、G和B值。你也可以点击右边显示颜色的方块，弹出一个颜色选择器对话框来选择新的颜色。

注意 该值为RGB颜色值，不含Alpha分量。

注意 此调试控制跨越两列。

 

语法:

dbg_colour(ref[, label]);

参数 类型 描述
ref Reference 使用ref_create对Colour变量创建的引用
label String 可选显示在颜色控件旁边的标签

 

返回:

N/A

 

例子:

Create Event

my_view = dbg_view($"Settings for {id}", true);
my_section = dbg_section("Colours");
dbg_colour(ref_create(self, "image_blend"), "Blend Colour");

Clean Up Event

dbg_section_delete(my_section);
dbg_view_delete(my_view);

上面的代码被添加到对象的Create事件中。执行代码的此对象的每个实例使用dbg_view创建新的调试视图，将其存储在实例变量my_view中，并使用dbg_section向其添加名为"Colours"的节，该节存储在变量my_section中。然后，它们中的每一个都使用dbg_colour添加一个颜色控件，以更改该实例的image_blend颜色。

在清理事件中，此对象的每个实例都将删除其创建的调试视图和节(分别为my_view和my_section)。
