camera_set_view_target

您可以使用此函数设置房间内视图相机的跟随目标。提供唯一的相机 ID 值(由不同的 camera_create() 函数返回)，然后提供要将相机视图设置为的目标实例或对象 ID。请注意，如果您选择一个对象 ID，并且房间存在该对象的多个实例，GameMaker 无法知道您希望跟随哪个实例，因此可能是其中的任何一个。

 

语法:

camera_set_view_target(camera_id, instance_id/object_id)

参数 类型 描述
camera_id Camera ID 创建相机时返回的唯一相机 ID 值。
instance_id/object_id Object Asset or Instance ID 要使相机目标跟随的实例或对象

 

返回:

N/A

 

例子:

camera_set_view_target(view_camera[0], id);

上述代码将分配给 viewport[0] 的相机视图的相机目标设置为运行代码的实例视图的相机目标。
