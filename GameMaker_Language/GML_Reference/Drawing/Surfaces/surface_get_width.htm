surface_get_width

此函数只是返回索引表面的宽度(以像素为单位)。应该注意，如果在使用surface_resize()更改其大小后立即调用此选项来检查application_surface，则不会返回新值，因为更改需要一两个步骤才能完全处理。在等待一个步骤之后，它应该正确返回新的大小。

注意：使用表面时，由于表面存储在纹理内存中，因此表面可能随时停止存在。在直接引用表面之前，应该始终使用surface_exists()检查表面是否存在。

 

语法:

surface_get_width(surface_id);

参数类型 描述
surface_idSurface ID 要获取宽度的表面的ID。

 

返回:

Real

 

例子:

sw = surface_get_width(surf);

以上代码将变量"washing"中索引的表面宽度存储在变量"swo"中。
