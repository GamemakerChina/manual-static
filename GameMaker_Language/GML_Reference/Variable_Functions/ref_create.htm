ref_create

此函数用于在结构或实例中创建对变量的引用。

您可以为变量所属的结构或实例、或提供对它的引用，以及变量的名称(作为字符串)、或 对存储名称或索引的变量的引用。

对于数组，可以将索引作为第三个参数提供，该参数告知要创建对哪个索引的引用。

可以创建更复杂的引用。有关各种可能性的完整列表，请参见示例。

注意 可以通过调试覆盖层中的引用更改该值。

警告 您无法创建对局部变量的引用，因为它们只是暂时存在，不能被引用。

 

语法:

ref_create(dbgrefOrStruct, dbgrefOrIndex[, index]);

参数 类型 描述
dbgrefOrStruct Reference or Struct 包含要引用的变量或对该变量的引用的结构或实例
dbgrefOrIndex Reference or String 变量的名称为字符串，或对其的引用
index Real 可选 数组中的索引，如果引用的是数组

 

返回:

Reference

 

示例1：实例变量的基本引用

text = "This is some text";
ref_to_text = ref_create(self, "text");

上面的代码首先在创建事件中创建一个 实例变量 text ，然后使用存储在变量 ref_to_text 中的 ref_create 创建对它的引用。

 

示例2：对数组索引的基本引用

array = [1, 2, 3, 4, 5];
ref_to_index = ref_create(self, "array", 2);

上面的代码首先在执行代码的实例中创建一个数组 array，该数组包含 5 个元素。然后，它使用 ref_create 创建对索引 2 (第三个元素) 的引用。该引用存储在新的实例变量 ref_to_index 中。

 

示例3：复杂引用

the_struct = {a: "text", b: 485};
ref_to_struct = ref_create(self, "the_struct");
ref_to_struct_var = ref_create(ref_to_struct, "a");

上面的代码首先在执行具有两个变量的代码的实例中创建一个结构 the_struct。然后，它使用 ref_create 创建对该结构的引用，然后将该引用作为 dbgrefOrStruct 参数传递给下一次调用 ref_create。对 ref_create 的最后一次调用创建了对结构变量 a 的引用，并将其存储在变量 ref_to_struct_var 中。使用之前存储在变量 ref_to_struct 中的引用，结构本身被间接地传递给函数。

 

示例4：复数数组引用

array = [3, 4, 1, 7, 8, 2];
index = 4;
ref_to_array = ref_create(self, "array");
ref_to_index = ref_create(self, "index");
ref_to_array_at_index = ref_create(ref_to_array, ref_to_index);

上面的代码创建了对数组 array 的引用，其中 ref_create (要引用的数组) 的第一个参数和第二个参数 (用作数组的索引) 本身都是引用。这意味着，如果稍后为变量 index 赋值不同的值，则由 ref_to_ray_at_index 索引的数组元素也会更改为新值集。如果将一个全新的数组赋给实例变量 array，则 ref_to_array_at_index 将引用该数组中相同索引处的元素。
