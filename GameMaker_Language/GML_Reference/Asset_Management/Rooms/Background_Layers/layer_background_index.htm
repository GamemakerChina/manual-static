layer_background_index

此函数可用于设置具有多个子图像的背景精灵的图像索引。给定背景元素 ID(当使用layer_background_create()创建背景元素时或使用函数layer_background_get_id()时获得)，然后设置要使用的图像索引。如果将值设置在子图像范围之外，则图像索引将循环。

 

语法:

layer_background_index(background_element_id, image_index);

参数 类型 描述
background_element_id Background Element ID 要更改的背景元素的唯一ID值
index Real 用于背景的图像索引

 

返回:

N/A

 

例子:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

上面的代码将获取名为"Background_trees"的层的层ID，然后使用该层来获取该层上的背景元素的ID。然后，此 ID 用于更改元素图像索引。
