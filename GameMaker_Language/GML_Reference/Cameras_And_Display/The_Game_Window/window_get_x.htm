window_get_x

使用此函数，如果是 HTML5 游戏，您可以在浏览器中获取 X 位置(以像素为单位)，如果是 Windows、Ubuntu 或 MacOS 游戏，则可以在显示器中获取 X 位置(以像素为单位)。

 

语法:

window_get_x();

 

返回:

Real

 

例子:

wx = window_get_x();

上述代码将当前 X 轴窗口位置存储在变量 "wx" 中。
