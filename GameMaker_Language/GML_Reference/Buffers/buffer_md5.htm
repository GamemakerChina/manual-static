buffer_md5

在密码学中，MD5(Message-Digest 5 算法)是一种广泛使用的加密哈希函数，具有 128 位哈希值，已用于各种安全应用程序。它也常用于检查数据的完整性。此函数将从给定缓冲区(或部分缓冲区)获取输入数据，并返回该数据独有的 32 个字符的十六进制 MD5 哈希。通过这种方式你可以生成一种安全密钥，可以存储并用于检查发送到(或从)外部服务器(例如)接收的信息的完整性。使用此函数将其应用于缓冲区时，必须将缓冲区的缓冲区 ID 指定为使用，然后是起始位置的偏移值(以字节为单位)，然后是要进行哈希处理的区域的大小(同样以字节为单位)。

注意MD5 并不完全安全并且可能被破解。请参阅 此页面 了解更多信息。

 

语法:

buffer_md5(buffer, offset, size);

参数 类型 描述
buffer Buffer ID 要使用的缓冲区的索引。
offset Real 数据偏移值。
size Real 缓冲区的大小。

 

返回:

String

 

例子:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

上述代码将为存储在由变量"buff"索引的缓冲区中的完整数据创建一个md5哈希，并将返回的哈希存储在变量"check_string"中。
