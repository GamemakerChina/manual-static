audio_get_listener_info

此函数将创建一个DS map，并用给定监听器的信息填充它。

注意 您应负责使用适当的函数销毁归还的 DS 映射。

DS映射将包含以下键值：

  • "name"-监听器的名称，作为字符串，" default "是大多数目标平台上的标准监听器名称
  • "mask"-监听器的位掩码
  • "index"-监听器的唯一索引值

掩码值可用于将声音或发射器设置为一次从多个监听器播放，只需使用按位或"|"为声音生成掩码(请参阅下面的示例代码)，而索引可用于使用audio_listener_set_position()等函数为给定监听器设置位置或速度等属性。

 

语法:

audio_get_listener_info(num);

参数 类型 描述
num Real or Audio Listener ID 要为其获取数据的监听器编号。

 

返回:

DS Map

 

例子:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

上面的代码检查可用的监听器的数量，然后循环通过它们，得到它们的掩码位，然后组合这些掩码位以创建应用于全局监听器的单个位掩码。
