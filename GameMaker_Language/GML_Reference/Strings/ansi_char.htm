ansi_char

该函数返回一个包含有原始BYTE值设置的字符的字符串。 这不会也不应该被显示出来，但它会正确地保存在磁盘上用于编码。

 

语法:

ansi_char(val);

参数 类型 描述
val Real 原始字节值。

 

返回:

String (Single character)

 

例子:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

上面的代码从原始字节数据创建一个字符串，并将其写入(先前打开的)文件。
