room_last

此 只读 变量返回游戏中最后一个房间的索引 (这由房间在 房间管理器 中的显示顺序定义，而 不是 按创建房间的顺序定义)。请注意，此变量不会识别或考虑使用 room_add() 或 room_duplicate() 动态添加的房间。

 

语法:

room_last;

 

返回:

Room Asset

 

例子:

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

上面的代码检查是否按下了某个键，如果是，则转到游戏中的最后一个房间。
