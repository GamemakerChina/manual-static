样条线编辑器首选项

样条线编辑器首选项主要用于定义路径编辑器中的平滑路径的外观，尽管它们会影响 IDE 的某些其他方面(如物理形状编辑器)。存在以下选项：

  • 插入路径点精度(像素)：此选项设置在路径上插入点的精确值(以像素为单位)，即：鼠标必须在此范围内才能插入点，而不是添加新点。默认大小为 20px。
  • 锚点宽度：设置锚点的宽度，单位为像素。默认为 8px
  • 锚点高度：设置锚点的高度，单位为像素。默认为 8px
  • 路径颜色：编辑时用于显示路径的颜色。默认为 (ARGB) #FFFFFFFF。
  • 突出显示路径段颜色：用于突出显示任何选定路径段的颜色。默认为 (ARGB) #FFFFFFFF。
  • 无效路径颜色：用于突出显示无效路径的颜色。默认为 (ARGB) #FF0000FF。
  • 锚点颜色：用于绘制路径锚点的颜色。默认为 (ARGB) #FFC0C0C0。
  • 无效锚点颜色：用于绘制路径锚点的颜色。默认为 (ARGB) #FF800000。
  • 选定锚点颜色：用于绘制已突出显示(选定)的路径锚点的颜色。默认为 (ARGB) #FF00A4FF。
  • 选择无效的锚点颜色：用于绘制突出显示但无效的锚点的颜色。默认为 (ARGB) #FFA400FF。
  • 鼠标悬停锚点颜色：用于绘制活动锚点的颜色。默认为 (ARGB) #FFFFFFFF。
  • 将鼠标悬停在无效锚点颜色上：用于绘制活动但无效的锚点的颜色。默认为 (ARGB) #FF0000FF。
  • 蒙版覆盖颜色：设置蒙版的覆盖颜色(如果启用)(请参见上文)。默认为 (ARGB) #FF404040。
