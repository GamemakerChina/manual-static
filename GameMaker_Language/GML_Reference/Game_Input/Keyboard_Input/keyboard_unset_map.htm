keyboard_unset_map

有了这个函数，您可以清除所有重新映射的键，以便它们返回到默认状态，即: 所有要映射到自己的键。

 

语法:

keyboard_get_map();

 

返回:

N/A

 

示例:

if (keyboard_check_pressed(vk_escape))
{
    keyboard_unset_map();
}

如果用户按下 “escape” 键，上面的示例代码会将所有映射的键重置为其默认设置。
