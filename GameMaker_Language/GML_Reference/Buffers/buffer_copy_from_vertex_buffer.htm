buffer_copy_from_vertex_buffer

此函数可用于将存储在一个顶点缓冲区中的部分(或全部)顶点数据复制到先前创建的常规缓冲区中。使用此函数从顶点缓冲区复制到常规缓冲区时，两个缓冲区必须事先创建过(例如，使用vertex_create_buffer()和buffer_create()函数)。您可以指定要复制到缓冲区的顶点数据的范围，其中起始顶点可以介于 0 和顶点数 1 之间的任何位置，并且可以给出从该点开始复制的顶点数。你可以使用顶点缓冲区上的函数vertex_get_number()来获取存储的顶点总数。最后，你给出要将顶点数据复制到的缓冲区索引，以及一个数据偏移量，用于定义将顶点数据复制到目标缓冲区中的位置。

 

语法:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

参数 类型 描述
vertex_buffer Vertex Buffer ID 要从复制的顶点缓冲区的索引。
start_vertex Real 起始顶点。
num_vertices Real 要使用的顶点总数。
dest_buffer Buffer ID 要复制到的缓冲区索引。
dest_offset Real 将数据复制到(以字节为单位)的偏移位置。

 

返回:

N/A

 

例子:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

上述代码将复制存储在变量"model_buffer"中索引的顶点缓冲区中的顶点数据，然后将其粘贴到变量"player_buffer"中索引的缓冲区中。
