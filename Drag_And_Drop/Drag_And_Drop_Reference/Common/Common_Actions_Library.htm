通用动作库

顾名思义，通用动作库就是您可以找到在创建项目时最常使用的动作的地方。这些动作通常与其他动作一起使用，并且都与游戏内部逻辑的运作有关。可采取的行动如下:

 

Assign Variable
Declare Temporary Variable
Set Global Variable
Get Global Variable
If Variable
If Expression
If Undefined
Else
Execute Code
Execute Script
New
Function Call
Return
Apply To...
Macro
Exit
Declare A New Function
