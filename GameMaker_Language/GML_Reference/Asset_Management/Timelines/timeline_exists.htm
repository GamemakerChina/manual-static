timeline_exists

使用此函数，您可以检查并查看时间线是否存在 (返回true) 或不存在 (返回false)。当使用timeline_add()函数动态创建时间线时，这特别有用，但是您应该注意，如果搜索未初始化的变量 (否则将分配给时间线的索引)，则会引发错误。

 

语法:

timeline_exists(ind);

参数类型 描述
indTimeline Asset 要检查的时间线的索引。

 

返回:

Boolean

 

例子:

if (timeline_exists(global.tl))
{
    timeline_delete(global.tl);
}

上面的代码检查时间线是否存在并在全局变量" TL "中索引，如果存在，则删除该时间线。
