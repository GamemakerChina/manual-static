sprite_width

此只读变量返回已分配给实例的精灵的宽度。此宽度以像素为单位返回，并将取决于image_xscale。如果你需要未缩放的宽度，你应该使用sprite_get_width()。

 

语法:

sprite_width;

 

返回:

Real

 

例子:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

上面的代码对照精灵资源的宽度检查分配给运行该代码的实例的精灵的宽度，如果不相同，则将图像_xscale重置为1。
