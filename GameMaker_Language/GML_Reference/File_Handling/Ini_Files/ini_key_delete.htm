ini_key_delete

使用这个函数，您可以从 ini 文件中删除选定的键(及其对应的值)。

 

语法:

ini_key_delete(section, key);

参数类型 描述
sectionString 要从中删除键的节。
keyString 要删除的键。

 

返回:

N/A

 

例子:

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

这个示例将打开 "savedata.ini" 并将一个值写入 "save1">"Score"。然后，它会删除 "Score" 键并关闭 .ini 文件。
