ini_read_string

可以使用此函数从 INI 数据文件读取字符串 (文本)。

Ini 文件被分割成 节 (sections) ，然后每个节又被分割成 键 (key)- 值 (value) 对。所以一个典型的 ini 文件看起来是这样的：

 

语法:

ini_read_string(section, key, default);

参数 类型 描述
section String 要从中读取的 .ini 部分。
key String 要从中读取的 .ini 相关部分中的键。
default String 在定义的位置找不到字符串（或.ini文件不存在）时返回的字符串。 必须是字符串。

 

返回:

String

 

例子:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name", "Player1");
ini_close();

这将打开 "savedata.ini" 并将 global.name 设置为其中 "save1">"Name" 下的字符串，然后再次关闭 .ini。如果 "save1">"Name" 下没有字符串，或者没有名为 "savedata.ini" 的文件，则 global.name 将设置为 "Player1"。
