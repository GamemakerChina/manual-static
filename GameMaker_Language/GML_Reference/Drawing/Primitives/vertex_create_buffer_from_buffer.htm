vertex_create_buffer_from_buffer

与函数vertex_create_buffer()一样，此函数将创建一个新的顶点缓冲区，只是现在它存储的顶点数据是从指定为源的常规缓冲区复制的。创建的缓冲区是一个特殊的增长缓冲区，该缓冲区使用顶点格式预先设置，构建用于(例如)着色器的基元。此函数要求您提供指向以前创建的常规缓冲区的指针，以及应该应用于复制数据的顶点格式。

 

语法:

vertex_create_buffer_from_buffer(buffer, format);

参数 类型 描述
buffer Buffer ID 用于创建顶点缓冲区的缓冲区。
format Primitive Type Constant 要使用的基元顶点格式。

 

返回:

Vertex Buffer ID

 

例子:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_texcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

上述代码将创建一个新的顶点格式，然后从先前创建的常规缓冲区创建一个新的顶点缓冲区，并将自定义顶点格式应用于该缓冲区。
