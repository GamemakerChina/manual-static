layer_set_fx

此函数用于将 FX 结构指定给房间层。指定要修改的层的 ID 或名称，然后指定将应用于指定层的 FX 结构(从fx_create()或layer_get_fx()返回)，使效果在房间中可见。

如果指定的图层不是滤镜/效果图层，而是其他类型的图层(如实例图层、资源图层、图块图层等)，则效果将应用于该图层的内容及其下面的任何图层。

如果指定的层已经分配了FX结构，它将被新的替换。

注意在脚本的根级别调用此函数将无济于事，因为脚本在第一个房间加载之前执行。您可以先设置目标房间，然后调用此函数。

 

语法:

layer_set_fx(layer_name_or_id, filter_or_effect);

参数 类型 描述
layer_name_or_id String or Layer ID 要修改的层的名称或ID
filter_or_effect FX Struct 应用于层的 FX 结构

 

返回:

N/A

 

例子:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上面的代码从"_filter_tintfilter"效果创建一个新的FX结构，该效果是在房间编辑器中找到的"色调"过滤器。它为其" g_TintCol"参数分配一个值，该参数是色调的颜色，由于它在内部是vec4，它采用包含四个值(对应于其红色、绿色、蓝色和alpha值)的数组。然后，将色调的 FX 结构应用于 FX 层。
