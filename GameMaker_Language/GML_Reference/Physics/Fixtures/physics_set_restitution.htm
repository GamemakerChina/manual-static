physics_set_restitution

当您使用 physics_fixture_bind() 将装置绑定到实例时，此操作将返回绑定装置的 "id"。您可以使用此 ID 随时使用此函数设置绑定装置的补偿值，而不是" 基础 " 装置。"补偿" 通常设置为介于 0 和 1 之间的值，但如果需要，您可以使用更高的值，尽管结果可能不可预测。

 

语法:

physics_set_restitution(fixture, restitution)

参数类型 描述
fixturePhysics Fixture ID 绑定装置的 ID
restitutionReal 要应用的新补偿值

 

返回:

N/A

 

例子:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

上面的代码获取实例的绑定物理属性的当前补偿值，然后将其设置为不同的值。
