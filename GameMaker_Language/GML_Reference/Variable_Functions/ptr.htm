ptr

此函数将尝试将给定值转换为指针数据类型，其中值必须为real,字符串,int64,int32, 或ptr。其他任何事情都会导致游戏崩溃并显示错误消息。

 

语法:

ptr(n);

参数 类型 描述
n Real, String, or Pointer 要转换的值。

 

返回:

Pointer

 

示例:

if (!is_ptr(val))
{
    val = ptr(application_surface);
}

上面的代码检查变量 “val” 以查看它是否包含一个指针，如果没有，则将一个分配给它。
