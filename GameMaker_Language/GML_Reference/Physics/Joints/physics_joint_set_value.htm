physics_joint_set_value

即使在创建关节之后，也可以更改和设置某些关节特性。此函数中可以使用许多常量，这里可以找到这些常量：物理关节常量。

 

语法:

physics_joint_set_value(joint, field, value)

参数 类型 描述
joint Physics Joint ID 要更改的关节的索引
field Physics Joint Constant 要更改的关节特性的常量
value Real 关节特性的新值

 

返回:

N/A

 

例子:

if (physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2)
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

上述代码检查接头最大马达扭矩是否设置为小于 2，然后将其设置为 2。
