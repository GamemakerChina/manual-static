array_push

使用此函数，您可以将一个值 (或多个值) 推到数组的末尾，而不必知道数组的长度。该函数要求您提供一个变量，该变量保存数组以及至少 一个 值以推送，尽管您可以选择提供其他参数，并且它们都将按连续顺序推送到数组上。

 

语法:

array_push(variable, value, [value], [value], [etc...]);

参数 类型 描述
variable Array 保存数组的变量。
value Any 推送到数组末尾的值
[value], [value], [etc...] Any 可选 要推送到数组的其他值

 

返回:

N/A

 

例子:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

上面的代码将把四个值推到给定数组的末尾。
