GM_version

此常量保存每个目标平台的游戏选项中定义的版本号。该值存储为字符串。

 

语法:

GM_version;

 

返回:

String

 

例子:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

上面的代码获取GM日期和版本常量并将它们绘制到屏幕上。
