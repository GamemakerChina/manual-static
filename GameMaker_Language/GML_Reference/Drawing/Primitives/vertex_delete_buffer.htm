vertex_delete_buffer

此函数从系统内存中删除先前创建的顶点缓冲区，以及它包含的任何自定义图元的所有顶点数据。

 

语法:

vertex_delete_buffer(buffer);

参数 类型 描述
buffer Vertex Buffer 要删除的顶点缓冲区。

 

返回:

N/A

 

例子:

vertex_delete_buffer(v_buff);

上述代码将从内存中删除变量 v_buff 中存储的顶点缓冲区。
