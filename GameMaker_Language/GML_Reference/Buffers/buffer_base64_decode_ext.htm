buffer_base64_decode_ext

使用此函数，您可以将base64编码的字符串 (使用buffer_base64_encode()函数创建) 解码到缓冲区中。与函数buffer_base64_decode(),这将不是为您创建缓冲区，而是您应该已经创建了缓冲区 (请参阅buffer_create())，然后将其id用于此函数。"偏移量" 是缓冲区内的位置来解码给定的字符串 (以字节为单位)。

 

语法:

buffer_base64_decode_ext(buffer, string, offset);

参数类型 描述
bufferBuffer ID 将字符串解码为的缓冲区的索引。
stringString 要解码的base64编码字符串。
offsetReal 数据偏移值。

 

返回:

N/A

 

例子:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

上面的代码将创建一个缓冲区，并将其唯一的id存储在变量 "buff" 中，然后打开一个ini文件，并从中读取一个字符串到本地变量 "str" 中。然后将此字符串解码到新创建的缓冲区中再次关闭ini文件。
