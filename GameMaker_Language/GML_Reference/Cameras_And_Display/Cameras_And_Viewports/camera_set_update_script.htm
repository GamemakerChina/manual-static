camera_set_update_script

此函数可用于设置脚本函数，该函数将调用摄像机分配到可见和活动视口的每个游戏帧。提供唯一的相机 ID 值(由不同的 camera_create 函数返回)和要调用的函数的 ID。附加到相机的函数的顺序和相机视图的实际渲染顺序如下：

  • 所有可见和活动视口的相机都调用了其更新函数
  • 然后对于每个视图：
    • 该视图被称为相机的开始函数
    • 为该视图执行绘制事件
    • 被调用为相机的结束函数
  • 移动到下一个活动的可见视图并重复...

对相机应用自定义更新函数会覆盖其默认更新行为。例如，如果将相机设置为跟随对象 (例如，obj_player) 调用 camera_create_view() 时，或在"房间属性"中将其设置为跟随对象，则一旦设置了自定义更新脚本，相机将停止跟随该对象。

 

语法:

camera_set_update_script(camera_id, script);

参数 类型 描述
camera_id Camera ID 创建相机时返回的唯一相机 ID 值。
script Script Function 运行每个游戏帧的脚本函数 (或 -1 表示不设置更新脚本)

 

返回:

N/A

 

例子:

camera_set_update_script(view_camera[0], updateCamera);

上述代码为分配给 viewport[0] 的相机设置更新函数。
