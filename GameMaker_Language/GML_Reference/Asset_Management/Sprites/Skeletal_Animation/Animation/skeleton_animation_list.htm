skeleton_animation_list

使用此函数，您可以使用作为骨骼动画精灵的一部分的所有动画名称填充(预先创建的)DS list。名称将是字符串，然后可以在这些类型的精灵的其他动画函数中使用。

 

语法:

skeleton_animation_list(sprite, list);

参数 类型 描述
sprite Sprite Asset 要从中获取列表的脊椎骨骼动画的精灵索引。
list DS List 要用动画名称填充的DS列表的ID.

 

返回:

N/A

 

例子:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++;)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

上面的代码为实例使用的精灵创建了所有动画名称的DS列表。然后，它循环通过这些对象，并将第一个动画的所有对象的"混合"值设置为0.5。
