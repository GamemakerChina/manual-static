part_system_exists

使用此函数，可以检查游戏中是否存在给定的粒子系统索引。请注意，如果要检查的变量是未初始化的变量(否则粒子系统会将其索引指定给该变量)，则会引发错误。

 

语法:

part_system_exists(ind);

参数 类型 描述
ind Particle System Instance 要检查的粒子系统的索引。

 

返回:

Boolean

 

例子:

if (!part_system_exists(global.Sname))
{
    global.Sname = part_system_create();
}

以上代码检查全局变量中引用的粒子系统是否存在，以及是否创建了该粒子系统。
