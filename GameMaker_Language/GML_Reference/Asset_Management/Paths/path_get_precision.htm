path_get_precision

此函数返回已"平滑"的给定路径的精度，并将是一个从 1 到 8 的整数值。虽然您可以获取(并设置)直线路径的该值，但它对实例使用路径的方式没有影响，因为它仅在路径类型设置为"平滑"时才相关。

 

语法:

path_get_precision(index);

参数类型 描述
indexPath Asset 要检查的路径的索引。

 

返回:

Real

 

例子:

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

上面的代码检查在"pth_Patrol"中索引的路径，以查看它是什么类型，如果它是平滑的，则检查平滑有多精确。如果小于8，则设置为8。
