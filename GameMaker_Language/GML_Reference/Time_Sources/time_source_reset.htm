time_source_reset

此函数重置给定的时间源，重置其计时器和重复信息。

调用此函数后，您需要再次启动时间源，使其向下计数。

 

语法:

time_source_reset(id);

参数 类型 描述
id Time Source ID 要重置的时间源

 

返回:

N/A

 

例子:

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

此代码将在新房间启动时重置给定的时间源，然后再次启动时间源。
