array_delete

使用此函数，您可以从数组中的任何给定位置删除一个值(或多个值)。该函数要求您提供保存数组的变量、要从中删除的数组中的索引(位置)以及要删除的值的数量。

 

语法:

array_delete(variable, index, number);

参数 类型 描述
variable Array 保存数组的变量。
index Real 要从中删除值的数组中的索引(位置)。
number Real 要删除的值的数目。

 

返回:

N/A

 

例子:

array_delete(score_array, 0, 10);

上面的代码将删除给定数组中从索引0开始的前10个值(因此索引0到9将从数组中删除)。
