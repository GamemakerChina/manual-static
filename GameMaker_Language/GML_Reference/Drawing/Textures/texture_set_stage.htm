texture_set_stage

此函数将设置给定阶段 " 槽 " 要使用的纹理。

可用的阶段 " 槽 " 数量取决于您要编译的平台，Windows、Mac 和 Linux 最多可使用 8 个，但在低端 Android 设备上 (例如)，此数字可能低至 2 。还值得注意的是，第一阶段槽 (1) 始终由 GameMaker 自动使用。

注意 此函数在正在运行的着色器上下文之外不会执行任何操作！请参阅 着色器 了解更多信息。

 

语法:

texture_set_stage(stage, tex);

参数类型描述
stageShader Sampler Handle or Real要使用的纹理"槽"。
texTexture or int64 (signed 64-bit integer)要使用的纹理。

 

返回:

N/A

 

例子:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

上述代码将获取索引为 GlassShader 的着色器中采样器的 句柄 ，然后将该着色器常量设置为给定的精灵纹理。
