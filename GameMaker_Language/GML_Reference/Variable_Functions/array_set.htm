array_set

使用此函数，您可以将数组中索引的值设置为值。该函数要求您提供一个变量，该变量保存数组以及要设置的索引和要设置的值。该函数也可以用于多维数组，只要您在提供数组索引时指定要设置的维度，就可以按照以下模式进行:

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

 

语法:

array_set(variable, index, value);

参数 类型 描述
variable Array 保存数组的变量。
index Real 要设置的值的数组元素的索引。
value Any 要设置的值。

 

返回:

N/A

 

例子:

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
}

上面的代码会将给定数组中的前10个项目设置为一个值。
