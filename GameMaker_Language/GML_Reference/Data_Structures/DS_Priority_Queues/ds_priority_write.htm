ds_priority_write

此函数返回一个字符串，然后可以使用ds_priority_read()函数将该字符串存储或传输到另一个数据结构。

注意:返回的字符串不是人工可读字符串，而是数据结构内容的转储

 

语法:

ds_priority_write(id);

参数类型 描述
idDS Priority ID 要检查的数据结构的ID。

 

返回:

String

 

例子:

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

上述代码打开ini文件，然后写入一个字符串，其中包含变量"p_queue"中索引的DS优先级队列中存储的信息。然后清除优先级队列并关闭ini文件。
