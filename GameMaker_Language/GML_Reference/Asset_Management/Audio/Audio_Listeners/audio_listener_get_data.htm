audio_listener_get_data

此函数将创建DS map，并用给定监听器的位置、速度和方向值填充该映射。默认监听器索引为0，但可以使用函数audio_get_listener_info()获取目标平台可用的不同索引。如果您提供的监听器索引不正确，则函数将返回-1。

注意您负责使用相应函数销毁返回的DS map。

DS map将包含以下键值：

  • "x" - 监听器的 x 位置
  • "y" - 监听器的 y 位置
  • "z" - 监听器的 z 位置
  • "vx" - 收听者的 x 轴速度
  • "vy" - 收听者的 y 轴速度
  • "vz" - 收听者的 z 轴速度
  • "lookat_x" - 监听器的环视向量的 x 分量
  • "lookat_y" - 监听器的环视向量的 y 分量
  • "lookat_z" - 监听器的环视向量的 z 分量
  • "up_x" - 监听器向上向量的 x 分量
  • "up_y" - 监听器向上向量的 y 分量
  • "up_z" - 监听器向上向量的 z 分量

 

语法:

audio_listener_get_data(index);

参数 类型 描述
index Real or Audio Listener ID 要为其获取数据的监听器(默认值为0)。

 

返回:

DS Map

 

例子:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "x"] != 0
    {
        audio_listener_set_position(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

上面的代码检查可用监听器的数量，然后遍历它们，如果它们的X位置不是0，则将它们的位置设置为0，0，0。
