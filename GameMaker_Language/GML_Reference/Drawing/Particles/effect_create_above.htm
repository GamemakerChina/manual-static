effect_create_above

使用此函数，您可以在房间的所有实例上方创建一个简单的效果(它实际上是在-100000的深度处创建的)。如果效果是任何其他ef_rain或ef_snow，则可以定义x/ y位置来创建效果，并且大小可以是0、1或2的值，其中0表示小，1表示中等，2表示大。

值得注意的是，通过使用函数part_system_automatic_draw()和part_system_automatic_update()以及粒子系统索引的适当值(其中0表示下面的效果，1表示上面的效果)，可以打开和关闭这些效果的图形，以及暂停其图形。

不同粒子类型的可用常量包括：

效果类型常量
常量 示例 描述
ef_cloud 创建大小不同的随机云粒子的效果
ef_ellipse 创建展开椭圆的效果
ef_explosion 产生扩展衰落爆炸的效果
ef_firework 创建多个小粒子以产生烟火爆炸的效果
ef_flare 一种产生亮点的效果，亮起并淡出
ef_rain 产生从屏幕顶部下来的雨粒子的效果
ef_ring 产生扩展和衰落圆的效果
ef_smoke 产生轻微烟雾的效果
ef_smokeup 创建烟羽的效果，该烟羽会沿着屏幕上升
ef_snow 产生多个雪粒子从屏幕上落下的效果
ef_spark 产生小火花的效果
ef_star 产生恒星粒子的效果

 

语法:

effect_create_above(kind, x, y, size, colour);

参数 类型 描述
kind 效果类型常量 类型的效果(使用上面列出的个常量之一)。
x Real 效果的x定位(如果相关)。
y Real 效果的y定位(如果相关)。
size Real 效果的大小。
colour Colour 效果的颜色。

 

返回:

N/A

 

例子:

if health <= 0
{
    effect_create_above(ef_explosion, x, y, 1, c_yellow);
    instance_destroy();
}

如果"Health"变量小于或等于0，则上述代码将在实例上方创建中等、黄色的爆炸，并销毁它。
