show_error

此函数将自定义字符串显示为错误消息。

重要第二个参数仅用于向后兼容旧项目，对显示的错误对话框没有影响。

注意此函数仅供调试使用。

 

语法:

show_error(str, abort);

参数类型描述
strString要在弹出消息中显示的字符串。
abortBoolean未使用错误是否应中止游戏(true)或允许玩家忽略它(false)。

 

返回:

N/A

 

例子:

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

上面的代码将检查当前房间是否是RM_Dungeon，如果不是，将显示一条错误消息。
