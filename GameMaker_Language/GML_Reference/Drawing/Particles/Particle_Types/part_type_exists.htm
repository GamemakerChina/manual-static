part_type_exists

使用此函数，可以检查游戏中是否存在给定的粒子类型索引。请注意，如果要检查的变量是未初始化的变量(否则粒子类型会将其索引指定给该变量)，则会引发错误。

 

语法:

part_type_exists(ind);

参数 类型 描述
ind Particle Type ID 要检查的粒子类型的索引。

 

返回:

Boolean

 

例子:

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

上面的代码检查全局变量 "p1" 是否存储粒子类型的索引，以及是否创建索引。
