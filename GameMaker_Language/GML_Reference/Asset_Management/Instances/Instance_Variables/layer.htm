layer

这个内置变量是为房间中的每个实例创建的，它包含实例所分配到的层的层ID值。然后，可以在其他函数(如layer_get_depth())中使用此值，也可以更改此值以将实例移动到另一个层，但请注意，如果当前房间中不存在要指定的层，则将出现错误，这将强制关闭游戏。分配层时，必须提供唯一的层ID，该ID由函数layer_get_id()返回(使用命名的房间层时)，或由函数layer_create()返回(在运行时创建自己的层时)。

重要如果您使用instance_create_depth()函数创建了实例，或手动更改了深度变量，则分配给该实例的层将成为"托管"层，GameMaker将自动控制和管理该层。在这种情况下，层变量将返回-1。

 

语法:

layer;

 

返回:

Layer ID

 

例子:

layer = layer_create(-1000);

上面的代码将创建一个深度为- 1000的新层，然后将实例层变量设置为返回的层ID，将实例从其当前所在的层移动到正在创建的新层。
