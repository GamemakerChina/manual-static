font_get_sdf_spread

此函数用于返回用于给定字体的跨距值。字体必须为启用SDF的。

如果字体不存在，函数将返回 -1。

注意 HTML5 当前不支持 SDF 字体。

 

语法:

font_get_sdf_spread(ind);

参数 类型 描述
ind Font Asset 要获取其跨距值的字体的索引

 

返回:

Real

 

例子:

var _spread = font_get_sdf_spread(my_font);
shader_set_uniform_f(uni_spread, _spread);

上面的代码获取字体的跨距值，并将其传递到着色器统一。
