gpu_set_texfilter_ext

在GameMaker中使用着色器时，此函数可用于设置单个采样器插槽的线性插值。启用此函数时 (true) 将平滑采样器纹理，如果禁用此函数 (false)，则将基于最近的像素绘制图像。默认值由游戏的全局游戏选项设置，或者使用函数 gpu_set_texfilter() 设置。

注意：调用函数 gpu_set_texfilter() 时设置的值将覆盖此设置。

注意：在 HTML5 目标上，此函数只能在启用 WebGL 的情况下使用。

 

语法:

gpu_set_texfilter_ext(sampler_id, enable);

参数类型 描述
sampler_id着色器采样器控制柄 着色器中的采样器 ID。
enableBoolean 启用或禁用纹理过滤 (true/false)

 

返回:

N/A

 

例子:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texfilter_ext(s_tex))
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

上述代码检查特定采样器ID(存储在局部变量中)的纹理过滤是打开还是关闭，并相应地进行切换。
