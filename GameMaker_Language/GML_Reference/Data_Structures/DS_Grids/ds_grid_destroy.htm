ds_grid_destroy

此函数将从内存中删除给定的网格数据结构，释放其正在使用的资源并删除其包含的所有值。当您使用完 DS grid 后，应始终使用此功能，以防止内存泄漏导致游戏速度减慢和崩溃。

注意 调用此函数后，您应始终将保存数据结构引用的变量设置为 -1，因为该引用将不再有效。

 

语法:

ds_grid_destroy(index);

参数类型描述
indexDS Grid要销毁的网格的索引。

 

返回:

N/A

 

例子:

if (lives == 0)
{
    ds_grid_destroy(Wall_Grid);
    Wall_Grid = -1;
    room_goto(rm_Menu);
}

上面的代码将检查内置全局变量 lives 的值，如果为 0，则会销毁变量 Wall_Grid 中引用的 DS 网格，然后更改房间。
