draw_point

使用此函数，您可以使用当前设置的绘制颜色和Alpha在屏幕上的任意位置绘制单个像素。

 

语法:

draw_point(x, y);

参数类型 描述
xReal 要绘制的点的x坐标。
yReal 要绘制的点的y坐标。

 

返回:

N/A

 

例子:

draw_set_colour(c_yellow);
draw_point(100,100);

这将在位置 (100,100) 绘制一个黄色像素。
