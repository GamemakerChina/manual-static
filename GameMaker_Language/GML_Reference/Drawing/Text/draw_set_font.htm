draw_set_font

此函数用于设置绘制文本时使用的字体。

字体必须已添加到游戏的 字体资源 中，或已使用 font_add、font_add_sprite 或 font_add_sprite_ext 创建。

 

语法:

draw_set_font(font);

参数 类型 描述
font Font Asset 要使用的字体的名称，或 -1 表示默认字体

 

返回:

N/A

 

例子:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

上面的代码将使用变量 fnt_Game 中索引的字体绘制给定的文本，并涂上蓝色。
