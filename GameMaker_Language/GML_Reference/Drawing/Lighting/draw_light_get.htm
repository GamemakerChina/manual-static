draw_light_get

该函数将获得指定的光照参数，作为具有以下6个元素的数组-

  • [0] = 启用 / 禁用 (true/false)
  • [1] = 光照类型常数(参见下表)
  • [2] = x 位置
  • [3] = y 位置
  • [4] = z 位置
  • [5] = 光照半径 (仅适用于点光源，对于定向光源为 0)
  • [6] = 浅色 (实数)

光照类型(element[ 1 ])可以是以下两个常量之一：

光照类型常量
常量 描述
lighttype_dir The light is a directional light
lighttype_point The light is a point light

 

语法:

draw_light_get(ind);

参数 类型 描述
ind Real 光照的索引号(从0到7)

 

返回:

Array

 

例子:

light_a = draw_light_get(1);
if light_a[5] < 200
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

上述代码将获取用于将光源步进定义为"1"的值，然后检查半径，如果小于200，则会增加半径，并将光源半径设置为新值。
