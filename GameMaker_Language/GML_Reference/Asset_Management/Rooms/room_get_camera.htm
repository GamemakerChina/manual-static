room_get_camera

使用此函数，可以获取指定给房间中特定视图而不是当前视图的摄影机的唯一索引 ID。指定要使用的房间、要使用的视图端口(从 0 到 7)，函数将返回相机索引。

 

语法:

room_get_camera(rm, vind);

参数 类型 描述
rm Room Asset 要获取其视图摄像机的房间的索引
vind Real 要获取其摄像机的视口的索引

 

返回:

Camera ID

 

例子:

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
{
    room_set_camera(rm_Game, 0, global.MainCam);
}

上面的代码将新创建的房间中的摄像机分配给视口[0]。
