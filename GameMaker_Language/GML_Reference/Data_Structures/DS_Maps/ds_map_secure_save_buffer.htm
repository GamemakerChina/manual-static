ds_map_secure_save_buffer

此函数将先前创建的DS映射保存到缓冲区。提供DS映射ID值(由函数ds_map_create()返回)和要写入的缓冲区的ID(由函数buffer_create()返回)。请注意，如果要保存的DS映射包含数组，则在保存时将转换为DS列表。

注意: HTML5不支持此函数。

用法说明

  • 安全保存的 DS 映射文件只能加载到创建它的设备上，如果您尝试加载保存在其他设备上的文件，则它将无法工作。
  • 此函数将 DS 映射写入当前查找位置的缓冲区 (请参见 buffer_tell)，并按写入的字节数递增。

 

语法:

ds_map_secure_save_buffer(filename);

参数 类型 描述
id DS Map ID DS映射ID值。
buffer Buffer ID 要保存到的缓冲区。

 

返回:

N/A

 

例子:

buff = buffer_create(128, buffer_grow, 4);
var _map = ds_map_create();
ds_map_add(_map, "bob", "ajob");
ds_map_add(_map, "money", 10);
ds_map_secure_save_buffer(_map, buff);
ds_map_destroy(_map);

上述代码将创建一个缓冲区和一个DS映射，然后使用一些值填充映射，并在删除映射之前将其写入缓冲区。
