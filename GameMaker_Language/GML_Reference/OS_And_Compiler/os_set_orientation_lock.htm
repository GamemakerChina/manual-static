os_set_orientation_lock

此功能更改游戏允许的方向。您可以分别启用或禁用横向和纵向。

调用此函数将禁用两个方向的翻转版本，这些版本可在 Android/iOS 游戏选项中找到。

 

语法:

os_set_orientation_lock(landscape_enable, portrait_enable);

参数 类型 描述
landscape_enable Boolean 设置为 true 或 false 可启用或禁用横向方向。
portrait_enable Boolean 设置为 true 或 false 可启用或禁用纵向。

 

返回:

N/A

 

例子:

os_set_orientation_lock(true, true);

这可以实现横向和纵向。
