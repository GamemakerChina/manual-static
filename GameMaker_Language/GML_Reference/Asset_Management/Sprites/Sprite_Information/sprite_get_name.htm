sprite_get_name

该函数将返回名称作为指定精灵的字符串。此名称是在主GameMaker窗口的资源树中为精灵指定的名称。请注意，这只是一个字符串，不能用于直接引用精灵-为此，您需要精灵索引。但是，您可以使用此字符串来获取sprite索引使用返回的字符串以及函数asset_get_index()。

 

语法:

sprite_get_name(index);

参数 类型 描述
index Sprite Asset 要获取其名称的精灵的索引。

 

返回

String

 

示例:

str = sprite_get_name(sprite_index);

上面的代码将获取运行代码的实例的精灵索引的名称，并将返回值存储在变量 “str” 中。
