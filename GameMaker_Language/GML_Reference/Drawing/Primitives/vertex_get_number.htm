vertex_get_number

使用此函数可以查找在任何给定顶点缓冲区中定义的单个顶点的数量。

 

语法:

vertex_get_number(buffer);

参数 类型 描述
buffer Vertex Buffer ID 要检查的顶点缓冲区。

 

返回:

Real

 

例子:

v_num = vertex_get_number(v_buffer);

上述代码将把存储在给定顶点缓冲区中的顶点数存储到变量中。
