精灵实例变量

默认情况下，这些变量包含在GameMaker对象中，并且都与分配给房间中实例的精灵相关。这些变量可以在实例的代码或操作中随时访问，并用于更改绘制的内容和方式。

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale
  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
