matrix_stack_set

此函数用指定矩阵覆盖矩阵堆栈的当前顶部。

 

语法:

matrix_stack_set(matrix);

参数类型 描述
matrixMatrix Array 要使用的矩阵索引。

 

返回:

N/A

 

例子:

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

上面的代码将构建一个新矩阵并将生成的矩阵索引存储在变量“m”中，然后用它替换矩阵堆栈的顶部。
