date_time_string

使用此函数，您可以创建一个包含给定时间的字符串，该字符串在调用此函数时为运行游戏的系统或设备格式化。

 

语法:

date_time_string( date );

参数类型 描述
dateDatetime 要使用的日期时间。

 

返回:

String

 

例子:

str = date_time_string(date_current_datetime());

这会根据显示日期/时间的系统设置以及当前日期和时间，将给定的变量设置为类似“11:36.00”的值。
