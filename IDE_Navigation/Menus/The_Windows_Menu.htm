窗口菜单

此菜单用于打开常规停靠的 IDE 窗口。由于码头可以重新排列，内容可以作为独立窗口删除，甚至完全关闭，如果您发现有任何需要，此窗口允许您重新打开它们。可以从此处打开的窗口是资源浏览器窗口和所有不同的输出和窗口(输出、搜索、源代码控制、断点和编译/语法错误)。

接下来，您可以打开检查器窗口。此窗口用于检查当前选定的 IDE 元素的属性，然后可以根据需要编辑这些属性。

在检查器选项之后，您会看到最近窗口列表，还有一个选项可以打开环境变量窗口，在该窗口中，您可以看到首选项中使用的各种环境变量所指向的不同路径。这些变量不能编辑(它们的名称或它们的路径/设置)，但您可以选择并复制/粘贴它们中的任何一个，以便在其他地方使用。

最后，您可以选择打开教程窗口。这将重新打开与您已打开的教程项目相关的所有窗口，但前提是该项目是作为教程的一部分创建的(否则，此选项将不可用)。有关创建您自己的教程项目的信息，请参阅市场上的部分(因为所有教程都是从那里创建、托管和下载的)。
