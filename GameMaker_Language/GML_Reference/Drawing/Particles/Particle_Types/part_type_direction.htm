part_type_direction

该函数用于确定粒子创建时的方向，也可用于使粒子在其寿命期间增加或减少其方向。最小和最大方向值默认为0，这意味着粒子将向右移动而没有任何变化，但是可以更改这些值以限制沿标准GameMaker方向移动的方向，即0向右，90向上，180向左，270向下。如果将它们设定为相同的值，粒子将仅沿一个方向移动。

您还可以设置一个增量值，该值将在其使用寿命期间向方向添加(如果为正数)或减去(如果为负数)度数。此值最小可为(+/-) 0.01。

最后，您可以设置"摆动"因子。该值将随机添加或减去到粒子寿命的每个步骤的方向。显然，较大的值比较小的值更明显，此值甚至可以是负值，最大范围在-20和20之间。

注意此粒子类型属性的*_incr和*_wiggle参数会影响给定粒子类型的现有粒子。

 

语法:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

参数类型描述
indParticle Type ID要更改的粒子类型的索引。
dir_minReal粒子可以开始的最小方向。
dir_maxReal粒子可以开始的最大方向。
dir_incrReal粒子方向每步应增加或减少多少。
dir_wiggleReal每个步骤应从粒子的方向随机添加或减少多少。

 

返回:

N/A

 

例子:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

上述代码将设置各种粒子值，包括0到359度之间的随机值的方向。每个步骤还将上升1度，并且每个步骤还将随机添加0到20之间的任意数量。
