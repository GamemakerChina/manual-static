ini_read_real

可以使用此函数从 ini 数据文件中读取数字.ini 文件被分割成节(sections)，然后每个节又被分割成键(key)-值(value)对。所以一个典型的ini文件看起来是这样的:

语法:

ini_read_real(section, key, default);

参数类型 描述
sectionString 要从中读取的.ini 部分。
keyString 要从中读取的.ini 相关部分中的键。
defaultReal 如果在定义的位置没有找到值(或.ini文件不存在)，则返回的值。一定是实数。

 

返回:

Real

 

例子:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

这将打开“ savedata.ini”并将得分设置为其中“ save1”> “ score”下的值，然后关闭。又是我。如果在“ save1”> “ score”下没有值，或者没有“ savedata.ini”文件，score 将被设置为0。
