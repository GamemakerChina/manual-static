The Help Menu

此菜单与使用GameMaker时获取帮助有关。它包含以下菜单项：

  • 打开手册 - 这将打开帮助手册。请注意，单击菜单栏中的帮助按钮 也将打开文档，按下 也将打开文档。有关手册的配置，请参阅 帮助首选项 。
  • 知识库 - 这将带您进入 GameMaker 知识库 ，您可以在其中找到有关平台设置、已知问题、使用 GameMaker 和许多其他内容的文章。
  • 联系我们 - 这将带您进入 联系我们 页面。
  • 创建您的联系我们包 - 这将创建一个 .zip 文件，其中包含提交有用的错误报告所需的基本信息，包括您的日志、许可证信息和项目 (可选)。 提交错误报告 时，请包括此 zip。
  • Report A GameMaker Bug - You can use this option to report bugs directly from within the IDE, in case you find any bugs or have any issues while using GameMaker. If you're not signed in, a message is shown to ask you to Sign In/Register. It's also possible to Continue Anonymously.
    In the Report a Bug window you fill in a Title and Description and optionally the Steps To Reproduce the issue. It's also possible to assign an Issue Category (either None, IDE, In-Game or Building), Attach additional files that might be useful, Include the project and Make the report private. Finally you can submit the report by clicking the Submit button.
  • 在资源管理器中打开日志 - 这将打开您系统上的目录，您可以在其中找到 ui.log 文件，该文件通常用于提交与 IDE 相关的错误报告。
  • GameMaker.io-打开GameMaker网站。
  • GameMaker 社区 - 这将在您的浏览器中打开 GameMaker 社区论坛 。请注意，这是一个 社区运行的 论坛，因此，如果您有错误报告或需要技术帮助，应首先访问官方支持网站 (见上文)。
  • 发行说明 - 这将在浏览器中打开 GameMaker 的 发行说明。如果需要回滚更新，可以通过单击此页面上的版本号链接获取以前版本的安装程序。
  • 最新版本说明 - 由于 运行时 可以独立于 IDE 进行更新，因此它有自己的一组 版本说明 。选择此选项将在浏览器中打开这些版本说明，但请注意，最新的运行时可能不是您正在使用的运行时。要了解您使用的运行时，或有关如何添加、删除或激活运行时的信息，请参阅 " 订阅源首选项 " 部分。
  • 所需SDK - 由于 GameMaker 的跨平台特性，需要许多第三方 SDK。单击此链接将转到一篇支持文章，其中详细介绍了您在每个平台上需要什么。
  • 在资源管理器中打开项目 - 这将在操作系统文件资源管理器中打开当前加载的项目的保存位置。
  • 刷新系统字体 - 这将刷新 GameMaker 能够看到的系统字体，并且对于检测 IDE 打开时安装的新字体非常有用。
  • 许可证 - 这将打开一个窗口，其中包含 GameMaker 中使用的所有第三方库的许可证。
  • 关于 - 这将打开"关于 GameMaker"窗口，您可以在其中获取产品的版本信息(报告任何错误或问题所必需的)并查看制作者名单。
