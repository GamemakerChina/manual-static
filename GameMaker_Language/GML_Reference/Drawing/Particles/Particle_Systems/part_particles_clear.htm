part_particles_clear

使用此函数，可以从房间中清除系统当前创建的所有粒子。它不会重置或移除粒子类型本身，只是重置或移除粒子类型本身的可视表示形式，如果有来自发射器的任何对象流式粒子，这些粒子将消失，但在调用此代码后的下一步中将再次出现。

 

语法:

part_particles_clear(ind);

参数 类型 描述
ind Particle System Instance 粒子系统的索引。

 

返回:

N/A

 

例子:

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

上面的代码将检查变量"life"的值，如果它等于0，则清除系统中的所有粒子，然后更改空间。
