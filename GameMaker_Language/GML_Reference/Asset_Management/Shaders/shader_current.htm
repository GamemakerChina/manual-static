shader_current

此函数将返回当前用于渲染的着色器的索引 ID 值，或者如果未使用着色器，则返回 - 1。

 

语法:

shader_current();

 

返回:

Shader Asset or -1 (if no shader is assigned)

 

例子:

if (shader_current() == -1)
{
    shader_set(sh_warp)
}

上面的代码将检查当前着色器是什么，如果它返回-1(未使用着色器)，则设置着色器。
