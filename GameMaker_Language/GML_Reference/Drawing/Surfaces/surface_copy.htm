surface_copy

此函数只需从一个表面获取图像，并将其复制到该表面内指定局部位置(其中(0,0 )位置是目标表面的左上角)的另一个表面上。如果目标表面已有信息，则副本将覆盖此信息，并且函数不会以任何方式更改源表面。

注意：使用表面时，由于表面存储在纹理内存中，因此表面可能随时停止存在。在直接引用表面之前，应该始终使用surface_exists()检查表面是否存在。

 

语法:

surface_copy(destination, x, y, source);

参数 类型 描述
destination Surface ID 要将另一个表面复制到的表面的ID。
x Real 要复制到的x位置。
y Real 要复制到的y位置。
source Surface ID 要复制的表面的ID。

 

返回:

N/A

 

例子:

if view_current == 0
{
    surface_copy(surf, 0, 0, temp_surf);
}
else
{
    draw_surface(surf, 0, 0);
}

上述代码将检查正在绘制的当前视图，如果它是view[ 0 ]，则将变量"temp_surfing"中的索引表面复制到变量"beach"中的索引表面上。如果当前视图不是view[ 0 ]，则会将表面"浏览"绘制到屏幕上。
