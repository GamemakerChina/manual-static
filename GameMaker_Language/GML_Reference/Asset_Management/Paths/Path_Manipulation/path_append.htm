path_append

使用此代码，您可以将一条路径附加到另一条路径上，从而有效地将它们连接在一起，为此，两条路径都必须事先存在。通常，您可能希望在使用 path_add() 创建的路径上使用此函数，因为如果您在路径资源上使用它， 它将永久影响游戏中实例的路径 ，从使用该函数的那一刻起直到游戏结束。

注意不会删除游戏中附加的路径。它仍然在那里，只是它现在没有点，如果您不想再使用它，您应该用函数path_delete()删除它。

 

语法:

path_append(index, path);

参数类型 描述
indexPath Asset 路径的新索引。
pathPath Asset 将附加到" index "的路径的索引。

 

返回:

N/A

 

例子:

path_append(mypath, mppath);
path_delete(mppath);

上述代码将变量"mppath"中索引的路径数据附加到变量"mypath"中索引的路径，然后从存储器中删除"mppath"。
