layer_sequence_is_finished

使用此函数，您可以提供由layer_sequence_create()或层元素函数之一返回的序列元素ID，它将检查序列是否完成播放，如果是，则返回true;如果未完成，则返回false。注意，这仅适用于在回放模式下序列未设置为循环或乒乓的情况。

 

语法:

layer_sequence_is_finished(sequence_element_id)

参数类型 描述
sequence_element_idSequence Element ID 序列元素到目标的唯一ID值

 

返回:

Boolean

 

例子:

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

上述代码检查存储在变量"title_sequence"中的序列元素ID是否完成，如果是，则开始重新播放并设置报警。
