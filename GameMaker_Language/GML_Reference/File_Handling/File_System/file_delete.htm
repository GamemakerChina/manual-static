file_delete

该函数将从系统中删除指定的文件。

需要注意的是，这个函数只会删除那些 GameMaker 能够创建和解析的文件 :
ini 文件、文本文件和二进制文件，或者那些用于存储游戏创建资源 (如精灵或表面) 的文件。
但是，它将 不会 删除除以上文件之外的任何其他文件。

如果文件已成功删除，该函数还将返回 true，或者在任何其他情况下返回 false。

警告 由于 GameMaker 处于沙盒状态，这可能无法按您的预期工作！请参阅有关 文件系统 的部分了解更多信息。

 

语法:

file_delete(fname);

参数类型描述
fnameString要删除文件的名称。

 

返回:

Boolean

 

例子:

if (file_exists("level.txt"))
{
    file_delete("level.txt");
}

这将检查一个文件，如果它存在，它将被删除。
