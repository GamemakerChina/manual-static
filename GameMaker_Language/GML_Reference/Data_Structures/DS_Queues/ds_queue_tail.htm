ds_queue_tail

此函数将只读取队列的最后一个值(即"尾部")。它不会将值出列，这意味着此函数或ds_queue_dequeue()将来仍可以读取该值。

 

语法:

ds_queue_tail(id);

参数类型 描述
idDS Queue ID 要读取的数据结构的ID。

 

返回:

Any (Data type value stored in the queue)

 

例子:

num = ds_queue_tail(control_queue);

上述代码将从变量"control_queue"中索引的队列读取值，并将返回值存储在变量"number"中。
