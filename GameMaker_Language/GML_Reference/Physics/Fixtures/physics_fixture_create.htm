physics_fixture_create

设置装置的第一步是使用此函数创建装置。返回的索引应该存储在一个变量中，用于定义和使用此装置的所有其他函数。

 

语法:

physics_fixture_create()

 

返回:

Physics Fixture ID

 

例子:

fix_Ball = physics_fixture_create();

上述代码将创建一个装置并将其索引存储在变量 "fix_Ball" 中。
