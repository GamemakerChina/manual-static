文本

有许多函数可用于绘制各种形式的文本。需要注意的是，其中一些函数在绘制时会以动态方式更改文本颜色和大小，但在使用html5或设备时，这些函数可能会降低游戏速度或看起来很差，因此最好使用各种大小的字体资源，而不是使用按比例绘制，或者使用大字体资源，然后将其按比例缩小而不是放大。此外，如果需要给大块文本着色，最好先使用draw_set_colour()和draw_set_alpha()函数以及简单的drow_text函数，而不要使用那些具有颜色组件的扩展函数。

有关使用字体和使用 SDF 功能的信息，请参见字体。

以下函数用于设置绘制时要使用的字体，以及绘制时文本的定位：

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

 

这些函数可用于将文本绘制到屏幕，可以是简单的，也可以是带有变换和颜色的：

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

 

还存在以下特定于绘制使用内置高分系统创建的高分的函数：

  • draw_highscore
