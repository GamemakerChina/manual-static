directory_create

此函数将在保存区域中创建具有给定名称的目录。

注意 : 由于无法在浏览器本地存储中创建或删除目录，此函数在 HTML5 上无法工作。

警告！ 由于 GameMaker 被沙盒化，此函数可能无法按您预期的那样工作！ 有关详细信息，请参阅 文件系统 部分。

 

语法:

directory_create(dname)

参数类型 描述
dnameString 要创建的目录的名称。

 

返回:

N/A

 

例子:

if !directory_exists("Games")
{
    directory_create("Games");
}

这将检查指定的目录是否存在于本地数据文件夹中，如果不存在，它将为您创建该目录。
