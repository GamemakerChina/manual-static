gpu_set_state

此函数将使用传入的 DS 映射 设置当前 GPU 状态。可以使用函数 gpu_get_state() 创建提供的映射。

 

语法:

gpu_set_state(ds_map);

参数类型描述
ds_mapDS Map要设置为 DS 映射 的 GPU 状态。

 

返回:

N/A

 

例子:

gpu_set_state(gpu_map);

上述代码使用变量 "gpu_map" 中提供的映射设置 GPU 状态。
