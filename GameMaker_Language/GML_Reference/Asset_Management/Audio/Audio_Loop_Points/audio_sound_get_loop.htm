audio_sound_get_loop

此函数用于返回正在播放的给定声音实例是否设置为循环。

 

语法:

audio_sound_get_loop(index);

参数 类型 描述
index Sound Instance ID 声音实例的索引

 

返回:

Boolean

 

例子:

var _loop = audio_sound_get_loop(snd_car);

上述代码在现有的声音资源 snd_car 上调用 audio_sound_get_loop，并将结果存储在一个临时变量 _loop 中。
