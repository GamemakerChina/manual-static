clickable_set_style

这个函数允许您通过提供的DS Map中的键/值对为给定的按钮设置CSS样式属性。您需要在之前创建按钮元素(使用clickable_add())和DS Map 并将存储的索引作为参数提供给它们。

 

语法:

clickable_set_style(index, map)

参数 类型 描述
index Clickable ID 要样式化的可单击图标的索引。
map DS Map ID 要从中设置样式的 DS Map 的索引。

 

返回:

N/A

 

例子:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

上面的代码将改变可点击按钮的样式与索引存储在变量“按钮”。
