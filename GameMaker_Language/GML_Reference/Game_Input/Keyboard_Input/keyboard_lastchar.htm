keyboard_lastchar

此变量存储最后按下的键的字符串。此变量是不是只读的，您可以更改它，例如，如果您已经处理过它，则将其设置为 "" (空字符串)。

 

语法:

keyboard_lastchar;

 

返回:

String

 

示例:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

上面的代码检查lastkey变量是否大于-1，如果是，它将最后一个键作为字符串添加到变量 "str" 中。
