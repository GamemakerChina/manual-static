os_get_region

此函数返回一个字符串，其中包含运行游戏的操作系统的两个或三个字母区域代码，由ISO3166-1标准设置。如果信息不可用，它将只保存一个空字符串""。

注意：这不是返回的位置区域代码，而是操作系统的区域语言代码。

 

语法:

os_get_region()

 

返回:

String

 

例子:

switch (os_get_language())
{
    case "zh":
        var region = os_get_region();
        if (region == "HK" || region == "MO" || region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

上面的代码检查操作系统语言，如果是中文，它会检查操作系统区域，打开不同的 *.ini 文件取决于返回的值。
