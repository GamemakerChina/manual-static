method_get_index

此函数为您提供给定方法的 脚本函数 引用。此函数可用于检索绑定到实例或结构的方法后面的原始函数。

当传递了一个无效的值(例如，不是一个方法)时，它返回undefined。

 

语法:

method_get_index(method);

参数类型描述
methodMethod要检查的方法变量

 

返回:

Script Function

 

例子:

var _index = method_get_index(light_setup);
if (_index != undefined)
{
    show_debug_message(script_get_name(_index));
}

上面的代码将获取方法light_setup的函数索引，然后如果它不是undefined，它将输出函数的名称到控制台。
