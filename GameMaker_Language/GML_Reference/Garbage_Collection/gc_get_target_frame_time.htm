gc_get_target_frame_time

使用此函数，您可以检索垃圾收集器的当前目标帧值。返回的值以微秒为单位(其中 1,000,000 微秒等于一秒)，默认目标帧时间为 100 微秒。如果你想更改这个值，那么你应该使用函数gc_target_frame_time()。

 

语法:

gc_get_target_frame_time(time);

 

返回:

Real

 

例子:

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

上面的代码检查垃圾收集器的当前帧时间目标，如果不是 50 微秒，则将其设置为该值。
