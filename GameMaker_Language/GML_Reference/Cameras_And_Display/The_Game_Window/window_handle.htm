window_handle

使用此函数，您可以获取内部 Windows ID 值 (HWND，一个指针)。此函数实际上仅对需要窗口句柄调用 DLL 代码中 Windows API 的扩展写入程序有用(返回的指针应转换为字符串，然后在 C++ 中转换为HWND)。下表显示了支持的平台及其返回内容：

平台 返回
Windows Window HWND
macOS NSWindow class
Ubuntu (Linux) XWindow handle
HTML5 Canvas ID

 

语法:

window_handle();

 

返回:

Pointer

 

例子:

win_id = windows_handle();

以上代码将游戏窗口 ID 存储在变量中。
