skeleton_get_bounds

此函数将返回与当前指定的骨骼动画精灵的任何给定边界框相关联的值数组。您为边界框提供索引号以获取的详细信息(您可以使用函数skeleton_GET_NUM_BOUNDS()来检索精灵的边界框总数)，该函数将返回包含以下元素的数组：

  • [0]-边界框上的点数(实数)
  • [1]-边界框的名称(字符串)
  • [2]-第一个点的X位置
  • [3]-第一个点的Y位置
  • [等等]-进一步的X/Y位置数据，直到边界框上的点数

注意Spine 精灵的边界框是在Spine本身中设置的，而不是在GameMaker中。

 

语法:

skeleton_get_bounds(index);

参数类型描述
indexRealThe bounding box index to get the values of.

 

返回:

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

 

例子:

 

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

上面的代码将遍历与当前指定的精灵相关联的每个边界框，然后将该数据作为字符串绘制到屏幕上。
