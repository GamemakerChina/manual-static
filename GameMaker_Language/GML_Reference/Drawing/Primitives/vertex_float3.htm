vertex_float3

该函数将三个浮点值添加到顶点数据中。

顶点缓冲区必须已正确格式化才能使用 vertex_format_add_custom 函数接受此设置。

 

语法:

vertex_float3(buffer, float, float, float);

参数 类型 描述
buffer Vertex Buffer 要写入信息的顶点缓冲区。
float Real 第一个输入值。
float Real 第二个输入值。
float Real 第三个输入值。

 

返回:

N/A

 

例子:

vertex_float3(buff, 0.05, 0.01, room_width / x);

上面的代码将三个浮点值添加到正在定义的顶点数据中。
