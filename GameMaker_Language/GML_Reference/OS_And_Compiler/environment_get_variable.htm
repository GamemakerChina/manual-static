environment_get_variable

此函数返回具有给定名称(也是字符串)的环境变量的值(字符串)。您可以通过在终端应用程序中键入"env"来获取macOS和Ubuntu(Linux)上的可用环境变量有关Windows环境变量的信息，如果您使用的是命令提示符，请键入"echo %PATH%"，使用PowerShell键入"ls env"。请注意，在macOS和Ubuntu(Linux)上，"HOME"环境变量将返回"~/" 路径，该路径在macOS上映射到"/Users/<username>"，在Ubuntu(Linux)上映射到"/home/<username>"。

注意：此函数仅适用于Windows、macOS和Ubuntu(Linux)。

 

语法:

environment_get_variable(name);

参数 类型 描述
name String 要检查的环境变量的名称(字符串)。

 

返回:

String

 

例子:

e_str = environment_get_variable("APPDATA");

上面的代码将返回环境变量"%appdata%"的完整路径，通常是"C:\Users\{username}\AppData\Roaming"。
