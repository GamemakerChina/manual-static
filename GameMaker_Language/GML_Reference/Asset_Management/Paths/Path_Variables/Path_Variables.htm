Path Variables

以下变量在创建时内置于所有实例中，可用于获取有关路径的信息或更改某些属性：

 

  • path_index
  • path_position
  • path_positionprevious
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

 

注意 您可以在此处找到有关实例变量的更多信息： 实例变量 。
