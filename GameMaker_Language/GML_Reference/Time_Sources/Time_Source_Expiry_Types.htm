时间源过期类型

时间源过期常量
常量 描述 值
time_source_expire_nearest The Time Source will expire on the frame nearest to its expiry time 0
time_source_expire_after The Time Source will expire on the first frame after its expiry time 1

使用秒作为单位的时间源不一定总是在游戏帧上过期。它们可能会在游戏帧被处理后但在新帧可以开始处理之前过期。

但是，时间源的回调只能在游戏帧上运行，因此时间源的过期类型控制其回调何时运行，以防在两个帧中间过期。

使用 time_source_expire_nearest 时，时间源的回调将在最接近其过期时间的帧上运行，该帧可能是它之前的帧，也可能是它之后的帧。

在之后使用 time_source_expire_after 时，时间源的回调将始终在其过期时间之后的帧上运行。

在一帧期间，时间源被"勾选"(即，已更新)，其回调在开始步骤和步骤事件之间执行(请参阅事件顺序)。
