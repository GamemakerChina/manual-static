path_reverse

使用此函数，您可以反转各个路径点的编号顺序，因此，例如，如果路径有 5 个点，点 0 将变为点 4，点 1 将为点 3，点 2 不会改变。实际位置点保持不变，仅更改其处理顺序。该函数更改实际路径资源，因此将永久影响从游戏中所有实例开始的路径使用方式。函数一直使用到游戏结束。如果这不是您需要的，那么您应该使用像path_duplicate()这样的函数来首先创建路径的副本，然后调用该函数在重复资源上(当不再需要时，别忘了在资源上调用path_delete())。

 

语法:

path_reverse(index);

参数类型描述
indexPath Asset要更改的路径的索引。

 

返回:

N/A

 

例子:

path_reverse(pth_AI);

这将颠倒处理给定路径上所有点的顺序。
