view_get_surface_id

此函数可用于检索指定给给定视口的表面的唯一 ID 值(如果未指定表面，将返回 -1)。

 

语法:

view_get_surface_id(view_port)

参数类型描述
view_portReal到目标的视口 (0 - 7)

 

返回:

Surface

 

例子:

if (view_get_surface_id(0) == -1)
{
    view_set_surface_id(0, global.vSurf);
}

以上代码将检查是否已将表面指定给 viewport[0]，以及是否尚未指定表面。
