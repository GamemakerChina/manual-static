path_position

此函数可用于获取或设置实例沿路径的位置。该值从0-1标准化，因此如果将其设置为0.5，则实例将精确移动到路径的中间。

 

语法:

path_position;

 

返回:

Real

 

例子:

path_position = random(1);

上面的代码将实例设置为当前路径上的任意位置。
