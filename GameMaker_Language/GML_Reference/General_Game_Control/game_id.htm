game_id 已删除

此只读变量返回您创建的游戏的唯一标识符。如果您需要一个唯一的文件名，或者需要其他任何东西来识别您的游戏，则可以使用此选项。这可以在游戏选项中设置。

 

语法:

game_id

 

返回:

Real

 

例子:

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

以上代码通过在保存之前向分数添加game_id来对分数执行非常基本的加密。在将其读回游戏中时，您将扣除game_id以再次获得正确的值。
