physics_particle_set_max_count

该函数将设置物理模拟中允许的粒子总数。如果将该值设定为 0，则粒子没有创建的封口，并且将其设定为任何其他值将在粒子总数等于该值时停止创建任何其他粒子。请注意，可以在模拟中创建的最小粒子数为 128，因此将此值设置为任何较低值都不会产生任何效果 (除非设置为 0)。

 

语法:

physics_particle_set_max_count(count)

参数类型描述
countReal允许的最大粒子数。

 

返回:

N/A

 

例子:

physics_particle_set_max_count(500);

上述代码将物理模拟的粒子数设置为 500。
