room_get_name

此函数可用于以字符串形式返回指定房间的名称。请注意，这只是字符串，不能用于直接引用房间，因为您需要房间索引。 但是，您可以使用此字符串获得房间索引，使用返回的字符串和函数asset_get_index()。

 

语法:

room_get_name(index);

参数类型 描述
indexRoom Asset 要检查名称的房间的索引。

 

返回:

String

 

例子:

var roomname = room_get_name(room);

draw_text(32, 32, roomname);

上面的代码将获取当前房间的名称并将其绘制在屏幕上。
