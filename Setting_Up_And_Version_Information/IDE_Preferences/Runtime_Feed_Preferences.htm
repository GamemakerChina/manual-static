运行时源首选项

运行时源首选项用于显示和更改当前安装的运行时以及当前使用的运行时。

GameMaker 可以同时安装多个不同的运行时版本，您可以从此菜单中设置要使用的版本。

理想情况下，您希望使用最新的运行时，但有时可能会出现问题，或者您可能需要使用较旧的运行时来实现兼容性等，因此您可以在此处更改 GameMaker 用来构建游戏的运行时工具。

记住上次选择的模块

如果以及如何显示运行时模块对话框，则此选项会更改。

可以在此处设置以下选项：

选项 描述
OK This option remembers your choice of modules and will not show the Runtime Modules Dialog each time you install any runtime. Instead, it will just automatically install those same runtime modules. It corresponds to the "Remember Selection" checkbox ticked in the Runtime Modules Dialog.
Show Message This option always show the dialog when installing a different runtime. It corresponds to the "Remember Selection" checkbox unticked in the Runtime Modules Dialog.

修改当前运行时模块

您可以点击此按钮，随时再次调出运行时模块对话框。通过此按钮打开对话框允许您更改为当前运行时安装的模块。

注意 当前运行时版本可在 IDE 右上角显示的版本详细信息中找到(请参阅起始页元素)。

该对话框显示您已经为当前运行时安装的模块，并允许您安装其他模块。

使用替代 DNS

GameMaker 允许您使用替代 DNS 服务器，如果您愿意或在获取更新的 RSS 提要时遇到延迟。

要使用此选项，请选中"使用替代 DNS "复选框并输入要使用的替代 DNS 服务器的 IP 地址：

注意 如果您将文本字段留空，则 GameMaker 将继续使用您的系统 DNS 设置。

单击"应用"保存更改。 GameMaker 将测试是否可以从给定的 IP 地址获得有效的 DNS 结果。如果不是，将显示错误消息： 

您可以使用的替代 IP 地址有 1.1.1.1 和 9.9.9.9。

运行时 RSS 源

运行时源首选项的这一部分与您从中提取运行时工具的位置相关，并要求您输入 RSS 源的 URL。然后将扫描此提要以查找运行时工具的更新，然后可以下载和安装这些工具。

默认情况下，此部分将显示主运行时源 URL，但您可以单击 添加 按钮来创建一个新的运行时源，您可以在其中提供 RSS 源的名称和 URL，然后单击应用以告知 GameMaker 访问它。您需要关闭并重新打开首选项窗口，才能显示新源 。
