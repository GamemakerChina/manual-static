draw_text

此函数使用当前的文本绘制设置，在房间内的任何位置绘制一个字符串。

绘制文本的设置为： 

  • 由 draw_set_font 设置的字体
  • 由 draw_set_color 和 draw_set_alpha 设置的颜色和 Alpha
  • 由 draw_set_halign 和 draw_set_valign 设置的水平和垂直文本对齐方式

用法说明

使用 string 函数将实数转换为用于绘制的文本，并使用 + 函数组合字符串。在字符串中使用 "\n" 可添加换行符，以便在多行中绘制 (有关如何正确格式化字符串以及可以使用哪些转义字符的信息，请参阅 字符串 页)。

 

语法:

draw_text(x, y, string);

参数 类型 描述
x Real 绘制字符串的x坐标。
y Real 绘制字符串的y坐标。
string String 绘制的字符串。

 

返回:

N/A

 

例子:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

上面的代码将在实例 x/y 位置绘制一个字符串，它将使用存储在全局变量 global.Name 中的字符串，并将其拆分成两行。
