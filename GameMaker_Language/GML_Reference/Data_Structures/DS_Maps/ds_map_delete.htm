ds_map_delete

使用此函数，您可以从给定的先前创建的Ds映射中删除任何给定的密钥(及其对应的值)。

 

语法:

ds_map_delete(id, key);

参数 类型 描述
id DS Map ID 要更改的映射的ID。
key String 要删除的密钥(及其关联值)。

 

返回:

N/A

 

例子:

ds_map_delete(inventory, "shield");

上面的代码将从 DS 映射 (inventory) 中删除键 "shield" (以及与之配对的值)。
