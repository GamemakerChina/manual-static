path_index

GameMaker中的所有资源都有唯一的标识号，path_index是一个只读变量，用于保存已使用path_start()函数分配给实例的给定路径资源的索引值。如果实例没有指定路径，变量将设置为-1。

 

语法:

path_index;

 

返回:

Path Asset

 

例子:

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

上面的代码检查路径是否已分配给实例，如果没有，则启动一个新路径(同时将其分配给path_index)。
