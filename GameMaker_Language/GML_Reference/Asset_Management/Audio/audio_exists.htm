audio_exists

此函数用于返回声音资源是否存在。

要检查的声音可以是声音的单个实例 (使用 audio_play_sound 或 audio_play_sound_at 函数) 或声音资源时，可以将播放的各个声音的索引存储在变量中。

注意 如果您搜索的索引以前未初始化，则此函数将导致错误，因为它正在搜索不存在的资源索引。

 

语法:

audio_exists(index);

参数类型描述
indexSound Asset检查声音的索引是否存在。

 

返回:

Boolean

 

例子:

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

上述代码检查是否存在声音。如果有，则设置为在循环中播放。
