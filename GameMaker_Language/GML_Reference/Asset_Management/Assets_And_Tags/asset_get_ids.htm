asset_get_ids

此函数获取对给定资产类型的资产的引用数组。

注意返回的数组包含您通过资产浏览器添加的资产以及您在运行时动态添加的资产。

注意对于资产类型asset_script，该函数返回脚本资产(您添加到资产浏览器的资产)及其包含的脚本函数。

 

语法:

asset_get_ids(asset_type);

参数 类型 描述
asset_type Asset Type Constant 数组中返回的资产类型

 

返回:

Array of Asset

 

例子:

var _arr_ids = asset_get_ids(asset_sprite);

上面的代码在调用函数时获取游戏中的所有精灵，并将它们存储在临时变量_arr_ids中。您在使用sprite_add_*函数之前添加的精灵也包含在内。
