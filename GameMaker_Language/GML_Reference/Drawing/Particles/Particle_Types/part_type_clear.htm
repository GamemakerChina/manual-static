part_type_clear

使用此函数可以"重置"粒子，将与粒子相关的每个函数(寿命、颜色、alpha、方向等)的所有值返回到其默认值。请注意，此函数不会从屏幕中移除房间中当前可见的任何粒子，因为您应该使用part_particles_clear()。

 

语法:

part_type_clear(ind);

参数 类型 描述
ind Particle Type ID 要清除的粒子类型的索引。

 

返回:

N/A

 

例子:

part_type_clear(global.explode_part);

以上代码将全局变量"explode_part"中索引的粒子类型清除为其默认值。
