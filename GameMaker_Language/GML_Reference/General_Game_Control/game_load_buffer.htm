game_load_buffer

该函数加载之前保存的游戏状态。游戏从之前创建的 " 增长 " 缓冲区 (请参阅 缓冲区 ) 加载，并且该缓冲区必须已使用 game_save_buffer 函数保存了游戏状态。

 

语法:

game_load_buffer(buffer);

参数 类型 描述
buffer Buffer 要从中加载的缓冲区。

 

返回:

N/A

 

例子:

if (keyboard_check_pressed(ord("L")))
{
    if (global.Checkpoint) game_load_buffer(save_buff);
}

仅当全局变量为 true 且玩家按下 "L" 键时，上述代码才会从变量 save_buff 中存储的缓冲区加载先前保存的游戏状态。
