ds_grid_sort

此函数可用于根据给定列中的值对DS网格进行排序(就像您在Os文件资源管理器中按日期、大小等对文件进行排序一样)。下图显示了一个示例：

 

语法:

ds_grid_sort(index, column, ascending);

参数类型 描述
indexDS Grid 要排序的网格的索引。
columnReal 用于对行进行排序的列
ascendingBoolean 是从最低到最高(true)还是从最高到最低(false)进行排序。

 

返回:

N/A

 

例子:

ds_grid_sort(grid, 3, false)

这将获取变量"grid"中索引的DS网格中的所有值，并根据网格第3列中的值对其进行排序(如上图所示)。
