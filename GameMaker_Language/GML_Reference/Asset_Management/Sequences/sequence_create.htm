sequence_create

使用此函数，您可以创建一个新的序列对象，然后可以向其中添加轨迹。该函数返回一个结构，然后您可以访问该结构来设置您创建的新序列。此结构的内容在此页上有详细说明。然后，使用函数layer_sequence_create()，可以使用序列对象结构在房间层上创建序列的实例。请注意，当以这种方式创建序列对象时，如果不再需要它们，则应该通过调用函数sequence_destroy()来再次删除它们，否则将会出现内存泄漏，这会降低速度并最终使游戏崩溃。

 

语法:

sequence_create();

 

返回:

Sequence Object Struct

 

例子:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

上面的代码创建了一个新的序列对象结构，并设置了它的播放长度和循环模式。
