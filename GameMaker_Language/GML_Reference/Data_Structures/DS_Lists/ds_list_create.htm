ds_list_create

此函数创建一个新的列表数据结构并向其返回一个 句柄 。该值应存储在变量中，并在与列表相关的所有进一步函数调用中使用。

注意 如果您在填充列表 之前 尝试访问列表中的值 (即列表为 " 空 ")，则返回值可能为 0，因为在内部是前几个条目列表中的内容在创建时进行设置，以最大程度地减少最初向列表添加项目时的性能问题 (尽管 ds_list_size 函数在新创建的列表上仍将返回 0)。如果您希望确保列表在创建时 " 真正 " 为空，那么您应该在创建列表后调用 ds_list_clear，这意味着为未填充的列表槽位返回的任何值都将为 不明确的 。

 

语法:

ds_list_create();

 

返回:

DS List

 

例子:

list = ds_list_create();

这将创建一个新列表并将其索引 ID 分配给实例变量 list。
