ds_grid_multiply_region

使用此函数，可以指定要将每个单元格值乘以给定量的栅格区域。

注意 此函数仅适用于实数，不适用于字符串。

 

语法:

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

参数类型描述
indexDS Grid网格的索引。
x1Real网格中区域左侧的x位置。
y1Real栅格中区域顶部的y位置。
x2Real网格中区域右侧的x位置。
y2Real网格中区域底部的y位置。
valReal要与区域单元格相乘的值。

 

返回:

N/A

 

例子:

ds_grid_multiply_region(mygrid, 5, 5, 10, 10, 2);

上述代码将采用在定义的矩形网格区域内找到的所有值，并将每个值乘以2。
