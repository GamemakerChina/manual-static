timeline_clear

使用此函数，您可以清除 "时刻" 的特定时间线，删除该时间线的所有代码和操作，并将其留空。

 

语法:

timeline_clear(ind);

参数 类型 描述
ind Timeline Asset 要清除的时间线的索引。

 

返回:

N/A

 

例子:

if (timeline_position > 200)
{
    timeline_clear(global.tl);
    timeline_index = -1;
}

上面的代码将清除由变量" global.tl "索引的所有时刻的指定时间线，当特定时刻已经过去。
