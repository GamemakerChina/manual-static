view_set_yport

此函数可用于设置给定视口的 Y 位置。提供视口索引(从0到7)以及放置它的新位置。

 

语法:

view_set_yport(view_port, y)

参数类型 描述
view_portReal 到目标的视口 (0 - 7)
yReal 新的 Y 位置

 

返回:

N/A

 

例子:

if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)
{
    view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));
}

上述代码将检查 viewport[0] 的 Y 位置，如果它不是要求的位置，则设置为该位置。
