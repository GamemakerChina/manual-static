throw

throw 运行时语句允许您使用以下语法生成自己的运行时异常、结束游戏并显示错误消息：

throw (<expression>);

使用的表达式可以是一个值或一个字符串或任何其他数据类型，然后这将生成一个异常错误，该错误默认情况下显示在屏幕上，关闭错误消息后游戏将结束。例如这样调用：

throw ("Hello World!");
