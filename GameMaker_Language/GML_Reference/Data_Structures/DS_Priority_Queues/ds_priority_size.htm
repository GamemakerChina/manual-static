ds_priority_size

此函数将返回优先级队列的"大小"，即优先级队列中已排定优先级的项目数。

 

语法:

ds_priority_size(id);

参数类型 描述
idDS Priority 要检查的数据结构的ID。

 

返回:

Real

 

例子:

if (!ds_priority_empty(control_priority))
{
    num = ds_priority_size(control_priority);
}

上述代码检查DS的优先级队列以查看其是否为空;如果不为空，则获取其包含的项目数，并将该值存储在变量中。
