IAP

此事件只能由市场上可用的不同目标平台的各种应用内购买(IAP) 扩展触发。此事件将始终生成此事件独占的 DS 映射，并存储在特殊变量 async_load 中。此 DS 映射将根据触发它的扩展函数具有不同的键 / 值对，但始终会有一个 "id" 键，然后可用于识别应用内购买事件的类型就是这样。有关生成此事件的可能返回值和函数的确切详细信息，请参阅每个扩展中包含的各个 PDF 手册。
