audio_sync_group_get_track_pos

此函数返回给定同步组的当前播放位置。组索引是使用函数audio_create_sync_group()创建组时返回的值，返回值是曲目播放的时间(以秒为单位)。

注意此函数不适用于HTML5目标平台。

 

语法:

audio_sync_group_get_track_pos(group_index);

参数 类型 描述
group_index Audio Sync Group ID 要获取位置的组索引。

 

返回:

Real

 

例子:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

上述代码获取变量"sg"中索引的同步组的时间位置，然后使用该值绘制屏幕上播放的时间。
