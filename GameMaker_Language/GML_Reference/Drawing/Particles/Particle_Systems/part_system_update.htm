part_system_update

该函数将在游戏时间内将给定粒子系统提前一步，并且可以用于模拟自动更新关闭时的粒子系统行为，或者可以用于将粒子系统提前到特定点，所有这一切都在一个步骤中完成。例如，如果您有一个用于雪的粒子系统，并希望它看起来从播放器进入房间的那一刻开始就在下雪，那么您可能希望在实例的创建事件中将雪粒子前进几秒钟，这样在第一次绘制雪粒子时，屏幕上到处都是粒子 (下面的代码示例显示了这一点)。当函数 part_system_automatic_update() 关闭系统的自动更新时，也可以使用此函数来移动系统。

 

语法:

part_system_update(ind);

参数 类型 描述
ind Particle System ID 要更新的粒子系统的索引。

 

返回:

N/A

 

例子:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

上述代码将把全球变量 "RainSys" 中索引的粒子系统提前 3 秒。
