audio_sync_group_is_playing

此函数可用于检查同步组中是否有音频正在播放。您需要提供函数audio_create_sync_group()返回的同步组 ID。

注意此函数不适用于HTML5目标平台。

 

语法:

audio_sync_group_is_playing(group_index);

参数 类型 描述
group_index Audio Sync Group ID 要检查的组索引。

 

返回:

Boolean

 

例子:

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

上述代码检查同步组group_one当前是否正在播放，如果正在播放，则组停止。
