vertex_position_3d

此函数将向当前自定义基元定义的顶点添加3D位置数据。您提供将数据写入的缓冲区以及用于绘制的x、y和z位置。

 

语法:

vertex_position_3d(buffer, x, y, z);

参数 类型 描述
buffer Vertex Buffer ID 写入信息的缓冲区。
x Real x位置。
y Real y位置。
z Real z位置。

 

返回:

N/A

 

例子:

vertex_position_3d(buff, x - 100, y - 125, 0);

上述代码将设置要定义的顶点的位置。
