dbg_view

此函数创建自定义调试视图并在 调试叠加层 中显示它。

调试视图是此覆盖中的一个自定义窗口，其中包含可以向其中添加控件的部分。控件可以更改它们引用的变量的值。

可以使用ref_create创建对变量的引用。

用法注释

  • 该函数返回一个指向调试视图的指针，该指针可与dbg_view_delete一起使用以再次删除该视图。
  • 调试视图包含一个或多个调试部分，您可以使用dbg_section添加这些部分。
  • 所有自定义调试视图都列在视图菜单下，可以在那里更改其可见性。

 

语法:

dbg_view(name, visible[, x, y, width, height]);

参数 类型 描述
name String 调试视图的名称
visible Boolean 打开调试覆盖时调试视图是否应可见
x Real 可选调试视图的x位置，默认值表示它可以放置在任何位置
y Real 可选调试视图的y位置，默认值表示它可以放置在任何位置
width Real 可选调试视图的宽度(默认值为500)
height Real 可选调试视图的高度(默认值为400)


返回:

Debug View Pointer

 

例子:

dbg_view("CustomDebugView", true);

上面的代码创建了一个名为"CustomDebugView"的新调试视图，并显示了调试视图可见的The Debug Overlay。
