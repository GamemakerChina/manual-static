skeleton_attachment_get

骨骼动画精灵可以将其他精灵添加为附件，并将这些精灵添加到命名槽(在动画程序中创建附件槽时给出名称)，它们将与当前精灵的动画一起绘制。使用此函数，您可以获取当前分配的精灵的给定槽的附件的名称(作为字符串)。请注意，附加的精灵是通过其在Spine中指定的名称字符串来引用的，或者是在调用skeleton_attachment_create()时引用的。

 

语法:

skeleton_attachment_get(slot);

参数 类型 描述
slot String 要获取其附件的插槽名称(字符串)。

 

返回:

String

 

例子:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

上面的代码将检查当前为名为"slot_leftHand"的插槽分配的附件名称，如果返回空字符串，则附加新的精灵。
