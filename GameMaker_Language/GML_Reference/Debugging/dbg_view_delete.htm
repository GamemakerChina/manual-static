dbg_view_delete

此函数删除使用dbg_view创建的现有调试视图。

 

语法:

dbg_view_delete(view);

参数 类型 描述
view Debug View Pointer 指向调试视图的指针

 

返回:

N/A

 

例子:

Create Event

debug_view = dbg_view("View", true);
show_debug_overlay(true);

Key Pressed Event - Space

dbg_view_delete(debug_view);

上面的代码首先在创建事件中使用 dbg_view 创建一个名为“view”的调试视图，并将指向该视图的指针存储在一个实例变量debug_view. 调试视图在之后使用show_debug_overlay来设置。

在 空格键按下事件 中，使用dbg_view_delete删除调试视图。
