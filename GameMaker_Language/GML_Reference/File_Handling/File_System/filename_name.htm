filename_name

使用此函数返回指定文件的名称部分，带有扩展名，但没有路径。

 

语法:

filename_name(fname);

参数类型 描述
fnameString 要使用的文件。

 

返回:

String

 

例子:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

以上代码获取在指定目录中找到的第一个“doc”类型文件的名称 (作为字符串)。
