speed

GameMaker中的所有实例都具有某些"内置"属性，您可以使用这些属性并设置这些属性来管理它们的外观和行为。speed是这些属性之一，定义实例每步移动多少像素。与hspeed和vspeed不同，speed没有与其关联的方向，因为这由实例的方向值控制，但它可以有一个负值，在这种情况下，实例将沿与方向函数(即：方向- 180°)。 请注意，设置速度和/或方向，也会修改hspeed和vspeed变量的值，重力、重力_方向和摩擦在游戏中使用时，都可以修改该变量的值。

 

语法:

speed;

 

返回:

Real (single precision floating point value)

 

例子:

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上面的代码将使用箭头键来设置实例的速度和方向。
