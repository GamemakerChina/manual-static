part_type_alpha3

此函数可用于为给定类型创建的每个粒子设置三个alpha(透明度)值梯度。第一个阿尔法是所有粒子开始的阿尔法，第二个阿尔法是在其寿命中混合到一半的阿尔法，第三个阿尔法是粒子结束的阿尔法。从一个粒子到另一个粒子的整个生命周期内，通过阿尔卑斯山脉会发生平滑的梯度变化。

 

语法:

part_type_alpha3(ind, alpha1, alpha2, alpha3);

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
alpha1 Real 粒子的起始Alpha。
alpha2 Real 粒子的中点Alpha。
alpha3 Real 粒子的末端alpha。

 

返回:

N/A

 

例子:

part_type_alpha3( part_Health, 0.5, 1, 0);

上述代码将使在变量"part_Health"中创建的索引类型的所有粒子在其生存期内具有从0.5到1到0的Alpha混合。
