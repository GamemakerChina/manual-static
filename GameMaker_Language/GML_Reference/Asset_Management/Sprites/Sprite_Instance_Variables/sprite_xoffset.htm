sprite_xoffset

此只读变量返回已分配给实例的精灵的局部xOffset(Sprite编辑器中定义的原点的X组件)。此XOffset以像素为单位返回，并将依赖于image_xscale，因此(例如)如果将正方形32x32 sprite定义为X原点为16，则当该sprite缩放2倍时，此变量将返回32。如果你需要未缩放的xoffset，你应该使用sprite_get_xoffset()。

 

语法:

sprite_xoffset;

 

返回:

Real

 

例子:

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

上面的代码检查实例的精灵的左边缘是否在房间外，如果在房间外，则移动实例以使整个sprite可见。
