draw_set_swf_aa_level

此函数可用于设置 SWF 格式矢量精灵的抗锯齿 (AA) 级别。这可以是从 0 到 1 的实数值，并将 " 平滑 " 这些精灵的边缘。请注意，要看到此效果，必须首先使用函数 draw_enable_swf_aa() 启用 AA。

 

语法:

draw_set_swf_aa_level(AA);

参数类型 描述
AAReal 要使用的0到1之间的抗锯齿值。

 

返回:

N/A

 

例子:

if (draw_get_swf_aa_level() == 0)
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上述代码将检查 SWF 格式精灵的 AA 值，如果为 0，则启用 AA 并将该值设置为 0.5。
