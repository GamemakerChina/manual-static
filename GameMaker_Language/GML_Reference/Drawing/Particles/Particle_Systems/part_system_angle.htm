part_system_angle

此函数更改渲染给定粒子系统的角度。

 

语法:

part_system_angle(ind, angle);

参数类型描述
indParticle System Instance要更改的粒子系统的索引。
angleReal粒子系统的新视角。

 

返回:

N/A

 

例子:

var _mouse_dir = point_direction(x, y, mouse_x, mouse_y);
part_system_angle(pt_sys, _mouse_dir);

这将获取从实例的位置到鼠标光标的方向，并将其应用为粒子系统的角度。
