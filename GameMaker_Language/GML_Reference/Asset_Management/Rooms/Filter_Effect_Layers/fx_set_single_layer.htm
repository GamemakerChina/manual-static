fx_set_single_layer

此函数用于启用或禁用滤镜/效果的"单层"模式。默认情况下，过滤器/效果将应用于指定给的层以及该层下的所有层，但是，您可以在FX结构上启用单层模式，以确保它仅应用于它指定给的层。

指定 FX 结构(从fx_create()或layer_get_fx()返回)和布尔值以启用(true)或禁用(false)单层模式。

下图显示了在禁用单层模式(这是所有FX层的默认行为)和启用单层模式时使用同一过滤器的过滤器：

注意为 FX 启用单层模式时，请确保将其应用于不是FX 层的层，因为 FX 层本身不包含任何内容，因此看不到过滤器/效果。

 

语法:

fx_set_single_layer(filter_or_effect, enable);

参数类型 描述
filter_or_effectFX Struct 要修改的FX结构
enableBoolean 用于启用(true)或禁用(false)单层模式的布尔值

 

返回:

N/A

 

例子:

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

上面的代码创建一个新的屏幕交换FX，在它上启用单层模式，然后应用到房间层。这意味着屏幕切换过滤器将只应用于"ShakeyThings"层。
