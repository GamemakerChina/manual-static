mouse_clear

此函数清除给定鼠标按钮的当前状态。这意味着，在玩家释放按钮并再次按下之前，检查是否按下按钮不会返回 true。

注意 此函数仅清除鼠标按钮的状态。当鼠标上的按钮被释放时，仍然会检测到释放。

您可以提供 鼠标输入 页面上列出的鼠标按钮常量之一。

 

语法:

mouse_clear(button);

参数类型描述
button鼠标按钮常数要检查哪个鼠标按钮常数。

 

返回:

Boolean

 

例子:

mouse_clear(mb_any);

上面的代码将清除所有鼠标按钮的向下状态。
