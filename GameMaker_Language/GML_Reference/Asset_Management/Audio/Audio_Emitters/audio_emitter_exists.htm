audio_emitter_exists

此函数返回是否存在音频发射器(true)(false)。 请注意，如果您搜索的索引之前尚未初始化，该函数将导致错误，因为它正在搜索不存在的索引。

 

语法:

audio_emitter_exists(index);

参数 类型 描述
index Audio Emitter ID 发射器的索引检查是否存在。

 

返回:

Boolean

 

例子:

if (audio_emitter_exists(s_emit))
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
    s_emit = audio_emitter_create();
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

上面的代码检查发射器是否存在，索引在(先前初始化的)变量"s_emit"中。如果有，则通过它播放声音，但如果没有，则创建该声音，然后播放该声音。
