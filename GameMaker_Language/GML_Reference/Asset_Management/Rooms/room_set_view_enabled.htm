room_set_view_enabled

此函数启用 (true) 或禁用 (false) 游戏中 除当前房间之外的任何房间中的视图。

 

语法:

room_set_view_enabled(index, val);

参数类型描述
indexRoom Asset房间要设置。
valBoolean是在给定房间中启用(true)还是禁用(false)视图。

 

返回:

N/A

 

例子:

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

这将启用在 global.myroom 中索引的房间中的视图。
