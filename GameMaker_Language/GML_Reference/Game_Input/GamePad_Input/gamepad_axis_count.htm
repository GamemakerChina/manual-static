gamepad_axis_count

此函数将返回正在检查的设备上的“轴”控件数量。这些控件是大多数控制器的模拟方向“拇指棒”。

 

语法:

gamepad_axis_count(device);

参数类型 描述
deviceReal 要检查哪个游戏设备“插槽”。

 

返回:

Real

 

例子:

axis = gamepad_axis_count(0))

上述代码存储与变量“轴”中的设备“插槽”0连接的游戏板可用的轴数。
