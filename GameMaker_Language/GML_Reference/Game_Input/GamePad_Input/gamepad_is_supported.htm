gamepad_is_supported

使用此函数，您可以确定目标平台是否支持游戏垫 (返回true) 或不支持 (返回false)。

 

语法:

gamepad_is_supported();

 

返回:

Boolean

 

例子:

global.GP = gamepad_is_supported();

上面的代码检查游戏手柄是否受支持，并将返回值存储在全局变量中以备将来检查。
