window_set_rectangle

通过这个函数，你可以设置游戏窗口在浏览器 (HTML5) 或显示器 (Windows、Ubuntu(Linux) 或 macOS) 中的位置，也可以设置窗口的尺寸。关于窗口位置和窗口大小的更多信息，请参见 window_set_position() 和 window_set_size()。

 

语法:

window_set_rectangle(x, y, w, h);

参数类型描述
xReal窗口的新的 X 坐标。
yReal窗口的新的 Y 坐标。
wReal窗口的新宽度。
hReal窗口的新高度。

 

返回:

N/A

 

例子:

window_set_rectangle(0, 0, display_get_width(), display_get_height());

上述代码将游戏窗口设置为占据整个显示区域(浏览器或屏幕，具体取决于目标平台)。
