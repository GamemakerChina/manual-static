游戏输入

使游戏成为 游戏 的根本在于与玩家的互动。这是通过多种方式实现的，在计算机上使用键盘和鼠标，在移动设备上通过触摸屏和设备倾斜，甚至通过所有平台上的游戏手柄！

首先在GameMaker中，您可以使用pc上的专用鼠标或键盘事件或移动设备上的手势事件，但有时您需要比这些事件提供的更多的控制权，或者您想要包括游戏手柄控件或多个不同的控件选项。为此，您可以使用不同的游戏输入函数来获取已按下或释放的按钮或键，以及检索屏幕上的触摸或移动了多少游戏手柄操纵杆。

在你的游戏中有大量的游戏输入和控制函数，所以它们被分成与特定输入法相关的部分，以便更容易找到你需要的东西。可用的部分是:

  • 键盘输入
  • 鼠标输入
  • 设备输入
  • 手势输入
  • 游戏手柄输入
  • 虚拟按键和虚拟键盘
