gpu_get_texfilter

使用此函数可以检查是否启用纹理过滤(线性插值)(returntrue)或不启用(returnfalse)。有关纹理过滤的详细信息，请参阅函数gpu_set_texfilter()。

注意:在HTML5目标上，此函数只能在启用WebGL的情况下使用。

 

语法:

gpu_get_texfilter();

 

返回:

Boolean

 

例子:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

以上代码检查纹理过滤是打开还是关闭，并相应地将其切换。
