layer_sequence_get_angle

使用此函数，您可以提供layer_sequence_create()或由其中一个层元素函数返回的序列元素 IDas，它将返回游戏室中序列元素的当前角度。注意，角度以度为单位返回，0o向右，90o向上，180o向左，270o向下。

 

语法:

layer_sequence_get_angle(sequence_element_id)

参数 类型 描述
sequence_element_id Sequence Element ID 序列元素到目标的唯一ID值

 

返回:

Real

 

例子:

var _ang = layer_sequence_get_angle(title_sequence)
if _ang > 0
{
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
}

上面的代码利用存储在变量 "title_sequence" 中的 ID 检索序列元素的当前角度，如果它不是 0，则从当前角度中减去 1。
