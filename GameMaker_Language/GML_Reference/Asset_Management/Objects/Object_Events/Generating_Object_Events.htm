生成对象事件

GameMaker 完全是事件驱动的，所有动作都是以特定顺序或游戏中特定事件触发的特定事件的结果。有许多不同的事件，每个事件都有特定的任务或时间，其中一些甚至被分成单独的 " 子 " 事件。通常，这些事件是由 GameMaker 在检测到游戏中的触发器或每个游戏 Tick 时运行的 (就像步和绘制事件一样)，但您也可以在一段代码内将事件应用于当前房间中的实例。

注意 有关事件的更多信息，请参阅 对象事件 部分。
