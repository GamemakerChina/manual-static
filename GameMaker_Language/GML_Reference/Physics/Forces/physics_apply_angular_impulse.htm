physics_apply_angular_impulse

此函数将为启用物理的实例提供角度脉冲。该脉冲将根据给定的量设置角度旋转，忽略当前扭矩，基本上设置固定装置具有的"旋转"量。如果希望使用扭矩向实例施加角力，则应使用函数 physics_apply_torque()。

 

语法:

physics_apply_angular_impulse(impulse)

参数类型 描述
impulseReal 要施加的脉冲(以牛顿/米为单位)

 

返回:

N/A

 

例子:

if (keyboard_check(vk_up))
{
    physics_apply_angular_impulse(10);
}

如果按下一个键，上面的代码将给物理启用的装置提供一个角度脉冲。
