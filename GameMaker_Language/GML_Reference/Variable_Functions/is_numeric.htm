is_numeric

此函数返回给定变量是否为数值 (实数、int32、int64或布尔值)。在某些情况下，您想检查并查看GameMaker中的变量是否包含任何数字值，这就是您使用此函数的时间。如果给定的输入是数字，则函数将返回true，否则返回false。

 

语法:

is_numeric(n);

参数类型描述
nAny要检查的输入。

 

返回:

Boolean

 

示例:

if (is_numeric(val))
{
    current_val += val;
}

上面的代码检查变量 “val” 以查看它是否是一个数值，如果是，则将其添加到给定的变量中。
