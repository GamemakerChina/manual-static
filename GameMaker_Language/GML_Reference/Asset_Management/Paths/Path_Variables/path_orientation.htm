path_orientation

此变量保存调用函数path_start()时分配给实例的路径的当前方向。创建路径时，其方向为默认方向0 degrees，但您可以将此值设置为您希望使用它的任何值。请记住，在GameMaker(除非使用物理)中，角度是逆时针计算的，因此设置路径方向到 90° 将路径向左旋转。

语法:

path_orientation;

 

返回:

Real

 

例子:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

上面的代码将一个随机的、预制的路径资源复制到变量"mypath"中。该新路径随后开始并旋转 90°。
