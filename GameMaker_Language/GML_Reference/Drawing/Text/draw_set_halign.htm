draw_set_halign

此函数用于沿水平轴对齐文本，更改水平对齐方式将更改绘制所有其他文本的位置和方向，默认值为fa_left。接受以下常量：

常量 对齐
fa_left
fa_center
fa_right

 

语法:

draw_set_halign(halign);

参数 类型 描述
halign Horizontal Alignment Constant 水平对齐常量(来自上表)。

 

返回:

N/A

 

例子:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

上述代码将在同一行上绘制两个字符串，分数左对齐，运行状况右对齐。
