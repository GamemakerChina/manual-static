part_type_speed

该函数用于在创建粒子时设置粒子的移动速度，也可用于使粒子在其寿命期间提高或降低其速度。最小和最大速度值默认为1，这意味着粒子将每步移动1个像素，如果它们设置为相同的值，则所有粒子的初始速度将相同，但如果设置为不同的值，则每个粒子将从给定范围内的随机速度开始。例如，如果最小值为3，最大值为6，则将以每步3到6个像素之间的速度创建所有粒子。

还可以设置一个增量值，使粒子在其生存期内移动更快(如果为正数)或更慢(如果为负数)，并且每一步都将给定的量添加到粒子速度中。此值最小可为(+/-) 0.01。

最后，您可以设置"摆动"因子。该值将随机添加或减去到粒子寿命的每个步骤的速度中。显然，较大的值比较小的值更明显，此值甚至可以是负值，最大范围在-20和20之间。

注意此粒子类型属性的*_incr和*_wiggle参数会影响给定粒子类型的现有粒子。

 

语法:

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
speed_min Real 粒子可以开始的最小速度。
speed_max Real 粒子可以开始的最大速度。
speed_incr Real 粒子速度每步应增加或减少多少。
speed_wiggle Real 应该从粒子每步的速度中随机添加或减去多少。

 

返回:

N/A

 

例子:

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

上述代码将设置各种粒子值，包括设置最小速度为5和最大速度为10的速度，每个步骤的速度减慢值为-0.2。每个步骤的转速也会随机摆动0到20之间的值。
