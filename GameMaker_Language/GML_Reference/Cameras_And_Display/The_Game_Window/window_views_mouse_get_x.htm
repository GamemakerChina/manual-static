window_views_mouse_get_x

此函数返回鼠标相对于所有活动视图的 X-坐标，并返回相同的值 mouse_x。

注意：有关常规鼠标函数，请参阅鼠标输入一节

 

语法:

window_views_mouse_get_x();

 

返回:

Real

 

例子:

myx = window_views_mouse_get_x();

这将在窗口中相对于所有活动视图将 myx 设置为鼠标的 X 坐标。
