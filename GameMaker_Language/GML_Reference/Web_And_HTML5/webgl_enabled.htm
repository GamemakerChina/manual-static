webgl_enabled

这个只读变量将返回你的游戏是否启用WebGL (true)或未启用(false)。它只适用于那些通过浏览器运行的游戏(如HTML5)，对于所有其他平台它将返回true。

 

语法:

webgl_enabled

 

返回:

Boolean

 

例子:

if (webgl_enabled)
{
    global.quality = 1;
}
else global.quality = 0;

上面的代码检查WebGL标志，然后相应地设置全局变量“quality”。
