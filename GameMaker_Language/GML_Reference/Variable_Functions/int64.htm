int64

此函数将尝试将给定值转换为带符号的64位整数，其中值必须是实数，字符串，int64,int32, 或ptr。其他任何事情都会导致游戏崩溃并显示错误消息。您可以使用函数is_int64()检查变量是否持有int64。

 

语法:

int64(val);

参数 类型 描述
val Real, String, or Pointer 要转换的值。

 

返回:

int64 (signed 64-bit integer)

 

例子:

steam_handle = int64(global.fileReadString);

上面的代码将全局变量中保存的值转换为64位整数。
