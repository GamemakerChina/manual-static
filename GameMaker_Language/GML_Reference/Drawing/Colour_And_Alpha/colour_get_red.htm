colour_get_red

此函数返回用于生成给定颜色的红色量，该值介于0和255之间，其中0表示无红色，255表示全部为红色。下图说明了这一点：

语法:

colour_get_red(col);

参数类型 描述
colColour 要检查的颜色

 

返回:

Real

 

例子:

r_comp = colour_get_red(c_teal);

上述代码将获得颜色常量"c_teal"的红色分量，并将该值存储在变量"r_com"中。
