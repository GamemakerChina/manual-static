layer_get_y

可以使用此函数检索当前范围房间内图层的 y 位置。提供层 ID(使用layer_create()创建层时获得的 ID)或层名称(作为字符串，这将对性能产生影响)，并且该函数返回层 x 位置相对于房间(0，0)位置的实数。默认值为0。

 

语法:

layer_get_y(layer_id)

参数类型描述
layer_idString or Layer要获取其Y位置的层的唯一ID值

 

返回:

Real

 

例子:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

上面的代码检查给定的层位置，如果未设置为(0，0)，则设置为该位置。
