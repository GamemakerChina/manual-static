is_real

此函数返回给定变量是否是实数 (单，双或整数)。在某些情况下，您想检查并查看变量是否包含实数，这就是您使用此函数的时间。它不会返回实数，而是true或false，因此例如 “fish” 的值将返回false，但是200的值将返回true。

注意对于任何存储为 int64 的 enum 值，此函数都会返回 false。 检查该数据类型的正确函数是 is_int64()。

 

语法:

is_real(n);

参数 类型 描述
n Any 要检查的参数。

 

返回:

Boolean

 

示例:

if is_real(val)
{
    score += val;
}

上面的代码检查变量 “val” 以查看它是否是实数，如果是，则将其添加到分数中。
