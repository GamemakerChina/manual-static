gif_open

使用此函数，您可以创建一个空的 GIF 格式图像，以便将数据添加到该图像中。调用该函数时，您需要指定 GIF 的宽度和高度(以像素为单位，除可用内存外没有大小上限)，并且该函数将返回用于在后续函数中标识礼物的唯一 ID 值，或者如果无法初始化 GIF，则返回 -1 (例如，如果宽度/高度对于可用内存来说太大)。您还可以指定一个可选参数来设置 GIF 的"清晰颜色"。这是一个 RGB 值(无 Alpha 分量)，并且将在添加任何内容之前将 GIF 清除为此颜色。如果不提供清晰的颜色参数，将使用默认的黑色。

请注意，使用此函数时，必须先调用 gif_save() 结束 GIF 的创建，然后再打开另一个文件进行录制(因此，对于每个 gif_open()，必须有一个伴随的 gif_save())。

 

语法:

gif_open(width, height, [clear_colour]);

参数 类型 描述
width Real 要创建的 GIF 的宽度
height Real 要创建的 GIF 的高度
[clear_colour] Colour 可选 要清除GIF的颜色 (RGB)。

 

返回:

GIF ID

 

例子:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

以上代码将创建从应用程序表面获取的具有 30 帧的 GIF 图像文件，然后保存该文件。
