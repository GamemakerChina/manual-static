skeleton_slot_colour_set

使用此函数，您可以设置附件槽的颜色和Alpha，以便使用这些混合值绘制附加到它的所有精灵。请记住，实例Sprite可以具有混合颜色和Alpha设置(image_blend和image_alpha，附件也可以(请参阅函数skeleton_attachment_create_colour()因此，为该插槽分配的附件精灵将具有的最终颜色和阿尔法将是所有这些值的合成。

 

语法:

skeleton_slot_colour_set(slot, colour, alpha);

参数类型描述
slotString要设置的插槽名称，字符串
colourColour要设置的颜色，可以是整数、常量或十六进制值。
alphaReal要设置的alpha(从0到1)。

 

返回:

N/A

 

例子:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

上面的代码创建了一个DS list，然后用鼠标在房间中的位置找到的插槽名称填充它。然后，它在插槽列表中循环，并将找到的插槽的颜色重置为白色，如果颜色还不是白色，则将alpha设置为1。
