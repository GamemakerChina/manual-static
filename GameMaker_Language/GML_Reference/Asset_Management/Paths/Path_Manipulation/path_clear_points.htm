path_clear_points

使用该函数，您可以从路径中删除所有点定义，有效地使路径"为空"。此不会删除路径，因为您应该使用path_delete()，但应注意此函数更改了实际路径资源，因此将永久影响从使用函数到游戏结束期间游戏中所有实例使用路径的方式。如果这不是您需要的，那么您应该使用像path_duplicate()这样的函数来首先创建路径的副本，然后在复制的资源上调用该函数(不要忘记在不再需要资源时调用path_delete())。

 

语法:

path_clear_points(index);

参数类型描述
indexPath Asset要清除的路径的索引。

 

返回:

N/A

 

例子:

if (path_get_number(mypath) > 0)
{
    path_clear_points(mypath);
}

上面的代码检查在变量" myPath "中索引的路径上是否有任何点，如果有，则清除该路径。
