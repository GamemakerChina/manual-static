draw_get_colour

此函数返回当前绘制颜色，该颜色用于绘制形式、文本、基本体和未纹理的3D模型。这可以使用draw_set_colour()函数进行设置。

 

语法:

draw_get_colour()

 

返回:

Colour

 

例子:

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

上述代码将当前绘制颜色存储到局部变量中，并根据实例变量更改绘制颜色。绘制某些文本后，它会将颜色重置为存储在局部变量中的值。
