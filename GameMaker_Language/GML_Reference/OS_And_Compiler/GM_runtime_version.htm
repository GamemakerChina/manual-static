GM_runtime_version

此常量将首选项中定义的运行时版本号保存为用于构建项目的运行时。该值存储为字符串。

 

语法:

GM_runtime_version;

 

返回:

String

 

例子:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

上面的代码获取GM日期、版本和运行时常量并将它们绘制到屏幕上。
