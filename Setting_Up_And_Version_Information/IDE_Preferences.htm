IDE 首选项

首选项窗口 是 GameMaker 的重要组成部分，因为您可以在其中使用 IDE 设置和定制您的体验，以及设置任何特定的平台 SDK 等。在本节中，我们将介绍首选项窗口的 IDE 特定区域-有关各个目标平台的信息，请参阅 目标平台首选项 一节。请注意，在从此窗口更改任何首选项后，您需要单击 应用 按钮才能使其生效。

在将此处的选项更改为根据您自己的喜好定制 GameMaker 之后，如果您希望备份您已设置的首选项，则需要转到 Windows 上的此文件夹：

%AppData%/GameMaker/<your_account_name>/

或 Mac 上的此文件夹：

~/.config/GameMaker

然后将以下两个文件复制到安全位置：

  • local_settings.json
  • cloud_settings.json

但是，如果要重置已更改的首选项，则需要单击窗口顶部的恢复默认设置按钮。请注意，这将仅恢复当前正在编辑的节的默认设置。

您可以从 IDE 左上角的 文件 菜单中打开 GameMaker 首选项：

首选项窗口打开后，您将有以下不同的选项(此列表忽略所有特定于平台的首选项，有关这些选项，请参阅此处)：

  • 常规
  • 开始屏幕
  • 图像编辑器
  • 精灵编辑器
  • 对象编辑器
  • 字体编辑器
  • 房间编辑器
  • 样条编辑器
  • 图块图块集编辑器
  • 时间线
  • 扩展编辑器
  • 文本编辑器
  • 语言首选项
  • 语法检查首选项
  • 市场
  • 插件
  • 运行时源
  • 重定义快捷键
  • 动画曲线编辑器
  • 资源浏览器
  • 序列编辑器
