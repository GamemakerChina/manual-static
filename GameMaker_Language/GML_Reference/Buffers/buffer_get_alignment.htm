buffer_get_alignment

此函数获取给定缓冲区的 字节对齐方式 。

 

语法:

buffer_get_alignment(buffer);

参数 类型 描述
buffer Buffer The buffer to check.

 

返回:

Real

 

例子:

alignment = buffer_get_alignment(buff);

上面的代码获取变量 buff 中存储的缓冲区的对齐方式，并将其存储在变量中。
