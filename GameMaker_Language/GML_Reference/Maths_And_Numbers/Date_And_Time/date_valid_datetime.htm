date_valid_datetime

使用此函数，您可以检查日期时间值以查看它是否有效(返回 true)或无效(返回 false)。 请注意，此函数只会将有效的日期时间视为1970 年 1 月 1 日之后，而在此之前的任何内容都将返回 false，因此您最早可以检查的是：

date_valid_datetime(1970, 01, 01, 0, 0, 0);

 

语法:

date_valid_datetime(year, month, day, hour, minute, second);

参数类型 描述
yearReal 要检查的年份。
monthReal 要检查的月份。
dayReal 要检查的天数。
hourReal 要检查的时间。
minuteReal 要检查的分钟。
secondReal 要检查的秒数。

 

返回:

Boolean

 

例子:

if date_valid_datetime(2011, 9, 15, 10, 3, 30)
{
    mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
}

这将把“mydatetime”设置为2011年9月15日，10:03.30，如果它是一个有效值(它是)。
