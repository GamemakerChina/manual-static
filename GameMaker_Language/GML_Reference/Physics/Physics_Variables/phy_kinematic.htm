phy_kinematic

如果将实例分类为运动学对象，则此只读变量将返回 true; 如果不是，则返回 false。运动学实例是具有无限质量 (密度为 0) 但可以移动的实例。因此，要使实例运动起来，首先要创建一个静态实例，然后设置一个或多个与移动相关的实例变量 (即：phy_speed_x, phy_speed_y 或 phy_angular_velocity)

 

语法:

phy_kinematic;

 

返回:

Boolean (or undefined if the instance is not physics enabled)

 

例子:

if !phy_kinematic
{
    phy_speed_x = 5;
}

以上代码检查实例是否为运动学实例，如果不是，则将水平速度设置为 5。
