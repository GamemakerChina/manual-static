fps

在 GameMaker 中，有两种主要的方法可以用来判断游戏运行的速度。游戏速度(在游戏选项中指定) 和 FPS (每秒帧数)。这些值经常被混淆，但基本上一个是 GameMaker 应该在一秒内完成的游戏步骤数，而另一个 (FPS) 是 GameMaker 在一秒内实际完成的 CPU 步骤数 直到房间速度本身的最大值 。以获得真正的 FPS，即。每个游戏步骤的实际 CPU 步骤数，使用 fps_real 变量。

此只读变量以整数值的形式返回当前的FPS.请注意，该函数只会在游戏的每一步更新一次，因此可能会出现从一个值"跳"到另一个值的情况，但这是很正常的。

 

语法:

fps

 

返回:

Real

 

例子:

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps));
}

上面的代码将检查游戏是否处于调试模式，如果是，它将在屏幕上显示当前的FPS.
