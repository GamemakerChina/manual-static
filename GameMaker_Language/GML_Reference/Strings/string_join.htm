string_join

该函数将传递给它的所有参数的字符串表示形式连接在一起，在每个参数之间插入给定的"分隔符"。该函数返回联接的字符串。

不是字符串的参数将具有隐式运行的 string() 函数。有关如何转换这些数据类型的信息，请参阅 从非字符串类型转换 。

 

语法:

string_join(delimiter, value1 [, value2, ... max_val]);

参数 类型 描述
delimiter String 要在值之间插入的字符串
value1 Any 要加入的第一个值
[, value2, ... max_val] Any 可选 要加入的其他值

 

返回:

String

 

例子:

countdown_message = string_join("... ", "Ready", "Set", "Go!");

上面的代码调用 string_join 从几个短语创建一个新字符串，这些短语由字符串 "..." 连接在一起。结果存储在名为 countdown_message 的变量中。
