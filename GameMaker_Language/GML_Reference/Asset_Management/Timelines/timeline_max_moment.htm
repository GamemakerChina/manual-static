timeline_max_moment

此函数将返回时间线执行其最终操作的时刻的值。因此，如果您有一个时间线，其中8个不同的动作相距20个时刻，则此函数将返回该时间线中最后一个动作的时刻值，这将是160的。此函数是一个很好的方法来测试时间线是否已经通过了运行时的最后一个活动时刻，因为时间线位置将继续递增游戏的每一步，无论是否有进一步的动作。

 

语法:

timeline_max_moment(ind);

参数类型描述
indTimeline Asset获取最后时刻的时间线的索引。

 

返回:

N/A

 

例子:

if (timeline_position > timeline_max_moment(timeline_index))
{
    timeline_running = false;
}

上面的代码将根据最大活动时刻检查当前时间轴位置，如果最大活动时刻更大，则将停止时间轴。
