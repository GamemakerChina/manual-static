part_type_size_x

此函数设置X轴上给定粒子类型的起始大小的范围，以及它在其生命周期内的变化方式。

注意如果两个轴上的尺寸必须相同，则使用part_type_size。

 

语法:

part_type_size_x(ind, size_min_x, size_max_x, size_incr_x, size_wiggle_x);

参数 类型 描述
ind Particle Type ID 颗粒类型指数
size_min_x Real 沿着x轴的最小尺寸，此类型的粒子可以使用该尺寸创建
size_max_x Real 沿着x轴的最大尺寸，此类型的粒子可以使用该尺寸创建
size_incr_x Real 沿着x轴添加每一步的大小
size_wiggle_x Real "摆动"值;每一步可以添加的最大随机值

 

返回:

N/A

 

例子:

 

Create Event

ps = part_system_create();
pt = part_type_create();
part_type_shape(pt, pt_shape_pixel);
part_type_size_x(pt, 1, 2, 0.2, 0.1);

Step Event

part_particles_create(ps, mouse_x, mouse_y, pt, 3);

Clean Up Event

part_system_destroy(ps);
part_type_destroy(pt);

上面的代码首先在Create事件中创建一个新的粒子系统，并使用part_type_create创建一个粒子类型。索引存储在实例变量pt中。然后使用part_type_size_x为新创建类型的粒子设置x轴上的大小：当它们被创建时，这种类型的粒子将在x轴上获得随机大小，该大小至少为1，最多为2。对于它们生命周期中的每一步，0.2将被添加到它们每个的x轴上的大小，沿着最多0.1的随机值。

在Step事件中，使用part_particles_create在粒子系统ps中的鼠标位置创建3个此新类型的粒子。

最后，在"清理"事件中，粒子系统和粒子类型都将被破坏。
