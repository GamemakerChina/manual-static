GameMaker 语言

您可以使用GameMaker 语言通过两种不同的方式对游戏进行编程：通过放置方块来直观地 “写入” 代码，或者手动编写代码。

第一个选项 GML 可视化 最适合想要快速开始制作自己的游戏的初学者和业余爱好者。

第二个选项 GML 代码 允许您通过手动编写代码来充分利用GameMaker语言的强大功能。

虽然 GML可视化 更易于开始使用，但 GML 代码 仍然是一种易于使用的编程语言，具有强大的功能（函数、方法、结构和构造函数等）。

创建可视化/代码脚本

在创建一个新的对象事件、一个新的脚本资源或任何导致脚本创建的东西时，您需要在 GML Code 和 GML Visual 之间做出选择:

除了在 GML 代码和 GML Visual 之间进行选择之外，此窗口还包含两个复选框:

  • 不再询问此项目：启用此选项后，它会将"默认脚本语言"游戏选项设置为您的新选择，这样您就不会被要求为当前项目再次选择。
  • 永远不再询问：启用此选项后，它会将"默认脚本语言"首选项 设置为您的新选择，这样您就永远不会再被要求为任何项目进行选择。
