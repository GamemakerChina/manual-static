vertex_texcoord

此函数将设置纹理坐标，以用于当前为自定义基元定义的顶点。提供用于写入数据的缓冲区以及要使用的纹理内的Uv位置。

 

语法:

vertex_texcoord(buffer, u, v);

参数 类型 描述
buffer Vertex Buffer ID 写入信息的缓冲区。
u Real 要使用的第一个纹理坐标(0-1)。
v Real 要使用的第二个纹理坐标(0-1)。

 

返回:

N/A

 

例子:

vertex_texcoord(buff, 0, 0);

上述代码将设置所定义顶点的Uv值。
