collision_circle_list

此函数与collision_circle()函数相同，只是不是一次只检测一个碰撞实例，而是检测多个实例。提供要检查的圆形区域中心的x/y位置以及半径和要检查的对象，并且可以将检查设置为精确(在这种情况下，所有要检查的实例都需要具有精确碰撞遮罩)，以及检查是否应包括调用实例。

您还提供了个 DS 列表 ，因此任何与调用实例碰撞的实例的 id 值都将添加到给定列表的末尾。您可以在调用此函数之前 清除 列表，以便它只包含此函数调用的结果。您还可以选择根据实例从圆形区域原点到其原点的距离对实例进行排序。函数返回找到的实例数，如果找不到实例，则返回 0。

请注意，您可以不提供对象索引，而是提供 实例关键字 all，以检查当前房间中的所有实例，其中可能包括运行代码的实例 (取决于 notme 参数)。

 

语法:

collision_circle_list(x1, y1, rad, obj, prec, notme, list, ordered);

参数 类型 描述
x1 Real 要检查的圆心的x坐标。
y1 Real 要检查的圆心的y坐标。
rad Real 半径(从中心到边缘的像素距离)。
obj Object Asset or Instance ID 检查实例碰撞的对象。
prec Boolean 检查是基于精确碰撞(true，速度较慢)还是基于其边界框(false，速度较快)。
notme Boolean 是否应排除调用实例(如果相关)(true)或(false)。
list DS List ID 用于存储碰撞实例ID的D列表。
ordered Boolean 列表是否应按距离(true)或不按距离(false)排序。

 

返回:

Real (The number of instances found to be in collision)

 

例子:

var _list = ds_list_create();
var _num = collision_circle_list(x, y, 100, obj_Enemy, false, true, _list, false);
if (_num > 0)
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

上面的代码将检查调用实例位置周围具有100像素半径的圆形区域是否与"obj_Enemy"实例发生碰撞。如果存在任何碰撞，则会循环显示预先创建的列表，并销毁碰撞中的每个实例。
