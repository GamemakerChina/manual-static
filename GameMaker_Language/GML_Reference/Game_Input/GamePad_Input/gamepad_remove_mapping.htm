gamepad_remove_mapping

此函数可用于从给定的游戏手柄时隙索引中删除当前设备映射。调用后，如果您希望能够使用游戏手柄常量正确检测输入，则需要使用gamepad_test_mapping()函数重新映射插槽索引(始终可以使用gamepad_axis/button/hat_count()和gamepad_axis/button/hat_value()函数一起检索直接输入)。

 

语法:

gamepad_remove_mapping(index);

参数类型 描述
indexReal 要从哪个游戏手柄索引 “插槽” 中移除映射。

 

返回:

N/A

 

例子:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

上面的代码将根据变量的值从给定的游戏手柄索引槽中删除映射。
