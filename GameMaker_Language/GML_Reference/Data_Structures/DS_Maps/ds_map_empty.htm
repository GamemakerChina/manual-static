ds_map_empty

如果指定的(以前创建的)DS映射包含任何键/值对，则此函数将返回false;如果不包含，则返回true。

 

语法:

ds_map_empty(id);

参数类型 描述
idDS Map ID 要检查的数据结构的ID。

 

返回:

Boolean

 

例子:

if ds_map_empty(inventory)
{
    weight = 0;
}

上述代码检查变量"inventory"中索引的DS映射是否具有任何密钥/值对，以及是否将变量"weight"设置为0。
