font_exists

此函数返回是否存在具有指定索引的字体。您可以检查从"资源浏览器"中定义的字体索引，或使用font_add()等函数添加的字体。

 

语法:

font_exists(ind);

参数类型描述
indFont Asset要检查的字体的索引。

 

返回:

Boolean

 

例子:

if (font_exists(fnt_Main))
{
    draw_set_font(fnt_Main);
}

这会将活动绘制字体设置为fnt_main(如果存在)。
