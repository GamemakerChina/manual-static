temp_directory

此函数返回每次运行游戏时为游戏创建的临时目录。

此目录将保存文件，并且可以在游戏运行时访问，但当游戏关闭时它将被删除 (连同其中包含的所有文件)。

注意temp_directory 变量存储路径 包括 最后的斜杠。

 

语法:

temp_directory

 

返回:

String

 

例子:

ini_open(temp_directory + "\temp_ini.ini");

这将在游戏的临时目录中打开一个 INI 文件 (如果尚不存在则创建它)。
