draw_vertex_texture

此函数定义基元的纹理顶点的位置。基元的最终外观将取决于选择绘制的基元类型(有关详细信息，请参阅draw_primitive_begin())、向其添加顶点的顺序以及为纹理设置的起点位置。要结束并绘制基元，您必须调用draw_primitive_end().

也可以通过设置要采样到的相应xtex/ytex位置(这些坐标也称为Uv点)来选择在基元表面上绘制纹理的部分。该点由介于0和1之间的值定义，要使用的图像的左上角为(0,0 )点，右下角为(1,1 )点。通过这种方式，可以仅使用源图像资源的一部分对基元进行纹理。

绘制带纹理的基元时，纹理将与draw_set_colour设置的颜色混合。这意味着要绘制不带混合的纹理，应将其设置为c_white，如下例所示。

注意要重复纹理，它的大小必须是2的幂，即：32x32、128x128等…

 

语法：

draw_vertex_texture(x, y, xtex, ytex)

参数 类型 描述
x Real 顶点的x坐标。
y Real 顶点的y坐标。
xtex Real 要使用的纹理采样的宽度。
ytex Real 要使用的纹理采样的高度。

 

返回：

N/A

 

示例：

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

上面的代码将绘制一个4顶点三角形条(制作一个矩形)，纹理保存在"tex"变量中，整个纹理将用于覆盖完成的基元。
