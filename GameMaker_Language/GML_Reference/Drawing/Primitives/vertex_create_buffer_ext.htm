vertex_create_buffer_ext

该函数创建一个新的顶点缓冲区并设置其初始大小 (以字节为单位)。

这是由 GameMaker 创建的特殊 grow 缓冲区，已预先格式化，可在构建与着色器一起使用的基元时使用。您可以指定缓冲区的初始大小 (以字节为单位)，它将返回缓冲区的索引，然后在对缓冲区的所有进一步调用中使用该索引。

 

语法:

vertex_create_buffer_ext(size);

参数 类型 描述
size Real 缓冲区的初始大小(字节)。

 

返回:

Vertex Buffer

 

例子:

v_buff = vertex_create_buffer_ext(1024 * 1024);

上述代码创建一个新的顶点缓冲区，最初大小为 1MB，并将其存储在变量 v_buff 中。
