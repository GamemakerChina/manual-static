dbg_add_font_glyphs

此函数添加 TTF 字体中的一系列字符，用于 调试叠加层 中的文本渲染。您以字符串形式提供字体名称，该名称必须存在于 包含的文件 中。

重要信息 您要添加的字形必须以给定的 TTF 字体出现。

可以使用以下值来指定范围：

字体范围常数
范围 描述
-1 Default Latin range
0 Greek
1 Korean
2 Japanese
3 Chinese Full
4 Chinese Simplified Common
5 Cyrillic
6 Thai
7 Vietnamese

 

语法:

dbg_add_font_glyphs(filename, [size], [font_range]);

参数 类型 描述
filename String 要使用的 TTF 文件的文件名
size Real 可选 字体大小 (以像素为单位 )( 默认值为 13)
font_range Real 可选 要添加的字符范围 (对于拉丁语范围，默认值为 -1)

 

返回:

N/A

 

例子:

dbg_add_font_glyphs("korean_font.ttf", 24, 1);

上述代码从 TTF 字体文件 "korean_font.ttf" 添加韩语字符范围的字形。使用的字体大小为 24。
