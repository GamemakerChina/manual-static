mouse_wheel_up

如果向上旋转鼠标滚轮，则此函数返回true，否则返回false。

 

语法:

mouse_wheel_up();

 

返回:

Boolean

 

例子:

if mouse_wheel_up()
{
    y -= 10;
}

如果鼠标滚轮向上旋转，这会在屏幕上移动当前实例。
