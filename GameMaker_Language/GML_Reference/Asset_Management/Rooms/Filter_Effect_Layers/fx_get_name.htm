fx_get_name

此函数用于从其结构中检索过滤器/效果的名称。指定要读取的 FX 结构(从fx_create()或layer_get_fx()返回)，并且函数以字符串形式返回其名称。然后可以将此名称传递到fx_create()中，以使用相同的过滤器/效果类型创建新的FX结构。

 

语法:

fx_get_name(filter_or_effect);

参数类型 描述
filter_or_effectFX Struct 外汇结构获取名称

 

返回:

String

 

例子:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

上面的代码运行for循环遍历房间中的所有层，并检查每个层的FX结构。如果层具有FX结构，则调用fx_get_name()来检查该过滤器/效果的名称; 如果它等于"_filter_tintfilter"，意味着它是"着色"滤镜，则它将其色调颜色更改为蓝色。
