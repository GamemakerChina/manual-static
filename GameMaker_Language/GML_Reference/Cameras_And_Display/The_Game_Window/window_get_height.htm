window_get_height

使用此函数，您可以获取游戏窗口的当前高度(以像素为单位)。

注意 在 GX.games 上，这将返回浏览器区域的高度。

 

语法:

window_get_height();

 

返回:

Real

 

例子:

hh = window_get_height();

上述代码将当前窗口高度存储在变量 "hh" 中。
