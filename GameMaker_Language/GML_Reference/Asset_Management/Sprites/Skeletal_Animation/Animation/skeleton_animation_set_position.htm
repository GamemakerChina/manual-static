skeleton_animation_set_position

此函数将指定给给定轨迹的动画设置为您提供的位置。位置应为介于0和1之间的实际值。如果提供的值超出此范围，则轨迹位置将设置为该值的小数部分。假设您使用的值为0.4、1.4、2.4……这些都会将位置设置为0.4。

 

语法:

skeleton_animation_set_position(track, position);

参数 类型 描述
track Real 要设置帧的动画轨迹。
position Real 动画帧的新位置。

 

 

返回:

N/A

 

例子:

skeleton_animation_set_position(1, 0.25);

上面的代码将骨骼动画的轨迹1的位置设置为0.25，或者换一种说法，设置为距离动画开始的25%。
