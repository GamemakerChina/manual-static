part_type_colour3

此函数可用于为给定类型创建的每个粒子设置三种颜色渐变。第一种颜色是所有粒子开始的颜色，第二种颜色是在其寿命的一半时间内混合的颜色，第三种颜色是粒子结束的颜色。从一种颜色到另一种颜色，粒子寿命期间的颜色会发生平滑的渐变。

 

语法:

part_type_colour3( ind, colour1, colour2, colour3 );

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
colour1 Colour 粒子开始的颜色。
colour2 Colour 粒子寿命中间的颜色。
colour3 Colour 粒子结束时的颜色。

 

返回:

N/A

 

例子:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

上述代码将使在可变"part_Health"中创建的索引类型的所有粒子在其生命周期内具有从红色到白色到褐色的颜色混合。
