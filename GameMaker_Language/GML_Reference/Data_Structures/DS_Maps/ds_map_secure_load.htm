ds_map_secure_load

此函数将从给定文件加载先前保存的 DS 映射。该文件必须是以前使用 ds_map_secure_save函数创建的，在加载时，该函数将返回从加载的数据创建的 DS 映射的 索引 。此 DS 映射索引应存储在一个变量中，并用于对此映射的所有其他函数调用。请注意，如果要加载的 DS 贴图与数组一起保存为关键值之一，则该数组将在加载时转换为 DS 列表。

重要 安全保存的文件的一个函数是它被锁定到创建它的设备上，因此您无法将保存在一个设备上的文件加载到另一个设备上运行的项目中。

 

语法:

ds_map_secure_load(filename);

参数 类型 描述
filename String 要从中加载地图数据的文件的名称。

 

返回:

DS Map

 

例子:

p_map = ds_map_secure_load("p_data.dat");

上述代码将加载安全保存的DS映射，并将其索引值存储在变量中以供将来使用。
