sqrt

该函数返回给定数字的平方根。

如果将一个数字与其自身相乘，则会得到该数字的平方，但有时您想要执行相反的操作并获得该数字的平方根。因此，为了查找哪个数字与其自身相乘以获得任何给定的 正 值，我们使用此函数。例如：sqrt(9) 将返回 3，因为 3 * 3 = 9。

注意 大于或等于负 epsilon 的负值被限制为 0，即它们被视为 0。请参阅 math_set_epsilon。

 

语法:

sqrt(val);

参数类型描述
valReal要获取其平方根的数。

 

返回:

Real

 

例子:

num = sqrt(val);

上述代码将设置变量 num 以保存 val 中包含的值的平方根。
