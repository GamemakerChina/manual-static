gpu_get_fog

此函数可用于检索雾设置。该函数返回一个4 element 1D数组，其中包含以下信息：

  • [0] = 启用切换(布尔值，true或false)，默认为false
  • [1] = 颜色，默认为 c_Black
  • [2] = 起始距离 (实数), 默认为 0
  • [3] = 终止距离 (实数), 默认为 1

请注意，您可以更改这些值并将完整数组作为单个参数传递给gpu_set_fog()函数。

 

语法:

gpu_get_fog();

 

返回:

Array (4 elements only; see above for details)

 

例子:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

以上代码获取当前雾设置，然后将数组的颜色元素设置为c_red，然后使用更改的数组再次设置雾。
