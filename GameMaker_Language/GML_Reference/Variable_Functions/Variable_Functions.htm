变量函数

本页中的函数旨在处理在游戏中使用GameMaker语言时所有不同的变量和变量类型。有关变量的更多信息，请参阅变量和变量范围页面。

以下可变函数可用:

  • variable_instance_exists 实例的变量是否存在
  • variable_instance_get_names 获取实例的变量名数组
  • variable_instance_names_count 获取实例的变量名数组长度
  • variable_instance_get 获取实例变量的值
  • variable_instance_set 设置实例变量的值
  • variable_global_exists 全局变量是否存在
  • variable_global_get 获取全局变量值
  • variable_global_set 设置全局变量值

信息函数

以下函数提供有关变量的一般信息：

  • nameof 获取任意名称
  • typeof 获取变量类型

方法函数

接下来这些函数都与使用方法变量:

  • method 返回结构与函数绑定引用后的新方法
  • method_get_self 获得方法的结构
  • method_get_index 获取方法的脚本
  • method_call 函数调用方法

结构函数

注意作为优化，当编译器在编译时检测到名称是常量时，编译器会将这些函数中的结构成员名称替换为它们的哈希值。

以下函数用于结构：

  • struct_exists 结构是否存在
  • struct_get 从结构获取值
  • struct_set 给结构设置值
  • struct_remove 从结构删除变量
  • struct_get_names 从结构获取键名数组
  • struct_names_count 从结构获取键名数组长度
  • is_instanceof 给定结构是否是某构造函数或子级创造
  • static_get 获取结构或函数的静态结构
  • static_set 设置静态结构
  • instanceof 获取构造函数名称
  • struct_foreach 结构遍历
  • struct_get_from_hash 从结构使用散列获取值
  • struct_set_from_hash 使用散列给结构设置值
  • variable_get_hash 返回给定变量名的哈希
  • variable_clone 克隆给定值

数据类型函数

最后，您还拥有以下与变量数据类型相关的函数：

注意有关数据类型的更多信息，请参阅数据类型，并参阅此处的类型表。

  • is_string 是否为字符串
  • is_real 是否为实数
  • is_numeric 是否为数值
  • is_bool 是否为布尔值
  • is_array 是否为数组
  • is_struct 是否为结构
  • is_method 是否为方法
  • is_callable 是否可调用
  • is_ptr 是否为指针
  • is_int32 是否为32位整数
  • is_int64 是否为64位整数
  • is_undefined 是否是undefined
  • is_nan 是否是nan
  • is_infinity 是否是infinity
  • is_handle 是否为句柄
  • bool 转换为布尔类型
  • ptr 转换为指针类型
  • ref_create 创建变量的引用
  • int64 转换为64位整数类型
  • handle_parse
