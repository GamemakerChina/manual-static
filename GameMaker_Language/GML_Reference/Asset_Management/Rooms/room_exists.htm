room_exists

该函数检查给定房间是否存在。

它采用房间 索引 ( 实数) 而不是房间名称 (字符串)。

当您使用函数 room_add 动态创建房间时，此函数非常有用，但您也可以使用 只读 变量 room_first 和 ro​​om_last 或函数 room_next 和 room_previous 用于获取特定房间索引，或提供具有存储游戏中任何其他房间的索引。

 

语法:

room_exists(index);

参数 类型 描述
index Room Asset 要检查的房间。

 

返回:

Boolean

 

例子:

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

上面的代码检查数组 global.rm 中索引的房间是否存在，如果存在则转到该房间。
