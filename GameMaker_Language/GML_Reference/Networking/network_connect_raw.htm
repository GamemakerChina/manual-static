network_connect_raw

该函数发送连接到服务器的请求。

它需要 套接字 ID 进行连接 (请参阅 network_create_socket)，并要求您提供要连接的 IP 地址 (字符串，可以是 IPv4 或 IPv6) 以及端口来连接通过。如果连接失败，将返回小于 0 的值。

该函数与 network_connect 的区别在于，该函数可以连接到任何服务器，并且对原始数据不执行任何操作，因此您必须在服务器端自行实现协议。

请注意，默认情况下该功能是同步的，因此您的游戏在建立连接时可能会 " 挂起"。您可以设置连接超时值，或者使用函数 network_set_config 使创建异步，或者使用函数 network_connect_raw_async。

注意您不能在HTML5上使用此函数。对于网络套接字，请使用异步函数。

注意 此函数使用类似 TCP 的套接字 (即 network_socket_tcp)。它也可能适用于其他套接字类型，但这不能保证。

 

语法:

network_connect_raw(socket, url, port);

参数 类型 描述
socket Network Socket ID 要使用的套接字的ID。
url String 连接到(字符串)的URL或IP。
port Real 要连接到的端口。

 

返回:

Real or Network Socket ID

 

例子:

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

上述代码将创建一个新的TCP套接字，然后尝试通过该套接字创建到端口6510上给定URL的"原始"连接。
