game_project_name

此只读变量为目标平台以"保存友好"格式返回游戏的显示名称。如果显示名称包含文件名不允许使用的任何字符，则将自动替换为"_"。可以在游戏选项中设置显示名称。请注意，由于HTML5的文件名没有限制，此字符串可能与由game_display_name返回。

 

语法:

game_project_name

 

返回:

String

 

例子:

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

以上代码获取项目名称，并使用此代码打开(或创建)一个ini文件，然后将其写入。
