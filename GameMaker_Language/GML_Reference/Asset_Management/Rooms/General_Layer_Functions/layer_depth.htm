layer_depth

此函数可用于更改给定层的深度，更改其内容将呈现到屏幕的顺序。提供层 ID(使用layer_create()创建层时获得的 ID)或层名称(作为字符串，这将产生性能影响)，然后给出新深度以将其设置为(整数值从 16000 到 16000)。层深度被定义为当"远离"相机时较高，而当"更靠近"相机时较低，因此，如果您有三个具有深度的层(100、0、100)，则层将按100、0、-100的顺序绘制，因此"顶部"层(即，最接近相机视图的层，因此在其它所有地方上绘制)将是具有-100深度的层。

下图显示了层的深度工作原理示意图：

请注意，您可以随时使用函数layer_get_depth()检查层的深度。 同时请注意，最小和最大层深度为-16000至16000，超出这些深度的任何内容都不会呈现。 如果需要超出该范围的深度，则需要使用函数layer_force_draw_depth()。

请记住，修改层的深度可能会改变哪些过滤器&效果应用于它，因为将深度更改为低于FX层的深度将不再将其效果应用于该层。

 

语法:

layer_depth(layer_id, depth)

参数 类型 描述
layer_id String or Layer ID 层的唯一ID值，用于设置的深度(或层名称为字符串)
depth Real 该层的新深度

 

返回:

N/A

 

例子:

if layer_get_depth(layer) != -100
{
    layer_depth(layer, -100);
}

上面的代码获取运行代码的实例所在的层的深度，如果它不是-100，则深度被设置为-100。
