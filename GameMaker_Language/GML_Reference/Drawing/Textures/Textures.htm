纹理

以下函数都与"纹理"相关，即存储在 VRAM 中的图像，用作游戏中的精灵或其他东西。

第一组可用函数用于检索有关特定图像及其在纹理页面上的位置或大小的信息，以及设置某些纹理函数：

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

 

GameMaker还具有许多与管理与单个纹理以及整个纹理组关联的 VRAM 相关的函数：

  • draw_texture_flush
  • texture_prefetch
  • texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

 

以下函数用于动态纹理页：

  • texturegroup_load
  • texturegroup_unload
  • texturegroup_set_mode
  • texturegroup_get_status

 

最后，我们具有以下函数，这些函数主要用于调试项目并确保纹理内存的有效使用：

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
