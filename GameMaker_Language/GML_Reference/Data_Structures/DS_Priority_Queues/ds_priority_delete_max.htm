ds_priority_delete_max

此函数将返回队列中优先级最高的值，然后从数据结构中删除该值(和优先级)。如果多个值具有相同的优先级，则可以按任何顺序返回其中的任何一个值，但具有相同优先级的所有其他值仍将保留在队列中。

注意 如果优先级队列为空，此函数返回0。

 

语法:

ds_priority_delete_max(id);

参数 类型 描述
id DS Priority ID 要使用的优先级队列的ID。

 

返回:

Any (Data type stored in the priority)

 

例子:

if ai_move
{
    script_execute(ds_priority_delete_max(ai_priority));
}

以上代码检查实例变量，如果返回个true，它将执行在优先级队列中具有最高优先级值的索引脚本，然后从队列中删除该脚本。
