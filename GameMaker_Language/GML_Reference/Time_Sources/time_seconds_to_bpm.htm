time_seconds_to_bpm

该函数以秒为单位获取一个节拍的长度，并返回每分钟的节拍值。

 

语法:

time_seconds_to_bpm(seconds);

参数 类型 描述
seconds Real 转换成每分钟节拍的秒值

 

返回:

Real

 

例子:

var _period = time_source_get_period(time_source);
var _bpm = time_seconds_to_bpm(_period);

show_debug_message(_bpm);

此函数获取时间源的周期，将其转换为 BPM 并将该值打印到输出日志中。

此代码遵循 time_bpm_to_seconds() 的示例。
