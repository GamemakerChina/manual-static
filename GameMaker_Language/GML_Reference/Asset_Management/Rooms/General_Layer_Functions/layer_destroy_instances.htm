layer_destroy_instances

此函数可用于销毁指定给给定层的所有实例。提供层 ID(使用layer_create()创建层时获得的 ID)或层名称(作为字符串，这将对性能产生影响)，然后层上的所有实例将从游戏中删除，触发其销毁和清除事件。

 

语法:

layer_destroy_instances(layer_id)

参数 类型 描述
layer_id String or Layer 层的唯一ID值(或作为字符串的层名称)

 

返回:

N/A

 

例子:

if (global.game_over)
{
    layer_destroy_instances(layer);
}

上面的代码将检查一个全局变量，如果它为 true，则位于调用实例层上的所有实例将被销毁(包括调用实例)。
