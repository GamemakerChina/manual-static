gpu_get_state

此函数将获取当前 GPU 状态，将其作为 DS 映射 返回。 然后可以操作甚至保存此映射，并且可以使用函数 gpu_set_state() 将此映射返回 GPU。

 

语法:

gpu_get_state();

 

返回:

DS Map ID

 

例子:

gpu_map = gpu_get_state();

上述代码将当前 GPU 状态存储在变量中。
