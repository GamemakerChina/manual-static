平台首选项

本手册的这一部分介绍了在将游戏编译到特定目标之前需要设置的不同平台特定的首选项。它们通常只需要设置一次(尽管未来对运行时的更新可能需要您更改某些内容，但这些将在发行说明中提到)，并用于指向所需的任何 SDK，并设置基本的编译选项，以确定您希望 GameMaker 如何为任何给定的平台编译游戏项目。请注意，可用的平台将取决于您拥有的许可证，并不是所有平台都可用。

以下是标准许可证的所有可用目标平台列表：

  • Android
  • HTML5
  • GX.games
  • iOS
  • tvOS
  • macOS
  • Windows

请注意，除了设置这些首选项之外，您还需要设置其中一些平台所针对的设备，否则您可能无法编译或测试您的项目。有关详细信息，请参阅编译一节和设备管理器一节。

由于不同公司的法律限制不同，该手册没有显示各种游戏机首选项(Xbox、PlayStation 和 Nintendo Switch)。如果您有游戏机许可证，则可以从 YoYo Games 帮助台获取此信息，但您必须首先使用您的游戏机 ID 电子邮件联系 YoYo Games，并请求允许查看不同的部分。
