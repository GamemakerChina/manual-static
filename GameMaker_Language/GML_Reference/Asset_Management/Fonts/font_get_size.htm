font_get_size

使用此函数，您可以获得任何字体资源的大小，即字体资源对话框中显示的点值。

 

语法:

font_get_size(ind);

参数 类型 描述
ind Font Asset 要获取大小的字体的索引。

 

返回:

Real

 

例子:

sz = font_get_size(font0);

这将获取由" font0 "变量索引的字体大小，并将其存储在变量" SZ "中。
