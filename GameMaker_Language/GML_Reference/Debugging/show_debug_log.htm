show_debug_log

此函数打开调试覆盖，但仅打开日志窗口。默认情况下禁用此覆盖。

此窗口允许您在底部的文本输入中输入基本命令。有关详细信息，请参阅日志部分。

注意所有脚本都在全局结构的范围内执行。

注意要在FPS窗口打开的情况下调出调试覆盖，请使用show_debug_overlay。

 

语法:

show_debug_log(enable);

参数 类型 描述
enable Boolean 是否显示调试日志

 

返回:

N/A

 

例子:

show_debug_log(true);

上面的代码打开调试覆盖并打开日志窗口。
