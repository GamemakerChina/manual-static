font_add_get_enable_aa

该函数可用于检查使用font_add()添加的字体是否启用了抗锯齿 (AA)。如果启用了AA，该函数将返回true，如果未启用，则返回false。注意，AA是默认启用的，但是您可以使用函数font_add_enable_aa()更改添加字体的AA状态，只要在添加字体之前调用即可。

 

语法:

font_add_get_enable_aa();

 

返回:

Boolean

 

例子:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

上面的代码检查添加字体的抗锯齿状态，如果未启用，则启用它。
