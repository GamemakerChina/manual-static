layer_background_get_blend

此函数可用于获取背景元素的混合颜色。给定背景元素 ID(当使用layer_background_create()创建背景元素或使用函数layer_background_get_id()时获得该 ID)，该函数将返回表示所分配颜色的真实值。

 

语法:

layer_background_get_blend(background_element_id);

参数类型 描述
background_element_idBackground Element ID 获取信息的背景元素的唯一ID值

 

返回:

Colour

 

例子:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_blend(back_id) == c_white
{
    layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

上面的代码将获取名为"Background_sky"的层的层ID，然后使用该层来获取该层上的背景元素的ID。然后使用此 ID 检查元素混合颜色，如果它等于常数c_white，则层混合设置为随机颜色。
