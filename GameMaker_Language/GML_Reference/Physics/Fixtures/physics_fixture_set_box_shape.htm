physics_fixture_set_box_shape

此函数定义装置的长方体形状。由于物理世界使用此值远远超过整个宽度/高度值来确定碰撞等事物，因此需要半宽度和高度。

 

语法:

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)

参数类型 描述
fixturePhysics Fixture ID 装置的索引
halfWidthReal 盒子的半宽度
halfHeightReal 盒子的半高度

 

返回:

N/A

 

例子:

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

上面的代码将对在变量 "fix_Border" 中索引的装置应用一个长方体形状，其宽度为房间宽度，高度为 20 像素。
