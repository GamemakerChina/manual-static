书签

导航工作区的另一种方法是使用 书签 。您可以在脚本或对象事件中向 IDE 窗口或代码段 /GML 可视化添加多达 9 个 " 书签"，然后您可以使用正确的组合键在它们之间快速跳过。

要添加书签，只需选择要标记的窗口或代码行，然后在 Windows 上选择 + + 数字键盘[1-9] 或在 Mac 上选择 + + 数字键盘[1-9]，这将设置书签，并在集成开发环境中对项目进行如下标记：

在上图中，我们为 IDE 的不同部分分配了编号为 1、2 和 4 的书签，现在我们可以离开这些窗口，甚至完全关闭它们，因为书签将允许我们在以后的任何时候再次直接返回它。这可以通过按  / +NumberKey[1-9] 来完成，如果书签项目已关闭，则会将其打开，否则工作区将平移到所选书签。

重要 书签不会与项目一起保存，也不会在 IDE 关闭时保存。它们仅对创建它们的会话有效。

如果要删除书签，只需选择书签窗口或代码行，然后重复您用来创建书签的  +  /  + 数字键盘[1-9]。
