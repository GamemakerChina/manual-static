part_system_drawit

使用此函数，可以强制粒子系统在屏幕上绘制所有当前粒子。如果 part_system_automatic_draw() 处于关闭状态，则在实例的绘制事件中使用时，此函数将显示粒子，或者在绘制目标设置为表面 (使用 surface_set_target()) 时，可以使用此函数将系统中的粒子绘制到该表面。

 

语法:

part_system_drawit(ind);

参数 类型 描述
ind Particle System ID 用于绘制粒子的粒子系统的索引。

 

返回:

N/A

 

例子:

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

上面的代码检查变量"washing"中索引的表面是否存在，如果存在，则将绘制目标设置为表面，绘制Id存储在全局变量中的粒子系统，然后重置绘制目标，以便所有正常绘制再次出现在屏幕上。
