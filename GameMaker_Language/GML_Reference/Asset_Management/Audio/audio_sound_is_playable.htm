audio_sound_is_playable

此函数可用于检查当前是否可播放给定声音索引。由于在HTML5目标平台上处理流和非流声音播放的方式不同，因此需要这样做，如果可以播放声音，则返回true;如果无法播放，则返回false。请注意，在HTML5以外的所有其他平台上，该函数将始终返回true。

 

语法:

audio_sound_is_playable(index);

参数 类型 描述
index Sound Asset 检查声音的索引。

 

返回:

Boolean

 

例子:

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

如果可播放，则上述代码将播放声音"snd_Shot"。
