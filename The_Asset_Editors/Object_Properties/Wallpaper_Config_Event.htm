壁纸事件

有两个事件用于制作动态壁纸。

壁纸配置

每当配套应用中墙纸的设置发生更改时，此事件就会运行。

您可以在wallpaper_config变量中获取更新的墙纸设置。

wallpaper_config

此变量是包含节的struct。 每个节都是一个结构，其中包含该节中的选项和节。

要访问此结构体中的选项，可以使用以下语法： 

wallpaper_config.section_name.option_name

或者，当使用嵌套节时：

wallpaper_config.section1_name.section2_name.option_name

以下是使用wallpaper_set_config页面上定义的相同设置的示例 ： 

var _new_colour = wallpaper_config.colours.blendColor;
obj_clock.colour = _new_colour;

壁纸订阅数据

当收到有关您订阅的指标的信息时，将触发此事件。有关此事件中包含的内容的详细信息，请参阅订阅指标。
