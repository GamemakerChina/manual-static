内置时间源

时间源常量
常量 描述 值
time_source_global The Global Time Source 0
time_source_game The Game Time Source 1

您可以使用全局时间源或游戏时间源作为自定义时间源的父级。时间源都是全局可用的。

全局时间源在主游戏循环之外运行，而游戏时间源作为游戏循环的一部分运行。

全局/游戏之间的选择不会影响时间源是否受游戏帧速率的影响，这将取决于您使用的单位。

次序

在游戏时间源之前处理从全局时间源继承的时间源。

注意 有关详细信息，请参阅 事件顺序 。

状态

游戏时间源具有状态，这意味着可以暂停。全局时间源没有状态，无法暂停。

无论游戏状态如何，您都可以使用全局时间源运行计时器，而使用游戏时间源运行与主游戏循环相关的计时器。然后，您可以暂停游戏时间源，暂停所有与游戏相关的计时器。
