animcurve_destroy

这个函数可以用来破坏任何以前用函数 animcurve_create() 创建的动画曲线。

调用此函数将从内存中删除动画曲线，并清理它包含的任何通道或点 (这些永远不需要手动清理)。请注意，您不能销毁在资源浏览器中创建的任何动画曲线，只有那些在运行时创建的曲线。

 

语法:

animcurve_destroy(curve_struct);

参数 类型 描述
curve_struct Animation Curve Struct 指向要销毁的曲线结构的指针。

 

返回:

N/A

 

例子:

animcurve_destroy(my_curve);

上面的代码会破坏在变量 “my_curve” 中索引的 (先前创建的) 动画曲线结构。
