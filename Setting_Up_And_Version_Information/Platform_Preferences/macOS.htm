macOS 首选项

macOS 首选项具有以下选项：

  • 默认团队识别符：您可以在此处添加 Apple 分配 给您的默认团队识别符。当您的游戏文件被发送到 Xcode 以构建应用程序时，将使用此团队 ID，并将允许 Xcode 生成所需的签名证书。请注意，默认情况下，此设置将应用于为 macOS 构建的所有游戏，但它可以从常规 macOS 游戏选项 中覆盖每个项目。
  • 自动打包选项：您可以在此处设置为 macOS 目标创建可执行文件的默认首选项。在创建可执行文件之前，"显示对话框"选项将始终显示一个对话框，要求您在 DMG 和 ZIP 之间进行选择，但您可以将此选项设置为首选项中的任一选项以跳过该对话框。请注意，在该对话框上启用"记住程序包选项"也会将此选项设置为所选的程序包类型。
