alarm_get

此函数可用于获取给定报警的当前值。提供从 0 到 11 的报警编号，此函数将返回当前报警打开的值。这是直接获取报警数组值的另一种方法。

 

语法:

alarm_get(index);

参数 类型 描述
index Real 要获取的警报索引，从0到11。

 

返回:

Real (integer)

 

例子:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

上面的代码检查调用实例中的所有警报，如果它们大于0，则将它们设置为-1，阻止它们进一步递减计数。
