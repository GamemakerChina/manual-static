ini_close

当您完成读取或写入任何打开的 ini 文件时，应该调用此函数。 如果在使用任何 ini 写入函数后不使用该函数，则不会将任何内容写入磁盘，因为文件信息一直保存在内存中，直到调用此函数，这会强制写入。 如果您尝试打开一个 ini 而之前没有关闭另一个 (或同一个)，您也会收到错误消息。

该函数还将返回一个字符串，其中包含已编码的 ini 文件。 然后可以将该字符串保存到服务器和 / 或与函数 ini_open_from_string()  一起再次使用以重新创建 ini。

 

语法:

ini_close();

 

返回:

String

 

例子:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

这将打开“savedata.ini”并使用键“score”从“save1”部分读取一个值，然后关闭 ini 文件。 如果“save1”下没有名为“score”的值，或者不存在“savedata.ini”文件，将返回 0 并创建一个新的 ini 文件。
