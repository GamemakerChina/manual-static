texturegroup_get_names

此函数返回一个数组，其中包含游戏中包含的所有 纹理组 的名称。

注意 包括 GameMaker 添加后备纹理的纹理组。空纹理组可能不包含在内，因为它们已被资源编译器过滤。

 

语法:

texturegroup_get_names();

 

返回:

Array of String

 

例子:

var _arr_names = texturegroup_get_names();
show_debug_message("Texture Groups:\n--------------");
array_foreach(_arr_names, show_debug_message);

上面的代码使用 texturegroup_get_names 获取所有纹理组的名称，并将它们列在调试输出中。
