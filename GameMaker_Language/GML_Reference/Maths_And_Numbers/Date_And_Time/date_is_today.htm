date_is_today

如果给定的datetime值是被检查的当天(例如:今天)，该函数将返回true，否则将返回false。这对于游戏中的复活节彩蛋或解锁季节性内容来说是一个非常方便的函数。请注意，此函数将受时区设置的影响(默认为本地时间)，您可以使用date_set_timezone()函数更改该设置。

 

语法:

date_is_today(date);

参数类型描述
dateDatetime要使用的日期时间。

 

返回:

Boolean

 

例子:

if (date_is_today(global.Halloween))
{
    global.Max_Levels = 200;
}

上面的代码将检查存储在全局变量 "Halloween" 中的日期时间，看看是否是今天，如果是，它将另一个全局变量设置为一个新值。
