ds_priority_delete_value

此函数将简单地从索引优先级队列中删除给定值及其优先级。

 

语法:

ds_priority_delete_value(id,val);

参数类型描述
idDS Priority要使用的优先级队列的ID。
valAny要从优先级队列中删除的值。

 

返回:

N/A

 

例子:

if (ai_move == false)
{
    ds_priority_delete_value(ai_priority, AI_Move);
}

以上代码检查实例变量，如果返回false，将从优先级队列中删除索引脚本函数。
