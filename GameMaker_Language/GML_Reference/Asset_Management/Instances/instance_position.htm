instance_position

使用此函数，可以检查与对象的另一个实例或所有实例碰撞的位置。使用该选项时，将检查房间中的实例或对象的单个点。如果已检查实例的精确冲突，则将针对实例的边界框或实例的掩码执行此检查，并将返回唯一实例id。 如果您不需要冲突实例的id，应该考虑使用position_meeting()。此函数还接受特殊关键字all和other，如果没有发生冲突，将返回关键字noone;如果确实发生冲突，则返回找到的实例的唯一 ID 值。

 

语法:

instance_position( x, y, obj );

参数 类型 描述
x Real 要检查实例的X位置。
y Real 要检查实例的Y位置。
obj Object Asset or Instance ID 要检查其实例的对象。

 

返回:

Instance ID or noone

 

例子:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst) image_index=1;
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

上述代码将检查与鼠标位置"obj_Pause_Button"实例的冲突，如果存在冲突，则在创建对象"obj_Menu"的新实例之前，它将使用返回的id将其image_index设置为新值。
