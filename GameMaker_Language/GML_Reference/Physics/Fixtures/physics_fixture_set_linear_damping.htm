physics_fixture_set_linear_damping

阻尼用于降低物理世界范围内实例的速度，与摩擦力的不同之处在于，摩擦力仅发生在具有装置的两个实例接触时。模拟阻尼也比模拟摩擦便宜得多，但请注意，阻尼不是摩擦的替代品;这两种效果可以并且应该一起使用。下面是一张图片来说明阻尼：

阻尼参数应介于 0 和无穷大之间，0 表示无阻尼，无穷大表示完全阻尼。通常，将使用介于 0 和 1 之间的阻尼值，但如果需要，可以使用任何非负值。

 

语法:

physics_fixture_set_linear_damping(fixture, damping)

参数类型 描述
fixturePhysics Fixture ID 装置的索引
dampingReal 装置的阻尼，通常在 0 和 1 之间

 

返回:

N/A

 

例子:

physics_fixture_set_linear_damping(fix_Ball, 0.1);

上面的代码将 "fix_ball" 中步进装置的线性阻尼设置为 0.1。
