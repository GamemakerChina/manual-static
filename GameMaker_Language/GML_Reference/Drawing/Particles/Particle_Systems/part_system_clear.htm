part_system_clear

使用此功能，可以将给定粒子系统清除为默认状态，删除所有发射器并重置深度和位置(如果已更改)。使用此函数时要小心，就好像您使用与此系统关联的发射器进行了任何实例设置或爆破或任何其他操作一样，除非您使用part_emitter_exists()函数进行检查，否则您将收到错误。也不需要调用part_emitter_destroy()函数，因为这也是自动处理的。

注意 ：此函数将清除房间中的可见颗粒，但 不会 清除颗粒属性，也不会将颗粒从内存中清除。为此，应使用函数 part_type_clear() 和 part_type_destroy()。

 

语法:

part_system_clear(ind);

参数 类型 描述
ind Particle System Instance 要清除的粒子系统的索引。

 

返回:

N/A

 

例子:

part_system_clear(global.Sname);

以上代码将全局变量"名称"中索引的粒子系统清除为其默认状态。
