get_integer

这将创建一个显示自定义消息的弹出窗口，其中有一个标记为" OK "的按钮，并提示用户输入整数值。该函数将返回键入的整数，如果未输入任何内容，则返回默认值。

注意此函数仅在Windows目标上用于调试，但在所有其他目标上已弃用。

 

语法:

get_integer(str, def);

参数 类型 描述
str String 要在弹出消息中显示的字符串。
def Real 文本框中的默认值。

 

返回:

Real

 

例子:

global.level = get_integer("Level to test?", 1);

上面的代码将显示一条消息，提示用户选择要测试的级别。返回值将存储在全局变量global.level中。
