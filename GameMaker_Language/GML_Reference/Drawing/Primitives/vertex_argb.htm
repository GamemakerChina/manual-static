vertex_argb

此函数将为当前为自定义基元定义的顶点设置ARGB值。提供写入数据的缓冲区，以及希望用作单个32位无符号整数Alpha样本(最高8位)的红色、绿色、蓝色和Alpha值，然后是红色样本、绿色样本，最后是蓝色样本(最低8位)。您可以使用十六进制表示法($AARRGGBB)实数或使用make_colour_*()函数中的任何一个来定义颜色值。

 

语法:

vertex_argb(buffer, argb);

参数 类型 描述
buffer Vertex Buffer ID 写入信息的缓冲区。
argb Colour 要设置的参数值。

 

返回:

N/A

 

例子:

vertex_argb(buff, $FFFFFFFF);

上述代码将被定义的顶点的Argb值设置为白色。
