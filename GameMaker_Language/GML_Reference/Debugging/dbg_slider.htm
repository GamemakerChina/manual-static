dbg_slider

此函数为当前调试部分中的实数值创建滑块控件。

您可以选择为滑块提供最小值和最大值。

注意 此调试控制跨越两列。

 

语法:

dbg_slider(ref[, minimum, maximum, label]);

参数 类型 描述
ref Reference 对实数值的引用，使用ref_create创建
minimum Real 可选滑块可以具有的最小值
maximum Real 可选滑块可以具有的最大值
label String 可选滑块旁边显示的标签

 

返回:

N/A

 

例子:

Create Event

my_var = 7;
dbg_slider(ref_create(self, "my_var"), 0, 10);

上面的代码在对象的Create事件或任何其他不连续执行的事件中执行。首先，设置实例变量my_var。然后，使用dbg_slider创建滑块控件。它接收由ref_create返回的变量引用，以及minimum和maximum参数的值0和10。拖动滑块会更改实例的my_var变量的值。
