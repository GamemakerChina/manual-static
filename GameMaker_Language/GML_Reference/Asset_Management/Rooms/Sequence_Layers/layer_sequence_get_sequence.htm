layer_sequence_get_sequence

With this function you supply the sequence element ID - as returned by layer_sequence_create() or by one of the layer element functions - and it will return the sequence object struct. This function bypasses the need to first get the sequence instance struct and permits you to access the sequence data directly. You can find out more about the format of the sequence object struct on The Sequence Object Struct page.

 

语法:

layer_sequence_get_sequence(sequence_element_id)

参数类型描述
sequence_element_idSequence Element ID序列元素到目标的唯一ID值

 

返回:

Sequence Object Struct

 

例子:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

上面的代码创建新的序列元素，然后检索其序列数据结构。然后用于更改播放速度和序列类型。
