path_set_closed

此函数可用于将给定路径标记为打开 (假) 或关闭 (真)。新创建的路径默认是关闭的 (true)。

闭合路径的起点与其终点相连，形成循环，而开放路径则具有确定的、不连接的起点和终点。

此函数会更改实际路径资源 ，因此将永久影响游戏中所有实例从使用该函数到游戏结束期间如何使用路径。

如果这不是您所需要的，那么您应该使用像 path_duplicate() 这样的函数首先创建路径的副本，然后在复制的资源上调用此函数 (不要忘记调用当不再需要资产时，对其进行 path_delete())。

 

 

语法:

path_set_closed(index, closed);

参数类型描述
indexPath Asset要更改的路径的索引。
closedBoolean路径是否关闭(true)或不关闭(false)。

 

返回:

N/A

 

例子:

path_set_closed(pth_Patrol, true);

这会将给定路径设置为闭合路径。
