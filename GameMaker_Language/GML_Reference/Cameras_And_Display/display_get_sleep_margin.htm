display_get_sleep_margin

此函数可用于获取用于游戏渲染计时的当前 sleep margin 值，并将返回毫秒值。有关显示计时的详细信息，请参阅 display_set_timing_method()。

 

语法:

display_get_sleep_margin();

 

返回:

Real

 

例子:

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

上述代码将检查定时方法，如果未将其设置为 tm_sleep，则会将其设置为该值，并将 sleep margin 设置为 20。
