audio_listener_set_orientation

使用此函数，您可以更改 3D 音频空间内给定收听者的方向。默认监听器索引为0，但可以使用函数audio_get_listener_info()获取目标平台可用的不同索引。

观察矢量和up矢量基于从给定的相对X、Y和Z位置解析的值，默认情况下，观察矢量为(0，0，1)，上方向矢量为(0，1，0)，如下图所示：

使用此函数更改给定的收听者方向将改变游戏室周围的音频发射器产生的声音被游戏玩家感知的方式。在下面的示例中，当收听者位于默认位置时，发射器产生的声音似乎来自收听者的右下方，但在收听者的新位置和方向下，它们现在将被感知为来自右上方。

 

语法:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

参数 类型 描述
index Real or Audio Listener ID 要设置其方向的监听器。
lookat_x Real X 环视矢量 (默认值为 0)。
lookat_y Real Y 环视矢量 (默认值为 0)。
lookat_z Real Z 环视矢量 (默认值为 1)。
up_x Real X 上方向向量 (默认值为 0)。
up_y Real Y 上方向向量 (默认值为 1)。
up_z Real Z 上方向向量 (默认值为 0)。

 

返回:

N/A

 

例子:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

上面的代码检索摄像机视图[0]的视图矩阵，然后使用它来设置音频监听器的位置和方向，该监听器的ID存储在全局变量"播放器_监听器"中。
