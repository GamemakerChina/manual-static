window_set_position

使用此函数，您可以将游戏窗口设置为显示器 (MacOS、Linux(Ubuntu) 和 Windows) 或浏览器 (HTML5) 中的特定位置。

注意：如果HTML5游戏使用自定义的 index.html，并将画布设置为固定位置，则此函数将不会影响窗口位置。

 

语法:

window_set_position(x, y);

参数类型 描述
xReal 窗口定位位置的 X 坐标。
yReal 窗口定位位置的 Y 坐标。

 

返回:

N/A

 

例子:

window_set_position(0, 0);

上面的代码将把游戏窗口定位在浏览器的左上角或显示(取决于使用的目标模块)。
