analytics_event OBSOLETE

这个函数将把指定的文本发送到你通过HTML5 游戏设置设置的分析提供程序。此函数可用于创建自定义分析，以跟踪所使用的提供者范围之外的内容。

 

语法:

analytics_event(string);

参数类型描述
stringString要发送给提供程序的字符串

 

返回:

N/A

 

例子:

if (hs_new)
{
    analytics_event("New hiscore of " + string (score));
}

上面的代码将检查一个变量，看它是否是true，如果是，则用指定的字符串触发一个特殊的分析事件。
