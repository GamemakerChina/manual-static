part_type_orientation

此函数用于确定粒子精灵在创建时的方向，也可用于使粒子方向在其寿命期间增加或减少。最小方向值和最大方向值默认为0，但这些值可以更改为沿标准GameMaker方向随机设置方向，0度为右，90度为上，180度为左，270度为下。如果将它们设定为相同的值，则将使用相同的方向创建所有粒子。

您还可以设置一个增量值，该值将在其整个生命周期内向方向添加(如果为正数)或减去(如果为负数)度数。此值最小可为(+/-) 0.01。

也可以设置"摆动"因子，与其他粒子函数相同。该值将随机添加或减去到粒子生存期的每个步骤的方向。显然，较大的值比较小的值更明显，此值甚至可以是负值，最大范围在-20和20之间。

最后，可以选择相对于或不相对于方向，这意味着当粒子具有方向(和速度)时，粒子精灵将围绕该向量定向。请记住，如果将粒子速度设置为减小，并且粒子速度达到0，则缺少速度会将方向设置为默认值0 °，因此相对方向将导致粒子精灵"跳转"到不同的角度。

注意此粒子类型属性的*_incr和*_wiggle参数会影响给定粒子类型的现有粒子。

 

语法:

part_type_orientation( ind, ang_min, ang_max, ang_incr, ang_wiggle, ang_relative );

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
ang_min Real 粒子的最小起始角度。
ang_max Real 粒子的最大起始角度。
ang_incr Real 每步角度的增加(如果设置为负值，则为减少)(以度为单位)。
ang_wiggle Real 角度每步顺时针/逆时针摆动的量。
ang_relative Real 是否设置其相对于粒子运动方向的角度(true)或不(false)。

 

返回:

N/A

 

例子:

part_type_shape(particle2, pt_shape_spark);
part_type_size(particle2, 0.10, 0.50, 0.01, 0);
part_type_scale(particle2, 0.30, 0.30);
part_type_colour1(particle2, 8454143);
part_type_alpha1(particle2, 0.50);
part_type_speed(particle2, 4, 4, -0.07, 1);
part_type_direction(particle2, 0, 359, 0, 20);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 5);

上述代码将设置各种粒子值，包括0 º到359 º之间的随机值的方向。每个步骤还将具有0到20之间的任意随机数量，并且方向是相对于运动方向的。
