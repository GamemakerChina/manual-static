camera_set_view_speed

您可以使用此函数更新房间内相机的速度。给出唯一的相机 ID 值(由不同的 camera_create() 函数返回)，然后给出 x 和 y (水平和垂直)设置为跟随给定实例时应该移动的速度。速度以每步像素数计算，并且可以设置为"-1"以使相机立即移动，但是如果相机未设置为跟随任何实例，则此处设置的值将不具有可见效果。

 

语法:

camera_set_view_speed(camera_id, xspeed, yspeed)

参数类型 描述
camera_idCamera ID 创建相机时返回的唯一相机 ID 值。
xspeedReal 视图应在水平(x)轴上移动的速度(每个游戏帧的像素数)
yspeedReal 视图应在垂直(y)轴上移动的速度(每个游戏帧的像素数)

 

返回:

N/A

 

例子:

camera_set_view_speed(view_camera[0], 5, 5);

上述代码将为分配给 viewport[0] 的相机设置视图的相机速度。
