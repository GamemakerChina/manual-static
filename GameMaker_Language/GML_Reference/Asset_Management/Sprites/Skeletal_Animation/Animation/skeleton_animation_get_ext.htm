skeleton_animation_get_ext

单个骨骼动画精灵可以具有各种动画集，并且可以为这些集指定不同的轨迹，以便您可以"混合和匹配"动画集。此函数将返回给定轨迹号当前使用的动画集的名称(由函数skeleton_animation_set_ext设置)。

 

语法:

skeleton_animation_get_ext(track);

参数类型描述
trackReal要获取其动画集名称的轨道编号。

 

 

返回:

String

 

例子:

if (skeleton_animation_get_ext(1) != "bodyfight")
{
    skeleton_animation_set_ext("bodyfight", 1);
}

上面的代码将把轨道1使用的动画集更改为" BodyFight "(如果尚未更改)。
