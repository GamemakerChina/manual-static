texture_global_scale

此函数允许您控制从WAD文件(作为最终游戏可执行文件的一部分)加载时纹理页面的缩放。输入值必须是两个值的幂，并且可以：

  • 若要将纹理页面设置为 1:1，请使用 texture_global_scale(1)
  • 始终使用 texture_global_scale(2) 将纹理减半
  • 要使其四分之一，请使用 texture_global_scale(4) 等 ...

通过这种方式，可以更好地控制具有低内存问题的平台上的纹理页面内存使用。请注意，纹理页面是根据Wad的需要创建的，因此您可以在游戏开始时调用它来加载所有缩放的纹理，或者可以在特定时间调用它以加载特定纹理，在这种情况下，您可能需要在调用此函数之前调用draw_texture_flush()，然后使用预取函数获取正确的纹理页面到内存中。

此功能仅适用于 PNG 纹理。您可以使用 纹理组 更改纹理格式。

 

语法:

texture_global_scale(pow2integer);

参数 类型 描述
pow2integer Real 要使用的比例因子(1、无比例、2、半比例、4、四分之一比例等)

 

返回:

N/A

 

例子:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

上述代码将刷新内存中的所有纹理，然后将纹理缩放设置为2(使纹理页面大小为一半)，最后预取精灵"spr_Trees"所在的纹理页面。
