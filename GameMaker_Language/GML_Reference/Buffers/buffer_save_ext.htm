buffer_save_ext

使用此函数，您可以将缓冲区的部分内容保存到文件中，以便使用buffer_load()函数将其读回内存。"偏移量" 定义缓冲区内的起始位置用于保存 (以字节为单位)，"大小" 是从该偏移量开始要保存的缓冲区的大小 (也以字节为单位)。

 

语法:

buffer_save_ext(buffer, filename, offset, size);

参数类型 描述
bufferBuffer ID 要保存的缓冲区的索引。
filenameString 要另存为的文件的名称。
offsetReal 要从中进行保存的缓冲区内的偏移量(以字节为单位)。
sizeReal 要保存的缓冲区的大小 (以字节为单位)。

 

返回:

N/A

 

例子:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

将带有存储在变量 "buff" 中的id的缓冲区当前内容的一部分保存到文件中。
