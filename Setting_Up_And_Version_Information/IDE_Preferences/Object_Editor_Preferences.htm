对象编辑器首选项

对象编辑器首选项用于定义对象编辑器；窗口的某些属性。主要部分包含以下选项：

  • 点击打开事件方法：此选项控制是否需要在对象事件上执行双击或单击才能打开它。默认情况下，此选项设置为双击。
  • 继承事件 Alpha：此选项允许您更改在对象的事件列表中绘制任何继承事件时使用的 Alpha。默认情况下，该值设置为 0.5 (值应介于 0 和 1 之间)。
  • 事件标签格式：您可以在此启用或禁用可应用于对象编辑器事件列表中的事件标签的不同可能文本。默认情况下，仅启用事件描述。
  • 添加默认事件内容：默认情况下，此选项处于启用状态，它所做的只是为所有新事件在代码编辑器中添加一个简单的注释，以提醒您可以在那里添加代码。
  • 默认事件内容：您可以在这里编辑新事件的代码窗口中默认添加的备注。

对象编辑器还存在以下子项：

确认对话框确认对话框 

本节介绍使用对象编辑器时可能显示的对话框，可用选项如下：

  • GML 可视化转换为 GML 代码的自动响应 ：您可以在 GML 可视化工作区内或在事件上使用鼠标右键 ，选择将该事件的所有 GML 可视化转换为 GML 代码的选项。这里的默认操作是显示一条警告消息，询问您是否要继续，但您可以将其设置为"OK"，并且在转换之前不会显示该对话框。
  • 删除事件时自动选择：此选项可用于控制在删除对象编辑器事件列表中的事件时显示的弹出消息。这里的默认操作是显示一条警告消息，但您可以将其设置为"OK"，对话框不会显示。
  • 自动响应超出物理顶点限制：此选项可用于控制当您超过物理形状编辑器中允许的顶点数时显示的弹出消息。这里的默认操作是显示一条警告消息，但您可以将其设置为"OK"，对话框不会显示。
