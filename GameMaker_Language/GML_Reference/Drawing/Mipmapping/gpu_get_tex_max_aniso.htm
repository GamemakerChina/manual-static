gpu_get_tex_max_aniso

使用此函数，您可以在使用tf_anisotropic过滤器模式时获得当前各向异性的最大级别(有关详细信息，请参阅gpu_get_tex_mip_filter())。返回的值将介于1和16之间。

 

语法:

gpu_get_tex_max_aniso();

 

返回:

Real (default: 16)

 

例子:

if (gpu_get_tex_max_aniso() != 8)
{
    gpu_set_tex_max_aniso(8);
}

上述代码将检查当前最大各向异性滤波电平，如果不是8，则设置为8。
