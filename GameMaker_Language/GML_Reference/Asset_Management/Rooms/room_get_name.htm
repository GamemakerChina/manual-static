room_get_name

此函数以字符串形式返回给定房间的名称。

请注意，这 只是 一个字符串，不能用于直接引用房间 - 为此，您需要 房间索引 。不过，您可以使用返回的字符串以及函数 asset_get_index()，使用该字符串来获取 房间索引 。

 

语法:

room_get_name(index);

参数类型描述
indexRoom Asset要检查名称的房间。

 

返回:

String

 

例子:

var _roomname = room_get_name(room);

draw_text(32, 32, _roomname);

上面的代码获取当前房间的名称并将其绘制在屏幕上。
