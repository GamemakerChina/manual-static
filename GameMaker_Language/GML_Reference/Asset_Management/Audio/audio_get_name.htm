audio_get_name

此函数将以字符串形式返回给定音频资源的名称。

"index" 值可以是资源本身的值 (如 资源浏览器 中所示)，也可以是使用 audio_play_sound 播放声音时给出的唯一 ID 值。

注意 返回的字符串与资源 ID 不一致，不能用于访问资源本身，只能用于显示或错误检查。

 

语法:

audio_get_name(index);

参数 类型 描述
index Sound Asset 检查声音的索引。

 

返回:

String

 

例子:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

上述代码播放从三种不同声音资源中选择的随机声音，然后显示带有其名称的调试消息。
