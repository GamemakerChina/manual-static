sequence_instance

这是一个内置变量，它是为游戏中每个对象实例创建的实例变量的一部分。如果实例受序列控制，该变量将保存控制实例的序列的序列实例结构，否则将未定义。 这是只读变量，无法更改。

请注意，即使其Sequence Element仍然存在，该变量在控制序列结束后将变为undefined，如果该Sequence元素被重放，该变量将再次获得该结构。

 

语法:

sequence_instance

 

返回:

Sequence Instance Struct

 

例子:

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

上面的代码检查in_sequence变量，如果它为真(表示实例由序列控制)，则它将该序列的速度标度更改为2。
