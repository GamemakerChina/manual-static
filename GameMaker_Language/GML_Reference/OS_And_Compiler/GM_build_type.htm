GM_build_type

此字符串常量保存构建的类型：创建的可执行文件的"exe"或从IDE开始的测试运行的"运行"。

 

语法:

GM_build_type;

 

返回:

String

 

例子:

draw_text(32, 32, GM_build_type);

上面的代码将绘制当前运行的可执行文件的构建类型。
