device_is_keypad_open

这会检查设备中的键盘，如果有键盘，则返回true，否则返回false。请注意，此函数主要用于Android设备

对于那些拥有 Sony Xperia Play 的用户，在 GameMaker 内有一个 set 按钮 / 键映射设置，因此您可以使用 键盘常量 vk_up,vk_down,vk_left,vk_right 表示 joypad 键，而 三角形 为 ord("T"),Square 是 ord("S"), 圆圈 是 vk_altvk_backspace 而 back 按钮是简单的 vk_backspace,Cross 是 vk_space,Select 是 vk_return 和 Start 为 vk_rshift,L 触发器 为 ord("L") 和 R 触发器 为 ord("R")。

 

语法:

device_is_keypad_open

 

返回:

Boolean

 

例子:

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

上面的代码检查键盘，然后根据返回值更改全局变量。
