os_powersave_enable

使用此函数，您可以打开或关闭设备的省电函数。这很重要，因为某些游戏(例如那些使用倾斜函数的游戏)可能不会生成操作系统可以解释为用户输入的事件，因此关闭屏幕或退出游戏。通过将此函数设置为false，您可以禁用省电函数并确保屏幕(和游戏)始终正常运行。

注意：这仅限于iOS和Android目标。

 

语法:

os_powersave_enable(flag)

参数类型 描述
flagBoolean 设置为true或false以启用或禁用省电函数。

 

返回:

N/A

 

例子:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

上面的代码检查操作系统类型，如果它是Android或iOS然后省电函数被停用。
