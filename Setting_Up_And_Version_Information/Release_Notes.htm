GameMaker更新日志

All changes for recent GameMaker versions are listed on this website. It lists release notes and downloads for the Monthly, LTS and Beta versions separately.

More in-depth changes can be found on GitHub. The linked page lists all major releases for GameMaker.

The links above contain changes for GameMaker 2023.8 and above. Release notes and downloads for older versions are available here: IDE / Runtime

Navigating GitHub

The linked GitHub page has two categories of releases: Open (containing in-progress and future releases) and Closed (which are already released to public).

Click on "[N] Closed" to see the released versions:

单击某个版本可查看其中的所有问题，这些问题可以是 Open( 表示该版本正在处理中) 或 Closed( 表示已完成)。

如果一个版本处于测试阶段，则关闭的问题可能已经进入最新的测试版本，或者将在下一个测试更新中进行。

GitHub 问题列表总是最新的，所以在报告之前检查已知问题是一个很好的地方。
