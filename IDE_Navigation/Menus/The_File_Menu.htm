文件菜单

在文件菜单中，您可以选择更改、保存或导入项目以及设置 GameMaker IDE 首选项。可用选项包括：

  • New Project - This option creates a new project from scratch in GameMaker. It will take you to The Start Page to start a new project. Choose a location to save the new project here and click Let's Go! and you'll be taken to a new, blank, workspace to start working on it. Project files are saved with the *.yyp file extension.
  • Open Project - This option opens a file explorer for you to select a previously created *.yyp project file for loading. Please note that on import, if the project was created using a version lower than 2.3.0, then you will get the following warning message: If you choose to proceed, a file explorer will be opened and you can give the project an updated name and location to save it, so the original project will not be overwritten. Once the project has been converted to the new 2.3.0+ format, it will open in the IDE and you will get a message saying that the conversion has succeeded. You should proceed to check the compatibility report that has been generated by the conversion process and see if there is anything that you need to fix before trying to run or compile the project. Note that if you get repeated errors while trying to open a project made with a version anterior to 2.3.0, then you should contact YoYo Games (using the Report a GameMaker Bug option from The Help Menu), supplying the original *.yyp file that you are trying to convert.

    WARNING A warning message is shown if GameMaker detects that the selected project is stored in a cloud storage location. Saving projects to cloud storage may cause problems when saving and building projects. See the Permissions Guide for more information.

  • 导入项目 - 在这里您可以导入以前作为 *.yyz 存档导出的项目 (请参阅下面的 导出项目 )，或者-如果您有一个使用传统 GameMaker 制作的项目：Studio 1.4 -您可以选择此选项从该产品线导入 *.gmx 或 *.gmz 文件。与打开项目一样，如果 *.yyz 是用 2.3.0 更新之前的 GameMaker 版本创建的，那么它将需要转换，因为项目文件格式随该更新而更改。

注意 要将以前的 GameMaker 项目导入到 GameMaker 中，它必须是使用 1.4 版本创建的，即使这样，它也可能无法直接导入。GameMaker 和 GameMaker：Studio 1.4 之间有很多不同之处，因此您的旧项目可能需要一些调整才能工作。请参阅文章 移植 GMS 1.4 游戏到 GameMaker，请参阅 GameMaker 知识库以及手册中涵盖 Objective Functions 和 Compatibility(兼容性 ) 的部分。

重要提示包含字体资源的 *.yyz项目可能要求您通过删除/重新添加字体，或通过在字体编辑器中更改字体然后重新设置字体来"重新生成"字体。这是因为导出 *.yyz时会保留现有字体信息，并且在以后导入或在其他计算机上导入时，项目可能还没有安装所需的字体(或者可能有同名的不同字体)。只有当你看到导入后字体呈现的问题时，才应该这样做。

  • 最近使用的项目 - 您可以使用此选项查看最近使用的项目。选择此选项将重新打开 起始页 ( 如果您已关闭该页)，您可以从此处找到的列表中选择最近使用的项目。
  • 保存项目 - 这会将您的游戏保存到您在创建项目时定义的位置(或从中打开它的位置)。
  • Save Project As - Same as above, only you specify the location and the new name of the project.

    WARNING A warning message is shown if GameMaker detects that the path you try to save the project to refers to a cloud storage location. Saving projects to cloud storage may cause problems when saving and building projects. See the Permissions Guide for more information.

  • 导出项目-这允许您将项目导出为文件或模板：
    • YYZ -这将导出项目为单个 *.yyz文件，并且对于向用户发送副本或保存备份非常有用。
    • 作为模板-这将允许您将项目导出为模板，稍后可以将其用作起始页上其他项目的基础。选择此选项将显示以下提示：
      您应该首先单击编辑模板信息以确保模板信息正确，然后在第二次打开此提示时选择导出以将其保存为模板。然后您将在开始屏幕上的模板菜单下看到它。

      任何您创建的模板将与您登录时使用的帐户绑定。如果您稍后通过其他帐户登录，将无法访问相同的自定义模板。
  • 新 IDE-选择此选项后，将打开IDE的一个新实例供您使用(它将在起始页上打开)。
  • 首选项 - 选择此选项将进入首选项窗口，您可以在其中设置 GameMaker 的外观和体验。
  • 退出 - 退出 GameMaker 到桌面。
