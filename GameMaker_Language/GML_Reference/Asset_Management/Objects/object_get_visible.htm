object_get_visible

此函数将通知您正在检查的对象是否已标记为"可见"(运行其绘制事件)或未(不运行其绘制事件)。请注意这不是实例函数！因此，您可以拥有一个可见对象和同一对象的不可见实例，反之亦然。您可以使用可见实例变量设置单个实例可见性。

 

语法:

object_get_visible(obj);

参数类型 描述
objObject Asset 要检查的对象的索引。

 

返回:

Boolean

 

例子:

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

上面的代码将检查运行它的实例以查看它是否可见，以及检查实例的对象索引以查看它是否被标记为可见。如果实例不可见，但对象索引标记为ON，则会将该实例的" visible "设置为true.
