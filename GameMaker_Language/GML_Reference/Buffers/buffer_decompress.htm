buffer_decompress

使用此函数，您可以使用zlib压缩解压缩先前压缩的缓冲区。您提供了要解压缩的缓冲区ID，该函数将返回一个包含未压缩数据的新缓冲区ID。如果解压缩失败 (例如，您提供的缓冲区尚未压缩)，则该函数将返回小于0的值。

 

语法:

buffer_decompress(buffer);

参数类型 描述
bufferBuffer ID 要解压缩的缓冲区的索引。

 

返回:

Buffer ID

 

例子:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

上述代码将首先加载一个保存的缓冲区，然后对其进行解压缩，最后从解压缩的缓冲区中将字符串数据读取到全局变量中。
