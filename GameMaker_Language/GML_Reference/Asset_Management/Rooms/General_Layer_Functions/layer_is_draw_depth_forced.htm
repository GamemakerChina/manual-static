layer_is_draw_depth_forced

此函数可用于检查和查看是否启用强制 Z 深度来渲染项目中的图层。有关详细信息，请参阅layer_force_draw_depth()。

 

语法:

layer_is_draw_depth_forced()

 

返回:

Boolean

 

例子:

if (!layer_is_draw_depth_forced())
{
    layer_force_draw_depth(true, 0);
}

上面的代码检查层Z深度是否被强制，如果不是，它将Z深度设置为0并启用它。
