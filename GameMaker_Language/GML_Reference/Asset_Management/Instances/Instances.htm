Instances

实例 是根据 对象 创建的，创建后，可以通过 GML 代码 和 GML Visual 操作更改、操作和销毁它们。本页面列出了与 GameMaker 中可用的实例相关的主要功能。

创建实例

您可以通过几种方式创建对象的实例：

  • 将其拖动到 房间编辑器 中的实例层上。
  • 使用函数 instance_create_depth 和 instance_create_layer 在运行时创建它。
  • 将其添加到 序列编辑器 中的序列，并在 房间编辑器 中或通过代码添加序列。
  • 使用 room_instance_add 将其添加到非当前 房间 的房间。

停用实例

您还可以 停用 实例，这意味着它们不会处理任何事件，尽管它们仍存在于游戏中。您可以在 停用实例 页面上了解其工作原理以及可以使用的功能。
