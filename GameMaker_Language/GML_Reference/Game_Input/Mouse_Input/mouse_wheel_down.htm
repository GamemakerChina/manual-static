mouse_wheel_down

如果向下旋转鼠标滚轮，则此函数返回true，否则返回false。

 

语法:

mouse_wheel_down();

 

返回:

Boolean

 

例子:

if (mouse_wheel_down())
{
    y += 10;
}

如果鼠标滚轮向下旋转，则会将当前实例向下移动。
