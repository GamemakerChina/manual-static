method_get_index

使用此函数，您可以检索定义方法的脚本的脚本索引。如果该方法未在脚本中定义，则该函数将返回-1，否则它将返回脚本的索引值。

 

语法:

method_get_index(method);

参数 类型 描述
method Method 要检查的方法变量

 

返回:

Script Asset or Real

 

例子:

var _index = method_get_index(light_setup);
if _index != -1
{
    show_debug_message(script_get_name(_index));
}

上面的代码将检查方法 “light_setup” 的脚本索引，然后如果不是-1，则将脚本名称输出到控制台。
