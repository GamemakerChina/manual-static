ds_priority_clear

使用此函数，您可以清除给定优先级队列数据结构中的所有数据。这不会不破坏数据结构(对于应使用ds_priority_destroy()的数据结构)，它只会擦除其中的所有数据，并返回一个空的优先级队列。

 

语法:

ds_priority_clear(id);

参数类型 描述
idDS Priority ID 要清除的数据结构的ID。

 

返回:

N/A

 

例子:

if (count = 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

以上代码检查变量是否已达到特定值，以及是否清除变量"command_queue"中索引的DS优先级队列，设置报警，并将变量重置为0。
