draw_highscore

此简单函数将使用指定矩形内当前设置的字体、颜色和字母值绘制内部存储的高分的当前列表。提供的坐标分别为矩形区域绘制文本，GameMaker将处理其余部分，并自动进行间距和位置调整。

 

语法:

draw_highscore( x1, y1, x2, y2 );

参数类型描述
x1Real高分矩形左侧的x坐标。
y1Real高分矩形顶部的y坐标。
x2Real高分矩形右侧的x坐标。
y2Real高分矩形底部的y坐标。

 

返回:

N/A

 

例子:

draw_highscore(100, 100, room_width - 100, room_height - 100);

这将在具有 100px 边框的房间中间的矩形中绘制高分表。
