设置 Git 插件

在使用 Git 插件之前，我们需要设置要使用的 Git 可执行文件，方法是进入首选项的插件-源代码管理 (Git)部分：

如果 GameMaker 找到 Git 可执行文件路径，则应自动填充该路径，但如果该字段为空，则可以单击 按钮并手动选择 Git 安装的 cmd 文件夹中包含的 git.exe 文件，该文件通常位于 C:\ProgramFiles\Git\ 中。

有关 Git 选项的其余内容，请参阅插件首选项页面。
