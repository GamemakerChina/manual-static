Windows 首选项

Windows 首选项具有以下选项：

  • Visual Studio 位置 ：此选项用于将 GameMaker 指向生成 PC 上的 Microsoft Visual Studio 安装位置。
  • 默认打包类型：构建最终的 Windows 可执行程序包时，您可以选择创建安装程序(使用 NSIS) 或压缩的 ZIP 文件，该文件将包含游戏和其中的任何其他文件。默认情况下，会显示一个对话框让您在编译时在这两个选项之间进行选择，但您可以将此首选项设置为始终使用其中一个。请注意，当此选项设置为显示对话框时，在编译时弹出的实际对话框和窗口中，启用"记住打包选项"会将此首选项设置为您选择的选项。
