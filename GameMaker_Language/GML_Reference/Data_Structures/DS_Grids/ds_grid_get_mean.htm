ds_grid_get_mean

此函数可用于查找在网格的已定义区域内找到的所有单元格的平均值(将所有单元格值相加，然后除以构成该区域的单元格总数)，如下图所示：

 

语法:

ds_grid_get_mean(index, x1, y1, x2, y2);

参数类型 描述
indexDS Grid ID 网格的索引。
x1Real 区域的左侧单元格列。
y1Real 区域的顶部单元格行。
x2Real 区域的右侧单元格列。
y2Real 区域的底部单元格行。

 

返回:

Real or String

 

例子:

val = ds_grid_get_mean(grid, 0, 0, 5, 5)

上述代码将变量"值"设置为包含在变量"栅格"中索引的D栅格的给定区域内的平均值。
