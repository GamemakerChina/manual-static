display_set_ui_visibility

此函数可用于显示或隐藏 Android 和 iOS 上的系统用户界面。

  • Android：您可以提供一个或多个系统标志整数值。当使用多个标志时，需要使用位"或"合并这些标志，如下例所示。您可以在此处找到 Android 系统标志列表。
  • iOS：传入 0 可以隐藏状态栏，传入 1 可以显示状态栏。

 

语法:

display_set_ui_visibility(flags);

参数 类型 flags
flags Real 要使用的系统标志(整数值)。

 

返回:

N/A

 

例子:

var flags = 1024|4096;
display_set_ui_visibility(flags);

上述代码将使用 Android 系统标志 "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" 和 "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" 来设置显示器的可见性。
