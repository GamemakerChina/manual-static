shop_leave_rating

此函数会打开一个依赖于操作系统的对话框，您可以在其中要求用户对特定页面发表评级或评论。 您可以定义要出现在对话框中的文本，以及您希望出现在两个按钮上的文本和必须发布评论的 URL。

NOTE This function is only for Android and iOS targets.

重要 On iOS, when using TestFlight, in development the prompt will be displayed but you won't be able to actually do anything. Once your app is in the App Store, this button becomes active and the user is given the option to write a review or leave a rating. Note that if you distribute the game through TestFlight, this window may not even be shown.

 

语法:

shop_leave_rating(text, yes_string, no_string, url)

参数 类型 描述
text String 显示在对话框上的文本。
yes_string String 出现在“是”按钮上的文本。
no_string String 出现在“否”按钮上的文本。
url String 必须发送评论的完整 URL。

 

返回:

N/A

 

例子:

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

如果变量“timer”倒计时到 0，上面的代码将要求用户发表评论。
