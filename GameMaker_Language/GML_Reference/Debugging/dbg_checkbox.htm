dbg_checkbox

此函数在当前调试节中创建复选框控件。

复选框采用布尔变量的值。单击它可切换变量。

当前调试节是上次使用dbg_section创建的调试节。

注意 此调试控制跨越两列。

 

语法:

dbg_checkbox(ref[, label]);

参数 类型 描述
ref Reference 对使用ref_create创建的布尔变量的引用
label String 可选复选框旁边显示的标签

 

返回:

N/A

 

例子:

Create Event

toggle = false;

var _ref_to_toggle = ref_create(self, "toggle");
dbg_checkbox(_ref_to_toggle, "The Toggle Switch");

上面的代码在Create事件中设置了一个布尔变量toggle。它使用ref_create创建对变量的引用，将其存储在局部变量_ref_to_toggle中，并将其传递给dbg_checkbox。由于未调用dbg_view或dbg_section，因此将创建一个新视图"Default"和一个新节"Default"，并将复选框控件添加到其中。复选框旁边将显示文本标签"切换开关"。
