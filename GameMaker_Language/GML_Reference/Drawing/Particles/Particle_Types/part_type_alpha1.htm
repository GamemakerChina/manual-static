part_type_alpha1

该函数用于将粒子类型设置为在每个单个粒子的生命周期的总持续时间内具有单个Alpha值(透明度)，并且该值可以是从0(透明)到1(不透明)。

 

语法:

part_type_alpha1(ind, alpha1);

参数类型描述
indParticle Type ID要更改的粒子类型的索引。
alpha1Real粒子的alpha。

 

返回:

N/A

 

例子:

part_type_alpha1(global.Snow_Part, 0.5);

上述代码将在全局变量"Snow_Part"中为粒子类型创建索引的所有粒子设定为Alpha值0.5(半透明)。
