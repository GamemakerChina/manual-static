path_add

使用此函数，您可以在GameMaker中创建路径，而无需使用路径编辑器。此函数将返回路径的索引，该索引应存储在变量中，并从那时起用作该路径的引用。请注意，创建的路径是空的IE：它没有定义点，因此您必须使用其他可用函数来向路径添加点，或者使用MP网格来生成路径。一旦你完成了对路径的使用，或者希望创建一个新的路径并将其索引存储在相同的变量中，你应该首先用path_delete删除旧的路径，以防止内存泄漏，这最终会使你的游戏崩溃。

 

语法:

path_add();

 

返回:

Path Asset

 

例子:

global.newpath = path_add();

这将创建一个新路径，并将其索引分配给global.newPath.
