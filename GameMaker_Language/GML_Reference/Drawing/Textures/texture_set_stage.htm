texture_set_stage

此函数将为给定的舞台"时隙"设置要使用的纹理。可用的阶段"插槽"的数量将取决于您正在编译的平台，Windows、Mac和Linux最多可用8个，但在低端Android设备(例如)上，此数量可以低至2个。另外值得注意的是，GameMaker始终自动使用第一级插槽(1 )。

注意:此函数在运行着色器的上下文之外不会执行任何操作！有关详细信息，请参见着色器。

 

语法:

texture_set_stage(stage, tex);

参数 类型 描述
stage Real 要使用的纹理"槽"。
tex Texture 要使用的纹理。

 

返回:

N/A

 

例子:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

以上代码将获取着色器索引为"GlassShader"的采样器的句柄，然后将该着色器设置为给定精灵纹理的常量。
