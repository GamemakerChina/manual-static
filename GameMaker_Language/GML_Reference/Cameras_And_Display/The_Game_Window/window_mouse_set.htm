window_mouse_set

使用此函数，您可以更改或设置鼠标在游戏窗口中的位置，例如，这对于 FPS 游戏非常有用。该函数仅在游戏对焦时起作用，使用 Alt + Tab将解锁鼠标。

注意 此函数仅在桌面平台(即 Windows、Mac 和 Ubuntu)上可用。

注意 有关常规鼠标函数，请参阅鼠标输入一节。

 

语法:

window_mouse_set(x, y);

参数 类型 描述
x Real 窗口中的 X 坐标。
y Real 窗口中的 Y 坐标。

 

返回:

N/A

 

例子:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

上述代码将使鼠标在游戏窗口中居中。
