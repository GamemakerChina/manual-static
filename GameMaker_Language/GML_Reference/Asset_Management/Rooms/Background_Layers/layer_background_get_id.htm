layer_background_get_id

此函数可用于检索图层上背景元素的唯一 ID 值。提供层 ID(与layer_get_id()一起使用层名时获得)，函数将返回与层上背景元素关联的 ID 值。请注意，此函数专门设计用于在 IDE 中添加的背景，就好像您使用函数layer_background_create()通过代码将背景添加到层，那么它将返回所添加的背景元素的唯一ID。

 

语法:

layer_background_get_id(layer_id)

参数 类型 描述
layer_id String or Layer ID 目标层的唯一ID值

 

返回:

Background Element ID

 

例子:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

上面的代码将获取名为"Background_trees"的层的层ID，然后使用该层来获取该层上的背景元素的ID。然后，此 ID 用于更改元素 sprite。
