texturegroup_set_mode

此函数允许您设置三件事：

  1. 模式：您可以设置动态纹理组是使用隐式(默认值)还是显式加载，方法是：对隐式使用false，对显式使用true。

    有关模式的信息，请参阅：动态组模式
  2. 调试：您可以启用调试，这将显示游戏中的所有纹理页面以及游戏中的状态。
  3. 默认Sprite：当模式设置为"显式"，并且要绘制的纹理尚未加载时，将显示默认sprite。

    有关详细信息，请参阅：显式

 

 

语法:

texturegroup_set_mode(explicit, [debug=false, default_sprite=-1]);

参数 类型 描述
explicit Boolean 用于动态纹理组的模式：隐式(false)或显式(true)
debug Boolean 可选启用/禁用纹理组调试
default_sprite Sprite Asset 可选仅在显式模式下使用：未加载另一个纹理时用作"回退"纹理的精灵(绘制整个纹理页面)

 

返回:

N/A

 

例子:

texturegroup_set_mode(true, false, spr_fallback);

这将启用"显式"模式，禁用调试，并将spr_fallback设置为卸载纹理的回退子程序。
