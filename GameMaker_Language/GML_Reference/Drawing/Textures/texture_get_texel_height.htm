texture_get_texel_height

纹理或纹理元素是计算机图形中使用的纹理空间的基本单位。纹理由文本数组表示，就像图片由像素数组表示一样，此函数从所使用的图像资源的纹理页面返回单个文本的高度。

 

语法:

texture_get_texel_height(tex);

参数 类型 描述
tex Texture 要使用的纹理页面资源指针

 

返回:

Real

 

例子:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

以上代码将获取从精灵资源获取的纹理的纹理宽度和高度。
