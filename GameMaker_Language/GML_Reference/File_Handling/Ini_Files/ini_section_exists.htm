ini_section_exists

此函数检查当前打开的 ini 中是否存在节，如果存在，则返回 true ，否则返回 false 。这不是防止错误的必要检查，因为当一个节不存在时，从一个不存在的节读取只会返回一个默认值，但是查看 ini 文件是否保存了特定的数据会很有用。

 

语法:

ini_section_exists(section);

参数类型 描述
sectionString 要检查的 open.ini 文件中的节。

 

返回:

Boolean

 

例子:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

如果未找到“save1”，这将把变量global.savegame设置为“1”。使用for循环，这可以用作计算一个游戏有多少个保存游戏的基本方法（即：为一个游戏创建用于多个同时保存游戏的系统）。
