is_infinity

此函数返回给定变量是否为infinity(一个无限数)，如果是，则返回true，如果不是，则返回false。

 

语法:

is_infinity(n);

参数 类型 描述
n Any 要检查的参数。

 

返回:

Boolean

 

示例:

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

上面的代码检查全局变量 “value” 以查看它是否为无限，如果是，则显示调试消息。
