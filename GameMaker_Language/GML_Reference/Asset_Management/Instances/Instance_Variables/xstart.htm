xstart

此变量存储实例在房间中首次创建时的初始 x 位置。这不是只读变量，可以设置也可以读取。

 

语法:

xstart;

 

返回:

Real (single precision floating point value)

 

例子:

if (place_meeting(x, y, obj_spike))
{
    score -= 100;
    x = xstart;
    y = ystart;
}

上述代码将检查与"obj_spike"实例的冲突，如果存在冲突，则从分数中扣除100，并将该实例移回其起始位置。
