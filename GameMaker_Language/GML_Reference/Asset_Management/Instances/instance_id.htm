instance_id

此只读数组保存房间内每个活动实例的所有ids。这意味着，如果您使用了任何实例取消激活函数，则那些已取消激活的实例将不会包括在此数组中(如果您以前使用过此数组中的值，则现在将返回关键字noone)。

 

语法:

instance_id[num];

 

返回:

Instance ID

 

例子:

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

上面的代码将遍历房间内的所有实例，并将它们的速度增加0.1。
