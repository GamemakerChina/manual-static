temp_directory

这可用于在每次运行游戏时返回为您的游戏创建的临时目录 (根目录不包含最后的“\”)。 此目录将保存文件并可在游戏运行时访问，但在游戏关闭时它将被删除 (连同它包含的所有文件)。

警告！ 由于 GameMaker 被沙盒化，此函数可能无法按您预期的那样工作！ 有关详细信息，请参阅 文件系统 部分。

 

语法:

temp_directory

 

返回:

String

 

例子:

ini_open(temp_directory + "\temp_ini.ini");

这将在游戏的临时目录中打开一个 ini 文件 (如果它不存在则创建它)。
