vertex_ubyte4

此函数将四个无符号字节值 (0 - 255) 添加到顶点数据。

顶点缓冲区必须已正确格式化才能使用 vertex_format_add_custom 函数接受此设置。

 

语法:

vertex_ubyte4(buffer, byte, byte, byte, byte);

参数类型描述
bufferVertex Buffer要写入信息的顶点缓冲区。
byteReal第一个输入值。
byteReal第二个输入值。
byteReal第三个输入值。
byteReal第四个输入值。

 

返回:

N/A

 

例子:

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

上面的代码将四个值添加到正在定义的顶点数据中。
