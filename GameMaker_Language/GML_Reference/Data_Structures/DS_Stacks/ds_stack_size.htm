ds_stack_size

此函数将返回堆栈的"大小"，即已推入堆栈的项目数。

 

语法:

ds_stack_size(id);

参数类型描述
idDS Stack要检查的数据结构的ID。

 

返回:

Real

 

例子:

if (!ds_stack_empty(control_stack))
{
    num = ds_stack_size(control_stack);
}

上面的代码检查DS堆栈以查看它是否为空，如果不为空，则获取它包含的项目数，并将该值存储在变量中。
