draw_flush

使用此功能，您可以刷新整个绘制管道。这是一个仅限调试的函数，通常不应使用，除非得到 YoYo Games 支持人员的指示，因为不加选择地使用会导致严重的游戏性能问题。

 

语法:

draw_flush();

 

 

返回:

N/A

 

例子:

draw_flush();

以上代码将刷新绘制管线。
