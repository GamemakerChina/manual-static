is_array

此函数可用于检查变量是否包含数组（将返回 true）或不包含数组（在这种情况下，将返回 false）。

 

语法:

is_array(n);

参数 类型 描述
n Any 要检查的变量。

 

返回:

Boolean

 

例子:

if (is_array(a))
{
    a = -1;
}

上面的代码检查一个变量以查看它是否为数组，并且如果函数返回true, 则通过将变量设置为-1来删除该数组。
