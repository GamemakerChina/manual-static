path_set_kind

此函数可用于设置您希望指定资源的路径类型。这可以是直线路径(设置为0)或平滑路径(设置为1)，在这种情况下，也必须考虑路径精度(精度也可以使用path_set_precision())设置。 此函数改变实际路径资源因此将永久影响路径如何被使用的所有实例在游戏中从一刻函数一直使用到游戏结束。如果这不是您需要的，那么您应该使用像path_duplicate()这样的函数来创建路径的副本，然后调用该函数在重复资源上(当不再需要时，别忘了在资源上调用path_delete())。

 

语法:

path_set_kind(index, val);

参数类型 描述
indexPath Asset 要更改的路径的索引。
valReal 路径类型，0表示直线，1表示平滑。

 

返回:

N/A

 

例子:

path_set_kind(pth_Patrol, true);

这将将给定路径设置为"平滑"路径。
