gpu_get_colourwriteenable

此函数可用于检索当前彩色写入启用值。函数返回一个 4 元素 1D 数组，其中包含以下元素：true (已启用) 或 false (已禁用)。默认情况下，所有彩色写入均设置为 true：

  • [0] = 红色通道启用 / 禁用
  • [1] = 绿色通道启用 / 禁用
  • [2] = 蓝色通道启用 / 禁用
  • [3] = Alpha 通道启用 / 禁用

 

语法:

gpu_get_colorwriteenable();

 

返回:

Array (4 elements only; see above for details)

 

例子:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

以上代码获取当前颜色写入值，然后将Alpha组件设置为false。
