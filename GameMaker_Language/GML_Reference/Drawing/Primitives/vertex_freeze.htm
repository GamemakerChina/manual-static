vertex_freeze

此函数可用于"冻结"顶点缓冲区。此缓冲区变为只读，这意味着如果需要更改它，您必须删除整个缓冲区并重新创建它。冻结的缓冲区可以比普通的动态缓冲区更快地提交到着色器，建议用于那些在级别或游戏期间需要不变的自定义基本体的效果。

函数在成功冻结指定的顶点缓冲区时将返回0，但如果由于任何原因失败，将返回-1。

 

语法:

vertex_freeze(buffer);

参数 类型 描述
buffer Vertex Buffer ID 要冻结的顶点缓冲区。

 

返回:

Real (0 for success, -1 for failure)

 

例子:

// Create event
vertex_freeze(vbuff);

上述代码将在创建事件中冻结给定的顶点缓冲区，因此可以在其他事件中用作"静态"缓冲区，以便更快地访问。
