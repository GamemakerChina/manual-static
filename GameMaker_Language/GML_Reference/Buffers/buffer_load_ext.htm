buffer_load_ext

此函数将先前使用buffer_save()函数保存的缓冲区数据加载到现有缓冲区中。给定要加载的先前创建缓冲区的 ID，然后加载保存的缓冲区文件，最后给出要加载数据的缓冲区开始的偏移量(以字节为单位)。

请阅读buffer_load()页面了解平台特定注释。

 

语法:

buffer_load_ext(buffer, filename, offset);

参数类型 描述
bufferBuffer ID 要加载到的缓冲区的索引。
filenameString 要从中加载的文件名。
offsetReal 缓冲区内要加载到的偏移(以字节为单位)。

 

返回:

N/A

 

例子:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

上面的代码将首先获取在变量"player_buffer"中索引的缓冲区末端的位置，然后将数据从给定加载到该位置(请注意，本示例只适用于"增长"或"包装"缓冲区类型)。
