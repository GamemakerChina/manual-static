buffer_resize

使用此函数，您可以将给定缓冲区的大小调整为指定的大小(以字节为单位)。

 

语法:

buffer_resize(buffer, newsize);

参数类型 描述
bufferBuffer ID 要更改大小的缓冲区的索引。
newsizeReal 缓冲区的新大小(以字节为单位)。

 

返回:

N/A

 

例子:

if (buffer_get_size(buff) < 16384)
{
    buffer_resize(buff, 16384);
}

上面的代码将检查变量 "buff" 中索引的缓冲区的大小，如果它小于给定值，则调整缓冲区的大小。
