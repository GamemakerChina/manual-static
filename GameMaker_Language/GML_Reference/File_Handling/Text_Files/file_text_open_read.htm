file_text_open_read

这个函数打开指定文件名为只读的文本文件，返回文件的唯一 id，该 id 应该存储在一个变量中，因为它将用于处理该文件的所有进一步操作。如果文件不存在，函数将返回值 -1。

说明 : 同一时间最多只能打开 32 个文件。你还应该 总是 在完成时关闭文件，因为这会写入信息和释放与文件相关的内存。

警告 ! 由于 GameMaker 被沙盒化，这个函数可能不会像你期望的那样工作 ! 有关更多信息，请参见  文件系统  一节。

 

语法:

file_text_open_read(fname);

参数类型 描述
fnameString 要从中读取的文件的名称。

 

返回:

Text File ID or -1

 

例子:

file = file_text_open_read(working_directory + "level.txt");

这将从与游戏相同的目录中打开“level.txt”，并将文件 ID 存储在变量“ file”中。
