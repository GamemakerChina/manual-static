window_enable_borderless_fullscreen

此函数允许 在调用 window_set_fullscreen 时使用全屏无边框窗口，或者使用 Alt + Enter 组合键切换到全屏。

当游戏在全屏无边框窗口中运行时，它仍然在窗口中运行，但 GameMaker 会调整窗口大小，使其覆盖整个屏幕。启用无边框全屏后，window_get_fullscreen 也会返回 true。

重要信息 您应该在使用 window_set_fullscreen 切换到全屏 之前 调用此函数，因为当已经在全屏模式下运行时，此函数不会更改任何内容。

注意 此函数仅适用于 Windows。

 

语法:

window_enable_borderless_fullscreen(enable);

参数类型描述
enableBoolean是否在全屏模式下启用无边框窗口 (true) 或不启用无边框窗口 (false)

 

返回:

N/A

 

例子:

window_enable_borderless_fullscreen(true);
window_set_fullscreen(true);

上面的代码首先通过调用 window_enable_borderless_fullscreen 来启用无边框全屏窗口。然后使用 window_set_fullscreen 切换到全屏。
