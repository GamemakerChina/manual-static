application_surface_enable

使用此函数，您可以启用或禁用应用程序表面。默认情况下，应用程序表面处于启用状态，"绘制"事件的所有绘制都将完成，并且表面将绘制到屏幕。但是，在某些较旧的设备和特定的芯片组上，这可能会导致性能较差。在这些情况下，可以关闭对应用程序表面的绘制，然后所有绘制事件绘制都将直接对屏幕缓冲区执行。

您可以在游戏中的任何时候使用此函数启用或禁用应用程序表面，但应注意，要使更改生效，至少需要一个帧(步骤)。关闭将关闭目标平台的游戏选项中设置的所有纵横比校正。这意味着您应该使用某种方法将游戏扩展到所需的设备、窗口或屏幕尺寸。如果再次打开，游戏选项将在下一步再次生效。

建议您仔细使用此函数，因为它将影响所有内容绘制到屏幕的方式。一般情况下，您会希望在游戏菜单中将其作为一个选项，或者在游戏开始时只调用一次，并围绕是否启用或禁用编写代码。

注意 禁用应用程序图面将禁用所有 滤镜和效果。

 

语法:

application_surface_enable(enable);

参数 类型 描述
enable Boolean 设置为启用(true)或禁用(false)应用程序表面。

 

返回:

N/A

 

例子:

if keyboard_check_pressed(vk_space)
{
    if application_surface_is_enabled()
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

上述代码检查是否按键，以及是否根据应用程序表面的状态(如在选项菜单中)打开或关闭应用程序表面。
