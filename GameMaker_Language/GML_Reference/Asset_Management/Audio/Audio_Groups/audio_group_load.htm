audio_group_load

此函数会将标记为属于给定音频组的所有声音载入存储器。

如果启动加载，函数将返回 true，如果组ID无效或已标记加载，则函数将返回 false。

注意 该函数是异步的，因此在后台加载音频时，您的游戏将继续运行。这意味着它还会触发异步加载/保存事件，通知您整个组已加载到内存中，现在可以使用声音。

 

语法:

audio_group_load(groupID);

参数类型描述
groupIDAudio Group ID要加载的音频组的索引(在音频组窗口中定义)

 

返回:

Boolean

 

例子:

if (!audio_group_is_loaded(audiogroup_level1))
{
    audio_group_load(audiogroup_level1);
}

上面的代码检查是否已加载音频组，如果未加载，则进行加载。
