view_set_hport

此函数可用于设置给定视口的高度。提供视口索引(从0到7)和要使用的新高度。

 

语法:

view_set_hport(view_port, h)

参数类型 描述
view_portReal 到目标的视口 (0 - 7)
hReal 新的高度(以像素为单位)

 

返回:

Real

 

例子:

if view_get_hport(0) != (display_get_height () / 2)
{
    view_set_hport(0, display_get_height() / 2);
}

上述代码将检查 viewport[0] 的高度，如果它不是显示高度的一半，则设置为该值。
