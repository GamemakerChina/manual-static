is_debug_overlay_open

此函数返回调试覆盖当前是否打开。

 

语法:

is_debug_overlay_open();

 

返回:

Boolean

 

例子:

var _open = is_debug_overlay_open();
show_debug_overlay(!_open);

上面的代码首先调用is_debug_overlay_open并将结果赋给临时变量_open。然后，它通过将取反值传递给show_debug_overlay来切换调试覆盖。
