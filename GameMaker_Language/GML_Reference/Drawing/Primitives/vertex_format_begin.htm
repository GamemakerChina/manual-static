vertex_format_begin

该函数开始定义新的顶点格式。

在定义新的顶点格式之前，您必须告诉 GameMaker 您正在使用此函数执行此操作。您必须首先调用此函数，然后使用适当的函数定义格式值，最后调用 vertex_format_end 完成定义并返回新的顶点格式。

 

语法:

vertex_format_begin();

 

返回:

N/A

 

例子:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

上面的代码创建了一个仅包含颜色和位置值的新顶点格式，然后将该格式存储在变量 my_format 中。
