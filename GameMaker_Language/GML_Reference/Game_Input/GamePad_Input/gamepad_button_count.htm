gamepad_button_count

此函数将返回连接到给定设备“插槽”的游戏板可用的按钮总数。

 

语法:

gamepad_button_count(device);

参数类型描述
deviceReal要检查哪个游戏设备“插槽”。

 

返回:

Real

 

例子:

b_num = gamepad_button_count(0);

上面的代码将返回插入设备 “插槽” 0的游戏手柄上可用的按钮数量，并将该值存储在变量 “b_num” 中。
