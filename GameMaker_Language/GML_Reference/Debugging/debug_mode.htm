debug_mode 已删除

此只读变量在调试模式下玩游戏时返回true，在正常玩游戏时返回false。

 

语法:

debug_mode

 

返回:

Boolean

 

例子:

if (debug_mode)
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

上面的代码根据只读变量debug_mode的值打开不同的ini文件。
