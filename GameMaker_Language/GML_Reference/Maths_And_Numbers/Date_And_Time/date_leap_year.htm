date_leap_year

如果给定日期时间值的年份部分是闰年，此函数将返回 true，否则返回 false。 对于游戏中的复活节彩蛋或解锁特殊内容，这可能是一个方便的函数。

 

语法:

date_leap_year(date);

参数类型描述
dateDatetime要使用的日期时间。

 

返回:

Boolean

 

例子:

if (date_leap_year(date_current_datetime()))
{
    if (!global.ExtraContent)
    {
        global.ExtraContent = true;
    }
}

上面的代码将检查当前的日期时间，看看这一年是否是闰年。如果是，则设置全局变量。
