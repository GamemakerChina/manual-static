timeline_size

使用此函数，您可以获得时间轴的活动时刻总数 ("活动" 时刻是添加了代码或GML视觉函数的时刻)。使用timeline_moment_add_script()和timeline_moment_clear()函数创建动态时间线时，这很方便，因为您可以检查给定的时间线是否具有正确的活动时刻数或根本没有活动时刻数。

 

语法:

timeline_size(ind);

参数类型 描述
indTimeline Asset 时间线的索引从获取信息。

 

返回:

N/A

 

例子:

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

上面的代码检查给定的时间轴大小，如果它返回 0 (即：时间轴没有活动时刻)，它将添加一个脚本函数，在全局变量 "tl" 中索引的时间轴内的随机时刻使用。
