part_type_colour_hsv

此函数为给定类型的所有粒子设置色调、饱和度和值范围。

您为三个组件中的每一个组件提供最小值和最大值，创建的粒子将具有基于给定参数范围的随机颜色。通过这种方式，您可以创建具有相同色调但不同饱和度的粒子，或者具有不同色调但具有相同值(亮度)的粒子，等等。所有值必须介于0和255之间(含0和255)。

 

语法:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
hmin Real 最终颜色的色调分量可以是最小值。
hmax Real 最终颜色的色调分量的最大值。
smin Real 最终颜色的饱和度分量可以是最小值。
smax Real 最终颜色的饱和度分量可以是最大值。
vmin Real 最终颜色的值分量可以是最小值。
vmax Real 最终颜色的值分量可以达到的最大值。

 

返回:

N/A

 

例子:

pt_stars = part_type_create();
part_type_shape(pt_stars, pt_shape_star);
part_type_colour_hsv(pt_stars, 0, 255, 0, 255, 255, 255);

上面的代码创建了一个新的粒子类型，将其索引存储在变量pt_stars中，并为其提供星形pt_shape_star。然后，它将此粒子类型发射的每个粒子设置为具有不同的颜色和饱和度，但具有相同的值(亮度)。
