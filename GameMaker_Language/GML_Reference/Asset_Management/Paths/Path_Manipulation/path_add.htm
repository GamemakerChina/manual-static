path_add

通过此功能，您可以在 GameMaker 中创建路径，而无需使用路径编辑器。

该函数将返回对路径的引用，该引用应存储在变量中并在其他路径函数中使用。

请注意，创建的路径是 空 ，即：它没有定义点，因此您必须使用 其他可用函数 向路径添加点或使用 MP 网格 生成路径。它将被创建为 " 封闭 " 路径，您可以 更改 。

一旦您使用完路径，或者希望创建一个新路径并将其引用存储在同一变量中，您应该首先使用 path_delete 删除旧路径，以防止内存泄漏，从而最终导致您的系统崩溃。游戏。

 

语法:

path_add();

 

返回:

Path Asset

 

例子:

global.newpath = path_add();

这将创建一个新路径，并将其索引分配给global.newPath.
