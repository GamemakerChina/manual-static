图块集编辑器设置

图块集编辑器首选项具有以下自动图块选项：

自动图块背景颜色: 该选项在创建自动图块部分的图块集编辑器中设置背景颜色。默认颜色是 (RGBA) #808080FF。

图块设置编辑器首选项也有一个确认对话框子项:

这里可用的选项控制当你改变图块集的单元格大小时弹出的消息，这将影响房间图块图层的呈现方式(导致裁剪图层)。默认情况下，您将被提示继续或取消操作，该选项允许您通过选择 "OK" 来阻止此消息，该选项将继续进行图块单元格更改，而不管对房间图块图层的影响。默认值为"显示消息"。
