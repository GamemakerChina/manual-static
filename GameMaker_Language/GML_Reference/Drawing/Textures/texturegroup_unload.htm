texturegroup_unload

此函数用于将已加载的 动态纹理组 以原始压缩形式从内存卸载回磁盘。

不能卸载默认(非动态)纹理组。

有关其使用的信息，请参阅：动态纹理

 

语法:

texturegroup_unload(groupname);

参数 类型 描述
groupname String 作为字符串的纹理组的名称。这些在"纹理组"窗口中定义。

 

返回:

N/A

 

例子:

texturegroup_unload("tg_UI");

这将卸载名为"tg_UI"的动态纹理组。
