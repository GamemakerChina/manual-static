collision_point

collision_point检查参数x 1、y 1指定的点是否与参数"obj"指定的对象的any实例发生碰撞。此检查可以是精确的或不是精确的，但是要启用精确的碰撞，您要检查的对象或实例还必须为其精确碰撞启用。如果不是，则默认检查基于边界框。下图说明了此操作的工作原理：

请记住，要同时考虑精确碰撞对象精灵和碰撞函数，必须精确标记为"on"。还应当注意，函数的返回值可以是被认为处于碰撞中的任何一个实例的ID，因此，如果三个实例在该点上重叠，则它们的ID中的任何一个都可以是函数的返回值。

请注意，您可以提供实例id来检查特定实例，或者提供实例关键字all或其他(取决于事件和当前代码范围)，而不是对象索引。

 

语法:

collision_point(x, y, obj, prec, notme);

参数 类型 描述
x Real 要检查的点的x坐标。
y Real 要检查的点的y坐标。
obj Object Asset or Instance ID 要检查的对象或实例。
prec Boolean 检查是基于精确碰撞(true，速度较慢)还是基于其边界框(false，速度较快)。
notme Boolean 是否应排除调用实例(如果相关)(true)或(false)。

 

返回:

Instance ID or noone

 

例子:

if collision_point(x, y, obj_Cursor, false, true)
{
    score += 10S;
}

在这里，我们检查对象位置处的点，该点具有对象"obj_Cursor"的代码。如果有一个，那么我们在分数变量上加10。
