sign

此函数返回一个数是正数、负数还是两者都不是，分别返回 1、-1、0。

例如 sign(458) 将返回 1，sign(-5) 将返回 -1，sign(0) 将返回 0。

用法说明

  • sign(NaN) 将始终返回 -1.

 

语法:

sign(n);

 

参数 类型 描述
n Real 要获取其符号的数字。

 

返回:

Real

 

例子:

y += sign(y - mouse_y);

上述代码将根据 y - mouse_y 的结果将 1、-1 或 0 添加到 y 上。
