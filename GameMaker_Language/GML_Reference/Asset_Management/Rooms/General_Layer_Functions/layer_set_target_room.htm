layer_set_target_room

调用此函数时，您告诉GameMaker，所有其他层函数都应用于给定房间。 这样，您可以在非当前房间的房间中以程序方式更改或生成图层和图层内容。完成向房间添加图层或图层元素后，调用函数layer_reset_target_room()重置房间目标(或以房间参数 - 1)再次调用此函数。请注意，此函数只能在个房间(当前房间除外)上使用，并且旨在将图层和图层元素添加到当前运行的房间以外的房间。

警告！瞄准其他房间时，可以使用所有常规层函数，但您无法使用instance_create_layer()或instance_create_depth()创建实例，层函数layer_add_instance(){}也不可用。

 

语法:

layer_set_target_room(room)

参数类型 描述
roomRoom Asset 所有其他层函数的目标空间

 

返回:

N/A

 

例子:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    var _x = irandom(1000);
    var _y = irandom(1000);
    layer_sprite_create(l, _x, _y, spr_Trees);
}
layer_reset_target_room();

上述代码将目标房间设置为"rm_Game"房间，然后获取该房间中名为"SpriteAssets"的层的层ID。然后，在将层目标重置为当前房间之前，使用该层ID向该层添加50个随机精灵资源。
