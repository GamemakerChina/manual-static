audio_emitter_get_y

此函数返回给定音频发射器的当前 y 位置。

 

语法:

audio_emitter_get_y(emitter);

参数 类型 描述
emitter Audio Emitter ID 要使用的发射器的索引。

 

返回:

Real

 

例子:

if audio_emitter_get_y(emitter_player) != x
{
    audio_emitter_position(emitter_player, x, y, 0);
}

上述代码检查给定发射器的当前y位置，如果它不等于实例y位置，则将其设置为实例位置。
