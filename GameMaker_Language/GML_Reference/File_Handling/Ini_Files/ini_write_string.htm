ini_write_string

可以使用此函数将字符串 (文本) 写入 ini 数据文件。Ini 文件被分割成 节 (sections)，然后每个节又被分割成 键 (key)- 值 (value) 对。所以一个典型的 ini 文件看起来是这样的:

值得注意的是，在编写 ini 字符串时使用引号和转义字符可能会在以后回读数据时引起问题，因为它可能会过早地截断。例如，这样写会是一个问题:

ini_write_string("test2", "section", "hello \"Fritz\"");

 

 

语法:

ini_write_string(section, key, value);

参数 类型 描述
section String 要写入的 .ini 节。
key String 要写入的 .ini 的相关部分中的键。
value String 要写入相关目标的字符串。

 

返回:

N/A

 

例子:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

上面的代码打开一个 ini 文件进行读写，然后将存储在全局变量 "Name" 中的字符串写入 "Save" 部分，键为 "Player"，然后再次关闭该文件。
