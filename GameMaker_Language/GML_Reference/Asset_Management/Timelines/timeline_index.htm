timeline_index

此变量保存当前与实例关联的时间轴的索引。您可以将其设置为特定的时间线以使用该时间线，或者将其设置为-1以停止使用实例的时间线 (如果未为实例定义时间线，则也将返回-1)。请注意，这不会开始使用变量timeline_running的时间线。

 

语法:

timeline_index;

 

返回:

Timeline Asset

 

例子:

if (timeline_exists(global.tl))
{
    timeline_index = global.tl;
}

上面的代码将为运行该代码的实例分配一个在变量" global.tl "中索引的时间线(如果该时间线存在)。
