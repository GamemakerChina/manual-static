max

该函数返回输入值的最大值，最多可以有16个输入值。例如，max(12,96,32,75)将返回96，因为这是所有输入值中最高的。

 

语法:

max(val1, val2, ... max_val);

参数类型 描述
val0 ... max_valReal 要比较的值。

 

返回:

Real

 

例子:

x = max(x, 0);

这将阻止玩家从房间的左侧离开。它的工作原理是不断地将x设置为自身或0，取较大的值。如果玩家从左侧退出，它的x将小于0(即负)，所以它将直接返回。
