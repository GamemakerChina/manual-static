fx_get_parameters

此函数用于检索包含给定 FX 结构的所有参数的结构。指定 FX 结构(从fx_create()或layer_get_fx()返回)，此函数返回一个包含适用于该过滤器/效果的所有参数的变量的结构。使用向量(即值组)的参数将以数组的形式存在。

您可以通过调用fx_get_parameter_names()来获取单独数组中所有参数的名称，或在此页中找到它们：外汇类型 参数。

改变该结构中的值将不适用于过滤器/效果; 您需要调用fx_set_parameters()将修改的结构应用回过滤器/效果，以使任何更改生效。

 

语法:

fx_get_parameters(filter_or_effect);

参数 类型 描述
filter_or_effect FX Struct 将返回参数的FX结构

 

返回:

Struct (or -1 on failure)

 

例子:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

上面的代码获取假定对其应用了"Color Tint"过滤器的层的FX结构，并通过调用fx_get_parameters()来检索其参数结构。 然后使用sin()和current_time生成一个振荡值，作为色调效果的RGBA值，保持动画。RGBA 数组分配给参数结构中的g_TintCol变量，然后调用fx_set_parameters()将该结构应用回FX结构。
