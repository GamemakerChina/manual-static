keyboard_virtual_hide

此函数可用于隐藏运行游戏的设备上的虚拟键盘。调用此函数将生成系统异步事件，其中async_load DS 映射将填充以下键/值对：

  • "event_type"-当由虚拟键盘函数触发时，这将是 "虚拟键盘状态"。
  • "screen_height"-虚拟键盘的高度(以像素为单位)。如果键盘不可见，则该值将为 0。
  • "keyboard_status" - 键盘的当前状态，作为以下字符串之一返回：
    • "hiding"
    • "hidden"
    • "showing"
    • "visible"

 

语法

keyboard_virtual_hide();

 

返回:

N/A

 

例子:

if input == true
{
    input = false;
    keyboard_virtual_hide();
}

如果给定的变量没有设置为false，上面的代码将隐藏操作系统虚拟键盘。
