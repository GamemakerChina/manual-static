shop_leave_rating

此函数会打开一个依赖于操作系统的对话框，您可以在其中要求用户对特定页面发表评级或评论。 您可以定义要出现在对话框中的文本，以及您希望出现在两个按钮上的文本和必须发布评论的 URL。

注意 此函数仅适用于 Android 和 iOS 目标。

重要信息 在 iOS 上，使用 TestFlight 时，在开发过程中会显示提示，但您实际上无法执行任何操作。一旦您的应用程序出现在 App Store 中，此按钮就会变为活动状态，并且用户可以选择撰写评论或留下评分。请注意，如果您通过 TestFlight 分发游戏，则可能不会显示此窗口。

 

语法:

shop_leave_rating(text, yes_string, no_string, url)

参数类型描述
textString显示在对话框上的文本。
yes_stringString出现在“是”按钮上的文本。
no_stringString出现在“否”按钮上的文本。
urlString必须发送评论的完整 URL。

 

返回:

N/A

 

例子:

if (timer <= 0)
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

如果变量“timer”倒计时到 0，上面的代码将要求用户发表评论。
