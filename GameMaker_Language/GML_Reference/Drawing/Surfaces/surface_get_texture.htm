surface_get_texture

此函数返回表面纹理的特殊指针。然后，该值可用于其他绘制函数，特别是在常规3D和某些2D基本体函数中。

注意使用表面时，由于表面存储在纹理内存中，因此表面可能随时停止存在。在直接引用表面之前，应该始终使用surface_exists()检查表面是否存在。

注意在HTML5上，这将返回结构而不是纹理指针，因为指针不能在该平台上使用。但是，这不会更改返回值的使用，因为其在纹理函数中的用法仍然保持不变。

 

语法:

surface_get_texture(surface_id);

参数 类型 描述
surface_id Surface ID 要获取宽度的表面的ID。

 

返回:

Texture

 

例子:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

以上代码将绘制一个带有纹理的4顶点三角形条纹，该纹理保持在"tex"变量中，该变量本身取自以前创建的表面。
