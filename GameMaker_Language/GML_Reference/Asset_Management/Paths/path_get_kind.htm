path_get_kind

路径可以是平滑或直(平滑路径计算围绕定义点的弯曲路径，而直线路径只是从一个点直到另一个点)。这个函数可以判断给定路径是否平滑(true)或不(false)。

 

语法:

path_get_kind(index);

参数类型 描述
indexPath Asset 要检查的路径的索引。

 

返回:

Boolean

 

例子:

path_kind = path_get_kind(pth_Patrol);

这会将"路径_种类"变量设置为true或false，具体取决于给定路径索引的路径种类。
