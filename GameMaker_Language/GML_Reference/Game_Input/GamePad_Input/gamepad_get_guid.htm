gamepad_get_guid

使用此函数，您可以检索连接到给定插槽索引的游戏手柄的GUID。您提供了要检查的游戏手柄插槽 (从0- 11)，该函数将返回以下字符串之一:

  • “none”，如果没有GUID可用或没有连接游戏手柄
  • “device index out of range”，如果游戏手柄插槽索引超出范围
  • 如果插槽分配了游戏手柄并可以为其获取GUID，则GUID字符串 (类似于 “050000005e040000fd020000ffff3f00”)

此函数通常将与gamepad_get_description()结合使用，以生成Windows，Android和macOS上重新映射控制器所需的SDL字符串的一部分。

平台专用注释
平台Note
GX.gamesOn the GX.games target platform, this function will always return "none" for non-DirectInput devices

 

语法:

gamepad_get_guid(index);

参数类型描述
indexReal要检查哪个Gamepad“插槽”索引

 

返回:

String

 

例子:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

上面的代码获取GUID和name描述字符串，然后将它们连接起来，并将最终字符串存储在全局变量中，以备将来参考。
