highscore_add

使用此函数，您可以将名称和分数添加到内部全局高分列表中。无需检查该值是否足够高，即可进入分数列表，因为GameMaker只会存储那些大于存储的第十位的值。

 

语法:

highscore_add(str, numb);

参数 类型 描述
str String 要将分数归属到的字符串(名称)。
numb Real 要加的数字(分数)。

 

返回:

N/A

 

例子:

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

上述代码将检查生活，如果设置为0，则会将当前分数和全局变量"Name"中保留的字符串添加到高分列表中，然后最终将分数设置为0，并将播放器发送到rm_MainMenu中索引的房间。
