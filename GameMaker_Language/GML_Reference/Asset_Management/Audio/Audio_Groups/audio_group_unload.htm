audio_group_unload

此函数将把标记为属于给定音频组的所有声音卸载到内存中。

如果卸载已启动，该函数将返回 true，如果组 ID 无效或已标记为卸载，则返回 false。

注意 调用此函数时，当前正在播放的任何音频都将停止。

 

语法:

audio_group_unload(groupID);

参数类型描述
groupIDAudio Group ID要卸载的音频组的索引(在音频组窗口中定义)

 

返回:

Boolean

 

例子:

if (audio_group_is_loaded(audiogroup_level1))
{
    audio_group_unload(audiogroup_level1);
}

上面的代码检查是否已加载音频组，以及是否已卸载音频组。
