ds_priority_empty

使用此函数，您可以检查给定的D优先级队列，查看它是否为空(返回true)或不是(返回false)。

 

语法:

ds_priority_empty(id);

参数 类型 描述
id DS Priority 要检查的数据结构的ID。

 

返回:

Boolean

 

例子:

if (count == 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = game_get_speed(gamespeed_fps);
    ai_count = 0;
}

以上代码检查变量是否已达到特定值，以及是否清除变量"command_queue"中索引的DS优先级队列，设置报警，并将变量重置为0。
