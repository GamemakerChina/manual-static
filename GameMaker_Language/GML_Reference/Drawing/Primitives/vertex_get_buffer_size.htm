vertex_get_buffer_size

使用此函数可以获取给定顶点缓冲区的大小(以字节为单位)。

 

语法:

vertex_get_buffer_size(buffer);

参数 类型 描述
buffer Vertex Buffer ID 要获取大小的缓冲区。

 

返回:

Real

 

例子:

bufferBytes = vertex_get_buffer_size(buff);

以上代码将获取给定顶点缓冲区使用的字节数，并将该值存储在变量bufferBytes中。
