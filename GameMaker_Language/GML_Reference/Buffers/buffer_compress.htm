buffer_compress

使用此函数，您可以使用zlib compression压缩部分(或全部)缓冲区。提供要压缩的缓冲区ID(由buffer_create()返回)、要使用的缓冲区内偏移量(以字节为单位)以及要压缩的缓冲区数据大小(也以字节为单位)。该函数将为压缩缓冲区返回一个新的缓冲区Id值，如果由于任何原因失败，则返回一个小于0的值。此函数不会改变原始缓冲区。

 

语法:

buffer_compress(buffer, offset, size);

参数类型 描述
bufferBuffer ID 要压缩的缓冲区的索引。
offsetReal 要压缩的缓冲区内的偏移量 (以字节为单位)。
sizeReal 要压缩的缓冲区的大小 (以字节为单位)。

 

返回:

Buffer ID

 

例子:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

上面的代码将创建一个缓冲区，然后用字符串中的数据填充它。然后将此缓冲区压缩并保存，并删除源缓冲区和压缩缓冲区
