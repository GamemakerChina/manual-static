window_views_mouse_get_y

此函数返回鼠标相对于所有活动视图的 Y-坐标，并返回相同的值 mouse_y。

NOTE For regular mouse functions see the section on Mouse Input.

 

语法:

window_views_mouse_get_y();

 

返回:

Real

 

例子:

myy = window_views_mouse_get_y();

这将在窗口中相对于所有活动视图将 myy 设置为鼠标的 Y 坐标。
