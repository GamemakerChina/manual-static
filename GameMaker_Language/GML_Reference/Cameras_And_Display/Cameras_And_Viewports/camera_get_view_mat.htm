camera_get_view_mat

此函数可用于检索视图矩阵。该函数返回矩阵数组，该矩阵数组可用于其他矩阵函数或设置另一个相机的视图矩阵(使用 camera_set_view_mat())。请注意，此函数通常用于使用 camera_create() 创建的相机，但也可用于在房间编辑器中创建的相机(或与 camera_create_view() 一起创建的相机)，在这种情况下，返回的矩阵仅在给定相机首次在视口中使用后才有效。

 

语法:

camera_get_view_mat(camera_id)

参数类型描述
camera_idCamera ID创建相机时返回的唯一相机 ID 值。

 

返回:

Matrix Array

 

例子:

mat = camera_get_view_mat(camera_view[0]);

以上代码获取分配给 viewport[0] 的相机的视图矩阵。
