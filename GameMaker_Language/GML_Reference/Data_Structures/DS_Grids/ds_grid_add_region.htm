ds_grid_add_region

此函数可用于将给定的值(实数或字符串)添加到在网格定义区域内找到的所有单元格值中。要添加的值必须与网格单元格中的值具有相同的类型，即：不能将字符串添加到实数或反之亦然，对于字符串而言，这相当于连接。

 

语法:

ds_grid_add_region(index, x1, y1, x2, y2, val);

参数类型描述
indexDS Grid网格的索引。
x1Real网格中区域左侧的x位置。
y1Real栅格中区域顶部的y位置。
x2Real网格中区域右侧的x位置。
y2Real网格中区域底部的y位置。
valAny要添加到区域单元格的值。

 

返回:

N/A

 

例子:

ds_grid_add_region(grid, 2, 4, 5, 5, ".")

这会将"."添加到由变量"grid"引用的DS网格的定义区域内单元格中保存的所有字符串中。
