window_set_min_width

此函数可用于设置游戏的最小窗口宽度。如果在目标平台的"游戏选项"中启用窗口大小调整选项，则玩家可以将游戏窗口大小调整为他们希望的任何大小，但是通过使用此函数，您可以将最小宽度限制为您指定的大小。如果要返回到默认行为(即：无最小值)，请使用 -1 值。

注意 此功能仅在 Windows、macOS 和 Linux 平台上可用。

 

语法:

window_set_min_width(width);

参数 类型 描述
width Real 游戏窗口的最小宽度(单位：像素)。

 

返回:

N/A

 

例子:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

上述代码将设置游戏窗口的最小和最大宽度和高度。
