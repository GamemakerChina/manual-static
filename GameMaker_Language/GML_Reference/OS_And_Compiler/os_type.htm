os_type

这个只读变量包含各种常量之一,GameMaker必须告诉您游戏是为哪个操作系统创建的。请注意，这不一定与运行它的设备的操作系统相同，因为例如，您的游戏可能在Amazon Fire OS上运行，但将是为安卓平台构建的(在这种情况下os_type将是os_android)。

可以返回以下常量：

操作系统类型常数
常量 描述
os_windows Windows OS
os_gxgames GX.games
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos Apple tvOS
os_android Android
os_ps4 Sony PlayStation 4
os_ps5 Sony PlayStation 5
os_gdk / os_xboxseriesxs Microsoft GDK (Xbox One and Xbox Series X/S)
os_switch Nintendo Switch
os_unknown Unknown OS

 

语法:

os_type;

 

返回:

操作系统类型常数

 

例子:

switch (os_type)
{
case os_windows: global.Config = 0; break;
case os_android: global.Config = 1; break;
case os_linux: global.Config = 2; break;
case os_macosx: global.Config = 3; break;
case os_ios: global.Config = 4; break;
case os_winphone: global.Config = 5; break;
}

上面的代码检查运行游戏的操作系统并相应地设置一个全局变量。
