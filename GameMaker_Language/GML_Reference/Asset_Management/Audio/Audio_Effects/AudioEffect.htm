AudioEffect Struct

AudioEffect 结构存储音频效果使用的参数。您可以更改这些值以实时调整效果。

这个结构必须用audio_effect_create 函数创建。该结构中可用的参数取决于你传递给该函数的音频效果类型(见 AudioEffectType Enum)

NOTE 这是一个内置结构，因此GameMaker对其内容有更多控制。 这意味着对其成员的访问是有限的，如果分配了无效的值，一些值可能会自动改变。 你可能还需要通过 GameMaker提供的内置函数来创建和销毁这个结构，而不是通过new 和delete操作符。

这种类型的所有结构都有一个type和bypass属性。 所有其他属性都是特定于某些类型的效果。

下面是一个结构成员的列表，取决于创建的效果类型。

混响(AudioEffectType.Reverb1)

变量 类型 描述
type AudioEffectType Enum 一个只读属性，用于存储本结构中的音频效果类型。 它被设置为 AudioEffectType.Reverb1 用于混响。
bypass Boolean 效果是否应该被绕过(忽略)
size Real 空间的大小[范围:0.0 - 1.0]

较大的尺寸意味着更多的反射和更长的混响。
damp Real 较高频率应被衰减的量 [范围: 0.0 - 1.0].
mix Real 输出中原始/混响信号的比例 [范围: 0.0 - 1.0].

0.0的值可以得到100%的原始信号。值为1.0时，给出100%的混响信号。 一个0.5的值导致50%的原始信号和50%的混响信号。

延迟 (AudioEffectType.Delay)

变量 类型 描述
type AudioEffectType Enum 一个只读属性，用于存储本结构中的音频效果类型。 它被设置为 AudioEffectType.Delay 用于延迟
bypass Boolean 效果是否应该被绕过(忽略)
time Real 延迟的长度(以秒为单位)
feedback Real 被延迟的信号反馈到延迟线的比例[范围: 0.0 - 1.0]
mix Real 原始/延时信号在输出中的比例 [范围: 0.0 - 1.0]

值为0.0时给出100%的原始信号，值为1.0时给出100%的延迟信号。 值为0.5时，会出现50%的原始信号和50%的延迟信号。

失真 (AudioEffectType.Bitcrusher)

变量 类型 描述
type AudioEffectType Enum 一个只读属性，用于存储本结构中的音频效果类型。 它被设置为 AudioEffectType.Bitcrusher 的失真
bypass Boolean 效果是否应该被绕过(忽略)
gain Real The input gain going into the effect. There is a hard clipper (clipping at ±1.0) directly after this stage.
factor Real 原始信号被降采样的系数 [range: 0.0 - 100.0].

这被四舍五入为一个整数。
resolution Real 信号被重采样的比特深度 [范围: 1.0 - 16.0].

这被四舍五入为一个整数
mix Real 原始/失真信号在输出中的比例 [range: 0.0 - 1.0].

0.0的值给出100%的原始信号，1.0的值给出100%的失真信号。0.5的值会导致50%的原始信号和50%的失真的信号

低通滤波器(AudioEffectType.LPF2)

变量 类型 描述
type AudioEffectType Enum 一个只读属性，用于存储本结构中的音频效果类型。 它被设置为 AudioEffectType.LPF2 用于LPF。
bypass Boolean 效果是否应该被绕过(忽略)
cutoff Real 滤波器的截止频率。 高于截止点的频率将被衰减。
q Real 滤波器的质量系数 [范围: 1.0 - 100.0].

这是一个无尺寸的参数，表示频率在截止点附近的峰值(增益)程度。 数值越大，峰值越大。

高通滤波器 (AudioEffectType.HPF2)

变量 类型 描述
type AudioEffectType Enum 一个只读属性，用于存储本结构中的音频效果类型。 它被设置为 AudioEffectType.HPF2 用于HPF。
bypass Boolean 效果是否应该被绕过(忽略)
cutoff Real The cutoff frequency of the filter. Frequencies lower than the cutoff will be attenuated.
q Real 滤波器的质量系数 [范围: 1.0 - 100.0].

这是一个无尺寸的参数，表示频率在截止点附近的峰值(增益)程度。 数值越大，峰值越大。

增益 (AudioEffectType.Gain)

变量 类型 描述
type AudioEffectType Enum 一个只读属性，用于存储本结构中的音频效果类型。 它被设置为 AudioEffectType.Gain 的Gain。
bypass Boolean 效果是否应该被绕过(忽略)
gain Real 应用于输入信号的增益值

颤音 (AudioEffectType.Tremolo)

变量 类型 描述
type AudioEffectType Enum 存储此结构中存储的音频效果类型的只读属性。对于颤音，它设置为 AudioEffectType.Tremolo 。
bypass Boolean 效果是否应该被绕过(忽略)
rate Real 调制增益的 LFO 的频率 (0.0-20.0 赫兹)
intensity Real 应由 LFO 调制的输入信号的比例 (0.0-1.0)。换句话说，受 LFO 影响的是信号 / 样本的幅度的比例 (或分数)。
offset Real 应由 LFO 调制的输入信号的比例 (0.0-1.0)。换句话说，受 LFO 影响的是信号 / 样本的幅度的比例 (或分数)。
shape AudioLFOType Enum LFO 应输出的波形形状。
