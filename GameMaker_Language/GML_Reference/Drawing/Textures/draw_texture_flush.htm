draw_texture_flush

使用此函数，您可以从视频内存中删除所有纹理，然后将在首次使用时重新加载这些纹理。当游戏有多个纹理页面时，这是管理视频内存的唯一有效方法，并且应该刷新纹理内存并使用纹理组函数组织图形，以确保使用的纹理数量最少。

注意 在所有目标上，纹理页面 默认情况下 只会在需要时加载，您可以使用精灵的各种预取和刷新函数 ( 此处 ) 来加载纹理页面管理事物以及此功能。

 

语法:

draw_texture_flush();

 

返回:

N/A

 

例子:

Create Event

draw_texture_flush();

上述代码将刷新纹理数据的视频内存，并可能放置在房间中放置的对象的第一个实例的创建事件中。
