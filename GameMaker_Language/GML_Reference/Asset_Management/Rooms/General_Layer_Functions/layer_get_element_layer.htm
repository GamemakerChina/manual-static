layer_get_element_layer

此函数获取给定元素所在的图层 ID。

您提供唯一的元素 ID 值 (例如，由创建元素的函数或从 房间编辑器 返回的值)，该函数将返回找到该元素的图层的唯一 ID。如果给定的元素 ID 不是有效的，则该函数将返回 -1。

 

语法:

layer_get_element_layer(element_id)

参数类型描述
element_idLayer Element ID要获取其层的元素的唯一ID值

 

返回:

Layer or -1 (if the element is invalid)

 

例子:

element = layer_get_element_layer(asset_1);

上述代码获取元素的图层 ID，该 ID 存储在变量 asset_1 中，并将其存储在变量中。
