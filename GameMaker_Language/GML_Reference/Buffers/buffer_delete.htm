buffer_delete

此函数从内存中删除先前使用 buffer_create 创建的缓冲区，释放用于创建它的资源并删除它当前可能包含的任何数据。

注意 当您不再需要任何动态创建的资源时，务必从内存中删除它们，以防止内存泄漏，这一点很重要。

重要信息 创建缓冲区时，会返回对新缓冲区的引用。缓冲区被销毁后，我们建议您将保存缓冲区引用的变量设置为 -1。

 

语法:

buffer_delete(buffer)

参数类型描述
bufferBuffer要删除的缓冲区。

 

返回:

N/A

 

例子:

buffer_delete(player_buffer);
player_buffer = -1;

上述代码删除变量 player_buffer 中存储的缓冲区，然后将该变量设置为 -1。
