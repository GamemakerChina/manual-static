buffer_save

使用此函数，您可以将缓冲区的内容保存到文件中，准备使用buffer_load()函数读回内存。

NOTE On HTML5 the contents of the buffer will be saved as base64 encoded strings when using this function.

 

语法:

buffer_save(buffer, filename);

参数 类型 描述
buffer Buffer ID 要保存的缓冲区的索引。
filename String 要另存为的文件的名称。

 

返回:

N/A

 

例子:

buffer_save(buff, "Player_Save.sav");

使用变量" buff "中存储的ID将缓冲区的当前内容保存到文件中。
