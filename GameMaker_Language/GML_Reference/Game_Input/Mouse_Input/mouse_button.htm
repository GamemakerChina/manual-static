mouse_button

此 只读 变量返回当前按下的鼠标按钮 (当前，如本步骤所示)，并且可以返回除 mb_any 之外的任何特殊 鼠标常量 。

 

语法:

mouse_button

 

返回:

Mouse Button Constant (except mb_any)

 

示例:

if (mouse_button == mb_left)
{
    x -= 1;
}

如果按下鼠标左键，此代码会将当前实例向左移动。
