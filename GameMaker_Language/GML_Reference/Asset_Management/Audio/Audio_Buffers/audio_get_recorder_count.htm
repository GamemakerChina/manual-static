audio_get_recorder_count

此函数将返回录音源的数量(如麦克风等...)当前可用于您的游戏。因此，如果您的返回值为4，则源0、1、2和3上将有音频输入，其中一个值是使用函数audio_start_recording()开始录制的值。 当人们将麦克风或其他输入源插入/拔出设备时，此值随时可能改变。请注意，您可以使用函数audio_get_recorder_info()获取每个连接设备的信息。

重要此函数将始终在GX.Games目标上返回1。

 

语法:

audio_get_recorder_count();

 

返回:

Real

 

例子:

if audio_get_recorder_count() > 0
{
channel_index = audio_start_recording(0);
}

上面的代码检查可用的音频记录器，如果有1个或多个，则从索引的第一个(源0)开始记录。
