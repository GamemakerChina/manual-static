event_user

使用此函数，您可以告诉实例运行已放置在16个用户定义事件之一中的操作或代码。这些事件只能以这种方式调用，或使用event_perform函数调用。

See: Other Events

 

语法:

event_user(numb);

参数 类型 描述
numb Real 要调用的用户事件数，介于0和15之间。

 

返回:

N/A

 

例子:

event_user(4);

这将调用用户定义的事件4。
