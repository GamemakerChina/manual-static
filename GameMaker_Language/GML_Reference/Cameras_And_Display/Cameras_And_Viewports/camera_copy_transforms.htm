camera_copy_transforms

此功能将源相机的所有与变换相关的设置复制到目标相机。

更具体地说，它复制源相机的视图和投影矩阵，以及视图位置、大小、速度、边界和角度值。

注意 此函数主要供 GameMaker 在处理滤镜和效果时内部使用，但您也可以将其用作将一台摄像机的变换复制到另一台摄像机的便捷方式。

警告 该函数 不会 复制源相机的 开始、结束和更新脚本 。

 

语法:

camera_copy_transforms(dest_camera, src_camera);

参数 类型 描述
dest_camera Camera ID 将源相机的变换复制到的相机 (投影和视图矩阵)
src_camera Camera ID 包含要复制的投影和视图矩阵的相机

 

返回:

N/A

 

例子:

cam1 = view_camera[0];
cam2 = camera_create();
camera_copy_transforms(cam2, cam1);

上述代码将 GameMaker 内部为视图 0 创建的摄像头 (您可以首先在 房间编辑器 中配置) 分配给实例变量 cam1。然后，它使用函数 camera_create 创建一个新的 " 空 " 相机 cam2。然后，它通过调用 camera_copy_transforms 将 cam1 的变换相关设置复制到 cam2。
