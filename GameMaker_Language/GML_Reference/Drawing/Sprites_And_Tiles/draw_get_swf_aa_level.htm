draw_get_swf_aa_level

该函数可用于获取 SWF 格式向量精灵的抗锯齿 (AA) 级别。返回值将介于 0 和 1 之间，并显示将如何 " 平滑 " 这些精灵的边缘。您可以使用函数 draw_set_swf_aa_level() 设置 AA 级别。

 

语法:

draw_get_swf_aa_level();

 

返回:

Real

 

例子:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上述代码将检查Swf格式精灵的Aa值，如果为0，则启用Aa并将该值设置为0.5。
