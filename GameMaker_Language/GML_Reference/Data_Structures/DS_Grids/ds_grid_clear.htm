ds_grid_clear

此函数可用于将给定的DS网格清除为特定值。然后，网格中的所有单元格都将包含该值，该值可以是实数或字符串。下图说明了这种方法的工作原理：

 

语法:

ds_grid_clear(index, val);

参数类型 描述
indexDS Grid 此索引要清除的网格。
valAny 所有栅格单元的新值。

 

返回:

N/A

 

例子:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

上述代码将调整在全局变量"Grid"中索引的DS网格的大小，然后将其清除，以便每个单元格保留值-1。
