path_set_kind

此函数可用于设置您希望指定资源的路径类型。这可以是直线路径 (设置为 0) 或平滑路径 (设置为 1)，在这种情况下，还必须考虑路径精度 (也可以使用 path_set_ precision() 设置精度 )。此函数会更改实际的路径资源，因此将永久影响从使用该函数到游戏结束期间游戏中所有实例如何使用路径。如果这不是您所需要的，那么您应该使用像 path_duplicate() 这样的函数，首先创建路径的副本，然后对重复的资源调用此函数 (不要忘记调用 path_delete() 当不再需要该资产时)。

 

语法:

path_set_kind(index, val);

参数类型描述
indexPath Asset要更改的路径的索引。
valReal路径类型，0表示直线，1表示平滑。

 

返回:

N/A

 

例子:

path_set_kind(pth_Patrol, true);

这将将给定路径设置为"平滑"路径。
