vertex_argb

此函数为当前为自定义基元定义的顶点设置 ARGB 值。

您提供用于写入数据的顶点缓冲区以及您希望用作单个 32 位无符号整数的红色、绿色、蓝色和 alpha 值 - 最高 8 位中的 alpha 样本，后跟红色样本，绿色样本，最后是最低 8 位的蓝色样本。您可以使用十六进制表示法 ($AARRGGBB)、实数或使用任何 make_colour_* 函数来定义颜色值。

 

语法:

vertex_argb(buffer, argb);

参数 类型 描述
buffer Vertex Buffer 要写入信息的顶点缓冲区。
argb Colour 要设置的参数值。

 

返回:

N/A

 

例子:

vertex_argb(buff, $FFFFFFFF);

上面的代码将定义的顶点的 ARGB 值设置为白色。
