layer_sprite_get_blend

此函数可用于获取精灵元素的混合颜色。给定精灵元素 ID(使用layer_sprite_create()或使用函数layer_sprite_get_id()创建精灵元素时得到的 ID)，该函数将返回表示指定颜色的真实值。

 

语法:

layer_sprite_get_blend(sprite_element_id);

参数类型 描述
sprite_element_idSprite Element ID 获取信息的精灵元素的唯一ID值

 

返回:

Colour

 

例子:

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_blend(back_id) == c_white
{
    layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

上面的代码将获取名为"sprite_sky"的层的层ID，然后使用该代码来获取该层上的精灵元素的ID。然后使用此 ID 检查元素混合颜色，如果它等于常数c_white，则将元素混合设置为随机颜色。
