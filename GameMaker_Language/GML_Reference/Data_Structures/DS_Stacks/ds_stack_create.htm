ds_stack_create

此函数创建一个新的堆栈数据结构并向其返回一个 句柄 。该值应存储在变量中，并在与堆栈相关的所有进一步函数调用中使用。

 

语法:

ds_stack_create();

 

返回:

DS Stack

 

例子:

stack = ds_stack_create();

这将创建一个新堆栈并将其索引 ID 分配给实例变量 stack。
