game_save_id

此只读变量将返回游戏用于保存文件的目录的完整路径ID。此目录可能对其他应用程序可见，也可能不可见，具体取决于平台。

在HTML5目标上，它将返回一个空字符串。

注意此变量仅供参考，不应用于获取文件路径或任何类似内容，因为它不能在所有平台上正常工作。

 

语法:

game_save_id

 

返回:

String

 

例子:

save_dir = game_save_id;

这将存储用于将文件保存到变量以供将来使用的目录。
