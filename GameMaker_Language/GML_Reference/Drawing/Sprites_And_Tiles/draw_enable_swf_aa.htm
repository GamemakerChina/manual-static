draw_enable_swf_aa

使用此函数，您可以为 SWF 格式的矢量精灵启用或禁用抗锯齿 (AA)。AA 只是简单地平滑矢量图像的边缘，以给它们更好的外观。使用的 AA 数量取决于使用函数 draw_set_swf_aa_level() 设置的值。默认情况下，此选项处于禁用状态。

 

语法:

draw_enable_swf_aa(enable);

参数类型 描述
enableBoolean 对所有 SWF 精灵启用(true)或禁用(false) AA。

 

返回:

N/A

 

例子:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上述代码将检查 SWF 格式精灵的 AA 值，如果为 0，则启用 AA 并将该值设置为 0.5。
