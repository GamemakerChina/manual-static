window_get_showborder

此函数用于获取游戏是否显示在无边框窗口中。

它对应于 Windows 图形选项中的无边框和窗口选项，它会在游戏中获取该复选框的值。

注意 此函数仅适用于Windows。

 

语法:

window_get_showborder();

 

返回:

Boolean

 

例子:

var _border_shown = window_get_showborder();

上面的代码获取是否显示窗口边框并将结果存储在临时变量 _border_shown 中。
