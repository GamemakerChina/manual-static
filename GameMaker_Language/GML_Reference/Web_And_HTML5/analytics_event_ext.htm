analytics_event_ext

这个函数将把指定的文本发送到你通过HTML5 游戏设置设置的分析提供程序。这个函数可以用来创建一个自定义分析来跟踪某些东西并且还将接受自定义参数/值对，其中参数是字符串，值是实数。对于谷歌Analytics，你只能添加一个额外的pair，而Flurry将接受最多7.

 

语法:

analytics_event_ext(string, string_param, value);

参数类型 描述
stringString 要发送给提供程序的字符串。
string_param[0 -9]String 要发送(字符串)的参数。
value[0 - 9]Real 参数的值(实数)。

 

返回:

N/A

 

例子:

var num, time;
ini_open("Options.ini");
num = ini_read_real("Data", "Plays", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Plays", num, "Time", time);
ini_write_real("Data", "Plays", num);
ini_close();

上面的代码将从 ini 文件中获得播放信息以及当前时间，然后将这些详细信息发送给分析提供程序。
