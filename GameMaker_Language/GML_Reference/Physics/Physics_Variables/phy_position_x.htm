phy_position_x

此变量可用于获取 (或设置) 游戏室物理世界中实例的 X 位置。请注意，当通过直接设置此变量来移动实例时，物理世界可能会出现错误，因为它将中断连续模拟。此变量是与实例变量 x 等效的物理变量。

 

语法:

phy_position_x;

 

返回:

Real (single precision floating point value, or undefined if the instance is not physics enabled)

 

例子:

xx = phy_position_x;
yy = phy_position_y;

上述代码将实例 X 和 Y 位置存储在两个变量中。
