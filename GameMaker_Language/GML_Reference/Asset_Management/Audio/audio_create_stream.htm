audio_create_stream

使用此函数，您可以创建新的声音索引，然后可以在常规音频函数中使用，直接从外部 OGG 文件源传输音频。

该函数需要文件名(例如，该文件可以是包含的文件)，并将返回新的声音索引以供使用。

注意 在您不再需要声音后，应调用带有声音索引的函数audio_destroy_stream()将其从内存中移除，否则可能会出现内存泄漏，这会减慢并最终导致游戏崩溃。

注意 此函数不适用于 HTML5 目标平台。

 

语法:

audio_create_stream(filename);

参数 类型 描述
filename String 流传输音频的文件路径(仅限 OGG)。

 

返回:

Sound Asset

 

例子:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

上面的代码在变量"snd"中创建一个新的声音索引，然后播放这个声音。
