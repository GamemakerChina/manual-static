time_source_exists

此函数返回给定变量是否拥有 时间源 。

如果时间源被破坏，这将返回 false，但如果时间源被停止或仅仅过期，它将继续存在，这意味着此函数返回 true。

 

语法:

time_source_exists(id);

参数 类型 描述
id Time Source 要检查的时间源

 

返回:

Boolean

 

例子:

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

此代码检查时间源是否存在。如果是，则会检查是否按下了空格键。

在这种情况下，它会销毁时间源。
