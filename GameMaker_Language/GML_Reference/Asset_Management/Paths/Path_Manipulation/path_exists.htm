path_exists

此函数返回具有给定索引的路径是否存在。请注意，如果您检查是否存在尚未声明的变量的路径，这将抛出一个错误。

 

语法:

path_exists(index);

 

参数类型描述
indexPath Asset要检查的路径的索引。

 

返回:

Boolean

 

例子:

if (path_exists(my_path))
{
    path_delete(my_path);
}

这段代码检查给定的变量是否存储了路径索引，如果是，则从内存中删除该路径。
