sequence_exists

使用此函数，您可以检查是否存在序列对象。提供序列对象结构(由函数sequence_create()或sequence_get()返回)或序列 ID(由函数layer_sequence_get_sequence()或序列实例结构属性sequence或资源浏览器的索引返回)，如果序列对象存在，则函数将返回true;如果序列对象不存在，则函数将返回false。

 

语法:

sequence_exists(sequence_struct_or_id);

参数 类型 描述
sequence_struct_or_id Sequence Asset or Sequence Object Struct 要检查的序列可以是资源引用本身或其序列对象结构。

 

返回:

Boolean

 

例子:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

上面的代码检查给定的序列对象是否存在，如果存在，则将其销毁。
