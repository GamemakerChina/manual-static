ds_priority_find_min

使用此函数，您可以找到优先级最低的优先级队列中存储的值，如果多个值具有相同的优先级，则可以按任何顺序返回其中的任何一个。但是，与ds_priority_delete_min()不同的是，此函数不会从队列中删除该值。

注意 如果优先级队列为空，此函数将返回 undefined。

 

语法:

ds_priority_find_min(id);

参数 类型 描述
id DS Priority ID 要使用的优先级队列的ID。

 

返回:

Any (Data type stored in the priority)

 

例子:

if ai_move
{
    script_execute(ds_priority_find_min(ai_priority));
}

以上代码检查实例变量，如果返回true，它将执行在优先级队列中具有最低优先级值的索引脚本函数。
