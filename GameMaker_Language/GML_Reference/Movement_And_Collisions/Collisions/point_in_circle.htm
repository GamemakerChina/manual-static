point_in_circle

使用此函数时，定义一个圆形区域，GameMaker将确定给定点是否在其范围内。如果点落在定义的圆内，则函数将返回true，否则函数将返回false。

 

语法:

point_in_circle(px, py, x1, y1, rad);

参数类型 描述
pxReal 要检查的点的x坐标。
pyReal 要检查的点的y坐标。
x1Real 圆心的x坐标。
y1Real 圆心的y坐标。
radReal 圆的半径。

 

返回:

Boolean

 

例子:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
    over = true;
}
else
{
    over = false;
}

以上代码使用point_in_circle函数检查鼠标位置是否落在定义的圆形区域内，如果是，则将变量over"设置为真，否则设置为假。
