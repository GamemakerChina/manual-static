keyboard_string

该变量保存一个字符串，其中包含键盘上输入的最后 (最多 )1024 个字符。该字符串仅包含键入的可打印字符，但它 将 通过擦除最后一个字符来正确响应按退格键。该变量 不是 只读的，您可以更改它，例如，如果您已经处理过它，则将其设置为 ""( 空字符串)，并且您可以使用 字符串函数 来操纵它。

注意 当使用屏幕上的虚拟键盘时，此变量只在用键盘输入时进行更新。

注意 在 Xbox GDK 上，此变量仅在使用 虚拟键盘 时起作用。请参阅 这篇帮助台文章 了解更多信息。

 

语法:

keyboard_string

 

返回:

String

 

示例:

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

上面的代码将键盘字符串的长度限制为15个字符，通过将前15个字符复制回变量来删除超出该限制的字符。
