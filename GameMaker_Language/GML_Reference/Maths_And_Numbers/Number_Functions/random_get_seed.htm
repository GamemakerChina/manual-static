random_get_seed

要生成随机数GameMaker从随机种子数开始。通过这个函数，您可以检索该种子，然后将其存储起来以再现特定的随机事件系列。例如，假设你有一款程序生成游戏，但希望玩家能够回到之前发现的区域。在每个区域的开始，你可以使用这个函数存储随机种子，所以每次你需要回到那里时，你可以使用random_set_seed()将种子设置为之前的值，从而强制 GameMaker 重现它之前使用的同一组“随机”函数。

 

语法:

random_get_seed();

 

返回:

Real

 

例子:

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

以上代码将获取随机种子并将其存储在 ini 文件中。
