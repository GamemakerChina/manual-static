event_action 已删除

此只读变量返回当前正在执行的操作的索引，在以前版本的 GameMaker 上从 0 开始。然而，现在是GameMaker中的过时变量。 它只留作遗留支持，并且始终返回0，因为所有操作都连接在一起以提高执行速度。

 

语法:

event_action;

 

返回:

Real

 

例子:

num = event_action;

上述代码将当前操作编号存储在变量" num "中。
