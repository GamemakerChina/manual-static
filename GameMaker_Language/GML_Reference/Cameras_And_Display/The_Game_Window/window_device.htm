window_device 已删除

此函数将返回当前 d3d 设备指针，然后您可以(例如)通过该指针到达 Windows 和 MacOS 上的 DLL 或 Dylib。

注意 在 GameMaker 中不建议使用此函数，建议使用 os_get_info()。

 

语法:

window_device();

 

返回:

Pointer

 

例子:

gfx_pointer = window_device();

上述代码将把 d3d 设备指针存储在一个变量中。
