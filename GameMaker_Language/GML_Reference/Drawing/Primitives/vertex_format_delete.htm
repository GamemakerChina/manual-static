vertex_format_delete

每当您使用完任何创建的顶点格式后，都必须调用此函数。

您提供顶点格式 (由函数 vertex_format_end 返回)，该函数将释放与其关联的内存。请注意，如果在调用此函数后再次尝试使用此格式，则会出现错误。

 

语法:

vertex_format_delete(format_id);

参数 类型 描述
format_id Vertex Format 要删除的顶点格式

 

返回:

N/A

 

例子:

vertex_format_delete(my_format);

上述代码将从内存中删除存储在变量 my_format 中的顶点格式。
