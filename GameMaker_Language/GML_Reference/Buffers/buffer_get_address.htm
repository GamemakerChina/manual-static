buffer_get_address

此函数获取指向原始对齐缓冲区地址的指针。

这主要用于扩展，因为您可以将此值传递给它们，允许它们访问缓冲区数据。

使用说明

  • 您无法释放内存。
  • 您无法调整内存大小。
  • 您不能写入超过缓冲区地址加上缓冲区大小 (您可以使用 buffer_get_size 函数)，否则您将获得越界错误。

 

语法:

buffer_get_address(buffer);

参数 类型 描述
buffer Buffer The buffer to use.

 

返回:

Pointer

 

例子:

var _address = buffer_get_address(buff_model);
var _end_address = _address + buffer_get_size(buff_model);

上面的代码获取变量 buff_model 中存储的缓冲区的内存地址，然后通过添加由 buffer_get_size 返回的缓冲区大小来获取缓冲区末尾的内存地址，到地址。这两个值都存储在局部变量中以供进一步使用。
