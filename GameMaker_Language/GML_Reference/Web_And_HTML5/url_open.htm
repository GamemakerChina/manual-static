url_open

这将在所选目标设备的浏览器上打开指定的URL，或者(如果使用HTML5模块)在当前打开的浏览器中打开。

注意 安装在玩家设备上的杀毒软件可能导致URL无法打开，因此在使用此函数时请注意这一点。

 

语法:

url_open(url);

参数 类型 描述
url String 要链接到的URL(网址)。

 

返回:

N/A

 

例子:

url_open("http://gamemaker.io");

这将在当前窗口中打开 GameMaker 主页。
