merge_colour

使用此函数，您可以选择两种颜色，然后将它们合并在一起以生成新颜色。可以通过更改 "amount" 参数来定义每种组件颜色的数量，其中值 0 将返回第一种颜色 (col1)，值 1 将返回第二种颜色 (col2)，而中间的值将返回相应的混合颜色。例如，值为 0.5 时，两种颜色的混合程度相等。下图通过将红色和蓝色合并到一起说明了这种方法的工作原理：

语法:

merge_colour(col1, col2, amount);

参数类型 描述
col1Colour 要合并的第一种颜色
col2Colour 要合并的第二种颜色
amountReal 每种颜色应该合并多少。例如，0 将返回 col1，1 将返回 col2，0.5 将同等返回两种颜色的合并

 

返回:

Colour

 

例子:

col = merge_colour(c_lime, c_orange, 0.5);

上述代码使用该函数通过合并石灰和橙色50/50来创建一种颜色，然后将其值存储在变量"col"中以供以后使用。
