camera_set_view_pos

您可以使用此函数更新房间内相机视图的位置。给出唯一的相机 ID 值(由不同的 camera_create() 函数返回)，然后给出 x 和 y 位置设置相机。

 

语法:

camera_set_view_pos(camera_id, x, y)

参数类型 描述
camera_idCamera ID 创建相机时返回的唯一相机 ID 值。
xReal (在房间内)放置视图的 x 位置
yReal (在房间内)放置视图的 y 位置

 

返回:

N/A

 

例子:

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

上述代码将为分配给 viewport[0] 的相机设置视图的相机位置。
