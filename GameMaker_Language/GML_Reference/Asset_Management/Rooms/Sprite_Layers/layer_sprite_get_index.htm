layer_sprite_get_index

该函数可用于获取精灵元素的当前图像索引值。给定精灵元素 ID(使用layer_sprite_create()或使用函数layer_sprite_get_id()创建精灵元素时获得的 ID)，该函数将返回表示为 sprite 显示的图像索引的真实值。如果精灵元素不存在或者元素没有有效的精灵分配给它，函数将返回-1。

 

语法:

layer_sprite_get_index(sprite_element_id);

参数类型描述
sprite_element_idSprite Element ID获取信息的精灵元素的唯一ID值

 

返回:

Real or -1

 

例子:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_index(spr_id) < 4)
{
    layer_sprite_index(spr_id, 4);
}

上面的代码将获取名为"sprite_sky"的层的层ID，然后使用该代码来获取该层上的精灵元素的ID。然后使用此 ID 检查元素的图像索引是否小于 4，如果小于 4，则将其设置为 4。
