object_set_visible

使用此函数，您可以将特定对象标记为可见或不可见(有关"可见"标志的详细信息，请参阅对象编辑器上的部分)。 这意味着此对象的所有实例(在可见更改后创建)将使用此新状态创建，而房间中的实例可能不会受到影响。

请注意这不是实例函数！您可以使用visible变量设置单个实例的可见标志，因此即使相关对象在 IDE 中将标志设置为 false，仍有 10 个实例全部标记为可见，即使使用此函数将对象的可见标志更改为 true，房间中当前的所有实例将保持原样，只有调用函数后创建的实例才会以标记为 true 的可见实例开始。

 

语法:

object_set_visible( index, vis );

参数 类型 描述
index Object Asset 要更改的对象的索引。
vis Boolean 对象的新可见性(true=可见，false=不可见)。

 

返回:

N/A

 

例子:

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

上述代码检查对象"obj_Block"的可见标志，如果为false，则更改为标记为true。
