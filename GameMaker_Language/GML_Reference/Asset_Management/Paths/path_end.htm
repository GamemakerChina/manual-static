path_end

调用此函数将结束实例所遵循的当前路径，与调用函数path_start()时设置的路径相同。

 

语法:

path_end();

 

返回:

N/A

 

例子:

 

if place_meeting(x, y, obj_Blocker)
{
    path_end();
}

如果实例检测到与给定对象的任何实例的任何冲突，则上面的代码将结束当前路径。
