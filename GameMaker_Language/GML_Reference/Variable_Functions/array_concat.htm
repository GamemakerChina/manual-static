array_concat

此函数将多个数组作为参数，将它们联接在一起(按参数的顺序)，并返回联接的数组。

 

语法:

array_concat(array1, array2 [, array3, ... array_n]);

参数 类型 描述
array1 Array 第一个要连接的数组
array2 Array 要连接的第二个数组
[array3, ... array_n] Array 可选要连接的附加数组

返回:

Array (new array with all arrays concatenated)

 

例子:

array_1 = [1, 2, 3];
array_2 = [4, 5, 6];
array_3 = [7, 8, 9];
new_array = array_concat(array_1, array_2, array_3);

上面的代码首先创建了三个数组:array_1,array_2和array_3。然后，它使用array_concat将数组连接在一起，并将结果存储在new_array中。
