layer_sprite_index

此函数可用于设置层上具有多个子图像的精灵资源的图像索引。指定精灵元素 ID(使用layer_sprite_create()或使用函数layer_sprite_get_id()时获得的 ID)，然后设置要使用的图像索引。如果将值设置在子图像范围之外，则图像索引将循环。

 

语法:

layer_sprite_index(sprite_element_id, image_index);

参数类型描述
sprite_element_idSprite Element ID要设置的精灵元素的唯一ID值
indexReal要用于精灵的图像索引

 

返回:

N/A

 

例子:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

上面的代码将获取名为"sprite_trees"的层的层ID，然后使用该代码来获取该层上给定精灵元素的ID。然后，此 ID 用于更改元素图像索引。
