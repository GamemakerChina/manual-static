draw_vertex

此函数定义基元的顶点位置。基元的最终外观将取决于选择要绘制的基元类型以及向其添加顶点的顺序。有关的信息，请参见draw_primitive_begin()更多信息。要结束并绘制基元，必须调用draw_primitive_end()。

 

语法:

draw_vertex(x, y)

参数类型描述
xReal顶点的x坐标。
yReal顶点的y坐标。

 

返回:

N/A

 

例子:

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

上述代码将绘制一个简单的三角形基元。
