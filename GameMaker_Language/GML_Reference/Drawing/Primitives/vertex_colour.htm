vertex_colour

如果定义的顶点格式采用颜色值，则可以使用此函数将该数据添加到为当前基本体定义的顶点。该函数需要一个缓冲区来存储数据，并将采用颜色常数或十六进制值(使用标准GameMaker格式的BGR,例如：$FF0000用于蓝色)以及从0(透明)到1(完全不透明)的Alpha值。

 

语法:

vertex_colour(buffer, colour, alpha);

参数 类型 描述
buffer Vertex Buffer ID 写入信息的缓冲区。
colour Colour 此顶点的颜色(可以是常数值或十六进制值)。
alpha Real 顶点的Alpha值(从0到1)。

 

返回:

N/A

 

例子:

vertex_colour(b, c_white, 1);

以上代码将使用Alpha值1将定义为白色的顶点设置为白色。
