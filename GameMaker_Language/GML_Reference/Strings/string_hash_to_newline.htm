string_hash_to_newline

此函数返回一个字符串，其中 “#” 符号已转换为新行。所以一个字符串格式化为:

"Hello#World"

将在屏幕上打印为：

Hello
World

重要 提供此函数是为了实现 GameMaker 与以前版本之间的导入兼容性，因此，除非需要复制过时的函数，否则不应使用此函数。相反，您应该使用转义字符来标记新行。有关导入兼容性的详细信息，请参阅过时函数页。

 

语法:

string_hash_to_newline(string);

参数 类型 描述
string String 要在多行上转换的字符串

 

返回:

String

 

例子:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

上面的代码将带有哈希符号的字符串转换为拆分为两行的字符串，然后将其打印到屏幕上。
