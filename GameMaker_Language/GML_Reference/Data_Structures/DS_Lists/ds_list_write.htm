ds_list_write

此函数返回一个字符串，然后可以使用ds_list_read()函数将该字符串存储或传输到另一个数据结构。

注意 返回的字符串不是人类可读的字符串，而是数据结构内容的转储。

 

语法:

ds_list_write(id);

参数类型描述
idDS List要写入的数据结构的ID。

 

返回:

String

 

例子:

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

上述代码打开ini文件，然后写入一个字符串，其中包含变量"list"中索引的DS列表中存储的信息。然后清除该列表并关闭ini文件。
