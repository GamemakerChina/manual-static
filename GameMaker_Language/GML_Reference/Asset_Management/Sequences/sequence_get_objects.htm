sequence_get_objects

使用此函数，可以检索在给定序列内创建实例的所有对象索引的数组。您可以提供序列对象结构(由函数sequence_create()或sequence_get()返回)或序列 ID(由函数layer_sequence_get_sequence()或序列实例结构属性sequence返回)，函数将返回数组，其中数组中的每个项都是object_index，用于序列用于创建实例的不同对象。

 

语法:

sequence_get_objects(sequence_struct_or_id);

参数 类型 描述
sequence_struct_or_id Sequence Asset or Sequence Object Struct 要从中获取对象的序列对象结构或ID.

 

返回:

Array

 

 

例子:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

上面的代码获取序列对象的结构，然后检索它使用的对象，并将数组存储在变量中。
