device_get_tilt_y

此函数根据设备的 “倾斜” 角度返回-1到1之间的值。倾斜度与返回的值之间的实际相关性取决于它使用的设备和OS，但通常1或-1的值与/- 90 ° 相同。下图显示了每个可用函数与设备的关系:

语法:

device_get_tilt_y()

 

返回:

Real

 

例子:

if (display_get_orientation() = display_landscape)
{
    x += sign(device_get_tilt_y());
}
else
{
    x += sign(device_get_tilt_x());
}

上面的代码检查显示器的方向，然后使用相应的倾斜值沿x轴移动播放器。
