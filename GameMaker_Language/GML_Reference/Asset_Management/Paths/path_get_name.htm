path_get_name

此函数将返回作为字符串引用的路径的名称。名称是指在编辑器中用来定义路径的名称，或者(如果路径是通过代码函数创建的)，它将返回一个格式为"_newpathXX"，其中"XX"是生成的路径编号，从 0 开始，每次创建新路径时递增 1。请注意，这只是个字符串，不能用于直接引用该路径，因为您需要路径索引ID。 但是，您可以使用此字符串来获取使用返回的字符串的路径索引 ID函数asset_get_index()。

 

语法:

path_get_name(index);

参数类型 描述
indexPath Asset 要检查的路径的索引。

 

返回:

String

 

例子:

path_name = path_get_name(path_array[0]);

这将将"path_name"设置为在给定数组中位置 0 处索引的路径的名称。
