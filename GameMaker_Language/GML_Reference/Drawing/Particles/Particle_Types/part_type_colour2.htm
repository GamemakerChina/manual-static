part_type_colour2

该函数为给定类型创建的每个粒子设置两种颜色的渐变。

第一种颜色是所有粒子将开始的颜色，第二种颜色是粒子将结束的颜色，并且在粒子的寿命期间，颜色将发生从一种颜色到另一种颜色的平滑梯度变化。

 

语法:

part_type_colour2(ind, colour1, colour2);

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
colour1 Colour 粒子开始的颜色。
colour2 Colour 粒子结束时的颜色。

 

返回:

N/A

 

例子:

part_type_colour2(part_Health, c_red, c_white);

上述代码将使在变量"part_Health"中创建的索引类型的所有粒子在其生命周期内具有从红色到白色的颜色混合。
