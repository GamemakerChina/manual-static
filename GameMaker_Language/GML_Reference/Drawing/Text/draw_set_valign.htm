draw_set_valign

此函数用于沿垂直轴对齐文本，更改垂直对齐将更改绘制所有其他文本的位置和方向，默认值为fa_top。接受以下常量：

常量 对齐
fa_top
fa_middle
fa_bottom

 

语法:

draw_set_valign(valign);

参数 类型 描述
valign Vertical Alignment Constant 垂直对齐常量(来自上表)。

 

返回:

N/A

 

例子:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

上述代码将绘制以文本中心为中心的分数。
