window_set_showborder

此功能用于设置游戏是否显示在无边框窗口中。

它对应于 Windows 图形选项中的无边框和窗口选项，它会在游戏中更改该复选框的值。

注意 此函数仅适用于Windows。

 

语法:

window_set_showborder(show);

参数 类型 描述
show Boolean 是否显示窗口边框

 

返回:

N/A

 

例子:

window_set_showborder(!window_get_showborder());

上面的代码行用于切换窗口边框。
