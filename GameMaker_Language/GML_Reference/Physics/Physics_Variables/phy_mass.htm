phy_mass

此只读变量返回以公斤为单位的实例质量。此值是根据指定的装置的表面积及其密度值自动计算的，但可以使用函数 physics_mass_properties() 来更改。

 

语法:

phy_mass;

 

返回:

Real (single precision floating point value, or undefined if the instance is not physics enabled)

 

例子:

if (phy_mass < other.phy_mass)
{
    instance_destroy();
}

以上代码来自实例与另一个实例的碰撞事件，它比较每个实例的质量并销毁质量较小的实例。
