font_set_cache_size

当执行图像混合和着色时，HTML5不能像执行可执行文件那样动态地执行。因此，GameMaker必须临时保存图像的混合副本，并在需要时加载它们。此函数设置在覆盖旧字体之前可以缓存多少给定字体的混合副本。默认值为4。

注意对于精灵字体，您应该使用 sprites 的等效函数，sprite_set_cache_size()。

 

语法:

font_set_cache_size(ind, max);

 

参数 类型 描述
ind Font Asset 要更改其缓存大小的字体的索引。
max Real 可存储的字体缓存副本的最大数量。

 

返回:

N/A

 

例子:

font_set_cache_size(fnt_MainMenu, 2);

这将将变量"fnt_MainMenu"中索引的字体的字体缓存设置为2个副本。
