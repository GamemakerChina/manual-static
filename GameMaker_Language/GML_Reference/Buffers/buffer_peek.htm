buffer_peek

使用buffer_read()函数，您可以在当前"查找"位置从给定缓冲区读取数据，读取的每个数据片段将此位置向前移动读取或写入的字节。但是，您可能需要读取给定的数据，而不希望更改当前的寻道位置，这就是您使用此函数的时候。您只需为函数提供一个缓冲区ID，然后在该缓冲区内提供一个偏移位置(从缓冲区开始)以供读取，并提供要读取的数据类型。

NOTE Using the incorrect data type for the data being read will result in erroneous values!

如果函数成功，将根据数据类型返回一个值(实数、字符串、布尔值等)，而失败将仅返回0。

 

语法:

buffer_peek(buffer, offset, type);

参数 类型 描述
buffer Buffer ID 要使用的缓冲区的索引。
offset Real 缓冲区内用于读取给定数据的偏移位置(以字节为单位)。
type Buffer Data Type Constant 要从缓冲区读取的数据类型(请参阅此处的常量列表)。

 

返回:

Real, Boolean, String or 0 (if it fails)

 

例子:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

上述代码将从变量"buffe"中索引的缓冲区内的三个不同位置获得三个值，然后使用这些值设置实例的图像混合。
