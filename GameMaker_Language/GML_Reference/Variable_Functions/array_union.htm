array_union

此函数返回一个新的数组，它是所有提供的数组的并集。联合是指所提供的数组中的所有元素，但删除了重复项。

注意新数组中元素的顺序是没有保证的。如果您需要对数组进行排序，则需要在之后进行排序。

语法:

array_union(array0, [array1, ... array_n]);

参数 类型 描述
array0 Array 第一个数组
[array1, ... array_n] Array 可选 Any number of arrays for union (one array per argument)

返回:

Array (the union of the provided arrays)

 

例子:

a1 = [1, 2, 3, 4, 5];
a2 = [3, 4, 5, 6, 7, 8];
a3 = [5, 6, 7, 8, 9, 10, 11];

a4 = array_union(a1, a2, a3);

上面的代码首先创建了3个数组a1,a2和a3。然后，它将这些数组合并，并将结果存储在新的数组a4中。
