vertex_create_buffer

使用此函数可以创建新的顶点缓冲区。这是由GameMaker创建的特殊增长缓冲区，该缓冲区已预先格式化，可用于构建基本体(例如，用于着色器)。该函数将返回缓冲区的索引，然后应在对其进行的所有进一步调用中使用该索引。

使用使用此函数创建的顶点缓冲区时，只需调用vertex_begin()即可开始为其指定顶点数据，以开始定义自定义基元，然后将该基元保存在缓冲区中，以便提交到着色器。必要时可以重新使用缓冲区(除非您已使用vertex_freeze()函数)，每次调用vertex_begin()都会擦除先前的缓冲区数据，以便接受新数据。

 

语法:

vertex_create_buffer();

 

返回:

Vertex Buffer ID

 

例子:

v_buff = vertex_create_buffer();

上述代码将创建一个新的顶点缓冲区，并将其句柄存储在变量"v_buffs"中。
