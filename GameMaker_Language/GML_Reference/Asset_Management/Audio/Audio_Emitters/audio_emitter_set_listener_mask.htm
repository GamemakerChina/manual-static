audio_emitter_set_listener_mask

此函数可用于设置发射器的位掩码，以便通过发射器播放的所有声音将仅从指定的收听者播放。可以使用audio_get_listener_info()，然后使用 bitwise或(\"|\")来创建与需要播放声音的收听者的自定义掩码，然后将此自定义掩码应用于发射器。此掩码将覆盖默认掩码或使用函数audio_set_listener_mask()设置的掩码。

 

语法:

audio_emitter_set_listener_mask(emitterID, mask);

参数 类型 描述
emitterID Audio Emitter ID 要设置掩码的发射器的唯一ID
mask Real 要为声音设置的位掩码数据

 

返回:

N/A

 

例子:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

上面的代码创建一个音频发射器，然后设置收听者使用存储在全局变量中的掩码数据来播放。
