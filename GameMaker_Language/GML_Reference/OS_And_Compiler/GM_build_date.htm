GM_build_date

此常量保存GameMaker构建正在运行的可执行文件的日期和时间。可以使用Date和Time函数解析该值，并在编译时从系统UTC值中获取。

 

语法:

GM_build_date;

 

返回:

Datetime

 

例子:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

上面的代码将绘制游戏的版本号以及编译它的日期和时间。
