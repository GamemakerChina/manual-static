fx_set_parameter

此函数用于更改给定 FX 结构中的参数值。指定 FX 结构(从fx_create()或layer_get_fx()返回)、参数名称作为字符串及其新值。这将使您的更改在 FX 结构指定给的层中可见。

如果参数是向量，则可以为参数传递多个参数，但是也可以简单地指定一个数组作为唯一的参数，该参数将包含多个值。

在本页查找滤镜/效果的参数名称：效果类型参数。

 

语法:

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

参数类型描述
filter_or_effectFX Struct要修改的FX结构
parameter_nameString要更改的值的参数名称(作为字符串)
val...Real or Array该参数作为实数或数组的新值

 

返回:

N/A

 

例子:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上面的代码从"_filter_tintfilter"效果创建一个新的FX结构，该效果是在房间编辑器中找到的"色调"过滤器。它为其" g_TintCol"参数分配一个值，该参数是色调的颜色，由于它在内部是vec4，它采用包含四个值(对应于其红色、绿色、蓝色和alpha值)的数组。然后，将色调的 FX 结构应用于 FX 层。

请注意，您还可以将值作为单独的参数而不是数组传递给向量：

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
