gamepad_is_connected

此函数将返回游戏手柄是否连接到给定的 “插槽”(返回true)(返回false)。你通常会将此函数与gamepad_get_device_count()函数结合使用，以获得正确数量的可用游戏手柄和/或游戏手柄 “插槽”。请注意，用户连接游戏手柄和GameMaker检测到游戏手柄处于连接状态之间可能会有轻微的延迟(在处理连接蓝牙的控制器时尤其如此)。

注意: 在某些平台上-尤其是控制台-此函数可能会在已连接/选择游戏手柄后立即返回false，并且可能需要在几帧 (步骤) 后的警报中进行检查才能正确检测游戏手柄。

 

语法:

gamepad_is_connected(numb);

参数类型 描述
deviceReal 要检查哪个游戏手柄 “插槽”。

 

返回:

Boolean

 

例子:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

上面的代码循环通过可用的游戏手柄(或游戏手柄插槽)，然后检查每个连接的游戏手柄。然后使用返回值将全局数组设置为true或false，以便在将来的检查中使用。
