fx_set_parameters

此函数用于更改滤镜/效果的参数。指定 FX 结构(从fx_create()或layer_get_fx()返回)和包含其参数的结构(从fx_get_parameters()返回)。 这将使您的更改在 FX 结构指定给的层中可见。

参数结构不需要包含适用于滤镜/效果的每个参数，并且只有给定结构中存在的变量将被更新为滤镜/效果。

 

语法:

fx_set_parameters(filter_or_effect, parameter_struct);

参数类型 描述
filter_or_effectFX Struct 要修改的FX结构
parameter_structStruct 包含滤镜/效果参数的结构

 

返回:

N/A

 

例子:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

上面的代码获取假定对其应用了"Color Tint"过滤器的层的FX结构，并通过调用fx_get_parameters()来检索其参数结构。 然后使用sin()和current_time生成一个振荡值，作为色调效果的RGBA值，保持动画。RGBA 数组分配给参数结构中的g_TintCol变量，然后通过调用fx_set_parameters()将该结构应用回 FX 结构。
