part_particles_count

使用此函数，可以检查粒子系统当前是否在房间中创建了任何粒子，并且粒子系统还将返回粒子的数量。

 

语法:

part_particles_count(ind);

参数 类型 描述
ind Particle System ID 粒子系统的索引。

 

返回:

Real

 

例子:

if (part_particles_count(Sname) == 0)
{
    part_system_destroy(Sname);
    instance_destroy();
}

以上代码将检查在变量"Sname"中索引的局部粒子系统中的粒子数，如果没有，则会破坏系统，然后破坏实例。
