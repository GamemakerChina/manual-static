texturegroup_get_textures

此函数可用于检索构成纹理组的各个页面的纹理页ID。提供纹理组Id字符串(如"纹理组编辑器"(Texture Group Editor)中所定义)，该函数将返回一个1D数组，其中数组中的每个条目都是单个纹理页面ID。如果函数失败，即给定了无效的组，或者该组没有指定纹理，则数组将为空(长度为0)。

 

语法:

texturegroup_get_textures(tex_id);

参数类型描述
tex_idString要检查的纹理组的名称(字符串)

 

返回:

Array of Textures

 

例子:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

以上代码将检索纹理组"MainMenu"的纹理页面ID，然后检查它们是否已解包，如果是，则将它们放置到 VRAM 中。
