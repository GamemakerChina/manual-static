skeleton_skin_list

使用此函数，您可以使用骨骼动画精灵中包含的所有皮肤名称填充(预先创建的)DS列表。这些名称将是字符串，然后可以在这些类型的精灵的其他骨骼动画蒙皮函数中使用。

 

语法:

skeleton_skin_list(sprite, list);

参数类型描述
spriteSprite Asset要从中获取列表的脊椎骨骼动画的精灵索引。
listDS List要用动画名称填充的DS列表的ID.

 

返回:

N/A

 

例子:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

上面的代码创建了一个DS列表，然后用皮肤名称填充它。然后，在销毁列表之前，选择一个随机变量并将其应用于实例。
