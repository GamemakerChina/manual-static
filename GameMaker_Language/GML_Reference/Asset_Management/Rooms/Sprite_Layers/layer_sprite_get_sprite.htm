layer_sprite_get_sprite

此函数可用于获取精灵元素的当前精灵索引。您给出精灵元素 ID(当您使用 layer_sprite_create 或使用函数 layer_sprite_get_id) 创建一个精灵元素时，该函数将返回一个表示所显示的精灵索引的实际值。如果该元素没有指定精灵，则该函数将返回 -1。

 

语法:

layer_sprite_get_sprite(sprite_element_id);

参数 类型 描述
sprite_element_id Sprite Element ID 获取信息的精灵元素的唯一ID值

 

返回:

Sprite Asset or -1

 

例子:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
    layer_sprite_change(spr_id, spr_Clouds);
}

上面的代码将获取名为"sprite_sky"的层的层ID，然后使用该代码来获取该层上的精灵元素的ID。然后，此 ID 用于检查分配给元素的精灵，如果尚未设置，则将其设置为 sprite" spr_Clouds"。
