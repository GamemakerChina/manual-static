sequence_get

使用此函数，可以从序列资源索引中检索序列对象结构。为在资源浏览器中定义的序列提供资源索引，函数将返回然后可以访问的序列对象结构，或者如果序列不存在或给定的资源索引不是序列，则返回-1。

 

语法:

sequence_get(sequence_index);

参数 类型 描述
sequence_index Sequence Asset 在资源浏览器中定义的序列资源。

 

返回:

Sequence Object Struct or -1

 

例子:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

上面的代码从序列对象"seq_Loop"中检索序列对象结构，然后设置序列的循环模式(所有随后创建的序列实例现在将使用此循环模式)。
