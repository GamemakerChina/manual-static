string_format

将一个实数变成一个使用你自己格式的字符串，你可以选择多少个 "位 "被保存到字符串中，以及多少个小数位也被保存。这两种方法都很方便，有些游戏喜欢将分数显示为一个固定的数字，而对小数点位的控制可以达到string()的两个小数点位无法提供的高精确度。如果指定的位数大于要显示的数值，并且/或者指定的数值加上小数点后的位数小于总的位数，那么在数值前会加上空格以弥补差值(见下面的例子)。如果给定的数值小于总数和要包括的小数位数，小数点右边将加零。默认格式是左边没有多余的空格，右边只有两个小数位，例如 "265.73"。

 

语法:

string_format(val, tot, dec);

参数 类型 描述
val Real 要变成字符串的实数。
tot Real 要显示的主数字的总位数。零或空格将被插入，以符合这个要求。
dec Real 要包括的小数位数。

 

返回:

String

 

例子:

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

这会将 str1 设置为 “1234”， str2 设置为 “3.1415926535”，将 str3 设置为 “3.14159”。
