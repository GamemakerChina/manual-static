buffer_save

此函数将缓冲区的内容保存到文件中，准备好使用 buffer_load 函数读回内存。

注意 在 HTML5 上，使用此函数时，缓冲区的内容将保存为 base64 编码的字符串。

 

语法:

buffer_save(buffer, filename);

参数类型描述
bufferBuffer要保存的缓冲区。
filenameString要另存为的文件的名称。

 

返回:

N/A

 

例子:

buffer_save(buff, "Player_Save.sav");

上述代码将变量 buff 中存储的缓冲区的当前内容保存到文件中。
