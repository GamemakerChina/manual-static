vertex_normal

此函数将向当前为自定义基元定义的顶点添加表面法线数据。提供缓冲区以将数据写入法线的x、y和z分量部分。

 

语法:

vertex_normal(buffer, nx, ny, nz);

参数 类型 描述
buffer Vertex Buffer ID 写入信息的缓冲区。
nx Real 表面法线的x分量。
ny Real 表面法线的y分量。
nz Real 表面法线的z分量。

 

返回:

N/A

 

例子:

vertex_normal(buff, 0, 1, 1);

上述代码将设置所定义顶点的表面法线。
