audio_sound_get_track_position

此函数获取声音文件中播放声音的位置(以秒为单位)。

该函数可以采用声音资源或声音实例。它返回：

  • 对于声音资源：使用任意audio_play_sound_*函数播放该声音时的起始位置(或偏移量)。
  • 对于当前播放的声音(由audio_play_sound_*返回)：当前曲目位置。

 

语法:

audio_sound_get_track_position(index);

参数类型描述
indexSound Instance ID or Sound Asset要获取播放位置的声音资源或实例。

 

返回:

Real

 

示例1：

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

上面的代码检查轨道以获取其起始位置，如果不是 0 秒，则将其设置为 0 秒。
