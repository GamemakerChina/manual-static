move_snap

此函数用于将实例"捕捉"到给定大小的网格。它将被捕捉到hsnap和vsnap值定义的"不可见"网格上最近的相应位置。

 

语法:

move_snap(hsnap, vsnap);

参数类型 描述
hsnapReal 水平捕捉("单元格"之间的像素大小)。
vsnapReal 垂直捕捉("单元格"之间的像素大小)。

 

返回:

N/A

 

例子:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

上述代码检查"obj_Pieces"的所有实例，以查看它们是否捕捉到32 x 32像素的网格，如果它们不捕捉，则将它们捕捉到该网格中最近的位置。
