ord

此函数采用单字符输入字符串并返回该字符的 Unicode (UTF8) 值。请注意，当与 keyboard_check* 函数一起使用时，输入字符串的长度只能为一个字符，并且只能是从 0 到 9 的数字或从 A 到 Z 的 大写的 罗马字符。

 

语法:

ord(string);

参数 类型 描述
string String 查找Unicode值的字符串。

 

返回:

Real

 

例子:

if keyboard_check(ord("W"))
{
    y -= 4;
}

如果按住键W，这将使调用实例向上移动四个像素。
