image_speed

这个变量决定了GameMaker在当前实例精灵的子图像中循环的速度。给定的速度值是一个乘数，1是默认值，将其设置为0.5将使动画速度减半-如在精灵编辑器或图像编辑器中设置的-而将其设置为2将使其加倍。如果所使用的精灵没有子图像，则此变量将不起作用。

 

语法:

image_speed;

 

返回:

Real (single precision floating point value)

 

例子:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

上面的代码创建了对象“ OBJ_爆炸”的实例，并将其图像_速度设置为0.5(有效地将运行速度减半)。
