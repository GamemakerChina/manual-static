fx_create

此函数用于为给定效果类型创建新的 FX 结构，然后可以应用于层。

可以传递到此函数的过滤器/效果的名称在此页面中列出：外汇类型 参数。

用途

您可以通过在返回的结构上调用fx_get_parameter/s来读取所创建效果的参数，并通过调用fx_set_parameter/s为它们分配新值。

请注意，此函数返回的结构将不包含可见成员，只能通过Filter/Effect 函数之一进行修改。

函数故障

如果结构因任何原因无法创建，函数将返回-1; 如果指定的 FX 名称错误或 FX 未加载到游戏包中，则可能发生这种情况。

为确保此函数正常工作，请确保将所需的 FX 加载到游戏包中，如筛选器和效果函数页面所述：

请注意，目前只有两种方法可确保GameMaker在游戏中加载特定过滤器/效果：

  • 通过IDE在至少一个房间中添加滤镜/效果
  • 调用fx_create()，将过滤器/效果名称作为字符串literal

这意味着要在运行时使用特定的过滤器/效果，必须首先将其添加到房间(因此 GameMaker 知道要使用它)，或者在fx_create()调用中明确指定。

如果不满足这些条件，即您没有传入字符串文本并且过滤器/效果在任何房间中都不存在，则函数将无法加载该过滤器/效果，并将返回-1。

 

 

语法:

fx_create(filter_or_effect_name);

参数 类型 描述
filter_or_effect_name String 要创建的过滤器/效果的名称(从外汇类型获取)参数)。

 

返回:

FX Struct or -1 (on failure)

 

例子:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上面的代码从"_filter_tintfilter"效果创建一个新的FX结构，该效果是在房间编辑器中找到的"色调"过滤器。它为其" g_TintCol"参数分配一个值，该参数是色调的颜色，由于它在内部是vec4，它采用包含四个值(对应于其红色、绿色、蓝色和alpha值)的数组。然后，将色调的 FX 结构应用于 FX 层。
