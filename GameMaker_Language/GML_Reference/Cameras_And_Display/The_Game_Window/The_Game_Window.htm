The Game Window

您创建的游戏将在窗口中进行(即使是全屏)，并且此窗口具有许多属性，如位置、大小、是否为全屏等等。这些详细信息通常会根据启用的房间大小和视口以及目标平台的游戏选项中的设置自动为您设置，但您可以在游戏过程中使用此页面上列出的函数进行更改。

注意 这些函数仅适用于 Windows、Ubuntu、macOS、GX.games 和 HTML5 目标，不能在任何其他设备上使用。
