layer_get_target_room

此函数将返回层函数目标的当前房间。有关详细信息，请参见layer_set_target_room()。

 

语法:

layer_get_target_room()

 

返回:

Room Asset

 

例子:

if (layer_get_target_room() != room)
{
    layer_reset_target_room();
}

上述代码检查当前目标房间，如果不是当前房间，则重置层目标房间。
