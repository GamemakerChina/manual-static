vertex_end

使用此函数可以结束自定义基本体的构建。调用此命令后，可以在vertex_submit()函数中使用基元，以便在着色器中使用，或者可以使用vertex_freeze()冻结缓冲区(使顶点缓冲区使用只读且速度更快)。

 

语法:

vertex_end(buffer);

参数 类型 描述
buffer Vertex Buffer ID 要结束写入的缓冲区。

 

返回:

N/A

 

例子:

vertex_end(buff);

上述代码将结束自定义基元的定义。
