gesture_rotate_angle

此函数用于设置一对触摸必须超过的角度，以触发旋转开始手势。 角度测量单位为度，默认值为5°。

 

语法:

gesture_rotate_angle();

 

返回:

Real (inches)

 

例子:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

上述代码检查手势的旋转角度是否设置为5°，如果不是，则将其设置为该值。
