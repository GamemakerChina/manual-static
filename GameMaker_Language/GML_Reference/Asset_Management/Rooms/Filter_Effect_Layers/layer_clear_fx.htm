layer_clear_fx

此函数用于清除应用于指定层的 FX 结构。可以指定要修改的图层的 ID 或名称，此函数将删除应用于该图层的 FX 结构，从而完全禁用其效果。

如果您使用此函数临时禁用某个效果，并且需要恢复它的层，您应该首先通过调用layer_get_fx()为层保存FX结构，然后调用该函数以在层上清除它。稍后可以通过调用layer_set_fx()将其应用回层，这将重新启用效果。

 

语法:

layer_clear_fx(layer_name_or_id);

参数 类型 描述
layer_name_or_id String or Layer 清除FX的层的名称或ID

 

返回:

N/A

 

例子:

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

上面的代码检查是否按下"E"键，然后检查"EffectLayer"层是否应用了过滤器/效果; 在这种情况下，它将其FX结构存储在临时实例变量中，并清除所有过滤器/效果层。当按下"F"键时，它检查临时变量是否持有- 1以外的内容，在这种情况下，它将FX应用回层，并将临时变量重置为-1。所有这些实质上允许您在需要时启用和禁用效果，而不会丢失其参数。
