health

此变量在范围内为全局，用于保存通常用于玩家健康状况的数值。此变量仅用于支持GameMaker以前版本的旧项目，不应在新项目中使用，因为将来可能会弃用此变量。

 

语法:

health;

 

返回:

Real (single precision floating point value)

 

示例

if (health <= 0)
{
    global.state = "Game Over";
    instance_destroy();
}

以上代码检查health变量，如果它小于或等于0，则设置全局变量并销毁实例。
