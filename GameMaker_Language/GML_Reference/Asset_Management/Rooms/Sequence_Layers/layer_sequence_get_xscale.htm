layer_sequence_get_xscale

使用此函数，您可以提供由layer_sequence_create()或由其中一个层元素函数返回的序列元素 IDas，它将返回游戏室中沿序列元素 X 轴的当前比例。

 

语法:

layer_sequence_get_xscale(sequence_element_id)

参数 类型 描述
sequence_element_id Sequence Element ID 序列元素到目标的唯一ID值

 

返回:

Real

 

例子:

var _xs = layer_sequence_get_xscale(title_sequence)
if _xs < 1
{
    _xs += 0.01;
    layer_sequence_xscale(title_sequence, _xs);
}

上述代码利用存储在变量 "title_sequence" 中的 ID 沿着序列元素的 X 轴检索当前比例，如果它小于 1，则将 0.01 添加到当前 X 比例。
