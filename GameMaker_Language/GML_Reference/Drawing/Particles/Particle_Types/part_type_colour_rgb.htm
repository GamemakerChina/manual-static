part_type_colour_rgb

使用此函数，可以为创建的给定类型的所有粒子设定红色、绿色和蓝色的混合颜色。为三个组件中的每个组件提供最小值和最大值，创建的粒子将具有基于给定参数范围的随机颜色。所有值必须介于0和255之间。

 

语法:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
rmin Real 最终颜色的红色分量的最小值。
rmax Real 最终颜色的红色分量的最大值。
gmin Real 最终颜色的绿色分量可以是最小值。
gmax Real 最终颜色的绿色组件可以达到的最大值。
bmin Real 最终颜色的蓝色分量的最小值。
bmax Real 最终颜色的蓝色分量的最大值。

 

返回:

N/A

 

例子:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

上述代码将在全局变量"Blood_Part"中索引的粒子类型发射的每个粒子设定为仅为不同的红色着色。
