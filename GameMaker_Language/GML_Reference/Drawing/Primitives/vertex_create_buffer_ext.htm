vertex_create_buffer_ext

与函数vertex_create_buffer()一样，此函数将创建新的顶点缓冲区。这是GameMaker创建的特殊增长缓冲区，已预先格式化，以便在构建用于着色器的基本体时使用。您可以指定缓冲区的初始起始大小(以字节为单位)，然后它将返回缓冲区的索引，该索引应用于对缓冲区的所有进一步调用。

 

语法:

vertex_create_buffer_ext(size);

参数类型 描述
sizeReal 缓冲区的初始大小(字节)。

 

返回:

Vertex Buffer ID

 

例子:

v_buff = vertex_create_buffer_ext(1024 * 1024);

上述代码将创建一个新的顶点缓冲区，最初大小为1MB，并将其句柄存储在变量"v_buffe"中。
