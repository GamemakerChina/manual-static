phy_col_normal_x

此只读变量返回对应于 phy_collision_x 数组值的碰撞法线的 X 分量。对于每个接触点，都有一个关联的接触法线 (对于碰撞中的所有接触点，法线通常是相同的)。此接触法线是一个单位向量，它从碰撞中的一个实例指向另一个实例，并可用于计算解决碰撞的正确"推"方向

注意 该变量仅在物理启用的实例的碰撞事件中可用。

 

语法:

phy_col_normal_x

 

返回:

Real (single precision floating point value, or undefined if the instance is not physics enabled)

 

例子:

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris))
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

上述代码使用触点法线为在两个启用了物理的实例之间的碰撞事件中创建的实例设置移动方向。
