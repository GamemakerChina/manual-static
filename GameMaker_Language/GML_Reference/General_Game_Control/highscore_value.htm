highscore_value

使用此函数，您可以检索已存储在给定位置的高分列表中的分数值。如果未输入分数，则函数将返回0。

 

语法:

highscore_value(place);

参数类型 描述
placeReal 桌上的位置(1-10 )。

 

返回:

N/A

 

例子:

var i = 9;
repeat(10)
{
    scr[i] = highscore_value(i + 1);
    i -= 1;
}

以上代码将循环浏览高分列表，并将所有分数存储在一个数组中。
