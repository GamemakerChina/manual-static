工具菜单

此菜单允许您使用以下选项打开GameMaker中提供的各种独立工具：

  • 纹理组 - 打开 纹理组 窗口。
  • 音频组 - 这使您可以查看 音频组 窗口。
  • 混音器 - 此选项允许您查看 混音器 窗口。
  • 生成项目图像 - 这将打开 生成项目图像  窗口。
  • 创建本地包 - 这将打开 资产打包器 窗口，以便您可以制作和保存本地资产包以备将来使用。
  • 导入本地包 - 这将打开一个文件资源管理器窗口，然后您可以在其中选择 本地资产包 并将其导入，将其添加到当前项目或为其创建新项目。
  • Project Tool - This opens the Project Tool, which performs conversions between current and older GameMaker project format versions.
