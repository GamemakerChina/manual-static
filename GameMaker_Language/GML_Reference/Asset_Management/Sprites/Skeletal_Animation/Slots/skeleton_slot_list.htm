skeleton_slot_list

使用此函数，您可以使用作为骨骼动画精灵的一部分创建的插槽的所有名称填充(预先创建的)DS list。名称将是字符串，然后可以在这些类型的精灵的其他骨骼动画插槽函数中使用。

 

语法:

skeleton_slot_list(sprite, list);

参数 类型 描述
sprite Sprite Asset 要从中获取列表的脊椎骨骼动画的精灵索引。
list DS List 要用骨骼名称填充的DS列表的ID.

 

返回:

N/A

 

例子:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

上面的代码创建了一个DS列表，然后用插槽名称填充它以供以后使用。
