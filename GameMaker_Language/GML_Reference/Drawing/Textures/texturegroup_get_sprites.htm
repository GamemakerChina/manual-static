texturegroup_get_sprites

使用此函数，可以检索指定给给定纹理组内纹理页面的每个精灵的精灵索引。提供纹理组Id字符串(在纹理组编辑器中定义)，该函数将返回一个1D数组，其中每个条目都包含精灵资源的精灵索引。如果函数失败，即给定了无效的组，或者该组没有指定纹理，则数组将为空(长度为0)。

 

语法:

texturegroup_get_sprites(tex_id);

参数 类型 描述
tex String 要检查的纹理组的名称(字符串)

 

返回:

Array

 

例子:

var _tex_array = texturegroup_get_sprites( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Sprite " + string(i) + " Index:" + string(tex_array[i]));
}

以上代码将检索纹理组"MainMenu"的精灵索引，然后在控制台输出窗口中显示这些ID。
