layer_sequence_get_y

使用此函数，您可以提供layer_sequence_create()或层元素函数之一返回的序列元素 IDas，它将返回该序列在游戏室中的当前 Y 位置。

 

语法:

layer_sequence_get_y(sequence_element_id)

参数类型描述
sequence_element_idSequence Element ID序列元素到目标的唯一ID值

 

返回:

Real

 

例子:

if (layer_sequence_get_y(title_sequence) != room_height / 2)
{
    layer_sequence_y(title_sequence, room_height / 2);
}

上述代码检查存储在变量"title_sequence"中的序列元素ID的Y位置，如果它未设置为房间高度的一半，则设置为该值。
