camera_set_view_size

您可以使用此函数更新房间内相机的大小。给出唯一的相机 ID 值(由不同的 camera_create() 函数返回)，然后(以像素为单位)给出宽度和高度。

 

语法:

camera_set_view_size(camera_id, width, height)

参数类型 描述
camera_idCamera ID 创建相机时返回的唯一相机 ID 值。
widthReal 相机视图的宽度(以像素为单位)
heightReal 相机视图的高度(以像素为单位)

 

返回:

N/A

 

例子:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

上述代码将为分配给 viewport[0] 的相机设置视图的相机大小。
