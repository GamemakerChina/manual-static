ds_map_secure_load_buffer

此函数将从缓冲区加载安全保存的 DS 映射。您必须先前已将缓冲区加载到内存中 (使用 buffer_load)，然后将其传递到此函数将返回一个填充了缓冲区内容的 ds 映射。请注意，必须使用函数 ds_map_secure_save_buffer 创建缓冲区才能正常工作，还请注意，如果要加载的 DS 贴图包含数组，则在加载时会将其转换为 DS 列表。

注意: HTML5不支持此函数。

用法说明

  • 安全保存的文件的功能之一是它被锁定到创建它的设备上，因此您无法将保存在一个设备上的文件加载到在另一个设备上运行的项目中。
  • 缓冲区寻道位置应位于数据的开头。使用 buffer_seek 将其移动到使用 ds_map_secure_save_buffer 写入数据的正确位置。

 

语法:

ds_map_secure_load_buffer(buffer);

参数类型描述
bufferBuffer从中加载地图数据的缓冲区。

 

返回:

DS Map or -1 in case the data couldn't be read correctly from the buffer

 

例子:

var _buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(_buff);
buffer_delete(_buff);

上述代码将从缓冲区加载安全保存的DS映射，并将其索引值存储在变量中以供将来使用。
