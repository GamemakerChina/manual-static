window_set_cursor

使用此函数，您可以将游戏窗口的光标设置为下面列出的任何一个常量(要查找正在使用的当前光标，您可以使用函数 window_get_cursor()，该函数还将返回其中一个常量)：

常量 光标
cr_none  
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

 

语法:

window_set_cursor(cursor);

参数 类型 描述
cursor Cursor Constant 要为游戏窗口设置的光标。

 

返回:

N/A

 

例子:

if mouse_check_button_pressed(mb_left)
{
    window_set_cursor(cr_drag);
}

如果按下鼠标左键，则上述代码将窗口光标更改为标准窗口拖动光标。
