part_type_life

该函数控制所示类型的每个粒子在屏幕上停留的时间。 与其他粒子函数一样，提供最小值和最大值（步长），每个粒子寿命将是指定范围内的随机步长数。 若要使所有粒子具有相同的寿命，请将这两个值设置为相同。

 

语法:

part_type_life(ind, life_min, life_max);

参数 类型 描述
ind Particle Type ID 要更改的粒子类型的索引。
life_min Real 粒子的最短寿命。
life_max Real 粒子的最大寿命。

 

返回:

N/A

 

例子:

var _gamespeed = game_get_speed(gamespeed_fps);
part_type_life(part_Flare, _gamespeed, _gamespeed * 2);

以上代码将根据当前游戏速度，将所有创建的粒子类型(在变量part_Flare中索引)的粒子设置为具有1到2秒的寿命。
