vertex_get_number

此函数返回给定顶点缓冲区中定义的顶点数。

 

语法:

vertex_get_number(buffer);

参数类型描述
bufferVertex Buffer要检查的顶点缓冲区。

 

返回:

Real

 

例子:

v_num = vertex_get_number(v_buffer);

上述代码获取给定顶点缓冲区中存储的顶点数，并将该值存储在变量 v_num 中。
