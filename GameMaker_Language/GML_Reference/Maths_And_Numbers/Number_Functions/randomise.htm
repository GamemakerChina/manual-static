randomise

此函数将种子设置为随机值。 如果您需要在游戏的多次运行中保持一致的值，您应该使用 random_set_seed()。 请注意，在 GameMaker 中使用随机数函数时，初始种子始终相同，因为这使得跟踪错误和调试更加容易。 如果您希望使用真随机进行测试，您应该在游戏开始时调用此函数。 该函数将返回新的随机种子值(无符号 32 位整数)。

 

语法:

randomise();
// or
randomize();

 

返回:

Real (unsigned 32 bit value)

 

例子:

randomise();

上面的代码将随机化种子。
