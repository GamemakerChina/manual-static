extension_exists

此函数判断是否存在具有给定名称的扩展(true)(false)。

 

语法:

extension_exists(extension_name);

参数 类型 描述
extension_name String 作为字符串的扩展资源的名称

 

返回:

Boolean

 

例子:

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

此代码检查MyExtension扩展是否存在，如果存在，则获取其作为结构的选项，并从中读取值。
