surface_save

此函数将使用给定的文件名将表面保存到光盘。表面必须保存为*.png格式的文件。

注意 此函数仅适用于使用 surface_rgba8unorm (默认)表面 格式的表面 。

 

语法:

surface_save(surface_id, fname);

参数 类型 描述
surface_id Surface ID 要设置为绘制目标的表面的ID。
fname String 保存的图像文件的名称。

 

返回:

N/A

 

例子:

if keyboard_check_pressed(ord("S")
{
    surface_save(surf, "test.png");
}

上述代码将检查用户是否按下键盘上的"S"键，如果按下，则会将变量"washing"中的索引表面保存到光盘中。
