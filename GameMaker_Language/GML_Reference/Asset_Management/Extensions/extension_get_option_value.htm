extension_get_option_value

此函数返回具有给定名称的扩展中给定选项的值。

如果提供的扩展名无效，函数将返回undefined。

 

语法:

extension_get_option_value(extension_name, option_name);

参数 类型 描述
extension_name String 作为字符串的扩展资源的名称
option_name String 要读取的选项名称

 

返回:

Any

 

例子:

var _enabled = extension_get_option_value("MyExtension", "enabled");

此代码获取MyExtension扩展中的已启用选项的值。
