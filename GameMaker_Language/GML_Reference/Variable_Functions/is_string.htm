is_string

此函数返回给定变量是否为字符串号。在某些情况下，您想检查并查看GameMaker中的变量是否包含字符串而不是实数，这就是您使用此函数的时间。它不会返回字符串，而是true或false，因此，例如n的值为 “fish” 将返回true, 但n的值为200将返回false。

 

语法:

is_string(n);

参数 类型 描述
n Any 要检查的参数。

 

返回:

Boolean

 

示例:

if (is_string(val))
{
    name = "Player: " + val;
}

上面的代码检查变量 “val”，看看它是否包含一个字符串，如果它这样做，它添加到另一个字符串，然后分配给变量 “名称”。
