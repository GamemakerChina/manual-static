keyboard_clear

使用此函数，您可以清除当前的键盘状态，这基本上意味着，如果按键被按住，它将不再被识别，直到它再次释放 (这也不会触发键盘按键释放事件场合)，然后再次按下。该函数将采用函数 ord()( 仅 a-Z 的大写字母 或 0-9 的数字) 或列出的 vk_* 常量中的任何一个返回的 keycode 值在主键盘输入页面上。

 

语法:

keyboard_clear(key);

参数类型描述
keyVirtual Key Constant (vk_*)清除的键。

 

返回:

N/A

 

示例:

keyboard_clear(vk_space);

上面的代码清除了空格键的状态。
