注意事项

默认资源级增益是声音编辑器中"音量"滑块的值：

稍后仍可以在游戏中更改此设置，方法是使用声音资产ID 作为参数调用audio_sound_gain：

audio_sound_gain(snd_Explode, 0, 0);  // Set the asset-level gain to 0 (new instances of this sound play muted)
