gpu_get_texrepeat

使用此函数，您可以检查是否启用纹理重复(returntrue)或不启用(returnfalse)。有关纹理重复的详细信息，请参阅函数gpu_set_texrepeat()。

 

语法:

gpu_get_texrepeat();

 

返回:

Boolean

 

例子:

if (gpu_get_texrepeat())
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

上述代码检查纹理重复是打开还是关闭，并相应地将其切换。
