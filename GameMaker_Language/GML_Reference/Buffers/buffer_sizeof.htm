buffer_sizeof

此函数将返回给定缓冲区数据类型常量(此处列出)的大小(以字节为单位)。

 

语法:

buffer_sizeof(type);

参数 类型 描述
type Buffer Data Type Constant 要检查的数据类型(参见此处的常量列表)。

 

返回:

Real

 

例子:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

上面的代码首先计算要创建的缓冲区的大小，方法是将无符号的 8 位数据类型乘以 12(因为我们将使用缓冲区保存 12 条数据)，然后使用该值将固定缓冲区设置为正确的大小。
