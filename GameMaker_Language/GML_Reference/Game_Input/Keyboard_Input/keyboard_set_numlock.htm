keyboard_set_numlock

您可以使用此函数打开或关闭键盘数字锁定 (设置为true为on，设置为false为off)。

注意此函数仅在Windows目标版本中可用，并且在任何其他设备上都不起作用。

 

语法:

keyboard_set_numlock(value);

参数 类型 描述
value Boolean true为on，false为off

 

返回:

N/A

 

示例:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

上面的示例代码将获取 NumberLock 键的状态，如果它是 on(true)，则将其设置为 off(false)，反之亦然。
