os_get_config

此函数返回当前为您的游戏选择的配置的名称(作为字符串)。有关配置的更多信息，请参阅配置部分。

 

语法:

os_get_config()

 

返回:

String

 

例子:

if (os_get_config() = "Free_Version")
{
    global.Ads = true;
}
else global.Ads = false;

上面的代码将检查正在使用的配置，如果是名为"Free_Version"的配置，广告将在游戏中启用。
