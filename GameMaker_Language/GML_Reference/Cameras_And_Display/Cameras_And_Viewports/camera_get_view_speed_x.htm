camera_get_view_speed_x

此函数可用于检索给定相机沿 X 轴的移动速度(水平移动)。返回值将以每个游戏帧的像素为单位。

 

语法:

camera_get_view_speed_x(camera_id)

参数类型 描述
camera_idCamera ID 创建相机时返回的唯一相机 ID 值。

 

返回:

Real

 

例子:

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
    camera_set_view_speed(view_camera[0], 5, 5);
}

以上代码检索分配给 viewport[0] 的相机的水平速度(xspeed)和垂直速度(yspeed)，然后检查它是否与给定值匹配。如果没有，则将视图的相机设置为该速度。
