mp_grid_to_ds_grid

使用此函数，您可以将给定的 MP 网格复制到 DS 网格中。DS 网格的大小应与 MP 网格的大小相同，但不一定非得如此 (如果数据较小，则会丢失。如果数据较大，则所有额外的网格单元格都将为 0)。DS 网格中的单元格将包含值 -1 (如果在 MP 网格中标记为"已占用") 或 0( 如果未标记为"已占用")。

 

语法:

mp_grid_to_ds_grid(source, destination);

参数 类型 描述
source MP Grid ID 要使用的 mp_grid 的索引
destination DS Grid ID 要用于将网格数据复制到的 ds_grid 的索引。

 

返回:

Boolean

 

例子:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

上述代码将创建一个新的 DS 网格，然后将变量 "mp_grid" 中包含的 MP 网格数据复制到新的 DS 网格中。
