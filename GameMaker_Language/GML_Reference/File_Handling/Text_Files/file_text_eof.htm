file_text_eof

当到达给定打开的文本文件的结尾时，此函数返回 true ，如果没有，返回 false 。

 

语法:

file_text_eof(fileid);

参数类型 描述
fileidText File ID 要检查的文件的 ID。

 

返回:

Boolean

 

例子:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    str[num++] = file_text_readln(file);
}
file_text_close(file);

上面的代码打开一个用于写入的文件，然后循环遍历已经写入文件的文本行，直到文件到达末尾，并将每一行存储在数组“ str”中。
