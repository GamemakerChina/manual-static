audio_group_load_progress

此函数将检查音频组的加载进度，并返回0到100之间的(近似)值。

 

语法:

audio_group_load_progress(groupID);

参数 类型 描述
groupID Audio Group ID 要检查的音频组的索引(在音频组窗口中定义)

 

返回:

Real

 

例子:

if load
{
    var pc = audio_group_load_progress(audiogroup_level1);
    draw_text(32, 32, "Loading: " + string(pc));
}

上面的代码检查一个变量，如果它返回true，那么将绘制一些文本，显示正在加载的音频的进度。
