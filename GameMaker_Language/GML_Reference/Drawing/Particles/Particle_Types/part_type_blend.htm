part_type_blend

使用此函数，可以使用添加剂混合模式(true)或非(false)绘制粒子。添加剂混合是一种特殊的混合模式，可以在每个粒子重叠时添加它们的亮度值，这样当它们重叠时，浅色将逐渐变亮(直到它们显示为白色)，并且黑色几乎不可见时，深色将变得越来越透明。该函数可以提供一些漂亮的粒子效果，但并不总是必需的。例如，烟雾轨迹会有添加剂混合以使效果保持灰色，但火焰效果可能会启用以使粒子更加半透明和明亮。

 

语法:

part_type_blend(ind, additive);

参数类型描述
indParticle Type ID要更改的粒子类型的索引。
additiveBoolean这些颗粒是否应该添加(true)或正常(false)混合。

 

返回:

N/A

 

例子:

part_type_blend(part_Fire, true);

上述代码将设置在变量"part_Fire"中索引的粒子类型创建的所有粒子具有加性混合模式。
