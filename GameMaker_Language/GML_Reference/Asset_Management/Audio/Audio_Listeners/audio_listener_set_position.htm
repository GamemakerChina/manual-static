audio_listener_set_position

使用此函数，您可以更改给定收听器在3D音频空间中的位置。默认监听器索引为0，但可以使用函数audio_get_listener_info()获取目标平台可用的不同索引。下面的示例图像显示了收听者在音频空间中的默认位置：

正如您所看到的，默认位置是(0，0，0)，但您通常会使用此函数在游戏中移动监听器和播放器对象，从而更改玩家听到发射器创建的音频的方式，例如，在自顶向下游戏的下图中。播放器实例设置监听器，该监听器将使来自各个发射器的音频随着播放器在关卡中的移动而"改变"：

 

语法:

audio_listener_set_position(index, x, y, z);

参数类型描述
indexReal or Audio Listener ID要设置位置的监听器 (默认为 0)。
xReal监听器的 X 位置 (默认为 0)。
yReal监听器的 Y 位置 (默认为 0)。
zReal监听器的 Z 位置 (默认为 0)。

 

返回:

N/A

 

例子:

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

上述代码检索相机视图 [0] 的视图矩阵，然后使用它来设置带有全局变量 "Player_Listener" 中存储的 ID 的收听者的音频收听器位置和方向。
