audio_group_name

此函数将返回一个字符串，其中包含用于显示或检查的给定音频组的名称。

当你在游戏选项中定义一个音频组时，你给它一个唯一的"名称"，它实际上是一个常量，用作组的ID值。这个函数所做的全部工作就是获取ID并返回您给出的ID名称的字符串。

注意 默认音频组为audiogroup_default。

 

语法:

audio_group_name(groupID);

参数 类型 描述
groupID Audio Group ID 要检查的音频组的索引值常量(在音频组窗口中定义)

 

返回:

String

 

例子:

var _name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + _name);

上面的代码检索给定音频组常量的名称，并将其显示在屏幕上。
