is_bool

此函数返回给定变量是否为布尔值 (true 或 false)。在某些情况下，您想检查并查看 GameMaker 中的变量是否包含布尔值，这就是您使用此函数的时间。

 

语法:

is_bool(n);

参数 类型 描述
n Any 要检查的参数。

 

返回:

Boolean

 

示例:

if (is_bool(val))
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

上面的代码检查变量 “val”，看看它是否是一个实数，如果它是，然后使用它来设置一个全局变量。
