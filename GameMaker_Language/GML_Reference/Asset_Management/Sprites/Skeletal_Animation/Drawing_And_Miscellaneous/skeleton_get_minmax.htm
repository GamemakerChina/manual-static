skeleton_get_minmax

此函数将返回一个数组，其中包含分配给骨骼动画精灵的所有单个边界框的总边界框值。返回的数组将有4个元素：

  • [0]-所有边界框的最小X位置
  • [1]-所有边界框的最小Y位置
  • [2]-所有边界框的最大X位置
  • [3]-所有边界框的最大Y位置

注意Spine 精灵的边界框是在Spine本身中设置的，而不是在GameMaker中。

 

语法:

skeleton_get_minmax();

 

返回:

Array (4 elements: xMin, yMin, xMax, yMax)

 

例子:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
{
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

上面的代码将检查指定给轨迹0的动画的当前帧，然后从精灵的总边界框中间的一个点爆发一些粒子。
