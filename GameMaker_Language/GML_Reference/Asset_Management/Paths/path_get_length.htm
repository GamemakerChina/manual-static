path_get_length

您可以使用此函数获取路径的精确长度(以像素为单位)。这不是从点到点的近似长度，而是沿着路径形状的精确长度，即使路径是平滑的高弯曲精度。

 

语法:

path_get_length(index);

参数类型 描述
indexPath Asset 要度量的路径的索引。

 

返回:

Real

 

例子:

path_len = path_get_length(pth_AI);

这将将"path_len"设置为以"pth_AI"为单位索引的路径的总长度(像素)。
