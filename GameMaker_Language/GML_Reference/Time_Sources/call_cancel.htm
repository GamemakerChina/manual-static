call_cancel

此函数取消先前由 call_later 启动的回调函数。

 

 

语法:

call_cancel(handle);

参数 类型 描述
handle Time Source ID 由 call_later 返回时间源的句柄。

 

返回:

N/A

 

例子:

call_cancel(handle);
