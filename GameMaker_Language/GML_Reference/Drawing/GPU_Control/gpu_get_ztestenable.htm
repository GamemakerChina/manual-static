gpu_get_ztestenable

此函数可用于检索是否启用Z-测试(函数返回true)(函数返回false)。默认值为Z-测试被禁用，因此函数将返回false。

 

语法:

gpu_get_ztestenable();

 

返回:

Boolean

 

例子:

if (gpu_get_ztestenable() == false)
{
    gpu_set_ztestenable(true);
}

上述代码检查z-测试是否已启用，以及是否已禁用，然后再次启用。
