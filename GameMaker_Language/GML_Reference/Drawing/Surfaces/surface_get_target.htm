surface_get_target

使用此函数，可以获取要绘制到的当前表面的ID。如果使用surface_create()创建了该表面，则这将是正整数，它是该表面的唯一ID

如果未设置自定义表面，但仍启用默认application_surface，则该函数通常将返回0。但是，如果您以前一直在操作application_surface，则可能希望检查所获得的ID是否与application_surface变量的当前值不匹配(如果该值不再为0)。

如果未设置自定义表面，并且也禁用了默认的application_surface，则函数将返回-1。

 

语法:

surface_get_target();

 

返回:

Surface ID or -1 (if no target surface is set)

 

例子:

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

上述代码将检查当前渲染目标是否为表面，如果是，则会重置目标，然后绘制表面。
