ds_grid_set_region

此函数可用于将给定网格的矩形区域设置为指定值(可以是实数或字符串)，如下图所示：

 

语法:

ds_grid_set_region(index, x1, y1, x2, y2, val);

参数类型 描述
indexDS Grid 网格的索引。
x1Real 网格中区域左侧的x位置。
y1Real 栅格中区域顶部的y位置。
x2Real 网格中区域右侧的x位置。
y2Real 网格中区域底部的y位置。
valAny 要将区域单元格设置为的值。

 

返回:

N/A

 

例子:

ds_grid_set_region(grid, 5, 5, 10, 10, 99)

这将设置在变量"grid"(5，5 )到(10，10 )到99中索引的网格区域内的所有单元格。
