mp_grid_clear_all

使用此函数，您可以清除所有"禁止"单元格的 MP 网格。

 

语法:

mp_grid_clear_all(id);

参数类型描述
idMP Grid ID要使用的 mp_grid 的索引

 

返回:

Boolean

 

例子:

if (!instance_exists(obj_Player))
{
    mp_grid_clear_all(grid);
}

如果房间中不再存在对象 "obj_Player" 的实例，则上述代码将清除变量 "grid" 中索引的 mp_grid，并将所有单元格标记为可用。
