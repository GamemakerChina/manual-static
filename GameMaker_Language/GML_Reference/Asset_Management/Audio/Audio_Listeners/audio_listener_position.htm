audio_listener_position

使用此函数，您可以更改监听器在3D音频空间中的位置。下面的示例图像显示了收听者在音频空间中的默认位置：

NOTE 如果您有多个监听器，请使用函数audio_listener_set_position()。

正如您所看到的，默认位置是(0，0，0)，但您通常会使用此函数在游戏中移动带有播放器对象的监听器，从而更改玩家听到发射器创建的音频的方式，例如，在自顶向下游戏的下图中。播放器实例设置监听器，监听器将使来自各个发射器的音频随着播放器在关卡中的移动而"改变"：

语法:

audio_listener_position(x, y, z);

参数 类型 描述
x Real 监听器的X位置(默认值为0)。
y Real 监听器的Y位置(默认值为0)。
z Real 监听器的Z位置(默认值为0)。

 

返回:

N/A

 

例子:

if (speed > 0)
{
    audio_listener_position(x, y, 0);
}

上面的代码检查播放器实例速度是否超过0，如果是，则将音频监听器更新到当前的X/y位置。
