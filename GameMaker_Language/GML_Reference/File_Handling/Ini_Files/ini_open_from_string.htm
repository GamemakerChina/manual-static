ini_open_from_string

这个函数将从字符串中创建一个临时的 ini 文件，并打开它进行读写。字符串应该被正确格式化为 ini 文件 (即 : 带有节、换行键和值)，否则 ini 文件将无法正确创建。注意，这个 ini 文件是临时的，一旦关闭，它将从内存中删除，丢失其中存储的任何信息，但是 ini_close() 函数返回完整 ini 文件的字符串，然后可以将其保存到服务器或磁盘上。

 

语法:

ini_open_from_string(string);

参数类型 描述
stringString 包含所有 ini 信息的字符串。

 

返回:

N/A

 

例子:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

上面的代码设置了一个临时的 ini 来保存字符串“str”中的信息，然后从 ini 中读取信息，然后再次关闭它。
