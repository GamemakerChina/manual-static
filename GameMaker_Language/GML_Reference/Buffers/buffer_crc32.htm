buffer_crc32

此函数将从缓冲区获取输入数据，并返回CRC32校验和哈希。指定要使用的缓冲区的缓冲区ID，然后指定从何处开始的偏移值(以字节为单位)，然后指定要进行哈希处理的区域的大小(同样以字节为单位)，该函数将返回该区域32位整数值。

 

语法:

buffer_crc32(buffer, offset, size);

参数类型 描述
bufferBuffer ID 要使用的缓冲区的索引。
offsetReal 数据偏移值。
sizeReal 缓冲区的大小。

 

返回:

Real

 

例子:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

上述代码将为变量" Buff "索引的缓冲区中存储的完整数据创建CRC32校验和哈希值，并将返回的整数哈希值存储在变量"check_val "中。
