show_message

这个函数创建一个弹出消息框，显示给定的字符串和一个标记为" OK "的按钮来关闭它。

注意此函数仅在Windows目标上用于调试，但在所有其他目标上已弃用。

 

语法:

show_message(str);

参数类型描述
strString要在弹出消息中显示的字符串。

 

返回:

N/A

 

例子:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

上面的代码将遍历存储在数组" inv "中的值，并将它们添加到变量" tot "中，然后显示带有总数的消息。
