display_mouse_set

使用此函数，您可以更改或设置鼠标在游戏显示内的位置，例如，这对于 FPS 游戏非常有用。只有在游戏处于焦点状态且使用 Alt+ Tab 将解锁鼠标时，该函数才能正常工作。

注意 此函数仅在桌面平台(即 Windows、Mac 和 Ubuntu)上可用。

 

语法:

display_mouse_set(x, y);

参数类型 描述
xReal 显示器的 X 坐标。
yReal 显示器的 Y 坐标。

 

返回:

N/A

 

例子:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

上述代码将使鼠标在游戏显示器中居中。
