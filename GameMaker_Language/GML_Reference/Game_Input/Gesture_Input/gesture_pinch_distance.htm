gesture_pinch_distance

此函数用于设置在触发捏手势之前，您必须触摸/单击屏幕并用两根手指移动的距离。距离以英寸为单位，默认值为0.1。

 

语法:

gesture_pinch_distance();

 

返回:

Real (inches)

 

例子:

if (gesture_get_pinch_distance() != 0.1)
{
    gesture_pinch_distance(0.1);
}

上述代码检查手势的捏距离是否设置为0.1英寸，如果不是，则将其设置为该值。
