ceil

这个函数接受任意实数，并将其向上舍入到最接近的整数。在使用这个函数时应该小心，因为一个常见的错误是使用它四舍五入一个随机值，并期望它总是大于1，例如:

int = ceil(random(5));

现在，您可能期望这段代码总是给出1到5之间的整数，但这可能并不总是这样，因为随机函数返回0的可能性很小，并且对0取四舍五入仍然得到0。这种可能性很小，但在使用此函数时应予以考虑。

 

语法:

ceil(val);

参数类型 描述
xReal 要更改的数字。

 

返回:

Real

 

例子:

val = ceil( 3.4 );

这将把 val 设置为4。
