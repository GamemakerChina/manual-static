matrix_get

这个函数返回一个包含16个值的数组，对应于给定的4x4矩阵类型，其中第一行是元素[0 - 3]，第2行是元素[4 -7]，等等……(参见主页上的图片)。可用的矩阵是view， projection和world，对此你可以使用以下常量之一:

矩阵类型常量
常量 描述
matrix_view The current view matrix
matrix_projection The current projection matrix
matrix_world The current world matrix

 

语法:

matrix_get(type);

参数 类型 描述
type 矩阵类型常量 要获取的矩阵类型(请参见上面列出的常量)

 

返回:

Matrix Array

 

例子:

v_array = matrix_get(matrix_view);

上面的代码将获取当前视图矩阵的值，并用它们填充数组“v_array”。
