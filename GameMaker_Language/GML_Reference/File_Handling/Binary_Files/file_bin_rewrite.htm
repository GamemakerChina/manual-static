file_bin_rewrite

这个函数接受函数 file_bin_open() 返回的文件名句柄，然后重写文件，清除之前的所有数据，从文件开始写入。

注意 : 当目标模块是 HTML5 时，这些函数 不能工作。

 

语法:

file_bin_rewrite(binfile);

参数类型 描述
binfileBinary File ID 要重写的文件的 ID。

 

返回:

N/A

 

例子:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

这将从与游戏相同的目录中打开一个文件，并将其索引分配给变量“file”。然后它将重写文件 (清除其中的数据)。
