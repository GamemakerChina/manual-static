view_enabled

此变量控制是否启用房间内可见的任何视口。如果将视口设置为可见，然后禁用此选项，则会将整个房间绘制到按照窗口大小缩放的屏幕上，而不是通过视口绘制不同的相机。

 

语法:

view_enabled;

 

返回:

Boolean (true: enabled, false: disabled)

 

例子:

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

以上代码检查是否为房间启用了视图，如果没有启用视图，则确保 viewport[0] 可见，然后启用房间的视图。
