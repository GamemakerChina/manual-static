mouse_y

此只读变量返回鼠标在房间内的当前y轴位置。

 

语法:

mouse_y;

 

返回:

Real

 

例子:

y = median(64, mouse_y, room_height - 64);

上面的代码将使实例保持在鼠标y位置，只要它在房间顶部和底部64像素的限制内即可。
