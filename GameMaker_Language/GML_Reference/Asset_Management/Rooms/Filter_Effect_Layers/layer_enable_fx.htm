layer_enable_fx

此函数启用/禁用分配给房间层的过滤器/效果。您可以指定要修改的层的 ID 或名称，然后指定一个布尔值，说明 FX 是应启用(true)还是禁用(false)。

传入false不会从层中删除FX，而只是使它不可见。使用layer_clear_fx()从层中删除FX。

类似地，如果没有FX应用于层，传入true将不起作用。使用layer_set_fx()将FX应用于层。

 

语法:

layer_enable_fx(layer_name_or_id, enable);

参数 类型 描述
layer_name_or_id String or Layer 要修改的层的名称或ID
enable Boolean 是启用还是禁用 FX

 

返回:

N/A

 

例子:

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

如果实例的HP值小于或等于3，则上述代码启用Desaturate FX层，否则禁用该层。这通过使屏幕饱和，向玩家表明他们的气血量低。
