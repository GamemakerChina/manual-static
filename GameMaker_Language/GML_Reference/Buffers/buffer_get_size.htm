buffer_get_size

使用此函数，您可以获取给定缓冲区的大小(以字节为单位)。

 

语法:

buffer_get_size(buffer);

参数 类型 描述
buffer Buffer ID 要获取大小的缓冲区的索引。

 

返回:

Real

 

例子:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

上述代码将创建一个新的缓冲区并将其索引存储在本地变量"_temp"中，该新缓冲区的大小与变量"player_data"中索引的缓冲区的大小相同。
