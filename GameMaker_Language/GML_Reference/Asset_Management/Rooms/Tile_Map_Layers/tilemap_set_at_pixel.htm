tilemap_set_at_pixel
Click here to see this page in full context

13:00:27

此函数可用于使用要在房间内更改的图块的实际位置，将图层上的图块贴图元素内的单元格设置为新图块。可以给出图块贴图元素 ID(使用 layer_tilemap_create() 创建图块贴图元素时或使用函数 layer_tilemap_get_id() 时获得的 ID)，然后给出要设置的图块数据以及房间内的位置。

您可以使用函数 tilemap_get_at_pixel() 检索图块数据，然后使用 tile_ 函数更改它，然后使用此函数再次设置它。或者，可以将新的图块数据值创建为 0 (将是空图块)，或使用要使用的图块的正索引值。仍然可以对这些值运行 tile_ 函数。

如果成功设置块，函数将返回true;如果出现问题且未设置，函数将返回false。

 

语法:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

参数类型描述
tilemap_element_idTile Map Element ID要更改的图块图元素的唯一ID值
tiledataTile Data要设置的图块设置数据
xcellRealx位置(房间内)
ycellRealy位置(房间内)

 

返回:

Boolean

 

例子:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

上面的代码获取"Tiles_Sky"层上的图块映射的ID，然后使用该ID从位置(4，4)的图块中获取数据。然后翻转此图块数据，然后再用于设置图块贴图上的图块。
