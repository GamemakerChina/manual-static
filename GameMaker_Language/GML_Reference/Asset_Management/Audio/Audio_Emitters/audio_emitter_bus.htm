audio_emitter_bus

此函数将音频发射器连接到总线。

这将导致发射器的所有输出被发送到给定的总线。总线上的所有效果都将应用于发射器的输出。

可以将任意数量的发射器指定给总线。

 

语法:

audio_emitter_bus(emitter, bus);

参数 类型 描述
emitter Audio Emitter ID 音频发射器
bus AudioBus Struct 分配给发射器的音频总线

 

返回:

N/A

 

例子:

emitter = audio_emitter_create();
var _new_bus = audio_bus_create();
audio_emitter_bus(emitter, _new_bus);

上面的代码首先创建一个音频发射器，然后使用audio_bus_create创建音频总线。 它使用audio_emitter_bus将发射器分配给新总线。

通过将发射器的总线配置分配回audio_bus_main，可以重置该发射器的总线配置。
