room_set_camera

此函数将摄像机分配给当前房间以外的房间中的特定视口。

您提供房间索引、视图索引 (从 0 到 7)，然后提供要使用的摄像机的索引。

 

语法:

room_set_camera(rm, vind, camera);

参数类型描述
rmRoom Asset设置摄像机的房间
vindReal要将摄影机指定给的视口的索引
cameraCamera ID要指定的摄影机的索引

 

返回:

N/A

 

例子:

global.myroom = room_add();
room_set_camera(global.myroom, 0, global.MainCam);

上面的代码将新创建的房间中的摄像机分配给视口[0]。
