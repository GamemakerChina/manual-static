项目工具

项目工具是一个内置工具，用于在不同版本的 GameMaker 之间转换项目。

IDE在内部使用它将项目从旧的项目格式转换为当前格式，或者反过来。

可以通过 工具菜单 > 项目工具 选项从 IDE 访问该工具。

注意 有关当前项目格式的详细信息，请参阅 项目格式 页面。

工具窗口包含以下字段：

  • 源-此字段包含要转换的源项目或包文件的路径。请在此处粘贴有效的文件路径或单击按钮以调出文件选择器。
  • 目标 - 此字段包含要创建的目标项目的路径。在此处粘贴有效路径或单击按钮以调出文件选择器。当源项目是 .yyp 文件时，此字段可以为空，在这种情况下，项目工具执行就地转换(即转换源目录中的项目)。但是，如果源项目是包文件，则必须填写此字段。如果您尝试转换目标路径为空的包，项目工具将显示以下消息：
  • 项目格式 - 此下拉列表包含您可以转换为的可用项目格式。以下格式可用：
    • LTS22- 这是 2022.0 LTS 版本使用的项目格式。
    • NOV23- 这是 2023.11 版本使用的项目格式。
    • DEFAULT -这是当前默认的版本化项目格式。
  • 强制 VERS0 转换(如果需要)- 启用此复选框时，项目工具 强制转换为中间基础版本 0 格式，以最后尝试修复项目中的任何问题。VERS0 格式是一种便于在 NOV23 格式和版本化 DEFAULT 格式之间转换的格式。

    警告 启用此选项时应非常小心，因为它将对丢失的字段进行最佳猜测并剔除任何未知数据。
  • 清理转换文件夹 - 选中此复选框后，项目工具将在转换后清理转换文件夹。默认情况下启用此功能。

项目工具输出

所有项目工具输出都显示在项目工具输出中，您可以通过窗口菜单的项目工具输出选项手动打开该窗口。当项目工具运行时，也可以通过启用常规首选项中的显示项目工具输出来自动显示该窗口。
