mouse_lastbutton

此变量返回最后按下的鼠标按钮，并且可以返回除mb_any之外的任何特殊鼠标常量(您也可以将此变量设置为常量之一)。

 

语法:

mouse_lastbutton

 

返回:

Mouse Button Constant (except mb_any)

 

例子:

if (mouse_lastbutton == mb_left)
{
    x -= 1;
}

如果最后按下的按钮是鼠标左键，则此代码将当前实例向左移动。
