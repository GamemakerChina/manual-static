ds_list_sort

使用此函数，您可以按升序或降序对列表中的所有值进行排序。如果列表包含字符串，这些字符串将根据英语26个字母的字母表按字母顺序排序。

 

语法:

ds_list_sort(id, ascend);

参数类型 描述
idDS List 要排序的列表的ID。
ascendBoolean 值应按升序(true)还是降序(false)顺序排列。

 

返回:

N/A

 

例子:

if newgame
{
    ds_list_sort(name_list, true);
}

如果变量"newgame"标记为true，则上述代码将对变量"name_list"中的索引列表进行排序。
