layer_sequence_get_instance

With this function you supply the sequence element ID - as returned by layer_sequence_create() or by one of the layer element functions - and it will return the sequence instance struct. You can find out more about the format of the sequence instance struct on The Sequence Instance Struct page.

 

语法:

layer_sequence_get_instance(sequence_element_id)

参数 类型 描述
sequence_element_id Sequence Element ID 序列元素到目标的唯一ID值

 

返回:

Sequence Instance Struct

 

例子:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

上面的代码创建一个新的序列元素，然后检索其序列实例ID。然后用于更改序列实例的回放速度比例属性。
