part_type_subimage

如果使用part_type_sprite设置的自定义精灵，则此函数设置用于给定粒子类型的子图像(或帧)。

用法注释

  • 此函数的效果取决于传递给part_type_sprite的参数：
    • 如果粒子类型的精灵设置为使用随机子图像，则此函数将无效。
    • 如果精灵未设置为随机：
      • 当animate设置为true时，精灵动画将循环，从该函数设置的子图像开始。
      • 当animate设置为false时，粒子将在其生命周期内使用该函数设置的子图像。

 

语法:

part_type_subimage(ind, subimg);

参数 类型 描述
ind Particle Type ID 粒子类型的索引
subimg Real 要使用的子图像(帧)

 

返回:

N/A

 

Example 1: Non Animated Sprite Sub-image

pt = part_type_create();
part_type_sprite(pt, spr_cards, false, false, false);
part_type_subimage(pt, 2);

上面的代码创建了一个新的粒子类型，并将其索引存储在实例变量pt中。然后，它使用part_type_sprite配置粒子类型，以使用不播放其动画且不使用随机子图像的自定义sprite资源spr_cards。最后，要使用的子图像在调用part_type_subimage时被设置为2。因此，此类型的所有粒子将在其整个生命周期内显示spr_cards的子图像2。

 

示例2：在子图像处开始播放的动画精灵

pt = part_type_create();
part_type_sprite(pt, spr_cards, true, false, false);
part_type_subimage(pt, 2);

上面的代码创建了一个新的粒子类型，并将索引存储在实例变量pt中。然后，它使用part_type_sprite配置粒子类型，以使用将播放其动画的自定义sprite资源spr_cards。由于random参数设置为false，因此不会为精灵选择随机子图像。相反，使用调用part_type_subimage中提供的子图像。因此，此粒子类型的所有粒子都将使用精灵spr_cards，所有动画都从子图像(或帧)2开始。
