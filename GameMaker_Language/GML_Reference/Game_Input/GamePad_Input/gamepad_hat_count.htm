gamepad_hat_count

此函数将返回连接到给定设备"插槽"的游戏手柄可用的十字键总数。十字键通常(虽然不总是)是带有上/下/左/右按钮的方向垫。

注意 在 Windows 目标上，十字键仅在 DirectInput 控制器上可用 (因此，从插槽 4 开始)。

 

语法:

gamepad_hat_count(device);

参数类型描述
deviceReal要检查哪个游戏手柄设备 "插槽"。

 

返回:

Real

 

例子:

h_num = gamepad_hat_count(4);

上面的代码将返回插入设备 "slot"4 的游戏手柄上可用的十字键数量，并将该值存储在变量 h_num 中。
