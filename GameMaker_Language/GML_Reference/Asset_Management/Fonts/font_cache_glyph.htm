font_cache_glyph

此函数允许您预先缓存字体中的字符或字形。

如果不使用此函数预缓存角色，则会在第一次绘制之前自动缓存该角色。

此函数返回具有字体纹理页面上字形的 x 和 y 位置的结构，例如 {x:208，y:62}。要获取缓存字形所在的纹理页的句柄，可以使用 font_get_texture。

 

语法:

font_cache_glyph(font, glyph_index);

参数 类型 描述
font Font Asset 要缓存字符或字形的字体
glyph_index Real 要缓存的字形的索引 (或字符代码) (参见ord/string_ord_at)

 

返回:

Struct

 

例子:

var _glyph = font_cache_glyph(fnt_gui, 65);

上面的代码缓存字体资源中带有索引 65 的字形，并将返回的结构存储在局部变量中。
