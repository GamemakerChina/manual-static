game_display_name

此只读变量返回在游戏选项中设置的目标平台的游戏显示名称。

 

语法:

game_display_name

 

返回:

String

 

例子:

var name = game_display_name;
var ver = string(GM_version);
draw_text(32, 32, name + ":" + ver);

以上代码获取游戏的显示名称和版本号并绘制。
