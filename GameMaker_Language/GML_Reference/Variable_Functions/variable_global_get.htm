variable_global_get

使用此函数，您可以从给定的命名全局变量中获取值。您提供全局变量的名称以获取的值作为字符串(请参见下面的示例代码)，并且该函数将返回全局变量持有的值，或者如果变量不存在，则返回undefined。

重要!如果您获取的全局变量不存在，则函数将返回关键字undefined，并且您可能会收到错误，这些错误将使游戏无法正常运行，因此，如果有疑问，请首先使用函数variable_global_exists。

 

语法:

variable_global_get(name);

参数类型 描述
nameString 要获取的全局变量的名称 (作为字符串)

 

返回:

Any (any data type) or undefined (if the named variable does not exist)

 

例子:

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

上面的代码将检查是否存在全局变量，如果存在，则将其输出到控制台。
