layer_sprite_y

此函数控制层上资源子图元元素沿房间 y轴的位置。指定精灵元素 ID(使用layer_sprite_create()或使用函数layer_sprite_get_id()时获得)，然后设置要使用的 y 值(基于房间坐标)。

 

语法:

layer_sprite_y(sprite_element_id, y);

参数类型 描述
sprite_element_idSprite Element ID 要更改的精灵元素的唯一ID值
yReal 精灵的y位置

 

返回:

N/A

 

例子:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

上述代码获得分配给层"Asset_sky"的精灵资源"云"的ID值，然后将其y位置设置为50与房间高度减200之间的随机值。
