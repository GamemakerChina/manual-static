clipboard_set_text

该函数将设置剪贴板以保存已定义的字符串。您可以将其设置为空字符串 ""，以有效清除文本的剪贴板。

注意 此函数仅在Windows，Android，MacOS，iOS，HTML5和Opera GX目标上有效。在HTML5上，在Firefox不使用扩展的情况下板，不支持剪贴板 ，并且在Internet Explorer上完全不支持剪贴板。

 

语法:

clipboard_set_text(string);

参数 类型 描述
string String 设置剪贴板保存的文本 (作为字符串)。

 

返回:

N/A

 

例子:

if (clipboard_has_text())
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

上述代码检查剪贴板是否包含文本，如果其中包含任何文本，则会将其作为字符串读入变量"string"中。最后，通过将剪贴板设置为空字符串来清除剪贴板。
