font_get_italic

使用此函数，您可以检查任何字体资源，以查看其是否具有斜体标志。如果执行，函数将返回true，否则返回false。

 

语法:

font_get_italic(ind);

参数类型 描述
indFont Asset 要检查的字体的索引。

 

返回:

Boolean

 

例子:

if font_get_italic(fnt_Main)
{
    draw_set_font(fnt_Main);
}

如果在其字体属性中将其设置为斜体，则此操作会将活动图形字体设置为fnt_main。
