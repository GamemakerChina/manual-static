layer_vspeed

您可以使用此函数设置当前范围房间内图层的垂直速度(每游戏帧的像素数)。提供层 ID(使用layer_create()创建层时获得的 ID)或层名称(作为字符串，这将对性能产生影响)和要设置的速度值，其中正值向下，负值向上。

 

语法:

layer_vspeed(layer_id, vspd)

参数 类型 描述
layer_id String or Layer ID 要设置垂直速度的层的唯一ID值
vspd Real 要设置的垂直速度(每游戏帧像素)

 

返回:

N/A

 

例子:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

上述代码检查给定层水平和垂直速度，如果它们没有都设置为0，则将其设置为0。
