仓库选项

在为您的项目设置了 Git 可执行文件并启用了源代码管理之后，您现在需要将 GameMaker IDE 链接到仓库，这可以通过以下两种方式之一完成：

  • 创建项目仓库
  • 克隆现有仓库

在第一种情况下，创建项目仓库，这将在您的项目所在位置直接创建一个 Git 仓库。用 Git 的话说，这是一个"非裸库"，所以在这种情况下，推送/拉取没有任何意义，但您仍然可以提交和跟踪项目更改，以及在需要时恢复更改和修订。要进行此设置，您只需单击主菜单上的源代码管理>创建项目仓库。

作为一种方便的功能，您还可以通过 IDE 克隆外部仓库。同样，您需要在身份首选项中输入要连接到的仓库的用户名和密码，然后只需单击源代码管理>克隆仓库即可。填写URL(使用 HTTPS 变体而不是 SSH)，并说出您希望它转到的位置：

克隆完成后，它会自动打开一个文件浏览器，供您根据需要打开项目。
