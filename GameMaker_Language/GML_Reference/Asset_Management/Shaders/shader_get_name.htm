shader_get_name

使用此函数，您可以检索着色器资源的名称。您为着色器提供唯一的ID值以获取的名称，该函数将以字符串形式返回资源的名称。

 

语法:

shader_get_name(shader);

参数类型 描述
shaderShader Asset 要获取其名称的着色器的索引。

 

返回:

String

 

例子:

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

上面的代码将获取给定着色器的名称，并将其绘制到屏幕上。
