parameter_string

命令行参数是您可以添加到 exe 中以更改程序运行方式的额外命令，使用此函数您可以获取所选命令行参数作为字符串。您可以使用函数 parameter_count 查找当前游戏的参数数量，其中第一个参数的索引为 1，最后一个参数的索引为函数返回的索引 (0 值对于它是游戏可执行文件的文件名，包括路径)。需要注意的是，该函数适用于 HTML5 平台，检索 URL 参数。

 

语法:

parameter_string(n);

 

返回:

String

 

例子:

var p_num;
p_num = parameter_count();
if (p_num > 0)
{
    var i;
    for (i = 0; i < p_num; i += 1)
    {
        p_string[i] = parameter_string(i + 1);
    }
}

上面的代码将获取命令行参数的数量，如果有1个或更多，它将遍历它们并将它们存储为数组中的字符串。
