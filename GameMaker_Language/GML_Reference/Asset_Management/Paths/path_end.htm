path_end

此函数结束实例所遵循的当前路径，如调用函数path_start 时设置的那样。

警告 应从实例的范围内调用此函数。如果不是，则会抛出错误。

 

语法:

path_end();

 

返回:

N/A

 

例子:

 

if (place_meeting(x, y, obj_Blocker))
{
    path_end();
}

如果实例检测到与给定对象的任何实例的任何冲突，则上面的代码将结束当前路径。
