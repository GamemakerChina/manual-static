ds_list_add

此函数可用于向列表中添加一个新值(实数或字符串)，该值将在列表末尾添加。该函数可以采用更多的可选参数(根据需要选择多个参数)，允许您在一次调用中将多个值连续添加到列表中。

 

语法:

ds_list_add(id, val1 [, val2, ... max_val]);

参数类型描述
idDS List要添加到的列表的ID。
valAny要添加到列表中的值。
[val2, ... max_val]Any要添加到列表中的可选值。

 

返回:

N/A

 

例子:

ds_list_add(sc_list, score);

上述代码将把存储在"score"变量中的值添加到变量"sc_list"中的索引列表中。
