gpu_set_blendenable

此函数可用于打开和关闭Alpha混合。基本上，如果将此设置为false，则所有正在绘制的图像都将被绘制为100%不透明，这意味着精灵或背景的任何透明或半透明区域都将可视为纯色。最好尽可能关闭Alpha混合(尤其是针对移动设备进行开发时)，因为这会大大提高绘制速度。

语法:

gpu_set_blendenable(enable);

参数类型 描述
enableBoolean 启用或禁用Alpha混合值(true或false)。

 

返回:

N/A

 

例子:

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

上述代码关闭Alpha混合以绘制背景精灵，然后再次打开以继续绘制。
