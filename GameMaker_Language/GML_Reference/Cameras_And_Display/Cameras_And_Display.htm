Cameras And Display

窗口是 GameMaker 显示游戏的位置，根据目标设备的不同，可以使用不同的设备完成不同的操作，例如，是否将游戏设置为全屏。摄像头(和视口)，但是，控制您在该窗口中看到的内容，因此还有一系列专用于它们的函数。以下两个部分涉及显示游戏的以下两个方面：

  • 游戏窗口
  • 相机和视口

 

但是，有时仅仅获取有关游戏窗口和视图的信息是不够的，您确实需要了解显示器本身(实际上是游戏窗口显示的屏幕)的相关信息。以下函数均提供有关显示器大小和方向的详细信息、有关 GUI 层的详细信息以及有关鼠标的信息：

  • display_reset
  • display_get_width
  • display_get_height
  • display_get_orientation
  • display_get_dpi_x
  • display_get_dpi_y
  • display_get_gui_width
  • display_get_gui_height
  • display_get_timing_method
  • display_get_sleep_margin
  • display_get_frequency
  • display_mouse_get_x
  • display_mouse_get_y
  • display_mouse_set
  • display_set_gui_size
  • display_set_gui_maximise
  • display_set_ui_visibility
  • display_set_timing_method
  • display_set_sleep_margin

 

还有一些特殊函数可用于将显示器或其一部分保存到图像文件，如 *.png 或 *.gif：

  • screen_save
  • screen_save_part
  • gif_open
  • gif_add_surface
  • gif_save
  • gif_save_buffer
