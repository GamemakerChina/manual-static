audio_debug

This function can be used to display debug information about the audio system, with true switching it on and false switching it off. Enabling this will display the Audio window of The Debug Overlay.

注意 对于同步的音频组，您还可以使用函数 audio_sync_group_debug。

注意 此函数在HTML5目标平台上不起作用。

 

语法:

audio_debug(enable);

参数 类型 描述
enable Boolean 启用(设置为true)或禁用(false)音频调试。

 

返回:

N/A

 

例子:

if (debug_mode)
{
    audio_debug(true);
}
else
{
    audio_debug(false);
}

根据游戏是否在调试模式下运行，上述代码将打开或关闭音频调试叠加。
