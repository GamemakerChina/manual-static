skeleton_collision_draw_set

使用此函数，您可以打开(true)或关闭(false)绘制实例正在使用的当前骨骼动画精灵的碰撞数据。如果打开该选项，边界框和精确碰撞遮罩将绘制为精灵周围的轮廓。

仅当骨骼精灵绘制由对象处理时，此函数才起作用。若要在手动绘制骨骼精灵时绘制碰撞边界框，请使用draw_skeleton_collision()。

注意Spine 精灵的边界框是在Spine本身中设置的，而不是在GameMaker中。

 

语法:

skeleton_collision_draw_set(flag);

参数 类型 描述
flag Boolean 设置为true以打开绘制，设置为false以关闭绘制。

 

返回:

N/A

 

例子:

if (debug_mode == true)
{
    skeleton_collision_draw_set(true);
}

上面的代码检查游戏是否在调试模式下运行，如果是，则切换要显示的实例的骨骼碰撞数据。
