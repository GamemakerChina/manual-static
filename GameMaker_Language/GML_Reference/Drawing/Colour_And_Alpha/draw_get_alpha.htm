draw_get_alpha

此函数返回绘制Alpha的当前值，其范围为0(完全透明)和1(完全不透明)。绘制Alpha会影响所有绘制函数的透明度，并且可以使用draw_set_alpha()函数进行设置。

 

语法:

draw_get_alpha()

 

返回:

Real

 

例子:

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

上述代码将当前绘制Alpha存储到局部变量中，并基于实例变量更改绘制Alpha。绘制某些文本后，它会将Alpha重置为存储在局部变量中的值。
