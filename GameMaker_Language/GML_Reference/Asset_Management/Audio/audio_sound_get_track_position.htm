audio_sound_get_track_position

此函数将获取声音文件中播放声音的位置(以秒为单位)。声音只能是声音的单个实例(当使用audio_play_sound()或audio_play_sound_at()函数时，播放的单个声音的索引可以存储在变量中)。

 

语法:

audio_sound_get_track_position(index);

参数 类型 描述
index Sound Instance ID 声音的索引获取播放位置。

 

返回:

Real

 

例子:

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

上面的代码检查一个轨迹，以得到它的起始位置，如果它不是0秒，它将它设置为0秒。
