part_system_create

此函数用于创建新粒子系统并返回其唯一索引号。

系统将被分配一个托管层，并将深度设置为0。托管层对用户不可访问，并且仅在使用深度而不是层时用于内部管理。通常情况下，您将使用函数 part_system_create_layer而不是此函数。

也可以提供使用粒子系统编辑器创建的粒子系统资源。执行此操作将创建在该资源中定义的系统及其所有发射器。

 

语法:

part_system_create([partsys]);

参数类型描述
partsysParticle System Asset可选 要创建实例的粒子系统资源

 

返回:

Particle System Instance

 

例子:

global.p_sys = part_system_create();

这将创建一个新的粒子系统，并将索引存储在全局变量 p_sys 中。
