layer_get_visible

使用此函数，您可以检查层是否可见。提供层 ID(使用layer_create()创建层时获得的 ID)或层名称(作为字符串，这将对性能产生影响)，如果可见，函数将返回true，否则返回false。

 

语法:

layer_get_visible(layer_id)

参数 类型 描述
layer_id String or Layer 目标层的唯一ID值(或作为字符串的层名)

 

返回:

Boolean

 

例子:

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

上面的代码在房间编辑器中获取名为"Instances"的层的ID值，然后使用ID检查层是否可见，根据返回值切换层可见性。
