string_repeat

此函数仅返回相同的字符串，该字符串在自身上重复了给定的次数。

 

语法:

string_repeat(str, count);

参数 类型 描述
str String 要重复的字符串。
count Real 重复字符串的次数。

 

返回:

String

 

例子:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

上面的代码将 str2 设置为 "Hello WorldHello WorldHello World"。
