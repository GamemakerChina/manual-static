gc_is_enabled

使用此函数，您可以检查是否启用了垃圾收集器。如果启用，函数将返回true，否则返回false。

 

语法:

gc_is_enabled();

 

返回:

Boolean

 

例子:

if (!gc_is_enabled())
{
    gc_enable(true);
}

上面的代码检查垃圾收集器是否被启用，如果它不是，则它启用它。
