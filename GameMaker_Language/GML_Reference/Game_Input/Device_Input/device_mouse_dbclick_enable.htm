device_mouse_dbclick_enable

此函数可用于设置设备以将mb_left(鼠标左键) 的双击检测为mb_right(鼠标右键) 输入。

默认情况下，此设置为true, 这意味着每次用户连续快速地两次点击设备屏幕时，返回值与单击鼠标右键时相同。打开此函数时，第一个tap将被检测为mb_left, 第二个tap将被检测为mb_right, 因此请确保您使用的任何代码都考虑到这一点。

 

语法:

device_mouse_dbclick_enable(bool);

参数 类型 描述
bool Boolean 将双击检测设置为on(true) 或off(false)。

 

返回:

Boolean

 

例子:

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

上面的代码检查运行游戏的设备是Windows PC还是Mac，如果是其中任何一个，则会禁用双击函数。
