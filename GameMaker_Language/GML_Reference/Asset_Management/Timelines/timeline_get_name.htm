timeline_get_name

此函数可用于以字符串形式获取时间线的名称。如果时间线是使用timeline_add ()函数动态创建的，则返回的名称将包含format'__newtimelineN" 其中 "N" 是时间线的编号(从 0 开始)。请注意，这只是一个字符串，不能直接用于引用时间轴-你需要时间轴索引。但是，您可以使用此字符串使用返回的字符串以及函数asset_get_index ()来获取时间轴索引。

 

语法:

timeline_get_name(ind);

参数类型描述
indTimeline Asset要检查其名称的时间线的索引。

 

返回:

String

 

例子:

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

上面的代码获取当前分配的时间线的名称，并将其输出到控制台。
