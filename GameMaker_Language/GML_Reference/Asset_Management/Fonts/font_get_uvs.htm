font_get_uvs

此函数返回具有纹理页面上字体纹理的 UV 坐标的数组，并用以下值填充数组：

  • [0] = 左侧
  • [1] = 顶部
  • [2] = 右侧
  • [3] = 底部

然后，此值可用于其他绘制函数，特别是在使用基元以及着色器函数的一般绘制中。

 

语法:

font_get_uvs(font);

参数 类型 描述
font Font Asset 要使用的字体的索引。

 

返回:

Array

 

例子:

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

上面的代码将在局部数组中存储给定背景的UV坐标，然后将值分配给实例变量。
