string_byte_at

以给定字符串中给定位置的实值的形式返回原始字节值。

 

语法:

string_byte_at(str, index);

参数 类型 描述
str String 要检查的字符串。
index Real 从中获取字节的位置。

 

返回:

Real

 

例子:

newbyte = string_byte_at("Hello World", 5);

这将设置newbyte为 "Hello World "第六个字母的原始字节值。
