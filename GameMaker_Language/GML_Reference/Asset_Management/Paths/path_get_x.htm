path_get_x

使用此函数，您可以获得任何给定路径上某个位置的X坐标。0是路径的起点，1是路径的终点，两者之间的任何值都等于通过路径的距离。这不一定是路径的定义点，它可以是路径上的任何位置。

 

语法:

path_get_x(ind, pos);

参数 类型 描述
index Path Asset 要检查的路径的索引。
pos Real 通过路径检查有多远。介于0(开始)和1(结束)之间。

 

返回:

Real

 

例子:

x = path_get_x(pth_WallWalk, 0.5);

这会将调用实例的X设置为给定路径中间的点的X坐标。
