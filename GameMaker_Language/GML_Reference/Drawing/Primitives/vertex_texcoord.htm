vertex_texcoord

此函数设置用于当前为自定义图元定义的顶点的纹理坐标。

您提供要写入数据的顶点缓冲区以及要使用的纹理页上的 UV 位置。

注意 要获取纹理页面上特定资源的 UV 范围，您可以使用 sprite_get_uvs、font_get_uvs 和 tileset_get_uvs。

 

语法:

vertex_texcoord(buffer, u, v);

参数 类型 描述
buffer Vertex Buffer 要写入信息的顶点缓冲区。
u Real 纹理页上要使用的第一个纹理坐标 (0 - 1)。
v Real 纹理页上要使用的第二个纹理坐标 (0 - 1)。

 

返回:

N/A

 

例子:

vertex_texcoord(buff, 0, 0);

上面的代码将定义的顶点的 UV 坐标设置为用于绘制它的纹理页面的左上角。
