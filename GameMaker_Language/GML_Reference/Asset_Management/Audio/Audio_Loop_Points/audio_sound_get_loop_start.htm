audio_sound_get_loop_start

此函数返回给定声音资源或声音实例的循环起始点(以秒为单位)。


语法:

audio_sound_get_loop_start(index);

参数 类型 描述
index Sound Asset or Sound Instance ID 声音资源或声音实例的 ID

 

返回:

Real

 

例子:

var _loop_start_time = audio_sound_get_loop_start(snd_loop);

上述代码对现有的声音资源 snd_loop 调用 audio_sound_get_loop_start，并将返回值存入变量 _loop_start_time。
