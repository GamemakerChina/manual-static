audio_sync_group_is_paused

此函数可用于检查同步组中的音频是否暂停。

函数接收函数audio_create_sync_group返回的同步组ID。

注意此函数不适用于HTML5目标平台。

 

语法:

audio_sync_group_is_paused(group_index);

参数 类型 描述
group_index Audio Sync Group ID 要检查的组索引。

 

返回:

Boolean

 

例子:

if !audio_sync_group_is_paused(sync_group)
{
    audio_resume_sync_group(sync_group);
}

上述代码首先检查音频同步组sync_group是否暂停。如果是这种情况，则使用audio_resume_sync_group恢复播放同步组。
