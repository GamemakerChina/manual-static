video_get_volume

此函数返回当前加载的视频的音量，该值介于0和1之间。您可以使用video_set_volume()更改加载的视频的音量。

 

语法:

video_get_volume();

 

返回:

Real

 

例子:

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

上述代码将在Step事件中运行。它获取当前正在播放的视频的音量，如果低于1，将增加0.02，逐渐增加音量，直到达到最大音量水平。
