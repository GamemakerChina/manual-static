path_scale

此值可用于获取或设置实例当前分配路径的比例(由函数path_start()设置)，默认值为 1。这是一个标量值，所以1 是1：1的比例，而设置为2，例如，将是比例的两倍，而将其设置为0.5将使比例减半。

 

语法:

path_scale;

 

返回:

Real

 

例子:

path_scale = 1 + random(2);

上面的代码将当前路径的比例设置为1(其原始大小)和3(其原始大小的3倍)之间的随机大小。
